# <a name="wpd-drivers"></a>WPD ドライバー


Microsoft Windows ・ ポータブル デバイス (WPD) は、コンピューターが接続されているメディアおよびストレージ デバイスと通信を有効にします。 このシステムでは、Windows メディア デバイス マネージャー (WMDM) とコンピューターの柔軟で堅牢な方法を提供することによって Windows Image Acquisition (WIA) がミュージック プレーヤー、記憶域デバイス、携帯電話、およびさまざまな種類の接続されたデバイスと通信するよりも優先されます。

Microsoft では、標準のプロトコルと画像転送プロトコル (PTP) を含め、デバイス、メディア転送プロトコル (MTP) デバイス、および大容量記憶域クラス (MSC) デバイスをいくつかのドライバーを提供します。 デバイスは、独自のプロトコルをサポートする場合は、独自のドライバーを開発する必要があります。 このドライバーを作成するのにには、ユーザー モード ドライバー フレームワーク (UMDF) を使用します。 このフレームワークの詳細については、[UMDF 入門](https://docs.microsoft.com/windows-hardware/drivers/wdf/getting-started-with-umdf-version-2)を参照してください。

Windows ポータブル デバイス向けに作成されたアプリケーションの詳細については、、 [WPD SDK ドキュメント](https://go.microsoft.com/fwlink/p/?linkid=178695)を参照してください。

WPD ドライバーの開発と WPD アプリケーション開発の両方の詳細については、次を参照してください。、 [WPD ブログ](https://go.microsoft.com/fwlink/p/?linkid=178687)、Windows 10 の正確であります。
