# <a name="using-the-wpdinfo-tool"></a>WpdInfo ツールを使用します。


*WpdInfo.exe*ツールは Windows ベースのアプリケーションで、機能をテストするには、ドライバーで使用することができます。 このツールを使用するには、次のタスクを実行します。

-   開くか、デバイスを閉じる
-   作成またはデバイス上のオブジェクトの削除
-   作成、保存、またはオブジェクトのリソースを削除
-   デバイスにコマンドを送信します。
-   サポートされているデバイスのコマンドを表示します。
-   サポートされているデバイスのコンテンツの種類を表示します。
-   サポートされているイベントを表示します。
-   サポートされているプロパティを表示します。
-   特定のリソースの属性を表示します。
-   特定のプロパティの属性を表示します。
-   プロパティの一括操作を実行します。
-   開く、またはデバイスのサービスを閉じる
-   デバイスのサービスでメソッドを呼び出す

ツールを起動するときに使用可能な WPD デバイスのいずれかを選択するよう求められます。

![wpdinfo デバイスの選択](images/wpdinfo1.png)

デバイスを選択した後は、接続が確立され、WpdInfo アプリケーション ウィンドウが初期化されます。 メイン ウィンドウの左上のフレームには、サポートされているオブジェクトの階層が表示されます。 右上のフレームには、現在選択されているオブジェクトに関連付けられているリソースのデータが表示されます。 下の中央のペインには、オブジェクトでサポートされているプロパティが表示されます。 ウィンドウの下部ペインには、特定の操作中に、ドライバーによって返されるエラー情報が表示されます。

![既定の画面を wpdinfo](images/wpdinfo2.png)

指定したオブジェクト、リソース、または、選択した要素に適用されるオプションのメニューを表示するプロパティを右クリックすることができます。 たとえば、削除して、そのリソースに対して、これに関連付けられているプロパティを表示するには、左ペインに表示されるオブジェクトを右クリックできます。

![wpdinfo オプション](images/wpdinfo3.png)
