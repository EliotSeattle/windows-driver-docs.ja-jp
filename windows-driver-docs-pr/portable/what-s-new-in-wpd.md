# <a name="whats-new-in-wpd"></a>WPD の新機能


このトピックでは、Windows ポータブル デバイス (WPD) の機能強化、新しい機能をまとめたものです。

## <a name="span-idthenetmontoolspanspan-idthenetmontoolspanspan-idthenetmontoolspanthe-netmon-tool"></a><span id="The_NetMon_Tool"></span><span id="the_netmon_tool"></span><span id="THE_NETMON_TOOL"></span>NetMon ツール


*NetMon.exe*ツールは Windows ベースのアプリケーションで、WPD コンポーネントからのトレースの表示に使用することができます。 このツールは*WpdMon.exe*以前のバージョンのオペレーティング システムに同梱され必要があります。
