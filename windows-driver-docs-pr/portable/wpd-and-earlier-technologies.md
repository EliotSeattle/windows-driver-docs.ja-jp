# <a name="wpd-and-earlier-technologies"></a>WPD と以前のテクノロジ


WPD には、デジタルの静止カメラ、ポータブル デジタル メディア プレーヤーの多くの機能強化が含まれています。 WPD は、Windows XP 以降、これらのデバイスにアクセスするための推奨される方法です。 既存の Windows Image Acquisition (WIA) および Windows メディア デバイス マネージャー (WMDM) で Windows Api のサポートは引き続きし、WPD との互換性を互換性層によって提供されます。 WPD デバイスに透過的に公開できる WIA および WMDM の既存のアプリケーション WIA デバイスまたは WMDM サービス プロバイダーとしてそれぞれこれら互換性レイヤーを確認します。 ただし、これらのアプリケーションがない同じさまざまな機能、新しい WPD API と新しい MTP ベースのデバイスと同じよ。

WPD MTP ドライバーには、画像転送プロトコル (PTP) デバイスもサポートしています。 WIA でサポートされていた PTP デバイスは引き続き互換性レイヤーを通じて WIA アプリケーションで機能します。 マイクロソフトでは、アプリケーションを確実にコミットし、デジタルの静止カメラ互換性 Windows とこれらの新しいテクノロジの次のバージョンでを補完し、Windows で、デジタル カメラ エクスペリエンスを向上するように設計します。
