# <a name="wpd-drivers-overview"></a>WPD ドライバーの概要


Windows ポータブル デバイス (WPD) は、さまざまなメディアと非メディア コンテンツを格納する多機能デバイスを有効にするためのインフラストラクチャを提供します。 多機能デバイスの例は、組み込みのズーム カメラ (デジタル カメラ関数) が含まれていて、音楽の同期と再生 (ポータブル メディア プレーヤー関数) をサポートしている携帯電話です。 また、WPD、これらのデバイス アクセスできるように、Windows ベースのコンピューターから。

オブジェクトに基づくデバイス ドライバー インターフェイス (DDI) は、ポータブル メディア プレーヤー、デジタルの静止カメラや携帯電話などのスマート ストレージ デバイスに最初に重点を置いて、convergent のデバイスを使用できます。 と共に、DDI、WPD は、次の標準のプロトコルとトランスポート クラス ドライバー ソリューションを実装します。

-   USB、IP、および Bluetooth 経由で画像転送プロトコル (PTP)
-   USB、IP、および Bluetooth 経由でのメディア転送プロトコル (MTP)
-   USB 経由での大容量記憶装置クラス (MSC)
