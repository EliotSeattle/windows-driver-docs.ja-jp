# <a name="the-wpdwudfsampledriver-sample"></a>WpdWudfSampleDriver サンプル


WPD ドライバーのドキュメントのこのセクションでは、ドライバー、WpdWudfSampleDriver、Windows Driver Kit (WDK) に含まれているの包括的なサンプルについて説明します。

包括的な WPD サンプル ドライバー (WpdWudfSampleDriver) では、Microsoft Windows ポータブル Devides (WPD) のデバイス ドライバー インターフェイス (DDI) のほぼすべての側面を示します。 このドライバーは、あるプロセス WPD のコマンドは、設定も通常ユーザー モード ドライバー フレームワーク (UMDF) ドライバーとして構築されます。 このドライバーは、実際のハードウェアと対話しない、連絡先、画像、音楽、ビデオおよびサポートの電話をデバイスとの通信をシミュレートします。

このドライバーは、概念を説明する最も簡単な方法で記述されました。 そのため、サンプル ドライバーでは、操作を実行する場合がありますか、ドライバーでは運用効率性が低い方法で構造化します。 さらに、このサンプルでは、実際のハードウェアは使用しません。 代わりに、メモリ内データ構造を使用してデバイスをシミュレートします。 そのため、ドライバーの場合は、実稼働のハードウェアの現実的な方法で実装されている可能性があります。

高度な Windows ポータブル デバイス (WPD) ドライバー開発者向けの WpdWudfSampleDriver によって実行されるタスクの一部が書き込まれます。 これらのタスクは、このセクションの後半のトピックで説明します。
