# <a name="test-modes"></a>テスト モード


TAEF では、さまざまなテストのさまざまな方法でテストの実行動作を変更するモードを提供します。 TAEF の基本的な実行について理解しているかどうかを確認しを参照してください[オーサリング テスト](authoring-tests.md)と[テストの実行](executing-tests.md)、このセクションに進む前にします。

**注:** テスト モードは相互に互換性がありません。指定したテストの実行中に 1 つだけテスト モードを有効にすることがあります。

次のテスト モードは、現在 TAEF によって提供されます。

-   [ループ](#loop)
-   [ストレス](#stress)
