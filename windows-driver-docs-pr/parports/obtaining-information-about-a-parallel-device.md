# <a name="obtaining-information-about-a-parallel-device"></a>パラレル デバイスに関する情報の取得





加え[並列デバイスへの接続](connecting-to-a-parallel-device.md)クライアントは、次のデバイス制御要求を使用して並列デバイスに関する追加情報を取得します。

[**IOCTL\_IEEE1284\_GET\_MODE**](https://msdn.microsoft.com/library/windows/hardware/ff543975)

[**IOCTL\_PAR\_取得\_既定\_モード**](https://msdn.microsoft.com/library/windows/hardware/ff544061)

[**IOCTL\_PAR\_取得\_デバイス\_キャップ**](https://msdn.microsoft.com/library/windows/hardware/ff544067)

[**IOCTL\_PAR\_IS\_ポート\_FREE**](https://msdn.microsoft.com/library/windows/hardware/ff544073)

[**IOCTL\_PAR\_クエリ\_デバイス\_ID\_サイズ**](https://msdn.microsoft.com/library/windows/hardware/ff544080)

[**IOCTL\_PAR\_クエリ\_デバイス\_ID**](https://msdn.microsoft.com/library/windows/hardware/ff544076)

[**IOCTL\_PAR\_クエリ\_情報**](https://msdn.microsoft.com/library/windows/hardware/ff544086)

[**IOCTL\_PAR\_クエリ\_場所**](https://msdn.microsoft.com/library/windows/hardware/ff544091)

[**IOCTL\_PAR\_クエリ\_RAW\_デバイス\_ID**](https://msdn.microsoft.com/library/windows/hardware/ff544100)

[**IOCTL\_シリアル\_取得\_タイムアウト**](https://msdn.microsoft.com/library/windows/hardware/ff544120)
