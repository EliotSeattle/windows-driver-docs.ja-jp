# <a name="physical-configuration-of-parallel-ports"></a>パラレル ポートの物理的な構成





このセクションでは、パラレル ポートの通常の物理的な構成について説明します。

次の図は、2 つの一般的なパラレル ポート構成を示します。

![多機能周辺機器にパラレル ポートのバスのカード (左図) および pci/isa ブリッジ (適切な図) に統合されています](images/parport1.png)

パラレル ポートには、多機能の周辺機器 bus カード上の 1 つのデバイスまたはシステム バスに接続されたデバイスを指定できます。
