# <a name="introduction-to-parallel-ports-and-devices"></a>パラレル ポートとデバイスの概要





このセクションでは、パラレル ポートやデバイスの Microsoft Windows が提供するサポートについて、次のトピックについて説明します。

[パラレル ポートの物理的な構成](physical-configuration-of-parallel-ports.md)

[パラレル ポートに接続されたデバイスの物理的な構成](physical-configuration-of-devices-attached-to-a-parallel-port.md)

パラレル ポートは、以下をサポートする、マルチモードの通信のポートでは通常は。

-   バイト ストリームとニブル (読み取り) モード

-   IEEE 1284 互換性モード

-   拡張機能のポートのモード

-   強化されたパラレル ポートのモード

パラレル ポートとのデバイス標準については、次の仕様を参照してください。

-   *IEEE Std 1284 1994 年、IEEE 標準のパーソナル コンピューター用の双方向の平行周辺インターフェイスのメソッドをシグナル通知*

-   *IEEE P1284.3、インターフェイスとプロトコルの拡張に準拠している周辺機器の IEEE 1284 1994、ホスト アダプターの Standard ドラフト D6.00、1998 年 12 月 3 日*
