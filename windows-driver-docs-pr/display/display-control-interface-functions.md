# <a name="display-control-interface-functions"></a>表示コントロール インターフェイス関数


## <span id="ddk_display_control_interface_functions_gg"></span><span id="DDK_DISPLAY_CONTROL_INTERFACE_FUNCTIONS_GG"></span>


ここでは、ディスプレイのデバイス コンテキストを制御するアプリケーションで使用される表示コントロール インターフェイス (DCI) 関数のリファレンス ページを示します。 これらの関数が宣言されている、 *dciman.h*ヘッダー ファイル。 新しいアプリケーションでこれらの関数を使用しません。代わりに、DirectDraw でサポートされているドライバーと通信するには、最新の DirectX グラフィックスのインターフェイスを使用します。 DirectX グラフィックスのインターフェイスについては、DirectX SDK の最新のドキュメントを参照してください。
