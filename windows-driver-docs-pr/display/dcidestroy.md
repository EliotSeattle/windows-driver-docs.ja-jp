# <a name="dcidestroy"></a>DCIDestroy


## <span id="ddk_dcidestroy_gg"></span><span id="DDK_DCIDESTROY_GG"></span>


この関数を使用しないでください。代わりに、DirectDraw でサポートされているドライバーと通信するには、最新の DirectX グラフィックスのインターフェイスを使用します。

**DCIDestroy**関数は、ディスプレイ デバイスのプライマリ画面を破棄します。
