# <a name="opm-terminology"></a>OPM の用語


OPM で使用される主な用語を次に示します。

<span id="Connector"></span><span id="connector"></span><span id="CONNECTOR"></span>**コネクタ**  
物理は、グラフィックス アダプターとディスプレイ デバイス間の接続を出力します。

<span id="Protection_type"></span><span id="protection_type"></span><span id="PROTECTION_TYPE"></span>**保護の種類**  
グラフィックス アダプターのコネクタを通じて渡されるビデオ信号に適用できる保護の種類。 保護の 1 つ以上の種類は、1 つのコネクタに適用できます。

<span id="Protection_level"></span><span id="protection_level"></span><span id="PROTECTION_LEVEL"></span>**保護レベル**  
グラフィックス アダプターのコネクタを通じて渡されるビデオ信号に適用される保護レベル。 レベルの値では、保護の種類に依存します。 一部の保護の種類 (たとえば、高帯域幅デジタル コンテンツの保護 (HDCP)) では、(たとえば、またはオフ) に 2 つだけの保護レベルがあります。
