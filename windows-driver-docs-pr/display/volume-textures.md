# <a name="volume-textures"></a>ボリュームのテクスチャ


## <span id="ddk_volume_textures_gg"></span><span id="DDK_VOLUME_TEXTURES_GG"></span>


DirectX 8.0 では、ボリュームまたは 3D テクスチャのサポートを追加します。 このようなテクスチャの幅と高さだけでなく深さがあります。
