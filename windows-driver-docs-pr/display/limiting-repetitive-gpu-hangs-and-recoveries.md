# <a name="limiting-repetitive-gpu-hangs-and-recoveries"></a>繰り返し発生する GPU のハングと復元の制限


Windows Vista Service Pack 1 (SP1)、Windows Server 2008 以降では、ユーザー エクスペリエンスが改善されました頻繁かつ迅速に、GPU がハングする場合。 GPU の繰り返し発生するハングを示すグラフィックス ハードウェアが正常に回復がありません。 このような場合は、エンドユーザーがシャット ダウンし、グラフィックス ハードウェアを完全にリセットするオペレーティング システムを再起動する必要があります。 オペレーティング システムでは、1 分以内に 6 つまたは複数の GPU のハングおよび後続の復元が発生することが検出された場合、オペレーティング システムのバグ チェック GPU のハングを次にコンピューター。
