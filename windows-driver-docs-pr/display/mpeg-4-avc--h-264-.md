# <a name="mpeg-4-avc-h264"></a>MPEG-4 AVC (H.264)


MPEG-2 高度なビデオコーディング (AVC) は、ITU-T h.264 とも呼ばれ、従来の標準よりも大幅に低いビットレートで優れたビデオ品質を提供できるビデオ圧縮の標準です (たとえば、MPEG-2、263、MPEG-2 のビットレートの半分以下)。 MPEG 4 AVC は、このビデオ品質を実現します。デザインの複雑さが大きくなることはありません。これにより、MPEG 4 の実装が現実的ではなく、非常に高価になります。 MPEG 4 AVC は、さまざまなネットワークやシステム上のさまざまなアプリケーションに適用できます。これには、低および高ビットレート、低解像度と高解像度のビデオ、ブロードキャスト、DVD ストレージ、RTP および IP パケットネットワーク、および ITU-マルチメディアテレフォニーシステムが含まれます。 DirectX VA が MPEG-2 AVC をデコードする方法については、「h.264 [/AVC デコード用の Directx Video アクセラレーション仕様」](https://www.microsoft.com/download/details.aspx?id=11323)をダウンロードしてください。
