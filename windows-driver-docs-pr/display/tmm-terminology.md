# <a name="tmm-terminology"></a>TMM の用語


TMM で使用される主な用語を次に示します。

<span id="Clone_View"></span><span id="clone_view"></span><span id="CLONE_VIEW"></span>**ビューを複製します。**  
プライマリを表示する表示モードは、グラフィックス アダプターに関連付けられているすべてのアクティブなモニタに表示されます。

<span id="COM"></span><span id="com"></span>**COM**  
コンポーネント オブジェクトのメソッドでは、コンポーネントをリンクするバイナリの標準を提供します。

<span id="External_Only"></span><span id="external_only"></span><span id="EXTERNAL_ONLY"></span>**外部のみ**  
内部のディスプレイ デバイスではなく、外付けディスプレイ デバイスに表示を表示する場所のラップトップ コンピューターで一般的な構成を表示します。

<span id="Single_View"></span><span id="single_view"></span><span id="SINGLE_VIEW"></span>**1 つのビュー**  
1 つのモニターに 1 つのビューを表示する場所の表示モード。

<span id="Topology"></span><span id="topology"></span><span id="TOPOLOGY"></span>**トポロジ**  
グラフィックス アダプターのターゲットに表示されるソースを指定する情報です。
