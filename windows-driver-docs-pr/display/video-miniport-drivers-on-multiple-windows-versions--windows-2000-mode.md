# <a name="video-miniport-drivers-on-multiple-windows-versions-windows-2000-model"></a>複数の Windows バージョン (Windows 2000 モデル) でビデオのミニポート ドライバー

NT ベースのオペレーティング システム以降で実行する Windows NT 4.0 向けに作成されたビデオのミニポート ドライバーを変更するプランを参照してくださいかどうか[Windows NT 4.0 のミニポート ドライバーを Windows 2000 に変換する](converting-a-windows-nt-4-0-miniport-driver-to-windows-2000.md)します。 現在のバージョンの Windows では、上のビデオのミニポート ドライバーを開発できますが、NT ベースの Windows の以前のバージョンで実行できます。 詳細については、[を使用して VideoPortGetProcAddress](using-videoportgetprocaddress.md)を参照してください。
