# <a name="using-multiple-multiple-head-adapters"></a>複数の複数ヘッド アダプターの使用


## <span id="ddk_using_multiple_multiple_head_adapters_gg"></span><span id="DDK_USING_MULTIPLE_MULTIPLE_HEAD_ADAPTERS_GG"></span>


ドライバーは、複数ヘッド、システムに 1 つ以上のヘッドが複数のカードが搭載されている場合のサポートを提供できます。 ドライバーが 1 つ以上のヘッドが複数のカードを所有している場合、ドライバーは、複数ヘッドの個別のカードが独立に実行されたことで確認する必要があります。
