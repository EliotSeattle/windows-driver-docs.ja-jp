# <a name="encoding-windows-7-display-driver-inf-files-in-unicode"></a>Windows 7 ディスプレイ ドライバーの INF ファイルの Unicode でのエンコード


Unicode; としてボックス内のすべての INF ファイルをエンコードする必要があります。ANSI はできません。

INF ファイルを Unicode (または、INF ファイルが unicode でエンコードされたかどうかを確認する) のエンコードで、次の手順を実行します。

1.  メモ帳を使用して、INF ファイルを開きます。

2.  ファイル メニューで、名前を付けて保存 をクリックします。

3.  ダイアログ ボックスのエンコードのフィールドに"ANSI"が表示されたら、"Unicode"にエンコードを変更し、新しい名前でファイルを保存します。
