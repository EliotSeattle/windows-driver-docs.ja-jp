# <a name="additional-encouraged-configuration-set"></a>その他の推奨構成セット


## <span id="ddk_additional_encouraged_configuration_set_gg"></span><span id="DDK_ADDITIONAL_ENCOURAGED_CONFIGURATION_SET_GG"></span>


ソフトウェアデコーダー用に追加の構成を実装することをお勧めします。 これらの構成はハードウェアに存在する可能性があり、[最小の相互運用性構成セット](minimal-interoperability-configuration-sets.md)の場合と比べて、パフォーマンスが大幅に向上します。

この追加の構成セットは、 [**DXVA\_Configピクチャデコード**](https://docs.microsoft.com/windows-hardware/drivers/ddi/dxva/ns-dxva-_dxva_configpicturedecode)構造体のメンバーの観点で定義されています。
