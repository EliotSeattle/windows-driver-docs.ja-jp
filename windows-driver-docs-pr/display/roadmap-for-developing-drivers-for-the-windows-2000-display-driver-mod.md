# <a name="roadmap-for-the-windows-2000-display-driver-model-xddm"></a>Windows 2000 Display Driver Model (XDDM) のロードマップ


![テキスト"wdk"を高速道路に重ね合わせて示すロードマップの図](images/wdkroadmap-th.png)Windows 2000 display driver model (XDDM) では、グラフィックス ハードウェアの製造元がペアになっているディスプレイ ドライバーとビデオのミニポート ドライバーを指定する必要があります。 カーネル モードで表示するため、これらのドライバーの両方を実行します。

**注**  XDDM と VGA ドライバーは、Windows 8 およびそれ以降のバージョンではコンパイルされません。 ディスプレイ ハードウェアが Windows 表示 Driver Model (WDDM) 1.2 以降をサポートするために認定されているドライバーがない Windows 8 コンピューターに接続されている場合は、Microsoft Basic 表示ドライバーを実行しているシステムが既定値です。

 

XDDM ディスプレイ ドライバーを Windows 7 と Windows の以前のバージョンを作成するダウンロードし Windows 7 Windows Driver Kit (WDK) (build 7600) をインストール、WDK のヘルプのドキュメントを開く、**開始**メニューのおよび、推奨フォロートピックの手順を[Windows Display Driver Model (WDDM) のドライバーの開発のためのロードマップ](roadmap-for-developing-drivers-for-the-windows-vista-display-driver-mo.md)します。
