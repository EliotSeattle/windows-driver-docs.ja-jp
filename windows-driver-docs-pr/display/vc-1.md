# <a name="vc-1"></a>VC-1


Vc-1 は、不連続のコサイン変換 (DCT) に基づいて、H.261、H.263、mpeg-1、mpeg-2 および mpeg-4 にも存在する従来のビデオ コーデック設計の進化したものです。 Vc-1 は代わりに、 [mpeg-4 AVC (H.264)](mpeg-4-avc--h-264-.md)標準ビデオ コーデックです。 Vc-1 には、ビデオのインター レースのシーケンスとしてエンコード プログレッシブ コーディング ツールが含まれています。 Vc-1 開発と標準化の主な目的は、プログレッシブにコンテンツを変換せずにインター レースされたコンテンツの圧縮をサポートするためにです。 このサポートは、ブロードキャストおよびビデオ業界のプロフェッショナルにとって魅力的な vc-1 をできます。 DirectX VA が vc-1 をデコードする方法については、ダウンロード[DirectX ビデオ アクセラレータ仕様の Windows Media ビデオ v8、v9、vA デコード (を含む SMPTE 421m"vc-1")](https://go.microsoft.com/fwlink/p/?linkid=141800)します。
