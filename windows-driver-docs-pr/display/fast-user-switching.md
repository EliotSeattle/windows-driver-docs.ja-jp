# <a name="fast-user-switching"></a>ユーザーの簡易切り替え


## <span id="ddk_fast_user_switching_gg"></span><span id="DDK_FAST_USER_SWITCHING_GG"></span>


ユーザーの簡易切り替え、Windows xp 以降の機能により、複数のユーザーに同じコンピューターにログオンしています。 特定のユーザーのデスクトップと、実行中のアプリケーションは、自分の次に、そのユーザーのログオン セッションから保持されます。

ユーザーの簡易切り替えは、同時に実行する複数の仮想ディスプレイ ドライバーを許可することで機能します。 (各仮想ディスプレイ ドライバーは、特定の PDEV に関連付けられます)。ただし、ビデオのミニポート ドライバーは 1 つのインスタンスとして存在します。 重大な問題が発生したり、ミニポート ドライバーがそのディスプレイ ドライバーのインスタンスがアクティブでなくなったときに、ディスプレイ ドライバーのコンテキストで渡されたメモリ アドレスへのアクセスを試みるとビデオのミニポート ドライバーのコールバックを呼び出す仮想ディスプレイ ドライバーのいずれかとカーネルのスレッド。 ディスプレイ ドライバー/ビデオのミニポート ドライバーのアーキテクチャの基本思想に従っては一方向のみで情報をフローする必要がある: ビデオのミニポート ドライバーにディスプレイ ドライバーから。
