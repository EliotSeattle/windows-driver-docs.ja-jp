# <a name="dcibeginaccess"></a>DCIBeginAccess


## <span id="ddk_dcibeginaccess_gg"></span><span id="DDK_DCIBEGINACCESS_GG"></span>


この関数を使用しないでください。代わりに、DirectDraw でサポートされているドライバーと通信するには、最新の DirectX グラフィックスのインターフェイスを使用します。

**DCIBeginAccess**関数は、指定した四角形に基づくフレーム バッファーを表示するアクセス ポインターを取得します。
