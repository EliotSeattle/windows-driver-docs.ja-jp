# <a name="windows-8-in-box-graphics-drivers-treated-as-generic-drivers"></a>Windows 8 インボックス グラフィックス ドライバーが汎用ドライバーとして扱われる


このシナリオで、MS 基本的なディスプレイ ドライバー (MSBDD) を含む Windows 8 インボックス グラフィックス ドライバーすべて扱わ汎用ドライバーにより、Windows と Windows Update。

つまり、Windows Update で新しい照合グラフィックス ドライバー パッケージとして提供されている、*重要*を更新します。

Windows 8 の Windows Update の既定の設定を使用している場合、重要なドライバーの更新をダウンロードして、ユーザーの介入なしにインストールし、多くの場合、ユーザーを確認しながらこの更新プログラムが発生しました。

Windows 8 が出荷されるときに、インボックス グラフィック ドライバーは Windows Update で最新のドライバーの更新プログラムよりも大幅に古いです。 この動作により、ユーザーが使用可能な最新/最高のグラフィック ドライバーを使用して Windows 8 を感じること。

**注:**  
Windows 8 認定の事前インストールされている Windows 8 で販売されている新しいコンピューターに提供されるグラフィック ドライバーを OEM にジェネリックは考慮されません。 Windows Update で一致する、新しいグラフィック ドライバーは、省略可能として提供する、このような場合に更新します。 ユーザーはオプションのドライバー更新プログラムをインストールする選択アクティブにする必要があります。
