# <a name="dcicreateprimary"></a>DCICreatePrimary


## <span id="ddk_dcicreateprimary_gg"></span><span id="DDK_DCICREATEPRIMARY_GG"></span>


この関数を使用しないでください。代わりに、DirectDraw でサポートされているドライバーと通信するには、最新の DirectX グラフィックスのインターフェイスを使用します。

**DCICreatePrimary**関数は、ディスプレイ デバイスのプライマリ画面を作成し、画面の情報を取得します。
