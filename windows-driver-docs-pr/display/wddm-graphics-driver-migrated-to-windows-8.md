# <a name="wddm-graphics-driver-migrated-to-windows-8"></a>WDDM グラフィックス ドライバーが Windows 8 に移行される


Windows 8 アップグレード インストールでグラフィックス ハードウェアの Window 8 ボックスのカバレッジがない場合は、Windows の以前のバージョンで使用された、WDDM 1.1 または WDDM 1.0 のグラフィック ドライバーは Windows 8 に移行されます。

Windows 8 のインストーラーは、問題の特定のドライバーをブロックできます。 このようなドライバーは、Windows 8 には移行されません。 ドライバーは、このようなブロックを Windows の製品利用統計情報を使用して、報告された問題に基づいて識別されます。 このような場合は、Windows 8 は、Windows Update によって、または OEM/IHV のサポート サイトからユーザーが新しいドライバーがインストールされるまで、MSBDD を使用します。
