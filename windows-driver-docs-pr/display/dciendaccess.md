# <a name="dciendaccess"></a>DCIEndAccess


## <span id="ddk_dciendaccess_gg"></span><span id="DDK_DCIENDACCESS_GG"></span>


この関数を使用しないでください。代わりに、DirectDraw でサポートされているドライバーと通信するには、最新の DirectX グラフィックスのインターフェイスを使用します。

**DCIEndAccess**関数は、フレーム バッファーを表示するアクセスを解放します。
