# <a name="dciopenprovider"></a>DCIOpenProvider


## <span id="ddk_dciopenprovider_gg"></span><span id="DDK_DCIOPENPROVIDER_GG"></span>


この関数を使用しないでください。代わりに、DirectDraw でサポートされているドライバーと通信するには、最新の DirectX グラフィックスのインターフェイスを使用します。

**DCIOpenProvider**関数は、ディスプレイのデバイス コンテキスト ハンドルを取得します。
