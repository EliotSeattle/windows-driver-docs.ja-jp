# <a name="dcicreateoverlay"></a>DCICreateOverlay


## <span id="ddk_dcicreateoverlay_gg"></span><span id="DDK_DCICREATEOVERLAY_GG"></span>


この関数を使用しないでください。代わりに、DirectDraw でサポートされているドライバーと通信するには、最新の DirectX グラフィックスのインターフェイスを使用します。

**DCICreateOverlay**関数は、ディスプレイ デバイスのオーバーレイ サーフェスを作成します。
