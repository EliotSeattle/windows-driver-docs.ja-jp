# <a name="display-driver-testing-tools"></a>ディスプレイ ドライバーのテスト ツール


## <span id="ddk_display_driver_testing_tools_gg"></span><span id="DDK_DISPLAY_DRIVER_TESTING_TOOLS_GG"></span>


Windows 2000 と以降のオペレーティング システム バージョンは、デバッガーを提供するだけでなく、テストとデバッグ、ディスプレイ ドライバーは、次のツールを提供します。

-   *newdisp* --システムを再起動しなくても、ドライバーを動的に再読み込みすることができます。 参照してください[NewDisp:ディスプレイ ドライバーの動的再読み込み](newdisp--dynamic-reload-of-a-display-driver.md)詳細についてはします。

-   コントロール パネルの表示プログラム--でスライダー コントロールでは、GDI を呼び出すには、ドライバーで加速度を動的に変更できます。 参照してください[動的変更の許可されているドライバーのアクセラレータ](dynamic-change-of-permitted-driver-accelerations.md)詳細についてはします。
