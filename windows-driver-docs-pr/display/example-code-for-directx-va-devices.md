# <a name="example-code-for-directx-va-devices"></a>DirectX VA デバイス用のコード例


## <span id="ddk_example_code_for_directx_va_devices_gg"></span><span id="DDK_EXAMPLE_CODE_FOR_DIRECTX_VA_DEVICES_GG"></span>


このセクションのサンプル コードの実装を示しています、[動き補正](motion-compensation-callbacks.md)コード テンプレートと認定の出力保護プロトコル (COPP) のビデオのミニポート ドライバー テンプレート。 動き補正コード テンプレートは、ProcAmp コントロール、デインター レース、および COPP 機能へのアクセスに使用されます。 COPP ビデオのミニポート ドライバーのテンプレートは、COPP 機能へのアクセスに使用されます。 これらのテンプレートを使用すると、ディスプレイ ドライバー、ビデオのミニポート ドライバーの開発を簡略化できます。 ただし、正常に動作するドライバーをこの方法で ProcAmp コントロール、デインター レース、および COPP 機能へのアクセスを実装する必要はありません。
