FP\_ALTERNATEMODE 偶ルールが次のように適用されます。 閉じたパスの外部で明らかにいくつかのポイントに閉じたパス内の任意の点から線を描画します。 行がパス セグメントの数が奇数を超える場合、開始点は閉じた領域の内側はで、塗りつぶし領域の一部であるためです。 交差するセグメント数が偶数では、ポイントを格納する領域にないことを意味します。
