# <a name="high-order-surface-dp2-stream-drawing-tokens"></a>高次サーフェスの DP2 ストリーム描画トークン

D3DDP2OP\_DRAWRECTPATCH トークンは、修正プログラムを四角形を描画するために、ドライバーに送信されます。 D3DDP2OP\_DRAWTRIPATCH トークンは、修正プログラムを三角形を描画するために、ドライバーに送信されます。
