# <a name="general-unicode-requirement-in-inf-files"></a>INF ファイルに Unicode の一般的な要件


INF ファイルを保存および Unicode (utf-16); としてエンコードする必要があります。ANSI または utf-8 する必要がありますはできません。

**INF ファイルで Unicode を確認するには**

1.  Microsoft メモ帳を使用して、INF ファイルを開きます。
2.  **[File]** メニューの **[Save As]** をクリックします。
3.  場合**ANSI**に表示されます、**エンコード**ダイアログのフィールド ボックスで、変更するエンコード**Unicode**し、新しい名前でファイルを保存します。
