# <a name="dcicloseprovider"></a>DCICloseProvider


## <span id="ddk_dcicloseprovider_gg"></span><span id="DDK_DCICLOSEPROVIDER_GG"></span>


この関数を使用しないでください。代わりに、DirectDraw でサポートされているドライバーと通信するには、最新の DirectX グラフィックスのインターフェイスを使用します。

**DCICloseProvider**関数は、ディスプレイのデバイス コンテキストを閉じます。
