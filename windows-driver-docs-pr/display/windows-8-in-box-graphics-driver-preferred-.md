# <a name="span-iddisplaywindows8in-boxgraphicsdriverpreferredspanwindows-8-in-box-graphics-driver-preferred"></a><span id="display.windows_8_in-box_graphics_driver_preferred_"></span>Windows 8 インボックス グラフィック ドライバーを優先


このシナリオでは Windows 8 の組み込みのグラフィック ドライバーが Windows 7 または古いグラフィック ドライバーを推奨されます。

Windows 8 アップグレードのインストールでは、グラフィックス ハードウェアは、Windows 8 のインボックス ドライバーでカバーされる場合、以前のバージョンの Windows のグラフィック ドライバーは、Windows 8 に移行されません。 これはでも古いグラフィックス ドライバーは、4 つの部分一致 (グラフィックス ハードウェア + pc が特定のモデル) と Windows 8 のインボックス ドライバーは、2 部一致 (グラフィックス ハードウェア固有のみ)。 Windows 8 のインボックス ドライバーには、任意のボックスでドライバーまたはロゴよりも、以前の Windows リリースの優れた特徴のスコアが含まれているためにです。 ドライバーの選択条件を理解するのを参照してください。[条件に一致するドライバー](driver-matching-criteria.md)します。 Windows 8 認定済みのドライバーが Windows 7、Windows 8 アップグレード インストール前にインストールされて、そのドライバーは移行されます。

**注**   A Windows 7 のグラフィックス ハードウェア用の Windows 8 ボックスのカバレッジがない場合でも、インボックス ドライバーは、Windows 8 に移行ことはありません。 この場合は、Windows 8 では、Microsoft の基本的なディスプレイ ドライバー (MSBDD) を使用します。
