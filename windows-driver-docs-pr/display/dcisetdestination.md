## <span id="ddk_dcisetdestination_gg"></span><span id="DDK_DCISETDESTINATION_GG"></span>


この関数を使用しないでください。代わりに、DirectDraw でサポートされているドライバーと通信するには、最新の DirectX グラフィックスのインターフェイスを使用します。

**DCISetDestination**関数は、領域の描画先を設定します。
