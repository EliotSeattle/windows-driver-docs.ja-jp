# <a name="direct3d-hardware-requirements-in-windows-8"></a>Windows 8 で Direct3D ハードウェア要件


このトピックでは、Windows 8 でマイクロソフトの Direct3D をサポートするハードウェアの要件について説明します。

独立系ハードウェア ベンダーは、次の表で指定されているハードウェアには、Windows 8 の Direct3D レンダリング要件に従う必要があります。 参照してください[Windows 8 の機能の改善を DirectX](directx-feature-improvements-in-windows-8.md)固有です。

**Direct3D レンダリング ハードウェア要件**

| Microsoft DirectX ハードウェア バージョン | 必須/オプション | Windows 8 のレンダリングの要件 |
|------------------------------------|-------------------|----------------------------------|
| D3D9                               | 必須          | D3D9 ハードウェア仕様                     |
| D3D10                              | 必須          | D3D9 ハードウェア仕様                     |
| D3D10                              | 必須          | D3D10 ハードウェア仕様                    |
| D3D10.1                            | 必須          | D3D9 ハードウェア仕様                     |
| D3D10.1                            | 必須          | D3D10 ハードウェア仕様                    |
| D3D10.1                            | 必須          | D3D10.1 ハードウェア仕様                  |
| D3D11                              | 必須          | D3D9 ハードウェア仕様                     |
| D3D11                              | 必須          | D3D10 ハードウェア仕様                    |
| D3D11                              | 必須          | D3D10.1 ハードウェア仕様                  |
| D3D11                              | 必須          | D3D11 ハードウェア仕様                    |
| D3D11.1                            | 必須          | D3D9 ハードウェア仕様                     |
| D3D11.1                            | 必須          | D3D10 ハードウェア仕様                    |
| D3D11.1                            | 必須          | D3D10.1 ハードウェア仕様                  |
| D3D11.1                            | 必須          | D3D11 ハードウェア仕様                    |
| D3D11.1                            | 必須          | D3D11.1 ハードウェア仕様                  |

 

次の表では、Windows 8 の Direct3D ハードウェア仕様の更新プログラムについて説明します。

**Windows 8 向けの Microsoft direct3d10 ハードウェア仕様の変更**

| 必須?      | 機能                            |
|----------------|------------------------------------|
| 必須       | ピクセル形式 (5551、565、4444) \* |
| 必須       | 同じ画面ブリット \*              |
| 実装されている場合 | Ops のロジック                          |

 

**Windows 8 の Direct3D 10.1 ハードウェア仕様の変更**

| 必須?      | 機能                            |
|----------------|------------------------------------|
| 必須       | ピクセル形式 (5551、565、4444) \* |
| 必須       | 同じ画面ブリット \*              |
| 実装されている場合 | Ops のロジック                          |

 

**Windows 8 向けの Microsoft direct3d11 ハードウェア仕様の変更**

| 必須?      | 機能                            |
|----------------|------------------------------------|
| 必須       | ピクセル形式 (5551、565、4444) \* |
| 必須       | 同じ画面ブリット \*              |
| 実装されている場合 | UAV MSAA                           |
| 実装されている場合 | 同時実行スレッドを作成します       |
| 実装されている場合 | スレッドのコマンド一覧            |
| 実装されている場合 | 倍精度サポート           |
| 実装されている場合 | Ops のロジック                          |

 

**Windows 8 のハードウェア仕様を Direct3D 11.1**

| 必須?      | 機能                                |
|----------------|----------------------------------------|
| 必須       | Ops のロジック                              |
| 必須       | ピクセル形式 (5551、565、4444) \*     |
| 必須       | 同じ画面ブリット \*                  |
| 必須       | すべてのステージにおいて UAVs                    |
| 必須       | UAV MSAA                               |
| 必須       | ターゲットに依存しないラスタライズ (TIR) |
| 実装されている場合 | 同時実行スレッドを作成します           |
| 実装されている場合 | スレッドのコマンド一覧                |
| 実装されている場合 | 倍精度サポート               |
