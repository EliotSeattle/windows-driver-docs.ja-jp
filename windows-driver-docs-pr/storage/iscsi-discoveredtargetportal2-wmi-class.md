# <a name="iscsi_discoveredtargetportal2-wmi-class"></a>ISCSI\_DiscoveredTargetPortal2 WMI クラス


ISCSI\_DiscoveredTargetPortal2 WMI クラスは、検出されたターゲットポータルに関連付けられている情報を提供します。 ターゲットポータルのネットワークアドレスに加えて、このクラスには、ターゲットポータルのセキュリティ特性に関する情報が含まれています。

Iscsi\_DiscoveredTargetPortal2 は[iscsi\_DISCOVEREDTARGETPORTAL wmi クラス](iscsi-discoveredtargetportal-wmi-class.md)に似ていますが、Iscsi\_DISCOVEREDTARGETPORTAL2 は Iscsi\_DiscoveredTargetPortal wmi によってターゲットポータル情報を格納します。クラスには、ポータルグループのタグ番号などのがありません。

ISCSI\_DiscoveredTargetPortal2 WMI クラスは、ターゲットポータルアドレスのみを定義する[iscsi\_TargetPortal WMI クラス](iscsi-targetportal-wmi-class.md)によって提供される情報のスーパーセットです。

ISCSI\_DiscoveredTargetPortal2 WMI クラスは、「 *Discover .mof*」で定義されている、パブリッシュされていないクラスです。

WMI ツールスイートは、このクラス定義をコンパイルするときに、 [**ISCSI\_DiscoveredTargetPortal2**](https://docs.microsoft.com/windows-hardware/drivers/ddi/iscsifnd/ns-iscsifnd-_iscsi_discoveredtargetportal2)データ構造を生成します。
