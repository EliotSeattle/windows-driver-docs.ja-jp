# <a name="removeradiusserver"></a>RemoveRADIUSServer


**Removerは**、イニシエーターとターゲットの間で認証ネゴシエーションを監督する radius サーバーの一覧から radius サーバーを削除します。

この WMI メソッドは、発行されていない[Msiscsi\_OPERATIONS WMI クラス](msiscsi-operations-wmi-class.md)に属しています。 **Removerの**パラメーターの説明については、「」および「\_[ **\_OUT**](https://docs.microsoft.com/windows-hardware/drivers/ddi/iscsiop/ns-iscsiop-_removeradiusserver_out)構造体」の「 [**removerの**](https://docs.microsoft.com/windows-hardware/drivers/ddi/iscsiop/ns-iscsiop-_removeradiusserver_in)メンバーの説明」を参照してください。

MSiSCSI\_Operations WMI クラスを実装するミニポートドライバーは、 **removerを**サポートするためには必要ありません。
