# <a name="removeradiusserver"></a>RemoveRADIUSServer


**RemoveRADIUSServer**メソッドは、イニシエーターとターゲット間の認証のネゴシエーションを監督する RADIUS サーバーの一覧から RADIUS サーバーを削除します。

この WMI メソッドは、パブリッシュされていないに属する[MSiSCSI\_操作 WMI クラス](msiscsi-operations-wmi-class.md)します。 パラメーターの説明については、 **RemoveRADIUSServer**メソッドのメンバーの説明を参照してください、 [ **RemoveRADIUSServer\_IN** ](https://msdn.microsoft.com/library/windows/hardware/ff564023)と[**RemoveRADIUSServer\_アウト**](https://msdn.microsoft.com/library/windows/hardware/ff564029)構造体。

ミニポート ドライバー、MSiSCSI を実装する\_操作 WMI クラスがサポートする必要はありません**RemoveRADIUSServer**します。
