# <a name="msfceventcontrol-wmi-class"></a>MSFC\_EventControl WMI クラス


## <span id="ddk_msfc_eventcontrol_wmi_class_kr"></span><span id="DDK_MSFC_EVENTCONTROL_WMI_CLASS_KR"></span>


MSFC\_EventControl クラスが WMI クライアントがリンク、ポート、およびターゲット イベントの配信を制御できるようにする WMI メソッドを定義します。

この WMI クラスには、データ ブロックがありませんが、WMI ツールのスイートで、クラスに属するメソッドのパラメーター データを格納する構造体の宣言を生成するためと、クラス自体に対応する構造体の宣言を生成しません。

このクラスに属している各メソッドの MOF 構文は、メソッドのリファレンス ページの説明です。 次のリファレンス ページには、これらのメソッドとそれらに付随する構造体について説明します。

[**AddLink**](addlink.md)

[**AddPort**](addport.md)

[**AddTarget**](addtarget.md)

[**RemoveLink**](removelink.md)

[**RemovePort**](removeport.md)

[**RemoveTarget**](removetarget.md)
