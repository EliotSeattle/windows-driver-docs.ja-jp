# <a name="msiscsi_bootinformation-wmi-class"></a>MSiSCSI\_BootInformation WMI クラス


MSiSCSI\_BootInformationWMI クラスは、ブートデバイスが含まれているノードに関する情報を公開します。

このクラスは発行されていません。 MSiSCSI\_BootInformation WMI クラスによって報告される情報の詳細については、 [**msiscsi\_bootinformation**](https://docs.microsoft.com/windows-hardware/drivers/ddi/iscsiop/ns-iscsiop-_msiscsi_bootinformation)構造のメンバーの説明を参照してください。

MSiSCSI\_BootInformation WMI クラスを実装するミニポートドライバーは、物理デバイスオブジェクト (PDO) のインスタンス名を使用してクラスを登録する必要があります。
