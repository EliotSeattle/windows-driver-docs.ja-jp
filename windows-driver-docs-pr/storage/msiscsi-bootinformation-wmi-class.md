# <a name="msiscsibootinformation-wmi-class"></a>MSiSCSI\_BootInformation WMI クラス


MSiSCSI\_BootInformationWMI クラスは、ブート デバイスを含むノードに関する情報を公開します。

このクラスは発行されません。 情報の説明を MSiSCSI\_BootInformation WMI クラスのレポートのメンバーの説明を参照してください、 [ **MSiSCSI\_BootInformation** ](https://docs.microsoft.com/windows-hardware/drivers/ddi/content/iscsiop/ns-iscsiop-_msiscsi_bootinformation)構造体。

ミニポート ドライバー、MSiSCSI を実装する\_BootInformation WMI クラスは、物理デバイス オブジェクト (PDO) のインスタンス名を使用して、クラスを登録する必要があります。
