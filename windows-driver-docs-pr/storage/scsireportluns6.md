**ScsiReportLuns**メソッドは、ターゲットで論理ユニットを SCSI レポート Lun のコマンドを発行し、結果が返されます。

この WMI メソッドは、パブリッシュされていないに属する[MSiSCSI\_操作 WMI クラス](msiscsi-operations-wmi-class.md)します。 パラメーターの説明については、 **ScsiReportLuns**メソッドのメンバーの説明を参照してください、 [ **ScsiReportLuns\_IN** ](https://msdn.microsoft.com/library/windows/hardware/ff564925)と[ **ScsiReportLuns\_アウト**](https://msdn.microsoft.com/library/windows/hardware/ff564936)構造体。

ミニポート ドライバー、MSiSCSI を実装する\_操作 WMI クラスをサポートする必要があります**ScsiReportLuns**します。
