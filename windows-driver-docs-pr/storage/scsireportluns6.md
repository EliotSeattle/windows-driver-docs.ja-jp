# <a name="scsireportluns"></a>ScsiReportLuns


**ScsiReportLuns**メソッドは、SCSI レポート lun コマンドをターゲットの論理ユニットに発行し、結果を返します。

この WMI メソッドは、発行されていない[Msiscsi\_OPERATIONS WMI クラス](msiscsi-operations-wmi-class.md)に属しています。 **ScsiReportLuns**メソッドのパラメーターの説明については、「」および「 [**ScsiReportLuns\_OUT**](https://docs.microsoft.com/windows-hardware/drivers/ddi/iscsiop/ns-iscsiop-_scsireportluns_out)構造体」の「メンバーの[ **\_** ](https://docs.microsoft.com/windows-hardware/drivers/ddi/iscsiop/ns-iscsiop-_scsireportluns_in)説明」を参照してください。

MSiSCSI\_Operations WMI クラスを実装するミニポートドライバーは、 **ScsiReportLuns**をサポートしている必要があります。
