# <a name="iscsi_discoveredtargetportalgroup-wmi-class"></a>ISCSI\_DiscoveredTargetPortalGroup WMI クラス


## <span id="ddk_iscsi_discoveredtargetportalgroup_wmi_class_kr"></span><span id="DDK_ISCSI_DISCOVEREDTARGETPORTALGROUP_WMI_CLASS_KR"></span>


ISCSI\_DiscoveredTargetPortalGroup WMI クラスには、ポータルグループ内のすべてのポータルに関する情報が含まれています。

このクラスは発行されていないため、 *Discover .mof*で定義されています。

WMI ツールスイートによってこのクラス定義がコンパイルされると、 [**ISCSI\_DiscoveredTargetPortalGroup**](https://docs.microsoft.com/windows-hardware/drivers/ddi/iscsifnd/ns-iscsifnd-_iscsi_discoveredtargetportalgroup)データ構造が生成されます。
