# <a name="addradiusserver"></a>AddRADIUSServer


**AddRADIUSServer**メソッドは、イニシエーターがターゲットを持つセッションを確立するために使用する RADIUS サーバーの一覧に RADIUS サーバーを追加します。 HBA イニシエーターは、不揮発性メモリに RADIUS サーバーの一覧を格納する必要があります。

この WMI メソッドは、パブリッシュされていないに属する[MSiSCSI\_操作 WMI クラス](msiscsi-operations-wmi-class.md)します。 パラメーターの説明については、 **AddRADIUSServer**メソッドのメンバーの説明を参照してください、 [ **AddRADIUSServer\_IN** ](https://msdn.microsoft.com/library/windows/hardware/ff550134)と[ **AddRADIUSServer\_アウト**](https://msdn.microsoft.com/library/windows/hardware/ff550135)構造体。
