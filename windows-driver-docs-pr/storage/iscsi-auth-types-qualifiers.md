# <a name="iscsiauthtypesqualifiers"></a>ISCSI\_AUTH\_型\_修飾子


## <span id="ddk_iscsi_auth_types_qualifiers_kr"></span><span id="DDK_ISCSI_AUTH_TYPES_QUALIFIERS_KR"></span>


ISCSI\_AUTH\_型\_修飾子 WMI プロパティ修飾子は、ログオンの接続を確立するために使用する認証方法の種類を示す状態値に対応します。

次の表に、ISCSI\_AUTH\_型\_修飾子の値。

<table>
<colgroup>
<col width="50%" />
<col width="50%" />
</colgroup>
<thead>
<tr class="header">
<th align="left">状態値</th>
<th align="left">説明</th>
</tr>
</thead>
<tbody>
<tr class="odd">
<td align="left"><p>0</p></td>
<td align="left"><p>認証の種類が指定されていません。</p></td>
</tr>
<tr class="even">
<td align="left"><p>1</p></td>
<td align="left"><p>認証の種類は、チャレンジ ハンドシェイク認証プロトコル (CHAP) 認証します。</p></td>
</tr>
<tr class="odd">
<td align="left"><p>2</p></td>
<td align="left"><p>認証の種類は、相互 CHAP 認証です。</p></td>
</tr>
</tbody>
</table>
