# <a name="iscsiinitiatorfailuretypequalifiers"></a>ISCSI\_イニシエーター\_エラー\_型\_修飾子


## <span id="ddk_iscsi_initiator_failure_type_qualifiers_kr"></span><span id="DDK_ISCSI_INITIATOR_FAILURE_TYPE_QUALIFIERS_KR"></span>


ISCSI\_イニシエーター\_エラー\_型\_修飾子 WMI プロパティ修飾子は、発信側エラーの種類を表す値のグループに対応します。

次の表に、ISCSI\_イニシエーター\_エラー\_型\_修飾子の値。

<table>
<colgroup>
<col width="50%" />
<col width="50%" />
</colgroup>
<thead>
<tr class="header">
<th align="left">開始側のエラー値です。</th>
<th align="left">意味</th>
</tr>
</thead>
<tbody>
<tr class="odd">
<td align="left"><p>LoginOtherFail</p></td>
<td align="left"><p>イニシエーターは、その他の理由ログインできませんでした。</p></td>
</tr>
<tr class="even">
<td align="left"><p>LoginAuthFail</p></td>
<td align="left"><p>イニシエーターは、イニシエーターの認証エラーのためにログインできませんでした。</p></td>
</tr>
<tr class="odd">
<td align="left"><p>LoginAuthenticateFail</p></td>
<td align="left"><p>イニシエーターは、ターゲットが失敗した認証の応答のためにログインできませんでした。</p></td>
</tr>
<tr class="even">
<td align="left"><p>LoginNegotiateFail</p></td>
<td align="left"><p>イニシエーターは、ログイン ネゴシエーションのパラメーターのためにログインできませんでした。</p></td>
</tr>
<tr class="odd">
<td align="left"><p>LogoutOthers</p></td>
<td align="left"><p>ログアウト中、イニシエーターが失敗しました。</p></td>
</tr>
