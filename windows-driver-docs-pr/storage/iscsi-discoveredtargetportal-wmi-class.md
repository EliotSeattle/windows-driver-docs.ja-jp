# <a name="iscsi_discoveredtargetportal-wmi-class"></a>ISCSI\_DiscoveredTargetPortal WMI クラス


## <span id="ddk_iscsi_discoveredtargetportal_wmi_class_kr"></span><span id="DDK_ISCSI_DISCOVEREDTARGETPORTAL_WMI_CLASS_KR"></span>


ISCSI\_DiscoveredTargetPortal WMI クラスは、検出されたターゲットポータルに関連付けられている情報を提供します。 ターゲットポータルのネットワークアドレスに加えて、このクラスには、ターゲットポータルのセキュリティ特性に関する情報が含まれています。

ISCSI\_DiscoveredTargetPortal は、ターゲットポータルのアドレスのみを定義する[iscsi\_TargetPortal WMI クラス](iscsi-targetportal-wmi-class.md)によって提供される情報のスーパーセットです。

ISCSI\_DiscoveredTargetPortal クラスは発行されていませ*ん。* また、「検出 .mof」で定義されています。

WMI ツールスイートによってこのクラス定義がコンパイルされると、 [**ISCSI\_DiscoveredTargetPortal**](https://docs.microsoft.com/windows-hardware/drivers/ddi/iscsifnd/ns-iscsifnd-_iscsi_discoveredtargetportal)データ構造が生成されます。
