# <a name="msiscsiadapterevent-wmi-class"></a>MSiSCSI\_AdapterEvent WMI クラス


## <span id="ddk_msiscsi_adapterevent_wmi_class_kr"></span><span id="DDK_MSISCSI_ADAPTEREVENT_WMI_CLASS_KR"></span>


MSiSCSI\_AdapterEvent WMI クラスは、ホスト記憶域アダプター (HBA) が追加または削除されるたびに、または HBA のターゲットの一覧が変更されたときに、システムが起動するイベントを定義します。

MSiSCSI\_AdapterEvent クラスは発行されませんしで定義されている*Operations.mof*します。

WMI ツールのスイートでは、このクラスの定義をコンパイルするときに生成、 [ **MSiSCSI\_AdapterEvent** ](https://docs.microsoft.com/windows-hardware/drivers/ddi/content/iscsiop/ns-iscsiop-_msiscsi_adapterevent)データ構造体。
