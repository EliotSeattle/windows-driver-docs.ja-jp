# <a name="msiscsi_adapterevent-wmi-class"></a>MSiSCSI\_AdapterEvent WMI クラス


## <span id="ddk_msiscsi_adapterevent_wmi_class_kr"></span><span id="DDK_MSISCSI_ADAPTEREVENT_WMI_CLASS_KR"></span>


MSiSCSI\_AdapterEvent WMI クラスは、ホスト記憶域アダプター (HBA) が追加または削除されるたび、または HBA のターゲットリストが変更されたときにシステムが起動するイベントを定義します。

MSiSCSI\_AdapterEvent クラスは発行されていないため、*操作 .mof*で定義されています。

WMI ツールスイートは、このクラス定義をコンパイルするときに、 [**Msiscsi\_AdapterEvent**](https://docs.microsoft.com/windows-hardware/drivers/ddi/iscsiop/ns-iscsiop-_msiscsi_adapterevent)データ構造を生成します。
