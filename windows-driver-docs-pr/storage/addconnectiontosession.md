# <a name="addconnectiontosession"></a>AddConnectionToSession


**Addconnectiontosession**メソッドは、ログオンセッションに新しい接続を追加するために ISCSI イニシエーター HBA を管理するミニポートドライバーに指示します。

[Msiscsi\_Operations クラス](msiscsi-operations-wmi-class.md)をサポートするミニポートドライバーは、少なくともこのメソッドにスタブを提供する必要がありますが、メソッドの機能の実装は省略可能です。 メソッドに接続の追加機能が実装されていない場合は、エラーが返されます。

**Addconnectiontosession**は、発行されていない msiscsi\_Operations WMI クラスに属しています。 **Addconnectiontosession**メソッドのパラメーターの説明については、addconnectiontosession のメンバーの説明、および[**addconnectiontosession\_OUT**](https://docs.microsoft.com/windows-hardware/drivers/ddi/iscsiop/ns-iscsiop-_addconnectiontosession_out)構造体の[ **\_** ](https://docs.microsoft.com/windows-hardware/drivers/ddi/iscsiop/ns-iscsiop-_addconnectiontosession_in)を参照してください。
