# <a name="addconnectiontosession"></a>AddConnectionToSession


**AddConnectionToSession**メソッド iSCSI イニシエーター ログオン セッションに新しい接続を追加する HBA を管理するミニポート ドライバーに指示します。

ミニポート ドライバーをサポートする、 [MSiSCSI\_操作クラス](msiscsi-operations-wmi-class.md)少なくともスタブは、このメソッドを指定する必要がありますが、メソッドの機能の実装はオプションです。 メソッドが追加の接続機能を実装していない場合、エラーが返されます。

**AddConnectionToSession**パブリッシュされていない MSiSCSI に属している\_WMI クラスを操作します。 パラメーターの説明については、 **AddConnectionToSession**メソッドのメンバーの説明を参照してください、 [ **AddConnectionToSession\_IN** ](https://docs.microsoft.com/windows-hardware/drivers/ddi/content/iscsiop/ns-iscsiop-_addconnectiontosession_in)と[ **AddConnectionToSession\_アウト**](https://docs.microsoft.com/windows-hardware/drivers/ddi/content/iscsiop/ns-iscsiop-_addconnectiontosession_out)構造体。
