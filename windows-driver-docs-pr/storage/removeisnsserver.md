# <a name="removeisnsserver"></a>RemoveiSNSServer


**RemoveiSNSServer**メソッドは、イニシエーターがターゲットのクエリを実行する iSNS サーバーの一覧から iSNS サーバーを削除します。 HBA イニシエーターは、不揮発性メモリに iSNS サーバーの一覧を格納する必要があります。

**RemoveiSNSServer** 、パブリッシュされていないに属している[MSiSCSI\_操作 WMI クラス](msiscsi-operations-wmi-class.md)します。 パラメーターの説明については、 **RemoveiSNSServer**メソッドのメンバーの説明を参照してください、 [ **RemoveiSNSServer\_IN** ](https://msdn.microsoft.com/library/windows/hardware/ff563981)と[ **RemoveiSNSServer\_アウト**](https://msdn.microsoft.com/library/windows/hardware/ff563982)構造体。

ミニポート ドライバー、MSiSCSI を実装する\_操作 WMI クラスは、このメソッドをサポートする必要はありません。
