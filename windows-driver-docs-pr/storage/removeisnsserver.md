# <a name="removeisnsserver"></a>RemoveiSNSServer


**Removeisnsserver**メソッドは、イニシエーターがターゲットを照会する isns サーバーの一覧から isns サーバーを削除します。 HBA イニシエーターは、iSNS サーバーの一覧を不揮発性メモリに格納する必要があります。

**Removeisnsserver**は、発行されていない[Msiscsi\_Operations WMI クラス](msiscsi-operations-wmi-class.md)に属しています。 **Removeisnsserver**メソッドのパラメーターの説明については、 [**removeisnsserver\_** ](https://docs.microsoft.com/windows-hardware/drivers/ddi/iscsiop/ns-iscsiop-_removeisnsserver_in)のメンバーの説明、および[**REMOVEISNSSERVER\_OUT**](https://docs.microsoft.com/windows-hardware/drivers/ddi/iscsiop/ns-iscsiop-_removeisnsserver_out)構造体を参照してください。

MSiSCSI\_Operations WMI クラスを実装するミニポートドライバーは、この方法をサポートするために必要ありません。
