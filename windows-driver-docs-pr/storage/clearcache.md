# <a name="clearcache"></a>ClearCache


**ClearCache**メソッドには、イニシエーターの iSCSI の認証と事前共有キーのキャッシュをクリアする HBA がように指示します。 このメソッドが*いない*その他の永続的な情報をオフにします。 具体的には、永続的なログオンは削除しないでください。

**ClearCache** 、パブリッシュされていないに属している[MSiSCSI\_SecurityConfigOperations WMI クラス](msiscsi-securityconfigoperations-wmi-class.md)します。 パラメーターの説明については、 **ClearCache**メソッドのメンバーの説明を参照してください、 [ **ClearCache\_アウト**](https://msdn.microsoft.com/library/windows/hardware/ff552447)構造体。
