# <a name="removepersistentlogin"></a>RemovePersistentLogin


**RemovePersistentLogin**メソッドは、永続的なログオンのイニシエーターの一覧からターゲットを削除します。

この WMI メソッドは、パブリッシュされていないに属する[MSiSCSI\_操作 WMI クラス](msiscsi-operations-wmi-class.md)します。 パラメーターの説明については、 **RemovePersistentLogin**メソッドのメンバーの説明を参照してください、 [ **RemovePersistentLogin\_IN** ](https://msdn.microsoft.com/library/windows/hardware/ff564003)と[ **RemovePersistentLogin\_アウト**](https://msdn.microsoft.com/library/windows/hardware/ff564008)構造体。

ミニポート ドライバー、MSiSCSI を実装する\_操作 WMI クラスをサポートする必要があります**RemovePersistentLogin**します。
