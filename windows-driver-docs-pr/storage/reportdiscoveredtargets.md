# <a name="reportdiscoveredtargets"></a>ReportDiscoveredTargets


**ReportDiscoveredTargets** WMI メソッドは、検出されたすべてのターゲットを報告します。

この WMI メソッドは、「 *Discover .mof*」で定義されている、公開されていない[Msiscsi\_discoveryoperations WMI クラス](msiscsi-discoveryoperations-wmi-class.md)に属しています。 **ReportDiscoveredTargets**メソッドのパラメーターの説明については、 [**ReportDiscoveredTargets\_OUT**](https://docs.microsoft.com/windows-hardware/drivers/ddi/iscsifnd/ns-iscsifnd-_reportdiscoveredtargets_out)構造体のメンバーの説明を参照してください。

MSiSCSI\_DiscoveryOperations WMI クラスを実装するミニポートドライバーは、 **ReportDiscoveredTargets**をサポートしている必要があります。
