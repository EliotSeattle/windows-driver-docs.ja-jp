# <a name="iscsi_discoveredtarget2-wmi-class"></a>ISCSI\_DiscoveredTarget2 WMI クラス


ISCSI\_DiscoveredTarget2 WMI クラスは、検出されたターゲットに関連する情報を提供します。

このクラスは[iscsi\_DISCOVEREDTARGET wmi クラス](iscsi-discoveredtarget-wmi-class.md)に似ていますが、Iscsi\_DiscoveredTarget2 には、Iscsi\_DiscoveredTarget wmi クラスに含まれていないターゲットポータル情報 (ポータルのタグ番号など) が含まれています。グループ.

ISCSI\_DiscoveredTarget2 クラスは発行されていませ*ん。* また、「検出 .mof」で定義されています。

WMI ツールスイートによってこのクラス定義がコンパイルされると、 [**ISCSI\_DiscoveredTarget2**](https://docs.microsoft.com/windows-hardware/drivers/ddi/iscsifnd/ns-iscsifnd-_iscsi_discoveredtarget2)データ構造が生成されます。
