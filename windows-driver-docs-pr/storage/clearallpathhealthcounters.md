# <a name="clearallpathhealthcounters-function"></a>ClearAllPathHealthCounters 関数


ClearAllPathsHealthCounters メソッドは、収集した MPIO によって管理されているすべてのパスの MPIO の正常性統計をすべてクリアに使用されます。

<a name="syntax"></a>構文
------

```ManagedCPlusPlus
unknown ClearAllPathHealthCounters(void);
```

<a name="parameters"></a>パラメーター
----------

この関数には、パラメーターがありません。

<a name="return-value"></a>戻り値
------------

なし

<a name="remarks"></a>注釈
-------

この WMI メソッドは、MPIO が属する\_WMI\_メソッド WMI クラスです。
