# <a name="msfcfibreportnpivmethods-wmi-class"></a>MSFC\_FibrePortNPIVMethods WMI クラス


NPIV WMI プロバイダーでは、MSFC\_FibrePortNPIVMethods クラスを仮想ポートを作成する管理クライアントを許可します。 これは移行するために特に役立ちますから別のホストに仮想マシンに関連付けられているファイル。

この WMI クラスは、メソッドのみが含まれていて、データ ブロックがありません。 そのため、WMI ツールのスイートには、このクラスに属するメソッドのパラメーター データを格納する構造が生成されますが、クラス自体に対応する構造を生成しません。

メソッドのリファレンス ページは、このクラスに属している各メソッドの MOF ファイルの構文を説明します。 次のトピックでは、これらのメソッドとそれらに付随する構造体について説明します。

[**CreateVirtualPort**](createvirtualport.md)

[**RemoveVirtualPort**](removevirtualport.md)
