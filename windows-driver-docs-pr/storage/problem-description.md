# <a name="problem-description"></a>問題の説明


Windows コンピューターのシステム、クラッシュ ダンプ、および休止状態のサポートは、特別な記憶装置ドライバー、ポートおよびミニポートの両方からのサポートのあるクラッシュ ダンプ ドライバーによって提供されます。 システムがクラッシュしたときに、ダンプ データは、一般的な記憶域スタックではなく、これらの特別なドライバーのセットを使用して、ディスクに書き込まれます。 これらのドライバーがファイル システム ボリュームまたはボリューム マネージャー ドライバーを使用せずに、データを書き込むし、サポートされていませんでした以前クラッシュ ダンプのスタックのすべてのドライバーを追加します。
