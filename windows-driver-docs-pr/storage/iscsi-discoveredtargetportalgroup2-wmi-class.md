# <a name="iscsi_discoveredtargetportalgroup2-wmi-class"></a>ISCSI\_DiscoveredTargetPortalGroup2 WMI クラス


ISCSI\_DiscoveredTargetPortalGroup2 WMI クラスには、ポータルグループ内のすべてのポータルに関する情報が含まれています。

このクラスは[iscsi\_DISCOVEREDTARGETPORTALGROUP wmi クラス](iscsi-discoveredtargetportalgroup-wmi-class.md)に似ていますが、Iscsi\_DiscoveredTargetPortalGroup2 には、Iscsi\_DiscoveredTargetPortalGroup wmi クラスには含まれていないターゲットポータル情報が含まれています。(ポータルグループのタグ番号など)。

ISCSI\_DiscoveredTargetPortalGroup2 クラスは発行されていませ*ん。* また、「検出 .mof」で定義されています。

WMI ツールスイートによってこのクラス定義がコンパイルされると、 [**ISCSI\_DiscoveredTargetPortalGroup2**](https://docs.microsoft.com/windows-hardware/drivers/ddi/iscsifnd/ns-iscsifnd-_iscsi_discoveredtargetportalgroup2)データ構造が生成されます。
