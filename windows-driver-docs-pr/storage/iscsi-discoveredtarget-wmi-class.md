# <a name="iscsi_discoveredtarget-wmi-class"></a>ISCSI\_DiscoveredTarget WMI クラス


## <span id="ddk_iscsi_discoveredtarget_wmi_class_kr"></span><span id="DDK_ISCSI_DISCOVEREDTARGET_WMI_CLASS_KR"></span>


ISCSI\_DiscoveredTarget WMI クラスは、検出されたターゲットに関連する情報を提供します。

このクラスは発行されていないため、 *Discover .mof*で定義されています。

WMI ツールスイートは、このクラス定義をコンパイルするときに、 [**ISCSI\_DiscoveredTarget**](https://docs.microsoft.com/windows-hardware/drivers/ddi/iscsifnd/ns-iscsifnd-_iscsi_discoveredtarget)データ構造を生成します。
