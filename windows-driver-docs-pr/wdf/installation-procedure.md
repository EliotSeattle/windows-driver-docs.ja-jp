# <a name="installation-procedure"></a>インストール手順


WDM ドライバーと同様、INF ファイルを使用してカーネル モード ドライバー フレームワーク (KMDF) とユーザー モード ドライバー フレームワーク (UMDF) ドライバーがインストールされます。 ただし、framework ベースのドライバーは、適切な共同インストーラーを使用する必要があります、KMDF または UMDF に必要なその他のサービスに関する情報を追加する必要があります。

実装の詳細については、[のビルド、インストール、およびテスト](building--installing--and-testing-a-wdf-driver.md)を参照してください。
