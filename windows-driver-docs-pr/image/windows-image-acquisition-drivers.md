# <a name="windows-image-acquisition-drivers"></a>Windows Image Acquisition ドライバー

Microsoft Windows Image Acquisition (WIA) がプログラミング インターフェイスは、Windows のイメージ (まだ STI) アーキテクチャです。 このドキュメントでは、デジタル カメラやスキャナーなどのイメージング デバイスへの通信をサポートしている WIA ドライバーを開発する方法について説明します。

WIA は、Windows STI モデルから進化してきたフル装備のイメージング アーキテクチャです。 WIA は、業界標準のプロトコルなど、画像転送プロトコル (PTP) カメラの堅牢性、信頼性、一貫性のあるユーザー エクスペリエンス、および組み込みのドライバー サポートを提供します。
