# <a name="wiaipsautodeskew"></a>WIA\_IP\_自動\_DESKEW


WIA\_IP\_自動\_DESKEW プロパティは、デバイスが傾斜の自動修正を使用するかどうかを示します。 WIA ミニドライバーは、作成し、このプロパティを保持します。

プロパティの種類:VT\_I4

有効な値 :WIA\_PROP\_一覧

アクセス権:読み取り/書き込み

<a name="remarks"></a>注釈
-------

次の表は、WIA で有効な定数\_IP\_自動\_DESKEW プロパティ。

<table>
<colgroup>
<col width="50%" />
<col width="50%" />
</colgroup>
<thead>
<tr class="header">
<th>値</th>
<th>定義</th>
</tr>
</thead>
<tbody>
<tr class="odd">
<td><p>WIA_AUTO_DESKEW_ON</p></td>
<td><p>傾斜の自動修正を使用します。</p></td>
</tr>
<tr class="even">
<td><p>WIA_AUTO_DESKEW_OFF</p></td>
<td><p>傾斜の自動修正を使用しないでください。</p></td>
</tr>
</tbody>
</table>
