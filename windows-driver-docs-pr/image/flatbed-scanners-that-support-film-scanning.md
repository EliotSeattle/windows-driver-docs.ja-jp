# <a name="flatbed-scanners-that-support-film-scanning"></a>フィルムのスキャンをサポートするためのフラット ベッド スキャナー





次の図は、フィルム ベッドからプラテン上にフィルムの画面をスキャンしてスキャンをサポートしているフラット ベッド スキャナーの想定される WIA 項目ツリーを示しています。 図では、物理デバイスとドキュメントも示します。

![プラテン専用のスキャンに関するフラット ベッド フィルム スキャナーの項目のツリーを示す図](images/art-flatbed-film.png)

上記の図では、左側のツリーは、スキャナーの項目のツリーを表します。 曲線要素に、右に描画されるシンボル、物理デバイスとこのアイテム ツリーによって表されるドキュメントで表します。

フィルム スキャナーの項目は、常にスキャン画面全体のフィルムを表します。 スキャンのプレビュー領域をスキャンする全体のフィルムを表す 1 つのイメージを常に存在するように、全体のスキャンの面にエクステント設定の有効な値を制限する必要があります。 この 1 つのイメージは、スキャンのサーフェイスに配置されているスライドまたはフィルムの表現が、ユーザーを表示するアプリケーションに適しています。 個々 のフレームの項目の範囲設定は、画面をスキャンするフィルムの物理的なサイズに制限されます。 ベッド サイズ項目のプロパティをフィルム[ **WIA\_DPS\_水平\_ベッド\_サイズ**](https://msdn.microsoft.com/library/windows/hardware/ff551399)と[ **WIA\_DPS\_垂直\_ベッド\_サイズ**](https://msdn.microsoft.com/library/windows/hardware/ff551445)領域をスキャンするフィルムの物理的なサイズを表す必要があります。 画面をスキャンするフィルムのエクステントの設定は、スキャン プラテン ベッドの途中である場合でも (0, 0) で起動に注意してください。 この番号では、画面をスキャンするフィルムがある配信元をスキャン ベッドの独立した独自の原点のためです。

**注**  オーバー ラップ フレームの選択範囲がセッションをスキャンするフィルムで許可されます。
