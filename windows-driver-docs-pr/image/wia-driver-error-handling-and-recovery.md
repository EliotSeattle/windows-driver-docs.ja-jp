# <a name="wia-driver-error-handling-and-recovery"></a>WIA ドライバー エラー処理と回復





エラーまたは遅延が発生できる WIA イメージの取得プロセスには、多くの点があります。 Microsoft Windows Millennium Edition および Windows XP では、エラーに関する追加情報を取得することができます。 Windows Vista およびそれ以降のオペレーティング システムでは、アプリケーションとユーザーを適切に処理し、場合によってはこれらのエラーまたは遅延から回復できるようにするメカニズムを使用することができます。 Windows Vista のメカニズムに依存、 [IStream データ転送](istream-data-transfers.md)、Windows XP と Windows Me がサポートされていません。
