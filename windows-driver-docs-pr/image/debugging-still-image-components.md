# <a name="debugging-still-image-components"></a>静止画像コンポーネントのデバッグ

静止画像のベンダーから提供されたコンポーネントのデバッグに役立てるため、静止のイメージ イベント モニターの動作を変更して、使用して、コマンド ライン オプションで、**実行**のオプション、**開始**メニュー。 次のオプションが使用できます。

<col width="50%" />
<col width="50%" />

<th>コマンド ライン オプション</th>
<th>定義</th>
</tr>
</thead>
<tbody>
<tr class="odd">
<td><p><strong>stimon /h</strong></p></td>
<td><p>メッセージ ウィンドウを非表示にします。</p></td>
</tr>
<tr class="even">
<td><p><strong>Stimon/r</strong></p></td>
<td><p>イベント モニターのデバイスの一覧を更新します。</p></td>
</tr>
<tr class="odd">
<td><p><strong>Stimon/t</strong><em>数</em></p></td>
<td><p>指定された秒数にポーリング間隔を変更します。<em>数</em>します。 通常のポーリング間隔を長く使用されます。</p></td>

<tr class="even">
<td><p><strong>Stimon/v</strong></p></td>

イベント モニターを停止およびコンピューターの管理ウィンドウを使用して開始できます。 イベント モニターは、「静止画像サービス」として表示されます。
