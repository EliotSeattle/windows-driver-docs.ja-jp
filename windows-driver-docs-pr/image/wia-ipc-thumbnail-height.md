# <a name="wiaipcthumbnailheight"></a>WIA\_IPC\_サムネイル\_高さ


WIA\_IPC\_サムネイル\_高さのプロパティには、デバイスに格納されているサムネイル イメージのピクセル単位で、現在の高さが含まれています。 WIA ミニドライバーは、作成し、このプロパティを保持します。

## <span id="ddk_wia_ipc_thumbnail_height_si"></span><span id="DDK_WIA_IPC_THUMBNAIL_HEIGHT_SI"></span>


プロパティの種類:VT\_I4

有効な値 :WIA\_PROP\_NONE

アクセス権:読み取り専用です。
