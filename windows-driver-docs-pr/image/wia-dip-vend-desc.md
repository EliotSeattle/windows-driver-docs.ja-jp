# <a name="wiadipvenddesc"></a>WIA\_DIP\_VEND\_DESC


WIA\_DIP\_VEND\_DESC プロパティには、WIA ミニドライバーの仕入先の説明文字列が含まれています。 WIA サービスは、作成し、このプロパティを保持します。

## <span id="ddk_wia_dip_vend_desc_si"></span><span id="DDK_WIA_DIP_VEND_DESC_SI"></span>


プロパティの種類:VT\_BSTR

有効な値 :WIA\_PROP\_NONE

アクセス権:読み取り専用かどうか

<a name="remarks"></a>注釈
-------

ベンダーの説明は、INF ファイルから取得されます。 アプリケーションの読み取り、WIA\_DIP\_VEND\_DESC プロパティ デバイス製造元の説明を取得します。
