# <a name="wiaipsfilmscanmode"></a>WIA\_IP\_フィルム\_スキャン\_モード


WIA\_IP\_フィルム\_スキャン\_モード プロパティには、フィルム スキャンの現在の構成設定が含まれています。 WIA ミニドライバーは、作成し、このプロパティを保持します。

プロパティの種類:VT\_I4

有効な値 :WIA\_PROP\_一覧

アクセス権:読み取り/書き込み

<a name="remarks"></a>注釈
-------

次の表は、WIA で有効な定数\_IP\_フィルム\_スキャン\_モード プロパティ

| スキャンの種類                  | 定義                                         |
|----------------------------|----------------------------------------------------|
| WIA\_フィルム\_色\_スライド    | スキャンは、色のスキャンになります。                     |
| WIA\_フィルム\_色\_負の値 | スキャンは、負の値の色のスキャンになります。       |
| WIA\_フィルム\_BW\_負の値    | スキャンを黒と白 (グレースケール) のスキャンとなります。 |

 

このプロパティは、フィルム スキャナーと透明度のアダプターの WIA 項目ツリーのルート項目に必要です。
