# <a name="wiaipsprinterendorsernumlines"></a>WIA\_IP\_プリンター\_裏書き\_NUM\_行


**WIA\_IP\_プリンター\_裏書き\_NUM\_行**プロパティは、それぞれの側の行のテキストを印刷したり、動作保証済みの最大数を指定します、・ インプリント ・/裏書き単位ドキュメントです。 このプロパティは初期化され、WIA ミニ ドライバーによって管理され、Windows 8 および Windows の以降のバージョンでは使用します。

プロパティの種類:VT\_UI4

有効な値 :WIA\_PROP\_NONE

アクセス権:読み取り専用です。

<a name="remarks"></a>コメント
-------

**WIA\_IP\_プリンター\_裏書き\_NUM\_行**プロパティは・ インプリント ・/裏書き項目のオプションです。 このプロパティはサポートされていません、ドキュメントの各側で 1 行のテキストのみが許可されます。
