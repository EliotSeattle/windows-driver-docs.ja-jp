# <a name="wiadpccontrast"></a>WIA\_DPC\_コントラスト


WIA\_DPC\_コントラスト プロパティは、キャプチャしたイメージの見かけ上のコントラストを示します。

## <span id="ddk_wia_dpc_contrast_si"></span><span id="DDK_WIA_DPC_CONTRAST_SI"></span>


プロパティの種類:VT\_I4

有効な値 :WIA\_PROP\_リストまたは WIA\_PROP\_範囲

アクセス権:読み取り/書き込み

<a name="remarks"></a>注釈
-------

WIA\_DPC\_コントラスト プロパティは、いずれかの値のリストまたは値の範囲に含めることができます。

WIA の値がサポートされる最小\_DPC\_コントラスト コントラスト、量が少なく、最大値が最もコントラストを示します。 通常、値の範囲は normal、または既定、コントラストの途中で。
