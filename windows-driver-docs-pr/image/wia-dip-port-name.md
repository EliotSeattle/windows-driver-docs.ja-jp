# <a name="wiadipportname"></a>WIA\_DIP\_ポート\_名


WIA\_DIP\_ポート\_NAME プロパティには、インストールされているデバイスのポート名には、デバイスを操作するカーネル モード ドライバーによって割り当てられているが含まれています。 WIA サービスは、作成し、このプロパティを保持します。
