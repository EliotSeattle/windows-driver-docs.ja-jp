# <a name="wiadpctimelapseinterval"></a>WIA\_DPC\_TIMELAPSE\_間隔


WIA\_DPC\_TIMELAPSE\_INTERVAL プロパティには、低速撮影のキャプチャ操作でのイメージのキャプチャとの間のミリ秒単位の時間が含まれています。

## <span id="ddk_wia_dpc_timelapse_interval_si"></span><span id="DDK_WIA_DPC_TIMELAPSE_INTERVAL_SI"></span>


プロパティの種類:VT\_I4

有効な値 :WIA\_PROP\_なし、WIA\_PROP\_リスト、または WIA\_PROP\_範囲

アクセス権:読み取り/書き込み
