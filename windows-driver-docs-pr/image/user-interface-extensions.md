# <a name="user-interface-extensions"></a>ユーザー インターフェイスの拡張機能





多くのカメラおよびスキャナー ベンダーは既定の WIA ユーザー インターフェイス (UI) を通してアクセスできない自分のデバイスで機能を指定可能性があります。 このような機能は、独自の形式で格納されているイメージの操作を含めることができます、self などの特殊なデバイス操作は、テストまたはズーム、および、具合です。 非標準の機能と特殊なデバイスおよびユーザー グループ UI に合わせて、するためには、WIA は、UI の拡張機能を提供します。 この機能により、コンテキスト メニューのプロパティ シート、イメージのシステム提供の UI のアプリケーション ダイアログ ボックスを拡張するためにベンダです。
