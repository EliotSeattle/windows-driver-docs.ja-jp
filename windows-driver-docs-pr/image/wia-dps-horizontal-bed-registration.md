# <a name="wiadpshorizontalbedregistration"></a>WIA\_DPS\_水平\_ベッド\_登録


WIA\_DPS\_水平\_ベッド\_REGISTRATION プロパティには、登録、または、フラット ベッド スキャナーの上に配置されているドキュメントの水平方向の配置が含まれています。 WIA ミニドライバーは、作成し、このプロパティを保持します。

## <span id="ddk_wia_dps_horizontal_bed_registration_si"></span><span id="DDK_WIA_DPS_HORIZONTAL_BED_REGISTRATION_SI"></span>


プロパティの種類:VT\_I4

有効な値 :WIA\_PROP\_NONE

アクセス権:読み取り専用かどうか
