# <a name="wiadipwiaversion"></a>WIA\_DIP\_WIA\_バージョン


WIA\_DIP\_WIA\_バージョン プロパティには (文字列) としてのコンピューターにインストールされている現在の WIA バージョン番号が含まれています。 WIA サービスは、作成し、このプロパティを保持します。

## <span id="ddk_wia_dip_wia_version_si"></span><span id="DDK_WIA_DIP_WIA_VERSION_SI"></span>


プロパティの種類:VT\_BSTR

有効な値 :WIA\_PROP\_NONE

アクセス権:読み取り専用かどうか

<a name="remarks"></a>注釈
-------

アプリケーションの読み取り WIA\_DIP\_WIA\_WIA コンピューターにインストールされているのバージョンを特定のバージョン。
