# <a name="wiadpcburstnumber"></a>WIA\_DPC\_バースト\_数

## <span id="ddk_wia_dpc_burst_number_si"></span><span id="DDK_WIA_DPC_BURST_NUMBER_SI"></span>


プロパティの種類:VT\_I4

有効な値 :WIA\_PROP\_なし、WIA\_PROP\_リスト、または WIA\_PROP\_範囲

アクセス権:読み取り/書き込み
