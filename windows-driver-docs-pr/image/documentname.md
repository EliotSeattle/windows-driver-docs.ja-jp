# <a name="documentname-element"></a>ドキュメント要素


必要な**ドキュメント**要素には、クライアントを提供するドキュメントの名前が含まれています。

<a name="usage"></a>使用方法
-----

```xml
<wscn:DocumentName>
  text
</wscn:DocumentName>
```

<a name="attributes"></a>属性
----------

属性はありません。

<a name="text-value"></a>テキスト値
----------

必須。 任意の文字の文字列。

## <a name="child-elements"></a>子要素


子要素はありません。
