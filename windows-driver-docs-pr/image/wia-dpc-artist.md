# <a name="wiadpcartist"></a>WIA\_DPC\_アーティスト


WIA\_DPC\_アーティスト プロパティには、デバイスの所有者 (つまり、現在のユーザー) の名前が含まれています。 デバイスでは、このプロパティを使用して、キャプチャする EXIF イメージごとにアーティスト フィールドを設定します。

## <span id="ddk_wia_dpc_artist_si"></span><span id="DDK_WIA_DPC_ARTIST_SI"></span>


プロパティの種類:VT\_BSTR

有効な値 :WIA\_PROP\_NONE

アクセス権:読み取り/書き込み
