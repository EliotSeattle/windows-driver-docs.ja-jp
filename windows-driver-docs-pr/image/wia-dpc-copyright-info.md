# <a name="wiadpccopyrightinfo"></a>WIA\_DPC\_著作権\_情報


WIA\_DPC\_著作権\_INFO プロパティには、著作権の通知が含まれています。

## <span id="ddk_wia_dpc_copyright_info_si"></span><span id="DDK_WIA_DPC_COPYRIGHT_INFO_SI"></span>


プロパティの種類:VT\_BSTR

有効な値 :WIA\_PROP\_NONE

アクセス権:読み取り/書き込み

<a name="remarks"></a>注釈
-------

WIA を使用するデバイス\_DPC\_著作権\_情報 プロパティをキャプチャするすべての EXIF イメージの 著作権 フィールドを設定します。
