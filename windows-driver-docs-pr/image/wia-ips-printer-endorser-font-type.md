# <a name="wiaipsprinterendorserfonttype"></a>WIA\_IP\_プリンター\_裏書き\_フォント\_型


**WIA\_IP\_プリンター\_裏書き\_フォント\_型**プロパティ プリンター/裏書きデバイスによって使用されるフォントの種類を構成します。 このプロパティは初期化され、WIA ミニ ドライバーによって維持されます。 この機能は、Windows 8 および Windows の以降のバージョンで使用できます。

プロパティの種類:VT\_I4

有効な値 :WIA\_PROP\_一覧

アクセス権:読み取り/書き込み

<a name="remarks"></a>注釈
-------

値、 **WIA\_IP\_プリンター\_裏書き\_フォント\_型**プロパティが次の表に示すようにします。

| 値                                        | 説明                       |
|----------------------------------------------|-----------------------------------|
| WIA\_印刷\_フォント\_標準                     | 通常のフォントです。                      |
| WIA\_印刷\_フォント\_太字                       | 太字のフォント。                        |
| WIA\_印刷\_フォント\_余分な\_太字                | 極太字のフォントです。                  |
| WIA\_印刷\_フォント\_斜体\_太字               | 斜体、太字のフォント。                |
| WIA\_印刷\_フォント\_斜体\_余分な\_太字        | 斜体、太字を追加します。          |
| WIA\_印刷\_フォント\_斜体                     | 斜体のフォントです。                      |
| WIA\_印刷\_フォント\_小さな                      | 小さいフォント。                       |
| WIA\_印刷\_フォント\_小さな\_太字                | 小規模の太字のフォント。                  |
| WIA\_印刷\_フォント\_小さな\_余分な\_太字         | 小さな極太字フォントです。            |
| WIA\_印刷\_フォント\_小さな\_斜体\_太字        | 斜体と太字のフォントを小さいです。       |
| WIA\_印刷\_フォント\_小さな\_斜体\_余分な\_太字 | 小さなし極太字、斜体フォントです。 |
| WIA\_印刷\_フォント\_小さな\_斜体              | 小規模の斜体のフォント。                |
| WIA\_印刷\_フォント\_大                      | 大きなフォントです。                       |
| WIA\_印刷\_フォント\_LARGE\_太字                | 大規模な太字のフォント。                  |
| WIA\_印刷\_フォント\_LARGE\_余分な\_太字         | 大規模な極太字フォントです。            |
| WIA\_印刷\_フォント\_LARGE\_斜体\_太字        | 大規模な斜体と太字のフォントです。       |
| WIA\_印刷\_フォント\_LARGE\_斜体\_余分な\_太字 | 大規模なし極太字、斜体フォントです。 |
| WIA\_印刷\_フォント\_LARGE\_斜体              | 大規模な斜体フォントです。                |

 

**WIA\_IP\_プリンター\_裏書き\_フォント\_型**プロパティは・ インプリント ・/裏書き項目のオプションです。 これはサポートされていない場合、プリンター/裏書きデバイスは、フォントの設定をサポートしていません。
