# <a name="wiadipbaudrate"></a>WIA\_DIP\_BAUDRATE


WIA\_DIP\_BAUDRATE プロパティには、デバイスの現在のボー レートの設定が含まれています。 WIA サービスは、作成し、このプロパティを保持します。

## <span id="ddk_wia_dip_baudrate_si"></span><span id="DDK_WIA_DIP_BAUDRATE_SI"></span>


プロパティの種類:VT\_BSTR

有効な値 :WIA\_PROP\_NONE

アクセス権:読み取り専用かどうか

<a name="remarks"></a>注釈
-------

値の WIA\_DIP\_BAUDRATE プロパティは、シリアル ケーブルでデバイスが接続されていない場合、「空」必要があります。
