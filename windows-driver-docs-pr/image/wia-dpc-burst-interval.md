# <a name="wiadpcburstinterval"></a>WIA\_DPC\_バースト\_間隔


WIA\_DPC\_バースト\_INTERVAL プロパティには、バースト操作中にイメージのキャプチャとの間のミリ秒単位の時間が含まれています。

## <span id="ddk_wia_dpc_burst_interval_si"></span><span id="DDK_WIA_DPC_BURST_INTERVAL_SI"></span>


プロパティの種類:VT\_I4

有効な値 :WIA\_PROP\_なし、WIA\_PROP\_リスト、または WIA\_PROP\_範囲

アクセス権:読み取り/書き込み
