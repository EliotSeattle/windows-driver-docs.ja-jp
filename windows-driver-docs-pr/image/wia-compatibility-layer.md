# <a name="wia-compatibility-layer"></a>WIA 互換性レイヤー


WIA 互換性レイヤーでは、従来のアプリケーションと Windows Vista でのデバイスを使用できるようにします。

WIA 互換性レイヤーの 2 つの関数があります。

1.  別の項目のツリーのアーキテクチャと従来のアプリケーションと Windows Vista のドライバーと Windows Vista のアプリケーションと従来のドライバーの間でのプロパティのマッピングです。

2.  Windows Vista の間のマッピング、ストリーム ベースのデータがモデルと従来の TYMED に基づくモデルを転送するようにします。
