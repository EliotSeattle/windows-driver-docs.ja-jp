# <a name="windows-2000-core-components"></a>Windows 2000 のコア コンポーネント





Windows 2000 に静止画像コア コンポーネントは、次の図に表示されます。

![windows 2000 の主要コンポーネントを示す図](images/stiwin2k.png)

Windows 2000 に静止画像コア コンポーネントとほぼ同じ、 [Windows 98 のコア コンポーネント](windows-98-core-components.md)します。 これらのオペレーティング システムの 3 つの違いは次のとおりです。

1.  *Stisvc.exe* Windows 98 を置き換える*stimon.exe*コンポーネント。

2.  システム提供のシリアル ポートのカーネル モード ドライバー *serscan.sys*します。

3.  *Scsimap.dll* Windows 2000 以降に存在しません。
