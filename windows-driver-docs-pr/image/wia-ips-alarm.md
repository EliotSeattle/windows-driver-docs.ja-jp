# <a name="wiaipsalarm"></a>WIA\_IP\_アラーム


**WIA\_IP\_アラーム**WIA ミニ ドライバーで、次の条件のいずれかで、デバイスによって生成される可聴アラーム (ビープ音) を構成するプロパティを使用します。

-   フィーダー項目でこのプロパティを実装する場合 (WIA\_カテゴリ\_フィーダー)、および複数のフィードの検出を有効に、複数のフィード条件が検出された場合、デバイスで可聴アラーム (ビープ音) サウンドを再生する必要があります。 (ときに、WIA\_IP\_マルチ\_フィードのプロパティがサポートされて、WIA でない値に設定\_マルチ\_フィード\_検出\_その複数のフィードを意味する、無効になっています。検出を有効にします。)
-   バーコード リーダー項目でこのプロパティを実装する場合 (WIA\_カテゴリ\_バーコード\_リーダー)、およびバーコードの検出が有効になっている、バーコードが正常にする場合は、デバイスで可聴アラーム (ビープ音) サウンドを再生する必要があります検出されました。
-   修正プログラム コード リーダー項目でこのプロパティを実装する場合 (WIA\_カテゴリ\_パッチ\_コード\_リーダー)、および修正プログラム コードの検出が有効になっている、デバイスで可聴アラーム (ビープ音) サウンドを再生するときに、修正プログラム コードが正常に検出されました。

このプロパティは初期化され、WIA ミニ ドライバーによって保持され、Windows 8 および Windows の以降のバージョンで利用できます。

プロパティの種類:VT\_I4

有効な値 :WIA\_PROP\_一覧

アクセス権:読み取り/書き込み
