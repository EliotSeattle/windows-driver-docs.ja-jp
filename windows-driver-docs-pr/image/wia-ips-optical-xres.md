# <a name="wiaipsopticalxres"></a>WIA\_IP\_光\_XRES


WIA\_IP\_光\_XRES プロパティにはでサポートされている最も高い水平光学解像度ドット/インチ (dpi) で、デバイスにはが含まれています。 WIA ミニドライバーは、作成し、このプロパティを保持します。

プロパティの種類:VT\_I4

有効な値 :WIA\_PROP\_NONE。

アクセス権:読み取り専用です。
