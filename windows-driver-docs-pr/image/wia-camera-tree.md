# <a name="wia-camera-tree"></a>WIA カメラ ツリー





次の図は、同じディレクトリに 2 つは、いくつかのイメージを含むカメラを示します。

![同じディレクトリに 2 つは、いくつかのイメージでのカメラを示す図](images/art-camera.png)

次の図では、WIA はカメラの項目のツリーとして、カメラと任意のフォルダーで撮った写真を前の図に示すようにカメラを表します。

![wia がカメラの項目のツリーとして、カメラと任意のフォルダーで撮った写真を前の図に示すようにカメラを表す方法を示す図](images/art-3.png)

カメラそのものである、ルート項目は、一般的なデバイスのプロパティ (カメラおよびスキャナーの両方に共通するプロパティ) とデバイスのカメラに固有のプロパティで構成されます。 同様に、各子項目は、カメラおよびスキャナー項目の両方に共通するプロパティとカメラのアイテムに固有のプロパティで構成されます。

WIA サービスを介して、アプリケーションは、カメラ項目から、次を要求できます。

-   カメラの機能のクエリを実行します。

-   デバイスのカメラのプロパティを設定します。

-   データ転送を要求します。

前の図では、カメラのルート項目が 3 つの子項目: 2 つの画像と 1 つのフォルダー。 フォルダーでは、両方の図は 2 つの子項目があります。 サウンド データまたはその他のデータ、デバイスがアプリケーションに提示するカメラのカメラの項目を表します。
