# <a name="wiadipstigencapabilities"></a>WIA\_DIP\_STI\_GEN\_機能


WIA\_DIP\_STI\_GEN\_機能プロパティには、ドライバーの INF ファイルから取得される、デバイスの場合の STI の汎用的な機能が含まれています。 WIA サービスは、作成し、このプロパティを保持します。

## <span id="ddk_wia_dip_sti_gen_capabilities_si"></span><span id="DDK_WIA_DIP_STI_GEN_CAPABILITIES_SI"></span>


プロパティの種類:VT\_I4

有効な値 :WIA\_PROP\_NONE

アクセス権:読み取り専用かどうか

<a name="remarks"></a>注釈
-------

アプリケーションの読み取り、WIA\_DIP\_STI\_GEN\_機能プロパティ、デバイスの汎用 STI 機能を判断します。
