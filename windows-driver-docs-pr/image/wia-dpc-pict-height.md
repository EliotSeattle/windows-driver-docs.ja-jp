# <a name="wiadpcpictheight"></a>WIA\_DPC\_PICT\_高さ


WIA\_DPC\_PICT\_高さのプロパティには、新しくキャプチャしたイメージを使用する、ピクセルの高さが含まれています。

## <span id="ddk_wia_dpc_pict_height_si"></span><span id="DDK_WIA_DPC_PICT_HEIGHT_SI"></span>


プロパティの種類:VT\_I4

有効な値 :WIA\_PROP\_リストまたは WIA\_PROP\_範囲

アクセス権:読み取り/書き込み

<a name="remarks"></a>注釈
-------

WIA の有効な値の一覧\_DPC\_PICT\_HEIGHT プロパティには、一対一で対応の有効な値の一覧で、 [ **WIA\_DPC\_PICT\_幅**](wia-dpc-pict-width.md)プロパティ。 個々 の幅と高さは直線的に設定可能で相互に直交場合は、範囲として表現できます。
