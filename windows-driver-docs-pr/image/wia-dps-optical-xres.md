# <a name="wiadpsopticalxres"></a>WIA\_DPS\_光\_XRES


WIA\_DPS\_光\_XRES プロパティにはでサポートされている最も高い水平光学解像度ドット/インチ (dpi) で、デバイスにはが含まれています。 WIA ミニドライバーは、作成し、このプロパティを保持します。

## <span id="ddk_wia_dps_optical_xres_si"></span><span id="DDK_WIA_DPS_OPTICAL_XRES_SI"></span>


プロパティの種類:VT\_I4

有効な値 :WIA\_PROP\_NONE。

アクセス権:読み取り専用かどうか
