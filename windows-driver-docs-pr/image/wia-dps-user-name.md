# <a name="wiadpsusername"></a>WIA\_DPS\_ユーザー\_名


WIA\_DPS\_ユーザー\_WIA ドライバーのルート項目の NAME プロパティは、ドメイン名またはコンピューターの名前と時間で特定の時点に WIA デバイスにアクセスしているユーザーのユーザー名を取得するドライバーを許可します。 WIA サービスは、作成し、このプロパティを保持します。

プロパティの種類:VT\_BSTR

有効な値 :WIA\_PROP\_NONE

アクセス権:読み取り専用かどうか
