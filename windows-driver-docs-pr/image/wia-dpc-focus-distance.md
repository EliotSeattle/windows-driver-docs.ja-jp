# <a name="wiadpcfocusdistance"></a>WIA\_DPC\_フォーカス\_距離


WIA\_DPC\_フォーカス\_距離のプロパティには、デジタル カメラのイメージ キャプチャ平面とフォーカスのポイントの間のミリメートル単位の距離が含まれています。

## <span id="ddk_wia_dpc_focus_distance_si"></span><span id="DDK_WIA_DPC_FOCUS_DISTANCE_SI"></span>


プロパティの種類:VT\_I4

有効な値 :WIA\_PROP\_範囲または WIA\_PROP\_一覧

アクセス権:読み取り/書き込み

<a name="remarks"></a>コメント
-------

値が 0 xffff は、WIA の\_DPC\_フォーカス\_距離のプロパティは、655 メーターよりも大きい設定に対応します。
