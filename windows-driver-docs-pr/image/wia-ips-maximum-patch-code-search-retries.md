# <a name="wiaipsmaximumpatchcodesearchretries"></a>WIA\_IP\_最大\_PATCH\_コード\_検索\_再試行


**WIA\_IP\_最大\_PATCH\_コード\_検索\_再試行**プロパティがない場合は、リーダーが試行の再試行の最大数について説明します修正プログラム コードの検出が有効な場合、修正プログラム コードを確認できます。
