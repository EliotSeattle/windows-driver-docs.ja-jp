# <a name="still-image-com-interfaces"></a>COM インターフェイスを静止画像します。





Microsoft STI には、次の 4 つの COM インターフェイスを定義します。

[COM インターフェイスの IStillImage](istillimage-com-interface.md) − 静止画像イベント モニターにアクセスを提供します。

[COM インターフェイスの IStiDevice](istidevice-com-interface.md) − 静止画像データ ストリームとデバイスのイベントにアクセスを提供します。

[COM インターフェイスの IStiUSD](istiusd-com-interface.md) − アクセスするためのユーザー モードは、ミニドライバーを静止画像します。

[COM インターフェイスの IStiDeviceControl](istidevicecontrol-com-interface.md)ミニドライバーに − 提供ユーティリティ操作。
