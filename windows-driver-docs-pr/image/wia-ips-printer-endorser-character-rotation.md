**WIA\_IP\_プリンター\_裏書き\_文字\_回転**で出力された個別の文字の回転を構成するプロパティを使用または動作保証済みのテキスト。 によって定義された現在の印刷されたまたは動作保証済みの領域の回転を個々 の文字の回転をさらに行うことができます、サポートされている場合、 [ **WIA\_IP\_回転**](wia-ips-rotation.md)プロパティ。 この機能は、Windows 8 および Windows の以降のバージョンで使用できます。
