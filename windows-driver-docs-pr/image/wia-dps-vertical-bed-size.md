# <a name="wiadpsverticalbedsize"></a>WIA\_DPS\_垂直\_ベッド\_サイズ


WIA\_DPS\_垂直\_ベッド\_1/1000 インチので、スキャナーのフラット ベッドの物理的な縦のサイズが SIZE プロパティに含まれています (. 001)。 WIA ミニドライバーは、作成し、このプロパティを保持します。

## <span id="ddk_wia_dps_vertical_bed_size_si"></span><span id="DDK_WIA_DPS_VERTICAL_BED_SIZE_SI"></span>


プロパティの種類:VT\_I4

有効な値 :WIA\_PROP\_NONE

アクセス権:読み取り専用です。
