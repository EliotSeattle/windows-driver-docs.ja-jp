# <a name="wiadpafirmwareversion"></a>WIA\_DPA\_ファームウェア\_バージョン


WIA\_DPA\_ファームウェア\_バージョン プロパティには、デバイスのファームウェア バージョンが含まれています。 ミニドライバーは、作成し、このプロパティを保持します。

## <span id="ddk_wia_dpa_firmware_version_si"></span><span id="DDK_WIA_DPA_FIRMWARE_VERSION_SI"></span>


プロパティの種類:VT\_BSTR

有効な値 :WIA\_PROP\_NONE

アクセス権:読み取り専用かどうか

<a name="remarks"></a>注釈
-------

値の WIA\_DPA\_ファームウェア\_バージョン プロパティは、「1.0.4」または"1.0abc"などの文字列値を指定する必要があります。
