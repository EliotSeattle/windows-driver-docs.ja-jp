# <a name="conditioncleartime-element"></a>ConditionClearTime 要素


必要な**ConditionClearTime**要素は、スキャン デバイスの状態がクリアされました日時を指定します。

<a name="usage"></a>使用方法
-----

```xml
<wscn:ConditionClearTime>
  text
</wscn:ConditionClearTime>
```

<a name="attributes"></a>属性
----------

属性はありません。

<a name="text-value"></a>テキスト値
----------

必須。 DateTime 型の任意の有効な値。 DateTime の詳細については、XML Schema Part 2 を参照してください。Datatypes Second Edition。**dateTimedateTime**

## <a name="child-elements"></a>子要素


子要素はありません。
