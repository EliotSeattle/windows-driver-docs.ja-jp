# <a name="wia-scanner-storage-compatibility-for-windows-xp-and-windows-me"></a>Windows XP の WIA スキャナー記憶域の互換性と Windows Me


Windows Vista では、WIA サービスには、新しい Windows Vista のアプリケーションおよびドライバーと従来の Windows XP ドライバーとアプリケーションを使用できるように内部互換性レイヤーにはが含まれています。 互換レイヤーは、フラット ベッドとフィーダー項目に対してのみ翻訳を提供します。 記憶域を実行しているため*いない*存在 Windows XP または Windows Me とは互換性レイヤーによって変換されないため、ストレージ アイテムは、従来の WIA アプリケーションを使用できません。

Windows XP および Windows Me の互換性に関する追加説は、[WIA フラット ベッド スキャナーの互換性の Windows と Windows XP](wia-flatbed-scanner-compatibility-for-windows-xp-and-windows-me.md)を参照してください。
