# <a name="wiaipasuppresspropertypage"></a>WIA\_IPA\_抑制\_プロパティ\_ページ


WIA\_IPA\_抑制\_プロパティ\_ページのプロパティでは、デバイス上のアイテムの [全般] プロパティ ページを抑制するかどうかを指定します。

## <span id="ddk_wia_ipa_suppress_property_page_si"></span><span id="DDK_WIA_IPA_SUPPRESS_PROPERTY_PAGE_SI"></span>


プロパティの種類:VT\_I4

有効な値 :WIA\_PROP\_NONE

アクセス権:読み取り専用かどうか
