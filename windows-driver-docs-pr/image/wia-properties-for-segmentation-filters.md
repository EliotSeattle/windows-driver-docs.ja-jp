# <a name="wia-properties-for-segmentation-filters"></a>セグメンテーション フィルターの WIA プロパティ





セグメント化フィルターは、いくつかの WIA プロパティに依存します。 次のようなクラスがあります。

[**WIA\_IP\_XPOS**](https://docs.microsoft.com/windows-hardware/drivers/image/wia-ips-xpos)

[**WIA\_IP\_YPOS**](https://docs.microsoft.com/windows-hardware/drivers/image/wia-ips-ypos)

[**WIA\_IP\_XEXTENT**](https://docs.microsoft.com/windows-hardware/drivers/image/wia-ips-xextent)

[**WIA\_IP\_YEXTENT**](https://docs.microsoft.com/windows-hardware/drivers/image/wia-ips-yextent)

Windows Vista 以降では、これら 2 つの追加のフィルター プロパティのフィルターと、セグメント化を使用します。

[**WIA\_IP\_DESKEW\_Y**](https://docs.microsoft.com/windows-hardware/drivers/image/wia-ips-deskew-y)

[**WIA\_IP\_DESKEW\_Y**](https://docs.microsoft.com/windows-hardware/drivers/image/wia-ips-deskew-y)

このセクションの残りの部分では、次の追加のトピックについて説明します。

[WIA\_IP\_DESKEW\_X と WIA\_IP\_DESKEW\_Y プロパティ](wia-ips-deskew-x-and-wia-ips-deskew-y-properties.md)
