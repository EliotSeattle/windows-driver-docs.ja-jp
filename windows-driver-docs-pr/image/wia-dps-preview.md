# <a name="wiadpspreview"></a>WIA\_DPS\_プレビュー


WIA\_DPS\_プレビュー プロパティは、デバイスのプレビュー モードを示します。 アプリケーションでは、プレビュー モードにデバイスを配置するには、このプロパティを設定します。

## <span id="ddk_wia_dps_preview_si"></span><span id="DDK_WIA_DPS_PREVIEW_SI"></span>


プロパティの種類:VT\_I4

有効な値 :WIA\_PROP\_一覧

アクセス権:読み取り/書き込み
