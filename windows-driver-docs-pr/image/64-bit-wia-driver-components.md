# <a name="64-bit-wia-driver-components"></a>64 ビット WIA ドライバーのコンポーネント


64 ビットの WIA ミニドライバーは、Windows オペレーティング システムの 64 ビット エディションの 64 ビット プロセスでは、WIA サービスのプロセスに読み込まれます。 その結果、Windows XP、Windows Server 2003、および Windows Vista の x64 ベース バージョンの WIA ドライバーは、64 ビット ドライバー コンポーネントのみを含めることができます。 たとえば、64 ビット ミニドライバー、64 ビット WIA ミニドライバー UI 拡張機能、および可能性のある、64 ビットのカーネル モード ドライバーでなければなりません。
