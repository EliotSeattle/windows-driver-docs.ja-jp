# <a name="wiadpczoomposition"></a>WIA\_DPC\_ズーム\_位置

## <span id="ddk_wia_dpc_zoom_position_si"></span><span id="DDK_WIA_DPC_ZOOM_POSITION_SI"></span>

有効な値 :WIA\_PROP\_NONE
