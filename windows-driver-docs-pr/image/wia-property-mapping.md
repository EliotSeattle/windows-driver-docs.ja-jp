# <a name="wia-property-mapping"></a>WIA プロパティ マッピング


このセクションでは、項目のツリー モデルとプロパティのマッピングについて説明します。 Windows XP から Windows Vista 対応の WIA プロパティのこのマッピングは、WIA 互換性レイヤーの一部です。

Windows Vista のプロパティに古い、予約されている、または無視するには、マークされている Windows XP のプロパティがマップされていないことに注意してください。
