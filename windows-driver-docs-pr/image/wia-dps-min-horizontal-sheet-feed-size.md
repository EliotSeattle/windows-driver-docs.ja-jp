# <a name="wiadpsminhorizontalsheetfeedsize"></a>WIA\_DPS\_MIN\_水平\_シート\_フィード\_サイズ


WIA\_DPS\_MIN\_水平\_シート\_フィード\_サイズ プロパティには、ドキュメント フィーダー付きのスキャナーをスキャンしてできますで最小のページの物理的な水平方向のサイズが含まれています。インチの部分の 1/1000 (. 001)。 WIA ミニドライバーは、作成し、このプロパティを保持します。
