# <a name="wiadpsopticalyres"></a>WIA\_DPS\_光\_YRES


WIA\_DPS\_光\_YRES プロパティにはでサポートされている最大垂直光の解像度ドット/インチ (dpi) で、デバイスが含まれています。 WIA ミニドライバーは、作成し、このプロパティを保持します。

## <span id="ddk_wia_dps_optical_yres_si"></span><span id="DDK_WIA_DPS_OPTICAL_YRES_SI"></span>


プロパティの種類:VT\_I4

有効な値 :WIA\_PROP\_NONE

アクセス権:読み取り専用かどうか
