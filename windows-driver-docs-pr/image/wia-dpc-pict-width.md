# <a name="wiadpcpictwidth"></a>WIA\_DPC\_PICT\_幅


WIA\_DPC\_PICT\_WIDTH プロパティには、新しくキャプチャしたイメージを使用する、ピクセルの幅がについて説明します。

## <span id="ddk_wia_dpc_pict_width_si"></span><span id="DDK_WIA_DPC_PICT_WIDTH_SI"></span>


プロパティの種類:VT\_I4

有効な値 :WIA\_PROP\_リストまたは WIA\_PROP\_範囲

アクセス権:読み取り/書き込み

<a name="remarks"></a>注釈
-------

WIA の有効な値の一覧\_DPC\_PICT\_WIDTH プロパティが有効な値の一覧に 1 対 1 の対応、 [ **WIA\_DPC\_PICT\_高さ**](wia-dpc-pict-height.md)プロパティ。 個々 の幅と高さは直線的に設定可能で相互に直交場合は、範囲として表現できます。
