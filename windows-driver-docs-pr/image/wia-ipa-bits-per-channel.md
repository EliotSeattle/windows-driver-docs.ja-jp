# <a name="wiaipabitsperchannel"></a>WIA\_IPA\_ビット\_1 秒あたり\_チャネル


WIA\_IPA\_ビット\_1 秒あたり\_チャネル プロパティには、イメージのチャネルあたりのビット数が含まれています。 WIA ミニドライバーは、作成し、このプロパティを保持します。

## <span id="ddk_wia_ipa_bits_per_channel_si"></span><span id="DDK_WIA_IPA_BITS_PER_CHANNEL_SI"></span>


プロパティの種類:VT\_I4

有効な値 :WIA\_PROP\_NONE

アクセス権:読み取り専用かどうか

<a name="remarks"></a>注釈
-------

WIA\_IPA\_ビット\_1 秒あたり\_チャネル プロパティは、 [ **WIA\_IPA\_RAW\_ビット\_ごと\_チャネル**](wia-ipa-raw-bits-per-channel.md)プロパティ (これは、生の形式の使用)。
