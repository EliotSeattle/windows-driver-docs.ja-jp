# <a name="scanner-microdriver-sample"></a>スキャナーのマイクロドライバーのサンプル





*Microdrv* WDK のディレクトリには、スキャナーのサンプル WIA microdriver が含まれています。 Microdriver はより完全な WIA ミニドライバーでは、開発する方が簡単ですが、基本的な機能のみを提供します。 参照してください[WIA スキャナー ドライバーを開発](developing-a-wia-scanner-driver.md)microdriver 制限します。 このサンプルでは、フラット ベッド スキャナー用 WIA microdriver を記述する方法を示します。 このサンプル ドライバーは、開発の開始点として使用できますが、ドライバーは、カーネル モード ドライバーを Windows に付属のいずれかで、スキャナーのハードウェアにアクセスする必要があります。
