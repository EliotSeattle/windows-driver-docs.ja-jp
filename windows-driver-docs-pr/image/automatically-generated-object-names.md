# <a name="automatically-generated-object-names"></a>自動的に生成されたオブジェクト名





場合、 **Filename** ObjectInfo データセットのフィールドが空で、Microsoft PTP WIA ミニドライバー ObjectFormat と ObjectHandle フィールドに基づいて、名前を作成します。 ObjectHandle 値 5 を使用したイメージは、Windows の英語版に Image5 を呼び出されます。
