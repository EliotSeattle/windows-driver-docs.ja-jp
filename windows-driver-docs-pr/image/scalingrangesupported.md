# <a name="scalingrangesupported-element"></a>ScalingRangeSupported 要素


必要な**ScalingRangeSupported**要素には、出力ドキュメントのスケーリングのスキャン デバイスをサポートする値の範囲がについて説明します。

<a name="usage"></a>使用方法
-----

```xml
<wscn:ScalingRangeSupported>
  child elements
</wscn:ScalingRangeSupported>
```

<a name="attributes"></a>属性
----------

属性はありません。
