# <a name="mass-storage-driver"></a>大容量記憶装置ドライバー





カメラは大容量記憶装置クラスの指定に基づいている場合は自動的に認識 WIA カメラ デバイスとして自動再生操作中に (場合、ユーザーは、まず、デバイスまたはデバイスにメディアを挿入プラグ)。 Microsoft 提供の WIA ミニドライバーを組み合わせて、WIA サービスは自動的にアタッチされ、スキャナーとカメラ ウィザードを使用してデバイスをサポートしています。 これにより、すぐに、指定の場所に、デバイスから画像を転送できます。
