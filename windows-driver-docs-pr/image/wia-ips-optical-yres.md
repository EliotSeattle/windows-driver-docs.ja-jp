# <a name="wiaipsopticalyres"></a>WIA\_IP\_光\_YRES


WIA\_IP\_光\_YRES プロパティにはでサポートされている最大垂直光の解像度ドット/インチ (dpi) で、デバイスが含まれています。 WIA ミニドライバーは、作成し、このプロパティを保持します。

プロパティの種類:VT\_I4

有効な値 :WIA\_PROP\_NONE

アクセス権:読み取り専用かどうか
