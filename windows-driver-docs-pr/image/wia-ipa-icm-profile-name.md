# <a name="wiaipaicmprofilename"></a>WIA\_IPA\_ICM\_プロファイル\_名


WIA\_IPA\_ICM\_プロファイル\_NAME プロパティが正しくイメージをデコードするために必要なイメージの色の管理 (ICM) プロファイル名が含まれています。

## <span id="ddk_wia_ipa_icm_profile_name_si"></span><span id="DDK_WIA_IPA_ICM_PROFILE_NAME_SI"></span>


プロパティの種類:VT\_BSTR

有効な値 :WIA\_PROP\_一覧

アクセス権:読み取り/書き込み

<a name="remarks"></a>注釈
-------

アプリケーションの読み取り、WIA\_IPA\_ICM\_プロファイル\_名プロパティに、イメージの処理に使用する ICM プロファイルを決定します。 WIA サービスは、作成し、ドライバーのインストール ファイルの ICMProfiles エントリに基づいてこのプロパティを保持します。
