# <a name="partner-app-development"></a>パートナー アプリ開発



パートナーは、Windows 10 設定アプリなど、特殊な用途のアプリケーションを作成できます。 パートナー固有の一部のアプリ開発シナリオには、一般的な機能を備えた API では使用できない機能 (システム API へのアクセスなど) が必要です。 このようなアプリには、使用が制限された機能または特殊な用途の機能が必要です。これにより、制御された方法でパートナーが機能を開発できるようになります。
