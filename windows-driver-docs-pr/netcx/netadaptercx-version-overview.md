# <a name="netadaptercx-version-overview"></a>NetAdapterCx バージョンの概要

このセクションでは、ネットワークアダプターの WDF クラス拡張 (NetAdapterCx) の新機能について説明します。 各バージョンの NetAdapterCx の変更点の詳細については、次のトピックを参照してください。

NetAdapterCx の現在のバージョンは**2.0**です。

次の表では、NetAdapterCx バージョンの Windows OS サポートについて説明します。

| オペレーティング システム | サポートされている NetAdapterCx バージョン | バージョンメモ |
| --- | --- | --- |
| Windows 10 バージョン2004 | 2.0 | 最初のリリース |

> [!IMPORTANT]
> 以前のバージョンで作成された NetAdapterCx クライアントドライバーは、現在のバージョンに再ターゲットされるまで、最新バージョンの Windows では機能しません。
