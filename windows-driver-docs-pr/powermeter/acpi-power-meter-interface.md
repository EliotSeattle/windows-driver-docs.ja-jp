# <a name="acpi-power-meter-interface"></a>ACPI 電源メーター インターフェイス


電力メーター インターフェイス (PMI) を提供する WDM ドライバーでは、基になるハードウェア プラットフォームから電力使用状況測定と予算 (PMB) の情報にアクセスします。 ドライバーは、ACPI PMI オブジェクトを使用しています。 プラットフォームでは、ACPI 4.0 Power メータリング オブジェクトを使用してこの情報を公開します。

このセクションでは、ACPI PMI の概要を提供します。
