# <a name="sensor-thresholds"></a>センサーのしきい値


このトピックでは、各センサーに関連するセンサー データ フィールド (変更の感度とも呼ばれます) に適用可能なしきい値の詳細についての情報を提供します。 このトピックで説明されているしきい値の値は、センサー ハードウェアがアプリケーションに送信されるデータにフィルター処理構成に使用されます。 アプリケーションは、ハードウェアを使用するしきい値を構成できますセンサー データの関係のない変更を除外します。 変更の関連性のレベルは、しきい値の値を使用して構成されます。
