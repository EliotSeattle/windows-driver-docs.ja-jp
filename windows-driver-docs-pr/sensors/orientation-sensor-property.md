# <a name="orientation-sensor-property"></a>方向センサーのプロパティ


方向センサーのプロパティのキー。

|プロパティのキー|種類|アクセス (R/O、読み取り/書き込み)|必須/オプション|説明|
|--|--|--|--|--|
|PKEY_OrientationSensor_GyroscopeUsed|VT_BOOL|R/O|必須|この方向センサーでは、ジャイロスコープなどがありますを使用するかどうかを示します。|

 

表示されるデータ型の詳細については、**型**列を参照してください[PROPVARIANT 構造](https://go.microsoft.com/fwlink/p/?linkid=313395)します。
