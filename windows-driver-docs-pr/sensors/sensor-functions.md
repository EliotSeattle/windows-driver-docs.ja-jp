# <a name="sensor-functions"></a>センサー関数

次の関数は、センサーの構造を操作またはセンサー クラスの拡張機能への呼び出しに使用されます。

## <a name="in-this-section"></a>このセクションの内容

|トピック|説明|
|---|---|
|[センサー DDSI 関数](sensor-ddsi-functions.md)|センサー デバイス ドライバー ソフトウェア インターフェイス (DDSI) 関数では、センサー ドライバーとクラスの拡張機能間のインターフェイスを形成します。|
|[センサーのヘルパー関数](sensor-helper-functions.md)|このセクションでは、センサー、構造体、それらについての情報を取得して、構造体を初期化するために使用するヘルパー関数の詳細についての情報を提供します。|
|[ヘルパー関数をマーシャ リング](marshalling-helper-functions.md)|このセクションでは、sensorsutils.h ヘッダー ファイルのマーシャ リングのヘルパー関数についての情報を提供します。|
