ユニバーサル センサー ドライバー モデルでは、Windows デバイスにインストールされているドライバーの特性を示すプロパティの使用をサポートします。

各プロパティがによって表される、*プロパティ キー*プロパティのカテゴリおよびプロパティの識別子を識別する GUID 値。 このトピックには、パートナーは、センサー ドライバーを識別するために使用できるプロパティに関する情報が含まれています。
