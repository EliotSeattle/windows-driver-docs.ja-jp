# <a name="sensor-enumerations"></a>センサー列挙型


さまざまなセンサー機能のさまざまな状態または値を示すために、次の列挙体が使用されます。

## <a name="in-this-section"></a>このセクションの内容

|トピック|説明|
|---|---|
|PROXIMITY_TYPE|この列挙体は、近接センサーが検出できる近接性の種類を表します。|
|[ACTIVITY_STATE](https://docs.microsoft.com/windows-hardware/drivers/ddi/sensorsdef/ne-sensorsdef-activity_state)|この列挙は、アクティビティ検出センサーによって報告されたアクティビティの状態を表します。|
|[PEDOMETER_STEP_TYPE](https://docs.microsoft.com/windows-hardware/drivers/ddi/sensorsdef/ne-sensorsdef-pedometer_step_type)|この列挙体は、pedometer によって報告されたステップの種類を表します。|
|[PEDOMETER_STEP_TYPE_COUNT](https://docs.microsoft.com/windows-hardware/drivers/ddi/sensorsdef/ne-sensorsdef-pedometer_step_type_count)|この列挙体は、pedometer によって検出できるステップの種類の数を表します。|
|[MAGNETOMETER_ACCURACY](https://docs.microsoft.com/windows-hardware/drivers/ddi/sensorsclassextension/ne-sensorsclassextension-magnetometeraccuracy)|この列挙体は、磁力計の精度の状態を表します。|
|SENSOR_DATA_TYPE|この列挙体は、サポートされるセンサーのデータ型について説明します。|
|[SENSOR_STATE](https://docs.microsoft.com/windows-hardware/drivers/ddi/sensorsdef/ne-sensorsdef-sensor_state)|この列挙は、センサーの有効な状態を表します。|
|SENSOR_TYPE|この列挙には、クライアントドライバーがセンサーの種類を説明するために使用する値が含まれています。|
|COORDINATE_AXIS|この列挙体は、3次元のユークリッド幾何座標軸の6つの異なる方向を表します。|
