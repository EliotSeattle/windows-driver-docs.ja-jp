# <a name="sensor-enumerations"></a>センサーの列挙型


次の列挙型を使用して、さまざまな状態またはさまざまなセンサーの特徴の値を指定します。

## <a name="in-this-section"></a>このセクションの内容

|トピック|説明|
|---|---|
|PROXIMITY_TYPE|この列挙体は、近接センサーを検出できます近接の種類を表します。|
|[ACTIVITY_STATE](https://docs.microsoft.com/windows-hardware/drivers/ddi/content/sensorsdef/ne-sensorsdef-activity_state)|この列挙体は、アクティビティの検出のセンサーで感知されたアクティビティの状態を表します。|
|[PEDOMETER_STEP_TYPE](https://docs.microsoft.com/windows-hardware/drivers/ddi/content/sensorsdef/ne-sensorsdef-pedometer_step_type)|この列挙体は、歩数計によって報告されたステップの種類を表します。|
|[PEDOMETER_STEP_TYPE_COUNT](https://docs.microsoft.com/windows-hardware/drivers/ddi/content/sensorsdef/ne-sensorsdef-pedometer_step_type_count)|この列挙体は、歩数計で検出できるステップの種類の数を表します。|
|[MAGNETOMETER_ACCURACY](https://docs.microsoft.com/windows-hardware/drivers/ddi/content/sensorsclassextension/ne-sensorsclassextension-magnetometeraccuracy)|この列挙体は、磁力計の正確性の状態を表します。|
|SENSOR_DATA_TYPE|この列挙体には、サポートされているセンサーのデータ型について説明します。|
|[SENSOR_STATE](https://docs.microsoft.com/windows-hardware/drivers/ddi/content/sensorsdef/ne-sensorsdef-sensor_state)|この列挙体は、センサーの有効な状態を表します。|
|SENSOR_TYPE|この列挙体には、クライアント ドライバーを使用してセンサーの種類について記述する値が含まれています。|
|COORDINATE_AXIS|この列挙体は、3 次元ユークリッド座標軸の 6 つの異なる方向を表します。|
