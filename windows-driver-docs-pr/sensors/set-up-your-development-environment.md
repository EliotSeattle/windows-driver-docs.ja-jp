# <a name="set-up-your-development-environment"></a>開発環境を設定します。


このトピックでは、サメ Cove ボードと Microsoft が提供するソフトウェア開発のツールを使用して、センサーのユニバーサル センサー ドライバーを構築する、開発環境を設定する方法を示します。

サメ Cove は、設計および x86 ベースのタブレット プラットフォームのコアになるように構成された開発基盤です。 これらのデバイスの新しいバージョンを開発して、サメ Cove を使用してテストできるように、ボードのデザインからなど、カメラ、センサー、表示などの追加のデバイスが残されています。

サメ Cove を注文する[Mouser Electronics](https://www.mouser.com/ProductDetail/CircuitCo/999-0005112/?qs=%2fha2pyFadujHMsjnM70kKGG23vMNVMdDa094WOKuFgB8eIBF6%252bBXww%3d%3d)します。

この演習を開始する前に、次のハードウェアがあることを確認します。

-   サメ Cove 掲示板、同梱の電源ケーブルとアダプター
-   テストにマウントされているセンサー ボード
-   電源の USB ハブ
-   USB キーボード
-   USB マウス
-   USB ネットワーク アダプター
-   (場合によっては HDMI ケーブル アダプター) で、モニター、HDMI ケーブル

接続し、上記のハードウェアを設定する方法の詳細については、次を参照してください。[サメ Cove ハードウェア開発ボード](https://msdn.microsoft.com/library/windows/hardware/dn745910.aspx)、まで、すべてのタスクを実行し、手順 4 を含むとします。

前の段落で推奨される手順を正常に完了する場合は、開発環境が、次が今すぐ必要です。

-   Microsoft キットとツールのダウンロードおよびインストール
-   インストールされている Windows の正しいバージョンのサメ Cove
-   サメ Cove の構成を受信し、デプロイされたドライバーをテストします。

次のトピックのガイダンスに従って[センサー テスト ボードの準備](prepare-your-sensor-test-board.md)サメ Cove ボードに接続します。
