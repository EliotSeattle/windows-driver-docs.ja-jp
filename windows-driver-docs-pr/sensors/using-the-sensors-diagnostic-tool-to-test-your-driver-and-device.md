# <a name="use-the-sensors-diagnostic-tool-to-test-your-driver-and-device"></a>センサー診断ツールを使用して、デバイスとドライバーをテストするには


センサー診断ツール (SDT) は、Windows Driver Kit に含まれているし、ドライバー、ファームウェア、ハードウェアの機能をテストするため、このツールを使用することができます。

>[!NOTE]
> センサー診断ツールは Windows 10 の非推奨となりました。 Microsoft Store で、すべてのセンサーのテストと診断の SensorInfo アプリを使用してください。

 

デバイスとドライバーが実行されると、テストをこのツールを使用することができます。

-   データの取得
-   イベント処理
-   レポート間隔
-   感度を変更します。
-   プロパティの取得
-   プロパティの設定

このスクリーン ショットは、SpbAccelerometer サンプル ドライバーと ADXL345 センサーがインストールされた後、SDT 実行を示しています。

![センサー診断ツール](images/sdt2.png)

詳細については、SDT を使用して、参照してください、[センサー ドライバーのマニュアル](https://docs.microsoft.com/windows-hardware/drivers/sensors/)します。
