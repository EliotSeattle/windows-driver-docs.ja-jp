# <a name="sensor-driver-design-guide"></a>センサー ドライバーの設計ガイド


このドキュメントでは、Windows で sensor and location プラットフォームと対話するデバイス ドライバーを作成する方法について説明します。 Windows ドライバー キットは、センサー ハードウェアのデバイス ドライバーを実装する最初の手順に従って、SensorsGeolocationDriverSample と呼ばれる基本的なサンプルを提供します。 さらに、Windows ドライバー キットは、実際のハードウェアをサポートするために必要なコードを示す SensorsHIDClassDriver と呼ばれるより完全なサンプルを提供します。

このセクションのトピックでは、Windows で sensor and location プラットフォームを使用するセンサー デバイスを有効にするドライバーを構築する方法について説明します。
