# <a name="proximity-sensor-property"></a>近接センサー プロパティ



このプロパティは、省略可能な列挙型プロパティです。

|プロパティのキー|種類|アクセス (R/O、読み取り/書き込み) |必須/オプション|説明|
|--|--|--|--|--|
|DEVPKEY_Sensor_ProximityType|VT_UI4|R/O|省略可能|検出されている近接の種類について説明します。 HumanProximity または ObjectProximity を指定できます。 詳細については、ProximityType 列挙型を参照してください。|

 

## <a name="requirements"></a>要件


**ヘッダー:** Sensorsdef.h
