# <a name="sensor-helper-functions"></a>センサーのヘルパー関数

このセクションでは、センサー、構造体、それらについての情報を取得して、構造体を初期化するために使用するヘルパー関数の詳細についての情報を提供します。

## <a name="in-this-section"></a>このセクションの内容

|トピック|説明|
|---|---|
|[SENSOR_COLLECTION_LIST_INIT](https://docs.microsoft.com/windows-hardware/drivers/ddi/content/sensorsdef/nf-sensorsdef-sensor_collection_list_init)|この関数では、SENSOR_COLLECTION_LIST 構造体を初期化します。|
|[SENSOR_COLLECTION_LIST_SIZE](https://docs.microsoft.com/windows-hardware/drivers/ddi/content/sensorsdef/nf-sensorsdef-sensor_collection_list_size)|この関数は、SENSOR_COLLECTION_LIST 構造体のサイズを返します。|
|[SENSOR_COLLECTION_LIST_CALCULATE_MAX_COUNT](https://docs.microsoft.com/windows-hardware/drivers/ddi/content/sensorsdef/nf-sensorsdef-sensor_collection_list_calculate_max_count)|この関数は、SENSOR_VALUE_PAIR SENSOR_COLLECTION_LIST 構造体の要素の数を計算します。|
|[SENSOR_PROPERTY_LIST_INIT](https://docs.microsoft.com/windows-hardware/drivers/ddi/content/sensorsdef/nf-sensorsdef-sensor_property_list_init)|この関数では、SENSOR_PROPERTY_LIST 構造体を初期化します。|
|[SENSOR_PROPERTY_LIST_SIZE](https://docs.microsoft.com/windows-hardware/drivers/ddi/content/sensorsdef/nf-sensorsdef-sensor_property_list_size)|この関数は、プロパティ リストのサイズを返します。|
|[SENSOR_PROPERTY_LIST_CALCULATE_MAX_COUNT](https://docs.microsoft.com/windows-hardware/drivers/ddi/content/sensorsdef/nf-sensorsdef-sensor_property_list_calculate_max_count)|この関数は、PROPERTYKEY 要素の数を計算します。|
|[SENSOR_CONFIG_INIT](https://docs.microsoft.com/windows-hardware/drivers/ddi/content/sensorscx/nf-sensorscx-sensor_config_init)|この関数では、SENSOR_CONFIG 構造体を初期化します。|
|[SENSOR_CONTROLLER_CONFIG_INIT](https://docs.microsoft.com/windows-hardware/drivers/ddi/content/sensorscx/nf-sensorscx-sensor_controller_config_init)|この関数では、SENSOR_CONTROLLER_CONFIG 構造体を初期化します。|
