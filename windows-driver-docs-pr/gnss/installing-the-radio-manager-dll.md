# <a name="installing-the-radio-manager-dll"></a>無線マネージャー DLL のインストール

> [!IMPORTANT] 
> このドキュメントと Windows 8.1 の地理的位置情報ドライバー サンプルが非推奨とされました。

ラジオ manager DLL をインストールするには、次の手順に従う必要があります。

1.  エラーのない DLL を構築することを確認します。
2.  ファイル install.cmd とから SampleRM.reg コピー、  **\\C++\\RadioManagerGPS**別のフォルダーにフォルダー。
3.  ビルド フォルダーからファイル SampleRM.dll を (上記) の手順 2 で作成した別のフォルダーにコピーします。
4.  管理者特権でコマンド プロンプトで、install.cmd を実行します。
