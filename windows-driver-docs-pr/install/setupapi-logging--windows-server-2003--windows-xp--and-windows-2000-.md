# <a name="setupapi-logging-windows-server-2003-windows-xp-and-windows-2000"></a>SetupAPI ログ (Windows Server 2003、Windows XP、および Windows 2000)





Windows セットアップとデバイスのインストーラー サービスとも呼ばれます SetupAPI は、セットアップとデバイスのインストールを制御する Windows 関数が含まれます。 セットアップ処理の進行状況、[一般的なセットアップ関数](https://msdn.microsoft.com/library/windows/hardware/ff544985)(**セットアップ * * * Xxx*関数) と[デバイスのインストール機能](https://msdn.microsoft.com/library/windows/hardware/ff541299)(* * SetupDi * * * Xxx*関数) デバイスのインストールに関する問題のトラブルシューティングに役立つ情報を提供するログ ファイルを作成します。

SetupAPI ログ ファイルに*systemroot*%\\*setupapi.log*します。 ログ ファイルは、プレーン テキスト ファイルです。 ログをリセットするには、変更するか、ファイルを削除します。

このセクションには、次の情報が含まれています。
