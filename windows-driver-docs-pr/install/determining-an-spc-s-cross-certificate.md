クロス証明書は、別の CA のルート証明書の公開キーを署名する CA によって発行された X.509 証明書です。 クロス証明書は、1 つ信頼された Microsoft ルート機関、SPCs を発行する商用の Ca を信頼チェーンを拡張する柔軟性も備えていて、カーネルを使用できます。
