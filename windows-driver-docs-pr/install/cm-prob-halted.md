# <a name="code-44---cm_prob_halted"></a>コード 44-CM_PROB_HALTED

このデバイスマネージャーエラーメッセージは、デバイスが停止したことを示します。

## <a name="error-code"></a>エラー コード

44

### <a name="display-message"></a>メッセージの表示

"アプリケーションまたはサービスがこのハードウェアデバイスをシャットダウンしました。 (コード 44) "

### <a name="recommended-resolution"></a>推奨される解決策

コンピューターを再起動する必要があります。
