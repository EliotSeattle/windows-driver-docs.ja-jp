# <a name="driver-packages"></a>ドライバー パッケージ


*ドライバーパッケージ*は、デバイスが Windows でサポートされるために提供する必要があるすべてのソフトウェアコンポーネントで構成されます。

デバイスまたはドライバーのインストールには、システムによって提供されるコンポーネントとベンダーが提供するコンポーネントが含まれます。 システムには、すべてのデバイスクラスに対する汎用インストールソフトウェアが用意されています。 ベンダーは、ドライバーパッケージ内にデバイス固有のコンポーネントを1つ以上指定する必要があります。

このセクションでは、ドライバーパッケージ内で提供するコンポーネントを決定するのに役立つ情報を提供します。
