# <a name="driver-packages"></a>ドライバー パッケージ


*ドライバー パッケージ*は、デバイスが Windows でサポートされるために指定する必要があるすべてのソフトウェア コンポーネントで構成されています。

デバイスまたはドライバーのインストールには、システム提供のコンポーネントとベンダー提供のコンポーネントが必要です。 システムには、すべてのデバイス クラスに対応する汎用インストール ソフトウェアが用意されています。 ベンダーは、デバイス固有のコンポーネントをドライバー パッケージ内で 1 つ以上指定する必要があります。

このセクションでは、ドライバー パッケージ内で指定するコンポーネントを決定するのに役立つ情報を提供します。
