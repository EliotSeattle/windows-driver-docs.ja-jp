# <a name="setupapi-device-installation-log-entries"></a>SetupAPI デバイス インストールのログ エントリ


Windows Vista および以降のバージョンの Windows、 [SetupAPI ログ](setupapi-logging--windows-vista-and-later-.md)デバイスのインストールを確認して、デバイスのインストールに関する問題のトラブルシューティングに使用できるプレーン テキスト ログ ファイルにデバイスのインストールに関する情報。 署名の問題がある場合は、SetupAPI はログ ファイルに署名の問題に関する情報をログインします。 このログ ファイルの名前は*SetupAPI.dev.log*、既定では、場所は、Windows の INF ファイルのディレクトリ (*%systemroot%\\inf*)。
