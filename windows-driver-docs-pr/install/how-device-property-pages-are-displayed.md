# <a name="how-device-property-pages-are-displayed"></a>デバイスのプロパティ ページを表示する方法


次の方法では、デバイスのプロパティ ページが表示されます。

-   デバイス マネージャーは、いつでも、デバイスのプロパティが表示されるプロパティ ページを表示します。

-   Windows 7 以降、**デバイスとプリンター**プロパティ ページをいつでも表示ユーザーがデバイスを右クリックします。 プロパティ ページを表示する**デバイスとプリンター**次の操作を行います。

    -   アイコンを右クリックして**デバイスとプリンター**システムに接続されているデバイスを表します。

    -   **[プロパティ]** を選択します。

    -   **プロパティ**ウィンドウで、をクリックして、**ハードウェア** タブをクリックして**プロパティ**します。
