# <a name="softwareinfo-xml-document"></a>SoftwareInfo XML ドキュメント


Windows 8 またはそれ以降は、メタデータ パッケージは、アプリケーションをデバイスに関連付けられているアプリをダウンロードして特権アクセスの情報を含む 1 つの Softwareinfo.xml ファイルを含めることができます。

Softwareinfo.xml ファイル内のデータはに基づいて書式設定、 [ **SoftwareInfo XML スキーマ**](https://docs.microsoft.com/previous-versions/windows/hardware/metadata/jj159075(v=vs.85))します。
