# <a name="specifying-hardware-ids-for-a-bluetooth-device"></a>Bluetooth デバイス用のハードウェア ID の指定


指定する[ハードウェア Id](hardware-ids.md)デバイス メタデータ パッケージ内での Bluetooth デバイスのデバイスがデバイスの Id (DID) のプロファイルをサポートする必要があります。 それ以外の場合、オペレーティング システムは選択し、Bluetooth デバイスの最適なデバイス メタデータ パッケージを読み込むことはできません。 すべての Bluetooth デバイスが DID のプロファイルをサポートすることを強くお勧めします。

プロファイルの詳細についてを参照してください、 [Bluetooth デバイス識別プロファイル仕様のバージョン 1.3](https://go.microsoft.com/fwlink/p/?linkid=145536)します。

Bluetooth 低エネルギー (LE) デバイス Id をサポートする方法の詳細についてを参照してください、[デバイス情報のサービスのバージョン 1.1](https://go.microsoft.com/fwlink/p/?linkid=242771)します。
