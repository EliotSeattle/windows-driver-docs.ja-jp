# <a name="code-55---cm_prob_console_locked"></a>コード 55-CM_PROB_CONSOLE_LOCKED

このデバイスマネージャーエラーメッセージは、グループポリシー/MDM (Intune) ポリシーが DMA 攻撃から保護するため、デバイスが列挙からブロックされていることを示します。




## <a name="error-code"></a>エラー コード

55

### <a name="display-message"></a>メッセージの表示

"ユーザーがログインしていないときに、このデバイスの起動がブロックされています。 (コード 55) "


### <a name="recommended-resolution"></a>推奨される解決策

カーネル DMA 保護の詳細については、「 https://docs.microsoft.com/windows/security/information-protection/kernel-dma-protection-for-thunderbolt」を参照してください。
