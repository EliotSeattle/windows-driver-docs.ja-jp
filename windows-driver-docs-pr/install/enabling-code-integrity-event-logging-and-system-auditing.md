# <a name="code-integrity-event-logging-and-system-auditing"></a>コードの整合性イベント ログとシステムの監査


コードの整合性は、ドライバー署名の検証を実装するカーネル モード コンポーネントです。 イメージの検証に関連するシステムのイベントを生成し、コードの整合性のログに情報をログします。

-   コードの整合性の操作ログ ビューには、イメージの検証エラー イベントのみが表示されます。

-   コードの整合性の詳細なログ ビューには、成功した署名の検証のイベントが表示されます。

コードの整合性イベント ログとシステムの監査の詳細については、次を参照してください。[コード整合性診断システムのログ イベント](code-integrity-diagnostic-system-log-events.md)します。

システム監査ログと詳細なログ記録を有効にする方法の詳細については、次を参照してください。[システム イベントの監査ログを有効にする](enabling-the-system-event-audit-log.md)します。
