# <a name="cmprobdevicereset"></a>CM_PROB_DEVICE_RESET

この関数は、システムの使用に予約されています。

## <a name="error"></a>エラー

54

### <a name="display-message-windows-10-and-later-versions-of-windows"></a>(Windows 10 および Windows の以降のバージョン) のメッセージを表示します。

"このデバイスが失敗し、リセットを実行中です。 (コード 54)"

### <a name="recommended-resolution-windows-10-and-later-versions-of-windows"></a>推奨される解決方法 (Windows 10 および Windows の以降のバージョン)

これは、ACPI の reset メソッドの実行中に割り当てられている、断続的な問題のコードです。 失敗したため、デバイスを再起動しない場合、この状態に陥る考えられるし、システムを再起動する必要があります。
