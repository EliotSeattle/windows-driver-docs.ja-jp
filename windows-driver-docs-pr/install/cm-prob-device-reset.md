# <a name="code-54---cm_prob_device_reset"></a>コード 54-CM_PROB_DEVICE_RESET

このデバイスマネージャーエラーメッセージは、デバイスに障害が発生し、リセット中であることを示します。

## <a name="error"></a>エラー

54

### <a name="display-message-windows-10-and-later-versions-of-windows"></a>メッセージの表示 (windows 10 以降のバージョンの Windows)

"このデバイスは失敗し、リセットされています。 (コード 54) "

### <a name="recommended-resolution-windows-10-and-later-versions-of-windows"></a>推奨される解決策 (windows 10 以降のバージョンの Windows)

これは、ACPI のリセット方法が実行されている間に割り当てられた断続的な問題コードです。 障害によってデバイスが再起動されない場合、デバイスはこの状態でスタックし、システムを再起動する必要があります。
