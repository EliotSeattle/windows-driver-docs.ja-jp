# <a name="cmprobtranslationfailed"></a>CM_PROB_TRANSLATION_FAILED

この関数は、システムの使用に予約されています。

デバイスのリソース変換に失敗しました。

## <a name="error-code"></a>エラー コード

33

### <a name="display-message"></a>メッセージを表示します。

"Windows を特定できませんこのデバイスで必要なリソースを。 (コード 33)"

### <a name="recommended-resolution"></a>推奨される解決方法

BIOS セットアップ ユーティリティを使用してお試しください。 または、BIOS を更新します。

構成、修復、またはハードウェアを交換します。
