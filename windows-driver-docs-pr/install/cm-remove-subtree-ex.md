# <a name="cmremovesubtreeex"></a>CM_Remove_SubTree_Ex

この関数は、システムの使用に予約されています。

この関数は廃止され、不要になった Windows 2000 以降のバージョンの Windows でサポートされています。 使用[ **CM_Query_And_Remove_SubTree_Ex** ](https://docs.microsoft.com/windows/desktop/api/cfgmgr32/nf-cfgmgr32-cm_query_and_remove_subtree_exw)代わりにします。
