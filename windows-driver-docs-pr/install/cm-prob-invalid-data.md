# <a name="code-9---cm_prob_invalid_data"></a>コード 9-CM_PROB_INVALID_DATA

このデバイスマネージャーエラーメッセージは、無効な[デバイス id](device-ids.md)が検出されたことを示します。

## <a name="error-code"></a>エラー コード

9

### <a name="display-message"></a>メッセージの表示

"有効なハードウェア識別番号がないため、Windows はこのハードウェアを識別できません。 (コード 9) "

"詳細については、ハードウェアの製造元に問い合わせてください。"

### <a name="recommended-resolution"></a>推奨される解決策

ハードウェアの製造元に問い合わせてください。 ハードウェアまたはドライバーに問題があります。
