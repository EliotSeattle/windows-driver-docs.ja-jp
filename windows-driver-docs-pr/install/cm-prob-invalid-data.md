# <a name="cmprobinvaliddata"></a>CM_PROB_INVALID_DATA

この関数は、システムの使用に予約されています。

無効な[デバイス Id](device-ids.md)が検出されました。

## <a name="error-code"></a>エラー コード

9

### <a name="display-message"></a>メッセージを表示します。

"Windows を識別できませんこのハードウェア、ハードウェアの有効な識別番号があるないためです。 (コード 9)"

「詳細については、ハードウェアの製造元に問い合わせてください。」

### <a name="recommended-resolution"></a>推奨される解決方法

ハードウェアの製造元に問い合わせてください。 ハードウェアまたはドライバーには、欠陥があります。
