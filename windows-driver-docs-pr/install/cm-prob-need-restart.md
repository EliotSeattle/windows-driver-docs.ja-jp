# <a name="code-14---cm_prob_need_restart"></a>コード 14-CM_PROB_NEED_RESTART

このデバイスマネージャーエラーメッセージは、システムを再起動する必要があることを示します。

## <a name="error-code"></a>エラー コード

14

### <a name="display-message"></a>メッセージの表示

"このデバイスは、コンピューターを再起動するまで正常に機能しません。 (コード 14)

"今すぐコンピューターを再起動するには、[コンピューターの再起動] をクリックしてください。"

### <a name="recommended-resolution"></a>推奨される解決策

**[コンピューターの再起動]** を選択すると、コンピューターが再起動されます。
