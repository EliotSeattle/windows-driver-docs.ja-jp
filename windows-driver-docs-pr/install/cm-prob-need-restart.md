# <a name="cmprobneedrestart"></a>CM_PROB_NEED_RESTART

この関数は、システムの使用に予約されています。

システムを再起動する必要があります。

## <a name="error-code"></a>エラー コード

14

### <a name="display-message"></a>メッセージを表示します。

"コンピューターを再起動するまで、このデバイスが正しく動作できません。 (コード 14)

「コンピューターを今すぐ再起動するには、コンピューターの再起動 をクリックします。」

### <a name="recommended-resolution"></a>推奨される解決方法

選択**コンピューターの再起動**、お使いのコンピューターを再起動します。
