# <a name="cmproboutofmemory"></a>CM_PROB_OUT_OF_MEMORY

この関数は、システムの使用に予約されています。

システム メモリ − 不足していますがおそらくシステム メモリ不足します。

## <a name="error-code"></a>エラー コード

3

### <a name="display-message"></a>メッセージを表示します。

"このデバイスのドライバーが壊れている可能性がある、またはシステムのメモリまたはその他のリソースが不足します。 (コード 3)"

### <a name="recommended-resolution"></a>推奨される解決方法

デバイス ドライバーが破損している場合は、ドライバーを更新します。

デバイス ドライバーを更新するには、次の手順を実行します。

1. [デバイス マネージャーを開く](using-device-manager.md)します。

2. デバイス マネージャー ウィンドウでデバイスを表すアイコンを右クリックします。

3. 表示されるメニューで、**ドライバーの更新**ハードウェアの更新ウィザードを起動します。

デバイスのドライバーが正常する場合は、コンピューターが、デバイスと同時に動作するには、その他のアプリケーションのメモリ不足のためにデバイスを操作するための十分なメモリを一部のアプリケーションを閉じる。

使用可能なメモリとシステム リソースを確認するには、次の手順を実行します。

1. をクリックして、**開始**メニュー。

2. 次の一連のサブメニューをポイントします。**すべてのプログラム**、**アクセサリ**、および**システム ツール**します。

3. **システム ツール** メニューのをクリックして**システム情報**使用可能な物理メモリの量がデバイスを操作するための十分なかどうかを判断します。

デバイスを操作する追加のランダム アクセス メモリ (RAM) をインストールする必要があります。
