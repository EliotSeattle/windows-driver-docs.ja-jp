# <a name="code-3---cm_prob_out_of_memory"></a>コード 3-CM_PROB_OUT_OF_MEMORY

このデバイスマネージャーエラーメッセージは、システムのメモリが不足していることが原因であることを示します。

## <a name="error-code"></a>エラー コード

3

### <a name="display-message"></a>メッセージの表示

"このデバイスのドライバーが壊れているか、システムでメモリまたはその他のリソースが不足している可能性があります。 (コード 3) "

### <a name="recommended-resolution"></a>推奨される解決策

デバイスドライバが破損している場合は、ドライバを更新します。

デバイスドライバーを更新するには、次の手順を実行します。

1. [デバイスマネージャーを開き](using-device-manager.md)ます。

2. [デバイスマネージャー] ウィンドウで、デバイスを表すアイコンを右クリックします。

3. 表示されるメニューで、 **[ドライバーの更新]** をクリックして、ハードウェアの更新ウィザードを開始します。

デバイスのドライバーが正しく動作していても、デバイスとその他のアプリケーションが同時に動作するためのメモリが不足している場合は、一部のアプリケーションを閉じて、デバイスを操作するための十分なメモリを確保することができます。

使用可能なメモリとシステムリソースを確認するには、次の手順を実行します。

1. **[スタート]** ボタンをクリックします。

2. 次の一連のサブメニューをポイントします。 **[すべてのプログラム]** 、 **[アクセサリ]** 、 **[システムツール]** 。

3. **[システムツール]** メニューの **[システム情報]** をクリックして、使用可能な物理メモリの量がデバイスを操作するのに十分であるかどうかを判断します。

場合によっては、デバイスを操作するために、追加のランダムアクセスメモリ (RAM) をインストールする必要があります。
