# <a name="guidclassusbhostcontroller"></a>GUID_CLASS_USB_HOST_CONTROLLER


GUID_CLASS_USB_HOST_CONTROLLER は古い形式の識別子、[デバイス インターフェイス クラス](https://msdn.microsoft.com/library/windows/hardware/ff541339)の[USB](https://msdn.microsoft.com/library/windows/hardware/ff538930)コント ローラー デバイスをホストします。 Microsoft Windows 2000 以降を使用して、 [ **GUID_DEVINTERFACE_USB_HOST_CONTROLLER** ](guid-devinterface-usb-host-controller.md)このクラスの新しいインスタンスのクラス識別子。

<a name="remarks"></a>注釈
-------

Microsoft Windows Driver Kit (WDK) が含まれています、 [USBVIEW サンプル アプリケーション](https://go.microsoft.com/fwlink/p/?linkid=256205)します。 USBVIEW サンプルでは、GUID_CLASS_USB_HOST_CONTROLLER を使用して、GUID_CLASS_USB_HOST_CONTROLLER デバイス インターフェイスのクラスのインスタンスを列挙します。
