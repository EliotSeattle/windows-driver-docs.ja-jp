# <a name="cmprobbiostable"></a>CM_PROB_BIOS_TABLE

この関数は、システムの使用に予約されています。

MP テーブルが無効であると、更新する必要があります。

## <a name="error-code"></a>エラー コード

35

### <a name="display-message"></a>メッセージを表示します。

"コンピューターのシステム ファームウェアでは適切に構成し、このデバイスを使用するための情報は含まれません。 このデバイスを使用するには、ファームウェアまたは BIOS の更新プログラムを取得するコンピューターの製造元にお問い合わせください。 (コード 35)"

### <a name="recommended-resolution"></a>推奨される解決方法

システムのベンダーから、新しい BIOS を取得します。
