# <a name="cmprobdriverblocked"></a>CM_PROB_DRIVER_BLOCKED

この関数は、システムの使用に予約されています。

記載されているためは、システムに、ドライバーは読み込まれません、 [Windows ドライバー保護](https://docs.microsoft.com/windows-hardware/drivers)によって提供されるデータベース[Windows Update](https://docs.microsoft.com/windows-hardware/drivers)*します。*

## <a name="error-code"></a>エラー コード

48

### <a name="display-message"></a>メッセージを表示します。

"このデバイスのソフトウェアを Windows に問題があることがわかっているので開始からブロックされています。 新しいドライバーについては、ハードウェア ベンダーにお問い合わせください。 (コード 48)"

### <a name="recommended-resolution"></a>推奨される解決方法

ハードウェア ベンダーから新しいドライバーを入手します。
