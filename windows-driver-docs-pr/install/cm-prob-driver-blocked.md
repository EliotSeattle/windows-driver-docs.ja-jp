# <a name="code-48---cm_prob_driver_blocked"></a>コード 48-CM_PROB_DRIVER_BLOCKED

このデバイスマネージャーエラーメッセージは、Windows Update によって提供される Windows Driver Protection データベースに一覧表示されているため、システムがドライバーを読み込まないことを示します。

## <a name="error-code"></a>エラー コード

48

### <a name="display-message"></a>メッセージの表示

"このデバイスのソフトウェアは、Windows で問題が発生していることがわかっているため、開始されませんでした。 新しいドライバーについては、ハードウェアベンダーに問い合わせてください。 (コード 48) "

### <a name="recommended-resolution"></a>推奨される解決策

ハードウェアベンダーから新しいドライバーを入手します。
