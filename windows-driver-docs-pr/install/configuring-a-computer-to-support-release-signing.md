# <a name="configuring-a-computer-to-support-release-signing"></a>リリース署名をサポートするコンピューターを構成します。


リリースで署名されたドライバー パッケージをインストールするコンピューターで、次のリリースの署名をサポートするコンピューターを構成する手順を実行します。

-   TESTSIGNING ブートの構成オプションを無効にします。 このオプションの詳細については、次を参照してください。 [TESTSIGNING ブートの構成オプション](the-testsigning-boot-configuration-option.md)します。

-   コードの整合性イベント ログとシステムのリリースで署名されたドライバーのインストールと読み込み中に問題のトラブルシューティングに役立つ監査を有効にします。 詳細については、次を参照してください。[コードの整合性イベント ログの有効化と、システムの監査](enabling-code-integrity-event-logging-and-system-auditing.md)します。

コンピューターは、リリース署名を有効にするように再構成後、は、変更を反映するには、コンピューターを再起動します。
