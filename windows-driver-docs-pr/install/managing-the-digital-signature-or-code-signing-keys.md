# <a name="managing-the-digital-signature-or-code-signing-keys"></a>デジタル署名またはコード署名キーの管理


暗号化キーは、Authenticode 署名プロセスの中核をも保護されていると、発行元の最も重要な資産として扱われる必要があります。 これらのキーは、組織の id を表します。 これらのキーで署名されたコードは、組織にトレースできる有効なデジタル署名が含まれているかのように Windows が表示されます。 キーが盗まれた場合は、悪意のあるコードを不正にサインインし、トロイの木馬を含むコードまたは正当なパブリッシャーから来るように表示されるウイルスの提供に発生する可能性がある使用可能性があります。
