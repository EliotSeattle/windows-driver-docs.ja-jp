# <a name="dif_destroywizarddata"></a>DIF_DESTROYWIZARDDATA


この差分コードは互換性のために残されており、Microsoft Windows 2000 以降のバージョンの Windows ではサポートされなくなりました。

Windows では、 [**DIF_NEWDEVICEWIZARD_FINISHINSTALL**](dif-newdevicewizard-finishinstall.md)などの DIF_NEWDEVICEWIZARD_*XXX*要求を代わりに使用します。
