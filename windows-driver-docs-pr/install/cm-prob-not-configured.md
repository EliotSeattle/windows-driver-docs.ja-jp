# <a name="cmprobnotconfigured"></a>CM_PROB_NOT_CONFIGURED

この関数は、システムの使用に予約されています。

デバイスが存在するシステムではない**ConfigFlags**レジストリ エントリ。 これは、ドライバーがインストールされていないことを意味します。 つまり、通常、INF ファイルが見つかりませんでした。

## <a name="error-code"></a>エラー コード

1

### <a name="display-message"></a>メッセージを表示します。

"このデバイスが正しく構成されていません。 (コード 1)"

"このデバイスのドライバーを更新するには、[ドライバーの更新] をクリックします。 うまく行かない場合を参照してください詳細については、ハードウェアのマニュアルです。"

### <a name="recommended-resolution"></a>推奨される解決方法

選択**ドライバーの更新**ハードウェアの更新ウィザードを開始します。
