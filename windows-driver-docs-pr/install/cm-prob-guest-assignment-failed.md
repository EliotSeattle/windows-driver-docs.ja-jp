# <a name="code-57---cm_prob_guest_assignment_failed"></a>コード 57-CM_PROB_GUEST_ASSIGNMENT_FAILED

このデバイスマネージャーエラーメッセージは、デバイスの割り当て中にエラーが発生したため、デバイスが開始されなかったことを示します。

## <a name="error-code"></a>エラー コード

57

### <a name="display-message"></a>メッセージの表示

"このデバイスをゲストパーティションに割り当てることができませんでした。 (コード 57) "

### <a name="recommended-resolution"></a>推奨される解決策

詳細については、「[個別のデバイスの割り当てを使用したデバイスの展開の計画](https://docs.microsoft.com/windows-server/virtualization/hyper-v/plan/plan-for-deploying-devices-using-discrete-device-assignment)」を参照してください。
