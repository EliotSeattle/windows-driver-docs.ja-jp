# <a name="cmprobnosoftconfig"></a>CM_PROB_NO_SOFTCONFIG

この関数は、システムの使用に予約されています。

デバイスでは、強制の構成が必要です。

### <a name="error-code"></a>エラー コード

34

### <a name="display-message"></a>メッセージを表示します。

"Windows は、このデバイスの設定を判断できません。 このデバイスに付属のマニュアルを参照し、構成を設定するリソース タブを使用します。 (コード 34)"

### <a name="recommended-resolution"></a>推奨される解決方法

(ジャンパーの設定や、ベンダーから提供されたユーティリティを実行する) をハードウェア設定を変更し、デバイス マネージャーを使用して**リソース**強制の構成を設定するタブ。
