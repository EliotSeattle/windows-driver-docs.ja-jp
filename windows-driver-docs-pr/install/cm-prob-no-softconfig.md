# <a name="code-34---cm_prob_no_softconfig"></a>コード 34-CM_PROB_NO_SOFTCONFIG

このデバイスマネージャーエラーメッセージは、デバイスに強制構成が必要であることを示します。

### <a name="error-code"></a>エラー コード

34

### <a name="display-message"></a>メッセージの表示

"このデバイスの設定を判断できません。 このデバイスに付属のドキュメントを参照し、[リソース] タブを使用して構成を設定します。 (コード 34) "

### <a name="recommended-resolution"></a>推奨される解決策

(ジャンパーを設定するか、ベンダーが提供するユーティリティを実行して) HW 設定を変更し、デバイスマネージャーの **[リソース]** タブを使用して強制構成を設定します。
