# <a name="cmprobfailedinstall"></a>CM_PROB_FAILED_INSTALL

この関数は、システムの使用に予約されています。

デバイスのドライバーがインストールされていません。

## <a name="error-code"></a>エラー コード

28

### <a name="display-message"></a>メッセージを表示します。

"このデバイスのドライバーはインストールされません。 (コード 28)"

### <a name="recommended-resolution"></a>推奨される解決方法

デバイスを製造した会社の web サイトにアクセスして、このデバイスの最新のドライバーを探してください。
