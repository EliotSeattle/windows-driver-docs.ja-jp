# <a name="hardware-based-dvd-decoding"></a>ハードウェア ベースの DVD のデコード





次の図は、ハードウェア ベースの DVD デコード ソリューションと、Windows Driver Model (WDM) アーキテクチャに組み込む方法を示します。 Microsoft Windows XP での既存の DVD テクノロジとハードウェア デコーダーの完全なサポートが表示されます。

白いボックスは、Microsoft によって提供されるソフトウェアを表し、影付きのボックスは、ハードウェア ベンダーによって提供されるコンポーネントを表します。 楕円は、Ihv と Oem によって提供されるハードウェアを表します。

Microsoft の DVD を使用してコンピューターが両方の DVD ストレージをサポートし、適切なハードウェアをデコードまたはソフトウェアが存在する、DVD のデコードおよび再生の場合。
