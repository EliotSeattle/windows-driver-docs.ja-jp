# <a name="ksproperty_bda_null_transform_stop"></a>KSK プロパティ\_BDA\_NULL\_変換\_停止


クライアントは、KSK プロパティ\_BDA\_NULL\_変換を使用して、その変換が無効になる前に\_停止します。この変換は、KSK プロパティ\_BDA\_NULL\_TRANSFORM を使用して行われます。7_ START プロパティ。
