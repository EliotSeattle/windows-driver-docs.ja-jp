# <a name="broadcast-driver-architecture-constants"></a>ブロードキャスト ドライバー アーキテクチャの定数


## <span id="ddk_broadcast_driver_architecture_constants_ks"></span><span id="DDK_BROADCAST_DRIVER_ARCHITECTURE_CONSTANTS_KS"></span>


BDA ミニドライバーは、次の定数を使用します。

<span id="BDA_Filter_Category_GUIDs"></span><span id="bda_filter_category_guids"></span><span id="BDA_FILTER_CATEGORY_GUIDS"></span>[BDA フィルターのカテゴリの Guid](bda-filter-category-guids.md)  
BDA ミニドライバーは、これらの Guid を使用して、BDA フィルターを作成の種類を指定します。

<span id="BDA_Node_Category_GUIDs"></span><span id="bda_node_category_guids"></span><span id="BDA_NODE_CATEGORY_GUIDS"></span>[BDA ノード カテゴリの Guid](bda-node-category-guids.md)  
BDA ミニドライバーは、これらの Guid を使用して、作成する BDA ノードの種類を指定します。

<span id="BDA_Pin_Name_GUIDs"></span><span id="bda_pin_name_guids"></span><span id="BDA_PIN_NAME_GUIDS"></span>[BDA 暗証番号 (pin) の名前の Guid](bda-pin-name-guids.md)  
BDA ミニドライバーでは、これらの Guid を使用して、サポートされる pin の名前を指定します。

<span id="BDA_Stream_Format_GUIDs"></span><span id="bda_stream_format_guids"></span><span id="BDA_STREAM_FORMAT_GUIDS"></span>[BDA Stream の形式の Guid](bda-stream-format-guids.md)  
BDA ミニドライバーは、これらの Guid を使用して、サポートされるデータ形式を指定します。

<span id="Broadcast_Network_Type_GUIDs"></span><span id="broadcast_network_type_guids"></span><span id="BROADCAST_NETWORK_TYPE_GUIDS"></span>[ブロードキャスト ネットワーク型の Guid](broadcast-network-type-guids.md)  
BDA ミニドライバーは、これらの Guid を使用して、スキャン操作をサポートするブロードキャスト ネットワークの種類を指定します。

**注**   BDA 定数は、Windows XP で以降。 これらの定数は、そのプラットフォームに DirectX 9.0 以降がインストールされている場合にのみ、Windows 2000 プラットフォームで使用できます。
