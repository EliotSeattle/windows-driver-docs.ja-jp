# <a name="usb-video-class-driver-stack"></a>USB ビデオ クラス ドライバー スタック


ベンダーは、Microsoft が提供する USB ビデオ クラス ドライバーを使用して、ときに、ハードウェアのみを提供します。 Microsoft は、すべてのドライバー サポートを提供します。

必要に応じて、ベンダーは、デバイス固有の機能を追加する USB ビデオ クラス ドライバーを拡張できます。
