# <a name="device-transform-manager-events"></a>デバイス変換マネージャーのイベント


このセクションには、デバイス変換マネージャーのイベントに関する情報が含まれています。

| event                                                                        | 説明                                                                                                               |
|------------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------|
| [METransformFlushInputStream](metransformflushinputstream.md)               | デバイス MFT の入力に接続されている devproxy の出力ストリームをフラッシュするデバイス Transform Manager に通知します。 |
| [METransformHaveOutput](metransformhaveoutput.md)                           | デバイス変換にその出力ストリームのいずれかの準備完了のサンプルがあることを示します。                                        |
| [METransformInputStreamStateChanged](metransforminputstreamstatechanged.md) | 入力ストリームの状態またはメディアの種類を変更する必要がありますのことを示します。                                                      |
| [METransformNeedInput](metransformneedinput.md)                             | デバイス変換が入力を必要があることを示します。                                                                         |
