# <a name="software-based-dvd-decoding"></a>DVD のソフトウェア ベースのデコード





次の図では、完全にソフトウェア ベース DVD デコードおよび再生ソリューション、現在はハードウェア ベースの DVD のデコードおよび再生より一般的であるがについて説明します。

白いボックスは、Microsoft によって提供されるソフトウェアを表し、影付きのボックスは、ソフトウェア ベンダーによって提供されるコンポーネントを表します。 楕円は、Ihv と Oem によって提供されるハードウェアを表します。
