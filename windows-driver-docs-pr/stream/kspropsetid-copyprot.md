# <a name="kspropsetidcopyprot"></a>KSPROPSETID\_CopyProt


## <span id="ddk_kspropsetid_copyprot_ks"></span><span id="DDK_KSPROPSETID_COPYPROT_KS"></span>


KSPROPSETID\_CopyProt プロパティ セットは、DVD のストリームの著作権保護のレベルを指定するプロパティを定義します。

KSPROPERTY\_COPYPROT 列挙体*Ksmedia.h*このセットのプロパティを指定します。

DVD デコーダー ミニドライバーは、次のプロパティのサポートを実装する必要があります。

[**KSPROPERTY\_DVDCOPY\_CHLG\_KEY**](ksproperty-dvdcopy-chlg-key.md)

[**KSPROPERTY\_DVDCOPY\_DVD\_KEY1**](ksproperty-dvdcopy-dvd-key1.md)

[**KSPROPERTY\_DVDCOPY\_DEC\_KEY2**](ksproperty-dvdcopy-dec-key2.md)

[**KSPROPERTY\_DVDCOPY\_タイトル\_キー**](ksproperty-dvdcopy-title-key.md)

[**KSPROPERTY\_コピー\_MACROVISION**](ksproperty-copy-macrovision.md)

[**KSPROPERTY\_DVDCOPY\_REGION**](ksproperty-dvdcopy-region.md)

[**KSPROPERTY\_DVDCOPY\_ディスク\_キー**](ksproperty-dvdcopy-disc-key.md)

DVD デコーダー ミニドライバーは、次のプロパティを必要に応じて実装できます。

[**KSPROPERTY\_DVDCOPY\_設定\_コピー\_状態**](ksproperty-dvdcopy-set-copy-state.md)
