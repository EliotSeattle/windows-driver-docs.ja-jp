# <a name="kseventconnectionpriority"></a>KSEVENT\_接続\_優先順位


**KSEVENT\_接続\_優先度**イベントにより、クライアントは優先度のエラーの通知を受信します。 このイベントは、リソースが優先度レベルによって失われた場合に発生します。 このイベントに対するパラメーターはありません。
