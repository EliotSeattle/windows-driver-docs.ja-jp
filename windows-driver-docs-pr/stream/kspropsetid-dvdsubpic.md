# <a name="kspropsetiddvdsubpic"></a>KSPROPSETID\_DvdSubPic


## <span id="ddk_kspropsetid_dvdsubpic_ks"></span><span id="DDK_KSPROPSETID_DVDSUBPIC_KS"></span>


KSPROPSETID\_DvdSubPic プロパティ セットは、色、コントラスト、およびサブピクチャ表示の出力を制御するプロパティを定義します。

KSPROPERTY\_DVDSUBPIC 列挙体*Ksmedia.h*このセットのプロパティを指定します。

DVD デコーダー ミニドライバーは、次のプロパティのサポートを実装する必要があります。

[**KSPROPERTY\_DVDSUBPIC\_パレット**](ksproperty-dvdsubpic-palette.md)

[**KSPROPERTY\_DVDSUBPIC\_HLI**](ksproperty-dvdsubpic-hli.md)

[**KSPROPERTY\_DVDSUBPIC\_複合\_ON**](ksproperty-dvdsubpic-composit-on.md)
