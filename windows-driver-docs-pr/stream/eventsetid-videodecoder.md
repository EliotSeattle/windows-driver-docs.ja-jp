# <a name="eventsetidvideodecoder"></a>EVENTSETID\_VIDEODECODER


## <span id="ddk_eventsetid_videodecoder_ks"></span><span id="DDK_EVENTSETID_VIDEODECODER_KS"></span>


**EVENTSETID\_VIDEODECODER**など、新しい物理入力コネクタを選択すると、ビデオ デコーダー変更イベントが発生するイベントのセットをクライアントに通知します。

このセット内のイベントの項目が、KSEVENT で指定された\_VIDEODECODER 列挙体。

このセット内の唯一のイベントは[ **KSEVENT\_VIDEODECODER\_CHANGED**](ksevent-videodecoder-changed.md)します。
