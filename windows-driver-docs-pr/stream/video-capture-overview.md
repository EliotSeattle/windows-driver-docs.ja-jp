# <a name="video-capture-overview"></a>ビデオのキャプチャの概要


ビデオ キャプチャ ミニドライバーは、主に、テレビのオーディオなどの補助的なデータと共に、ビデオのデータのストリームを生成または自分をハードウェア デバイスを制御するか、AVStream の Stream クラス インターフェイスと対話/FM チューナー機能。 ベンダーは、ビデオ キャプチャ ミニドライバーを記述します。

-   IEEE 1394、USB、S-ビデオなどのデジタルおよびアナログのビデオ ソースから、および非圧縮ビデオ ストリームと RCA ジャックにビデオをキャプチャします。

-   垂直帰線消去期間 (VBI) データをキャプチャします。

-   テレビのオーディオまたは AM などの補助的なデータのストリームをキャプチャ/FM チューナー オーディオです。

-   Timecode をキャプチャします。

-   ビデオ ポートを制御し、ビデオ ポート ストリームからビデオをキャプチャします。

-   テレビやラジオ チューナー、ルーティング デバイス (クロスバー)、テレビのオーディオ コントロール、および圧縮方法のビデオ信号などのビデオ ストリームに関連付けられているデバイスを制御します。

-   カメラ ズーム、パン、およびフォーカスなどのプロパティを制御します。

-   色合い、鮮やかさ、明るさ、および鮮明度などのビデオのプロパティを制御します。

-   WDM (カーネル モード) 用ストリームと DirectShow (ユーザー モード) 用の互換性を提供します。
