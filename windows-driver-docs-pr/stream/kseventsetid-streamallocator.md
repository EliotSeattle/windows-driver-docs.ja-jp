# <a name="kseventsetidstreamallocator"></a>KSEVENTSETID\_StreamAllocator


KSEVENTSETID\_StreamAllocator イベントのセットを 2 つのイベントを指定します。 1 つのイベントは、IRP インターフェイスを通じて行われた未処理の割り当て要求を処理する既定のアロケーターの実装で内部的に使用されます。 無料のフレームの可用性のクライアントに通知する、別のパブリック イベントの場合は、使用されます。

KSEVENTSETID\_StreamAllocator イベントのセットが含まれます。

[**KSEVENT\_STREAMALLOCATOR\_内部\_FREEFRAME**](ksevent-streamallocator-internal-freeframe.md)

[**KSEVENT\_STREAMALLOCATOR\_FREEFRAME**](ksevent-streamallocator-freeframe.md)
