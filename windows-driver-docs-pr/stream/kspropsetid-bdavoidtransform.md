# <a name="kspropsetidbdavoidtransform"></a>KSPROPSETID\_BdaVoidTransform


## <span id="ddk_kspropsetid_bdavoidtransform_ks"></span><span id="DDK_KSPROPSETID_BDAVOIDTRANSFORM_KS"></span>


KSPROPSETID\_BdaVoidTransform は BDA void 変換のプロパティ セット。 ノードの起動し、停止を制御するために使用されます。 ネットワーク プロバイダーのフィルターは、このプロパティは、ノードの操作を制御する設定を使用します。

次のプロパティを使用できます。

<span id="KSPROPERTY_BDA_VOID_TRANSFORM_START"></span><span id="ksproperty_bda_void_transform_start"></span>[**KSPROPERTY\_BDA\_VOID\_変換\_開始**](ksproperty-bda-void-transform-start.md)  
ノードの操作を開始します。

<span id="KSPROPERTY_BDA_VOID_TRANSFORM_STOP"></span><span id="ksproperty_bda_void_transform_stop"></span>[**KSPROPERTY\_BDA\_VOID\_変換\_停止**](ksproperty-bda-void-transform-stop.md)  
ノードの操作を停止します。
