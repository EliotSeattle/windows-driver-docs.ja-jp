# <a name="kseventconnectionpositionupdate"></a>KSEVENT\_接続\_POSITIONUPDATE


**KSEVENT\_接続\_POSITIONUPDATE**イベントにより、クライアントは、現在の位置で変更の通知を受信します。 変更のマージンとは、ストリームと依存フィルターです。
