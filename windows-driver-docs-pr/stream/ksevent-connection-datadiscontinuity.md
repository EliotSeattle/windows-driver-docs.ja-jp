# <a name="kseventconnectiondatadiscontinuity"></a>KSEVENT\_接続\_DATADISCONTINUITY


**KSEVENT\_接続\_DATADISCONTINUITY**イベントにより、クライアントはデータの不連続性の通知を受信します。 このイベントは、pin でのストリーム インターフェイスを通じてデータの不連続性フラグが送信されるときに発生します。 このイベントに対するパラメーターはありません。
