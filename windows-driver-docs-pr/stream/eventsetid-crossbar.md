# <a name="eventsetidcrossbar"></a>EVENTSETID\_クロスバー


## <span id="ddk_eventsetid_crossbar_ks"></span><span id="DDK_EVENTSETID_CROSSBAR_KS"></span>


**EVENTSETID\_クロスバー**ときに、新しいシグナル ルーティングが選択されているなど、クロスバー変更イベントが発生するイベントのセットをクライアントに通知します。

このセット内のイベントの項目が、KSEVENT で指定された\_クロスバー列挙体。

このセット内の唯一のイベントは[ **KSEVENT\_クロスバー\_CHANGED**](ksevent-crossbar-changed.md)します。
