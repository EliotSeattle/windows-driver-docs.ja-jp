# <a name="dvd-data-streams"></a>DVD データ ストリーム





DVD デコーダー ミニドライバーは、両方の入力をサポートし、データ ストリームを出力します。 ストリームは、次の 3 つの Guid (主な種類、マイナーの種類、および指定子) とブロックの書式設定で一意に定義されます。 Stream の形式は、"メディアの種類"とも呼ばれます。 形式のブロックとメディアの種類の詳細については、Microsoft Windows sdk で DirectShow を参照してください。
