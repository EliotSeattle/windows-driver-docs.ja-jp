# <a name="kseventsetidconnection"></a>KSEVENTSETID\_接続


KSEVENTSETID\_接続イベントのセットがフィルターと pin を有効にできる一般的なイベントを指定します。 次のイベントのセットで構成されます。

[**KSEVENT\_接続\_POSITIONUPDATE**](ksevent-connection-positionupdate.md)****

[**KSEVENT\_接続\_DATADISCONTINUITY**](ksevent-connection-datadiscontinuity.md)****

[**KSEVENT\_接続\_TIMEDISCONTINUITY**](ksevent-connection-timediscontinuity.md)****

[**KSEVENT\_接続\_優先順位**](ksevent-connection-priority.md)****

[**KSEVENT\_接続\_ENDOFSTREAM**](ksevent-connection-endofstream.md)****
