# <a name="eventsetidtuner"></a>EVENTSETID\_チューナー


## <span id="ddk_eventsetid_tuner_ks"></span><span id="DDK_EVENTSETID_TUNER_KS"></span>


**EVENTSETID\_チューナー**チャネルの切り替えなど、チューナーの変更イベントが発生し、デバイスがスキャン操作を完了すると、チューニング時にイベントのセットをクライアントに通知します。

このセット内の次のイベント項目が、KSEVENT で指定された\_チューナー列挙体。

[**KSEVENT\_チューナー\_CHANGED**](ksevent-tuner-changed.md)

[**KSEVENT\_チューナー\_開始\_スキャン**](ksevent-tuner-initiate-scan.md)

**注**   、KSEVENT\_チューナー\_開始\_スキャン イベントが Windows Vista の新機能です。
