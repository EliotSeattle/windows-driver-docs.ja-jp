# <a name="closed-captioning-streams"></a>クローズド キャプション ストリーム





サポートのクローズド キャプションが必要です。 DVD デコーダーのミニドライバーは、Microsoft 行 21 デコーダーのフィルターのクローズド キャプションの情報を提供する必要があります。 実装は MPEG2 デコーダーの内部でクローズド キャプション情報をデコードし、ビデオ ストリームに追加するだけでなく可能性があります。 DVD デコーダーのミニドライバーは、出力ピンとしてクローズド キャプション暗証番号 (pin) を提示する必要があります。 DVD デコーダーのミニドライバーが SRB を受け取るストリームを開いた後\_読み取り\_ストリームに対するデータ要求。 DVD デコーダーのミニドライバーは、閉じたキャプション データが利用可能になるまで、これらの要求をキューします。

DVD デコーダーのミニドライバーがユーザー データ (クローズド キャプション情報) を検索し、使用してストリーム要求の 1 つに、クローズド キャプションをブロック (される Srb) の存在情報を返します、ビデオ ストリームのスタート コードの画像 (GOP) のグループが処理されるときストリームのキューです。 クローズド キャプション暗証番号 (pin) をビデオ ピンからすべてのデータの不連続性と形式ブロック変更を反映する必要があります。
