# <a name="kspropsetidvramcapture"></a>KSPROPSETID\_VramCapture


ビデオ ストリーミングのサポートをピン留め、 **KSPROPSETID\_VramCapture**プロパティ セット。 このセット内のすべてのプロパティは VRAM 画面のキャプチャとトランスポートでは必須です。

これらのプロパティがサポートされていない場合、画面キャプチャの既定値はキャプチャおよびトランスポート システム メモリに基づいています。

このセット内のプロパティ項目が KSPROPERTY によって指定された\_VIDMEM\_列挙の値をヘッダー ファイルで定義されているトランスポート*Ksmedia.h*。

**KSPROPSETID\_VramCapture**プロパティ セットには、次のプロパティが含まれています。

[**KSPROPERTY\_表示\_アダプター\_GUID**](ksproperty-display-adapter-guid.md)

[**KSPROPERTY\_優先\_キャプチャ\_画面**](ksproperty-preferred-capture-surface.md)

[**KSPROPERTY\_現在\_キャプチャ\_画面**](ksproperty-current-capture-surface.md)

[**KSPROPERTY\_マップ\_キャプチャ\_処理\_TO\_VRAM\_アドレス**](ksproperty-map-capture-handle-to-vram-address.md)
