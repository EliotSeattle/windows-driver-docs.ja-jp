# <a name="kspropsetidaudiodecoderout"></a>KSPROPSETID\_AudioDecoderOut


## <span id="ddk_kspropsetid_audiodecoderout_ks"></span><span id="DDK_KSPROPSETID_AUDIODECODEROUT_KS"></span>


KSPROPSETID\_AudioDecoderOut プロパティ セット、DVD デコーダーから使用可能なオーディオ出力形式を指定するプロパティを定義します。

KSPROPERTY\_AUDDECOUT 列挙体*Ksmedia.h*このセットのプロパティを指定します。

DVD デコーダー ミニドライバーは、次のオーディオ ストリームのプロパティのサポートを実装する必要があります。

[**KSPROPERTY\_AUDDECOUT\_モード**](ksproperty-auddecout-modes.md)

[**KSPROPERTY\_AUDDECOUT\_CUR\_モード**](ksproperty-auddecout-cur-mode.md)
