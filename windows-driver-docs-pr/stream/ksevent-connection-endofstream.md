# <a name="kseventconnectionendofstream"></a>KSEVENT\_接続\_ENDOFSTREAM


**KSEVENT\_接続\_ENDOFSTREAM**イベントにより、クライアントはストリームのイベントの終了の通知を受信します。 このイベントが発生した標準のストリーミングするときにヘッダーが使用されるときにヘッダーが、KSTREAM バッファー\_ヘッダー\_OPTIONSF\_ENDOFSTREAM 変数には、処理が完了します。 *レンダリング*ストリーム イベントの終了は、クライアントで検出できるように、ポイントがこのイベントをサポートする必要があります。
