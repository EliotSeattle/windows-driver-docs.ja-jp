# <a name="sample-bda-drivers"></a>サンプル BDA ドライバー

Windows 7 に付属している Windows Driver Kit (WDK) バージョン7600には、Broadcast Driver Architecture (BDA) の原則と機能、および BDA ミニドライバーが AVStream クラスドライバーを使用する方法を示すサンプルが含まれています。 これらのサンプルは、独自の BDA ミニドライバーを作成するための出発点として使用できるソースコードと共に実装されています。

WDK バージョン7600をインストールすると、これらのサンプルをインストールできます。 各 BDA ドライバーサンプルは、WDK インストールのメインディレクトリの下にある \\src\\Swtuner\\ サブディレクトリにインストールされます。 これらのサンプルの詳細については、「WDK の各 BDA ドライバーサンプルのソースコード」を参照してください。

これらのサンプルは、GitHub の[Windows ドライバーのサンプル](https://github.com/Microsoft/Windows-driver-samples)では使用できません。
