このセット内のイベントの項目がで指定された、 **KSEVENT\_VOLUMELIMIT**列挙体。

このセット内の唯一のイベントは[ **KSEVENT\_VOLUMELIMIT\_CHANGED**](ksevent-volumelimit-changed.md)します。
