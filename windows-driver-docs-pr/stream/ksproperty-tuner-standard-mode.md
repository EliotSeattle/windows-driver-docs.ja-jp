# <a name="kspropertytunerstandardmode"></a>KSPROPERTY\_チューナー\_標準\_モード


KSPROPERTY\_チューナー\_標準\_モード プロパティは、ドライバーが、チューニング自体の信号から標準のチューナーを自動的に検出するデバイスを設定できるかどうかを示すブール値を取得します。 このプロパティは、必要に応じて実装できます。
