# <a name="windows-driver-kit-documentation"></a>Windows Driver Kit のドキュメント

Windows Driver Kit (WDK) 10 は Microsoft Visual Studio 2017 および [Debugging Tools for Windows](/windows-hardware/drivers/debugger/index) と統合されています。この統合環境には、ドライバーの開発、構築、パッケージ化、展開、テスト、およびデバッグに必要なツールが用意されています。 統合環境内で、多くの基本的な認定テストを実行できます。 Windows Driver Kit (WDK) には、[Windows Driver Frameworks](https://docs.microsoft.com/windows-hardware/drivers/wdf/)、ユニバーサル シリアル バス (USB)、印刷、ネットワーク、ファイル システム フィルターを含む、いくつかのテクノロジとドライバー モデルのテンプレートが含まれています。

<p><b>WDK 10 の実行時の要件</b></p>

WDK には Visual Studio が必要です。 詳細については、「[Visual Studio 2017 のシステム要件](https://www.visualstudio.com/productinfo/vs2017-system-requirements-vs)」を参照してください。

EWDK にはさらに .NET 4.6.1 が必要です。 .NET の要件については、「[.NET Framework のシステム要件](https://docs.microsoft.com/dotnet/framework/get-started/system-requirements)」を参照してください。 

WDK Insider Preview と EWDK Insider Preview を使用すると、次のオペレーティング システムで動作するドライバーを開発できます。

|  クライアントの OS  |         サーバーの OS          |
|-------------|----------------------------|
| Windows 10  |    Windows Server 2016     |
| Windows 8.1 |   Windows Server 2012 R2   |
|  Windows 8  |    Windows Server 2012     |
|  Windows 7  | Windows Server 2008 R2 SP1 |
