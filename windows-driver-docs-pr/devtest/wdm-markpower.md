# <a name="markpower-rule-wdm"></a>MarkPower ルール (wdm)


**MarkPower**ルールを指定する IRP\_MJ\_IRP で電源\_MN\_設定\_用の POWER **SystemPowerState** IRP をS0 が適して保留します。 このルールは、FDO および FIDO ドライバーのみに適用されます。

|              |     |
|--------------|-----|
| ドライバー モデル | WDM |
