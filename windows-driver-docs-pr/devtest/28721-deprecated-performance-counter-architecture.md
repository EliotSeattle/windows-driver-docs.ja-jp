# <a name="c28721"></a>C28721


C28721 を警告します。非推奨のパフォーマンス カウンター アーキテクチャ

非推奨のパフォーマンス カウンターのアーキテクチャを使用: &lt;*現象*&gt;します。

コードでは、以前のパフォーマンス カウンター アーキテクチャを使用して、アプリケーションまたはコンポーネントのパフォーマンス メトリックを公開します。 Windows Vista の新しいパフォーマンス カウンター アーキテクチャが導入されました。 アプリケーションに (Windows Vista と Windows の以前のバージョンの両方で実行する必要があります)、同時の下位レベル要件がない限り、新しいアーキテクチャに移行する必要があります。
