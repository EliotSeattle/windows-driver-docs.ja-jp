# <a name="static-driver-verifier-report-panes"></a>静的ドライバー検証ツールのレポートのウィンドウ


静的ドライバー検証ツールのレポートは、次のペインで構成されます。

-   [トレースのツリー ペイン](trace-tree-pane.md)

-   [ソース コード ウィンドウ](source-code-pane.md)

-   [状態ウィンドウ](state-pane.md)

同時に、**トレース ツリー**ウィンドウで、**ソース コード**ウィンドウで、および**状態**ペイン フォーム、[欠陥ビューアー](defect-viewer.md)ウィンドウ。

SDV の「不具合」(ルール違反) が検出された場合にのみ、参加解除ビューアーを開くことができます。
