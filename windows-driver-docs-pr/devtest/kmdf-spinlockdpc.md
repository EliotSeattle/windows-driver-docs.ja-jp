# <a name="spinlockdpc-rule-kmdf"></a>SpinlockDpc ルール (kmdf)


**SpinlockDpc**への呼び出し規則を指定[ **KeAcquireSpinLock** ](https://docs.microsoft.com/windows-hardware/drivers/ddi/content/wdm/nf-wdm-keacquirespinlock)または[ **KeAcquireSpinLockRaiseToDpc**](https://docs.microsoft.com/previous-versions/windows/hardware/drivers/ff551928(v=vs.85))と[ **KeReleaseSpinlock** ](https://docs.microsoft.com/windows-hardware/drivers/ddi/content/wdm/nf-wdm-kereleasespinlock)厳密な代替構成で使用されます。

|              |      |
|--------------|------|
| ドライバー モデル | KMDF |
