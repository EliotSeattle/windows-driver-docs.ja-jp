# <a name="using-function-role-type-declarations"></a>関数役割型宣言の使用


Sdv は、関数である関数の役割の種類の宣言を使用して宣言します。

関数の宣言で使用する定義済みの typedef には、関数のロールの種類の識別、基になる KMDF ドライバー モデルまたはディスパッチ関数で、基になる WDM ドライバー モデルでのイベントのコールバック関数に対して定義されている個別のロールまたはStorport の NDIS ドライバー モデルで異なるコールバック関数。 関数の役割の型の宣言では、これらの関数の従来の宣言を置き換えます。

これらの宣言を追加する方法の詳細については、次を参照してください。

[KMDF ドライバーのロールの種類の関数を使用して関数を宣言します。](declaring-functions-by-using-function-role-types-for-kmdf-drivers.md)

[WDM ドライバーのロールの種類の関数を使用して関数を宣言します。](declaring-functions-using-function-role-types-for-wdm-drivers.md)

[NDIS ドライバーのロールの種類の関数を使用して関数を宣言します。](declaring-functions-by-using-function-role-types-for-ndis-drivers.md)

[Storport ドライバーのロールの種類の関数を使用して関数を宣言します。](declaring-functions-by-using-function-role-types-for-storport-drivers.md)
