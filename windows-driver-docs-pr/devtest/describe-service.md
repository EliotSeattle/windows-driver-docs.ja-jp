# <a name="describe-your-service-in-the-mobile-broadband-metadata-authoring-wizard"></a>モバイル ブロードバンド メタデータ作成ウィザードでのサービスの説明

Windows では、モデルの名前と製造元などのサービスの詳細については、わかりやすい情報を提供できます。

### <a name="to-provide-descriptive-information-about-a-service"></a>サービスに関する説明情報を提供するには

1. をクリックして、**説明**タブ。
2. 次のフィールドに記入します。
    - **サービス名**します。 この省略可能なフィールドは、Windows 8 では使用されません。
    - **サービス プロバイダー**します。 Windows 接続マネージャーに表示される演算子の名前。
    - **サービス数**します。 サービスを一意に識別する GUID を指定します。 この GUID は、演算子の XML のプロビジョニングを使用する場合、演算子を識別するために使用されます。 デバイス メタデータ パッケージを更新する場合は、この GUID は同じになります。
        **注**エクスペリエンスの ID とデバイス メタデータ パッケージのファイル名と異なる値になります。 サービス数の GUID を選択する方法の詳細については、次を参照してください。[モバイル ブロード バンド メタデータを提供するためのガイド](https://go.microsoft.com/fwlink/p/?linkid=242064)します。

- **説明 1**します。 この省略可能なフィールドは、Windows 8 では使用されません。
- **説明 2**します。 この省略可能なフィールドは、Windows 8 では使用されません。

メタデータ プロパティの詳細については、次を参照してください。[サービス メタデータ パッケージ スキーマ リファレンス for Windows 8](https://go.microsoft.com/fwlink/p/?LinkId=226755)します。
