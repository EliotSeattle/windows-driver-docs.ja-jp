# <a name="driver-verifier-manager-windows-xp-and-later"></a>ドライバー検証マネージャー (Windows XP 以降)


## <span id="ddk_driver_verifier_manager_windows_xp_and_later__tools"></span><span id="DDK_DRIVER_VERIFIER_MANAGER_WINDOWS_XP_AND_LATER__TOOLS"></span>


Windows XP およびそれ以降のバージョンのドライバー検証マネージャーは、5 つのタスクのいずれかを選択することができる画面から始まります。 選択すると、どのタスクが表示されるため、一連の画面を使用して移動できる、**戻る**と**次**ボタン。

このシーケンスが完了したらを使用して、**完了**ドライバー検証マネージャーを終了して行った変更を保存するボタンをクリックします。 変更を破棄する場合を使用して、**キャンセル**ボタンをクリックします。

ドライバー検証マネージャーの最初の画面から、次のタスクを選択できます。

<span id="Create_standard_settings_______"></span><span id="create_standard_settings_______"></span><span id="CREATE_STANDARD_SETTINGS_______"></span>**標準的な設定を作成します。**   
Driver Verifier のオプションの標準セットを選択し、し検証するドライバーを選択することができます。 これらの設定は、[次へ] の起動後に有効になります。 参照してください[ドライバー検証ツールのオプションの選択](selecting-driver-verifier-options.md)と[検証するドライバーを選択すると](selecting-drivers-to-be-verified.md)詳細についてはします。

<span id="Create_custom_settings_______"></span><span id="create_custom_settings_______"></span><span id="CREATE_CUSTOM_SETTINGS_______"></span>**カスタム設定を作成します。**   
すべての Driver Verifier のオプションし、することができます、表示は、これらの中から選択します。 次に、検証するドライバーを選択することを許可します。 これらの設定は、[次へ] の起動後に有効になります。 参照してください[ドライバー検証ツールのオプションの選択](selecting-driver-verifier-options.md)と[検証するドライバーを選択すると](selecting-drivers-to-be-verified.md)詳細についてはします。

<span id="Delete_existing_settings_______"></span><span id="delete_existing_settings_______"></span><span id="DELETE_EXISTING_SETTINGS_______"></span>**既存の設定を削除します。**   
すべてのドライバー検証マネージャーのオプションを非アクティブ化し、検証されているドライバーの一覧を削除します。 これは、[次へ] の起動後に有効です。

<span id="Display_existing_settings_______"></span><span id="display_existing_settings_______"></span><span id="DISPLAY_EXISTING_SETTINGS_______"></span>**既存の設定を表示します。**   
アクティブにされるオプションと [次へ] の起動後に確認されるドライバーを表示します。 これは、揮発性の設定には含まれません。 参照してください[ドライバー検証ツールの設定を表示する](viewing-driver-verifier-settings.md)詳細についてはします。

<span id="Display_information_about_the_currently-verified_drivers_______"></span><span id="display_information_about_the_currently-verified_drivers_______"></span><span id="DISPLAY_INFORMATION_ABOUT_THE_CURRENTLY-VERIFIED_DRIVERS_______"></span>**現在検証ドライバーに関する情報を表示します。**   
これにより、Driver Verifier のアクションに関連する統計情報が表示されます。 参照してください[グローバル カウンターの監視](monitoring-global-counters.md)と[カウンターを個別に監視](monitoring-individual-counters.md)詳細についてはします。 表示、Driver Verifier の揮発性のアクションを変更すること許可されます。 参照してください[揮発性の設定を使用して](using-volatile-settings.md)します。
