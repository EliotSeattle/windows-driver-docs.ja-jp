# <a name="spchangeirql-rule-storport"></a>SpChangeIrql ルール (storport)


このルールは、呼び出される位置のレベルと同じ IRQL レベル、StorPort コールバック ルーチンを返すことを確認します。

Storport ミニポート エントリ ポイントまたはコールバック ルーチン内で IRQL レベルを変更する任意の DDI に適用されます。

|              |          |
|--------------|----------|
| ドライバー モデル | Storport |
