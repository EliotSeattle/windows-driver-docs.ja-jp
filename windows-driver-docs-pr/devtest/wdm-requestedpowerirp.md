# <a name="requestedpowerirp-rule-wdm"></a>RequestedPowerIrp ルール (wdm)


**RequestedPowerIrp**ルールでは、そのドライバーは呼び出しを指定します[ **PoRequestPowerIrp** ](https://docs.microsoft.com/windows-hardware/drivers/ddi/content/wdm/nf-wdm-porequestpowerirp)で、`*Irp`ポインター変数を設定**NULL**.

ポインターがによって返されるため、 [ **PoRequestPowerIrp** ](https://docs.microsoft.com/windows-hardware/drivers/ddi/content/wdm/nf-wdm-porequestpowerirp)が信頼できないドライバーしたりしないでください取得この値を使用します。
