# <a name="saving-a-filter"></a>フィルターを保存しています


フィルターを保存する特定のプロシージャはありません。 代わりに、ワークスペースで、トレース セッションを保存できます。 フィルターは、ワークスペースの一部として保存されます。 ワークスペースを保存するには。

1.  [トレース セッション リスト](trace-session-list.md)セッションで、任意のセルを右クリックし、クリックして**ワークスペースの保存**します。

2.  ページの下部で、**ワークスペース**ボックスで、ワークスペースの名前を入力し、をクリックして**OK**。

ワークスペースで、フィルターを保存するときに、保存するファイルを上書きしないように注意します。 ワークスペースを開く場合、トレース ログのパスとファイル名が、ワークスペースに保存されているため、既存のトレース ログ ファイルは上書きされます。 詳細については、次を参照してください。 [traceview でワークスペースを使用して](using-traceview-workspaces.md)します。
