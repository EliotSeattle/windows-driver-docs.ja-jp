# <a name="importing-certificates-from-a-windows-certificate-store"></a>Windows 証明書ストアからの証明書のインポート


記憶域証明書の管理の強化されたツールを使用すると、IEEE 1667 準拠 USB 記憶装置に、コンピューターで、プライベート証明書ストアから証明書をインポートします。

プライベート証明書ストアから証明書をインポートするを使用して証明書の名前を指定する必要があります、 **-格納**のパラメーター、 [ **/add** ](enhstor-add-switch.md)と[**置換/** ](-replace-switch.md)拡張記憶域証明書管理ツールのスイッチ。 ツールは、指定された証明書のプライベート証明書ストアのすべてを検索し、(場合が見つかりました) 指定された USB 記憶装置にインポートされます。

**注**  ツールが、Windows ルート証明書ストアから証明書をインポートできません。
