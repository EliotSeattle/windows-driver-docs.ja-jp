# <a name="c28645"></a>C28645


C28645 を警告します。これは推奨されなく疑問符のメッセージ シンボルを使用して MessageBox が呼び出されました

これは明らかに表しません特定の種類のメッセージ、およびメッセージの言い回し質問でした、メッセージ型に適用されないために、モーダル ダイアログ ボックスにある疑問符のメッセージ アイコンの使用は不要になったお勧めします。 さらに、ユーザーはヘルプ情報をメッセージのシンボルの疑問符を混同することができます。

したがって、メッセージ ボックスではこの疑問符のメッセージ シンボルを使用しないでください。 旧バージョンとの互換性のための組み込みをサポートするために、システムが続行されます。
