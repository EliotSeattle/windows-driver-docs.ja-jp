# <a name="c28266"></a>C28266


関数内のプロパティの注釈で構文エラーが見つかりました。

コード分析ツールは、注釈で構文エラーを検出した場合に、このエラーを報告します。 この警告は、注釈で、分析するコードではなくエラーを示します。
