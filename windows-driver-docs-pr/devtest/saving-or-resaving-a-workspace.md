# <a name="saving-or-resaving-a-workspace"></a>ワークスペースの保存または再保存

ワークスペースを保存するときに、traceview では、トレース セッションと、トレース メッセージの表示の現在の構成を保存します。 新しいワークスペースを保存したり、新しいプロパティを持つ既存のワークスペースを再度保存します。

### <a name="span-idtosaveaworkspacespanspan-idtosaveaworkspacespanto-save-a-workspace"></a><span id="to_save_a_workspace"></span><span id="TO_SAVE_A_WORKSPACE"></span>ワークスペースを保存するには

1.  [トレース セッション リスト](trace-session-list.md)トレース セッションを表す行の任意のセルを右クリックし、またはトレース ログの表示。

2.  クリックして**ワークスペースを保存**します。

3.  **新しいワークスペース**ボックスで、ワークスペースの名前を入力し、をクリックして**OK**します。

### <a name="span-idtoupdateasavedworkspacespanspan-idtoupdateasavedworkspacespanto-update-a-saved-workspace"></a><span id="to_update_a_saved_workspace"></span><span id="TO_UPDATE_A_SAVED_WORKSPACE"></span>保存したワークスペースを更新するには

1.  [トレース セッション リスト](trace-session-list.md)トレース セッションを表す行の任意のセルを右クリックし、またはトレース ログの表示。

2.  クリックして**ワークスペースを保存**します。

3.  **ワークスペースの保存**一覧で既存のワークスペースの名前を選択してクリックして**OK**。
