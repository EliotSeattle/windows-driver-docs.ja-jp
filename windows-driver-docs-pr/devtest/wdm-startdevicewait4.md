# <a name="startdevicewait4-rule-wdm"></a>StartDeviceWait4 ルール (wdm)


**StartDeviceWait4**規則は、ドライバーが START device IRP のコンテキストで[**KeWaitForSingleObject**](https://docs.microsoft.com/windows-hardware/drivers/ddi/wdm/nf-wdm-kewaitforsingleobject)を呼び出さないように指定します。

|              |     |
|--------------|-----|
| ドライバー モデル | WDM |
