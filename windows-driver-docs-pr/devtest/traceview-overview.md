# <a name="traceview-overview"></a>TraceView の概要

Traceview でのすべての機能を実行する 1 つのツールを使用して多くの従来型の単一関数のコマンド ライン トレース ツールを置換するもの、[トレース コント ローラー](trace-controller.md)と[トレース コンシューマー](trace-consumer.md)で、便利なグラフィック ユーザー インターフェイスです。

Traceview で、トレース セッションを開始でき、個別の書式設定手順を使用せず、すぐにリアルタイムのトレース セッションとトレース ログからのトレース メッセージを表示できます。 すべてのコントロールの Guid を含む、プロバイダーに関する必要な情報を抽出し、プロバイダーの PDB シンボル ファイルは、いくつかの手順を排除することから、データの書式設定するメッセージをトレースできます。

Traceview でウィンドウは、簡単に使用し、検索またはコマンド ライン パラメーターを記憶するを解放します。

Traceview でのトレース メッセージが表示されます、[トレース メッセージ一覧](trace-message-lists.md)ウィンドウ、表示、フィルター、およびトレース メッセージを簡単に並べ替えることができます。 また、ウィンドウから直接、トレース メッセージをコピーしてテキスト ファイル、スプレッドシート ファイル、またはデータベース ファイルを保存したり、後で処理に貼り付けます。
