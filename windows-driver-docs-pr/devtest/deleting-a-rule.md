# <a name="deleting-a-rule"></a>規則の削除


フィルターからルールを削除します。

1.  [トレース セッション リスト](trace-session-list.md)セッションで、任意のセルを右クリックし、クリックして**フィルターの管理**します。

2.  **フィルターの管理** ダイアログ ボックスで、**現在ルール**領域で、削除する規則を選択します。

3.  クリックして**Deleterule**します。

内のルールを保存した場合、[ワークスペース](using-traceview-workspaces.md)、する必要があります[ワークスペースを再度保存](saving-or-resaving-a-workspace.md)ルールを削除した後。 それ以外の場合、このルールは、ワークスペースからは削除されません。
