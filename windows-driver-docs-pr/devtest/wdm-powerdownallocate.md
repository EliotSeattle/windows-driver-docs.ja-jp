# <a name="powerdownallocate-rule-wdm"></a>PowerDownAllocate ルール (wdm)


**Powerdownallocate**ルールでは、FDO および FIDO ドライバが\_s0 から \[S1 に向かう**SystemPowerState**遷移に対して\_設定された IRP\_処理するときにメモリを割り当てないことを指定します。S5\]。

このルールは、FDO および FIDO ドライバーにのみ適用されます。

|              |     |
|--------------|-----|
| ドライバー モデル | WDM |
