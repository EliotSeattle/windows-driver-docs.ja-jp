# <a name="stopping-and-removing-a-workspace-session"></a>ワークスペース セッションの停止と削除

ワークスペースを開くときに、新しいトレース セッションまたは新しいログの表示を開始します。 セッションを停止して、任意のトレース セッションの場合と同様に、表示から削除するには、同じ手順を使用します。

### <a name="span-idtostopaworkspacesessionspanspan-idtostopaworkspacesessionspanto-stop-a-workspace-session"></a><span id="to_stop_a_workspace_session"></span><span id="TO_STOP_A_WORKSPACE_SESSION"></span>ワークスペースのセッションを停止するには

1.  [トレース セッション リスト](trace-session-list.md)トレース セッションを表す行の任意のセルを右クリックし、またはトレース ログの表示。

2.  クリックして**トレースを停止**します。

### <a name="span-idtoremoveaworkspacesessionspanspan-idtoremoveaworkspacesessionspanto-remove-a-workspace-session"></a><span id="to_remove_a_workspace_session"></span><span id="TO_REMOVE_A_WORKSPACE_SESSION"></span>ワークスペースのセッションを削除するには

1.  ワークスペースのセッションを停止します。

2.  [トレース セッション リスト](trace-session-list.md)トレース セッションを表す行の任意のセルを右クリックし、または停止したログの表示をトレースします。

3.  クリックして**ログ セッションを削除**します。
