# <a name="pnpcpu-overview"></a>PNPCPU の概要


PNPCPU は、次の機能を実行するコマンド ライン ツールです。

-   インストール
    -   ツールをインストールするには、Pnpcpu.exe でを実行、 **-インストール**オプション。
    -   Pnpcpu では、関連するすべてのドライバーをインストールします。
    -   Pnpcpu は、適切なパラメーターを使用して、ブート構成データ ストアを更新します。
-   Add
    -   ホットに PNPCPU 試行は、インストールされているエディションのライセンスでサポートされている最大まで、システムですべての論理プロセッサを追加します。
-   削除
    -   ツールを削除するには、実行と Pnpcpu.exe、 **-アンインストール**オプション。 これは、結果、によって実行されるすべての手順の完全な元に戻す **-インストール**します。
    -   このオプションは、後続の再インストールと使用のためのディスクにバイナリを残します。
