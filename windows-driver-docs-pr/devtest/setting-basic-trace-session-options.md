# <a name="setting-basic-trace-session-options"></a>トレース セッションの基本オプションの設定

## <span id="ddk_setting_basic_trace_session_options_tools"></span><span id="DDK_SETTING_BASIC_TRACE_SESSION_OPTIONS_TOOLS"></span>

オプションを設定する基本的なトレース セッション トレース セッションを作成するときにします。 オプションが表示される、**ログ セッション オプション**ページ。

## <a name="how-to-set-basic-trace-session-options"></a>トレース セッションの基本オプションの設定方法

この手順を使用すると、トレース セッションを作成するときに基本的なトレース セッションのオプションを設定できます。

1.  **ファイル** メニューのをクリックして**Create New Log Session**します。

2.  クリックして**プロバイダーの追加**プロバイダーを選択します。 必要に応じて、TMF ファイルまたは TMF ファイルのパスを指定します。

3.  追加のプロバイダーを追加するには、クリックして**プロバイダーの追加**します。 それ以外の場合、をクリックして**次**します。

4.  **ログ セッション オプション** ページで、基本的なトレース セッションのオプションを設定します。

基本的なトレース セッションのオプションの詳細については、次を参照してください。[トレース セッション オプションの基本的な](basic-trace-session-options.md)します。

高度なオプションについては、次を参照してください。[トレース セッション オプションの高度な設定](setting-advanced-trace-session-options.md)します。
