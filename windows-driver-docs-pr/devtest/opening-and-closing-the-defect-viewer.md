すべてのルールのトレース ビューアーを開くことができます、**結果**欠陥を報告するタブ。

トレース ビューアーを開きます。

-   [結果] タブでは、興味のあるをクリックした規則を探して**欠陥**します。

-   SDV が、必要な情報を取得する簡単な遅延の後、トレース ビューアーを開きます。 次に、拡張の遅延中に配置し、windows のサイズを変更します。 1 分以上の遅延は、通常と見なされます。

(1 つの規則の違反) の 1 つだけのトレース ビューアーは、一度に表示できます。 別の規則違反のトレース ビューアーを開くには、現在のルール違反のトレース ビューアーを閉じてください。

欠陥のビューアーを閉じるには、次のいずれかの操作を行います。

-   **ファイル**メニューの **終了**します。
