# <a name="initial-view"></a>初期ビュー


Traceview でウィンドウを開くときに、[トレース セッション リスト](trace-session-list.md)が表示されたらは空です。 次のスクリーン ショットは、空の traceview でウィンドウを示しています。

![空の traceview では、ウィンドウのスクリーン ショット](images/traceview-empty.png)

このビューを表示するときに、いずれかの行を右クリックしてまたはを使用して、**ファイル**メニュー[新しいトレース セッションを作成](creating-a-trace-session.md)、[トレース ログを表示](displaying-a-trace-log.md)、 [ワークスペースを開く](opening-a-workspace.md)、または[ワークスペースを削除する](deleting-a-workspace.md)します。
