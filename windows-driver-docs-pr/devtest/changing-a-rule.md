# <a name="changing-a-rule"></a>規則の変更


フィルターのルールを変更するには。

1.  [トレース セッション リスト](trace-session-list.md)セッションで、任意のセルを右クリックし、クリックして**フィルターの管理**します。

2.  **フィルターの管理** ダイアログ ボックスで、**現在ルール**領域で、変更するルールを選択します。

3.  ルールを適用する順序を変更する をクリックして、**上へ移動**または**下へ移動**ボタン。

4.  規則の要素を変更する、**ルールを変更または追加された**領域で、変更、[要素ルールの](filter-rule-elements.md)します。

5.  クリックして**規則を変更する**します。

内のルールを保存した場合、[ワークスペース](using-traceview-workspaces.md)、する必要があります[ワークスペースを再度保存](saving-or-resaving-a-workspace.md)ルールを変更した後。 それ以外の場合、元のルールは、ワークスペース内に残ります。
