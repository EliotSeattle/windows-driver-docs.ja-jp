# <a name="enhanced-storage-certificate-management-tool"></a>Enhanced Storage Certificate Management Tool


記憶域証明書の管理の強化されたツール (EhStorCertMgrCmd.exe) では、IEEE 1667 標準に準拠している USB ストレージ デバイスに証明書を管理します。 このツールは、次のグループで使用できます。

-   USB ストレージ デバイスの開発者にテストし、IEEE 1667 標準への準拠を確認します。

-   提供し、証明書を管理する IT 管理者は、IEEE 1667 準拠 USB ストレージ デバイスの企業環境で使用される格納します。
