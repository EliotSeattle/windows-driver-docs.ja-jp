# <a name="saving-a-group-log"></a>グループ ログの保存


トレース セッションのグループのメンバーシップでは、イベントのトレース ログ (.etl) ファイルまたは (.out) ファイルを一覧表示、グループ内のトレース セッションの traceview でのコンテンツは影響しません。 常に 1 つだけのトレース セッションに関するデータを記録します。

表示されるように、グループのメッセージのレコードを作成する、[トレース メッセージ一覧](trace-message-lists.md)、トレース メッセージの一覧のグループからメッセージをコピーして、テキストやスプレッドシート ファイルなど、保存できるドキュメントに貼り付けます。

トレースの到着にメッセージがあるため、[トレース セッション リスト](trace-session-list.md)トレース セッションごとに、可能性がありますが実際に到着して別のセッションからのトレース メッセージのメッセージ率、バッファー サイズ、およびフラッシュ タイマーによって影響を受けるまったく異なるタイミングで発生しました。 結合のトレース メッセージのより正確な表示は、タイムスタンプで並べ替えたりする (で、**システム時刻**列) の各メッセージ。
