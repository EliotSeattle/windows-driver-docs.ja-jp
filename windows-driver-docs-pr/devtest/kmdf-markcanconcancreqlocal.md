# <a name="markcanconcancreqlocal-rule-kmdf"></a>MarkCancOnCancReqLocal ルール (kmdf)


**MarkCancOnCancReqLocal**ルールを指定する[ **WdfRequestMarkCancelable** ](https://docs.microsoft.com/windows-hardware/drivers/ddi/content/wdfrequest/nf-wdfrequest-wdfrequestmarkcancelable)メソッドでは呼び出せません 2 つの連続する時間と同じ I/O 要求。

**MarkCancOnCancReqLocal**ルールは、既定の I/O キューのコールバック関数内でのみこのチェックを実行します。

|              |      |
|--------------|------|
| ドライバー モデル | KMDF |
