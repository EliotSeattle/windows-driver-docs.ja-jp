# <a name="ungrouping-trace-sessions"></a>トレース セッションのグループ化解除


トレース セッションのグループをグループ化を解除します。

1.  [実行中のすべてのトレース セッションを停止する](stopping-a-trace-session.md)グループ。

2.  [トレース セッション リスト](trace-session-list.md)、グループ化されたセッションで任意のセルを右クリックします。

3.  クリックして**セッションをグループ解除**します。

停止したトレース セッションのグループを解除するときに再実行を開始します。 トレース セッションを削除するには、それらを停止し、[削除](removing-a-trace-session.md)します。
