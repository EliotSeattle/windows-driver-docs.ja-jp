# <a name="specify-autoplay-handler-and-options-in-the-device-metadata-authoring-wizard"></a>デバイス メタデータ作成ウィザードでの自動再生ハンドラーとオプションの指定


切断された状態と自動再生ハンドラーなど、デバイスのオプションを指定する をクリックして、**オプション**タブ。

## <a name="span-iddisconnectedstatespanspan-iddisconnectedstatespanspan-iddisconnectedstatespandisconnected-state"></a><span id="Disconnected_State"></span><span id="disconnected_state"></span><span id="DISCONNECTED_STATE"></span>切断された状態


切断された状態で表示するデバイスを有効にするには、**切断状態で表示するデバイス**します。

## <a name="span-idautoplayhandlerspanspan-idautoplayhandlerspanspan-idautoplayhandlerspanautoplay-handler"></a><span id="AutoPlay_Handler"></span><span id="autoplay_handler"></span><span id="AUTOPLAY_HANDLER"></span>自動再生ハンドラー


デバイスが接続されているときに自動再生のオプションの一覧表示するデバイス用の UWP アプリを自動再生ハンドラーにできます。 アプリは自動的に起動しません。

次のオプションから選択したデバイスの自動再生ハンドラーを定義するには、**自動再生ハンドラー (省略可能)** します。

-   自動再生ハンドラーを定義しない場合は、選択**None**します。
-   次のフィールドに入力して、自動再生ハンドラーを定義する**自動再生ハンドラー**:
    -   **パッケージ名**
    -   **パブリッシャー**
    -   **アプリ ID**
    -   **動詞**
    -   **自動再生の種類**
        -   一覧から、次のように選択します。**デバイス**または**コンテンツ**します。
-   フィールドに入力すると、デスクトップ アプリの自動再生ハンドラーを定義するには、**デスクトップの自動再生ハンドラー**します。
    **注**  自動再生ハンドラーとデスクトップの自動再生ハンドラーの両方を定義することはできません。
