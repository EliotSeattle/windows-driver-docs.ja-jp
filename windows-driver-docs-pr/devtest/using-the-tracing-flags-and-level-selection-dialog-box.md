# <a name="using-the-tracing-flags-and-level-selection-dialog-box"></a>[Tracing Flags and Level Selection]\(トレース フラグとレベルの選択\) ダイアログ ボックスの使用


**トレース フラグとレベル選択** ダイアログ ボックスで、 **G** (GUID) のアイコンは、トレース プロバイダーを表します。 ファイル名の横にあるこのアイコンが表示されます、[トレース メッセージのコントロール (.tmc) ファイル](trace-message-control-file.md)トレース プロバイダー。 プロバイダーのトレース レベルおよびトレース フラグを選択するには、適切な検索**G**アイコンと、次の操作を行います。

-   を表示またはプロバイダーのトレース レベルを変更するには、次のいずれかの操作を行います。
    -   ダブルクリックして、**レベル**プロバイダーの項目し、プロバイダーに必要なレベルを選択します。 (1 つだけのレベルを選択できます)。
    -   右クリックし、**レベル**項目をポイントして、**レベル**、プロバイダーに必要なレベルをクリックします。
-   を表示またはプロバイダーのフラグを変更するには、次のいずれかの操作を行います。
    -   ダブルクリックして、**フラグ**プロバイダーに対して項目し選択するか、必要なフラグのチェック ボックスをオフにします。 (複数のフラグを選択できます。)
    -   右クリックし、**フラグ**項目をクリックして**すべてフラグの設定**または**すべてフラグをクリア**します。
-   変更を保存するには、次のようにクリックします。 **OK**します。
