# <a name="changing-the-text-size-in-the-defect-viewer"></a>欠陥ビューアーのテキスト サイズの変更


欠陥ビューアー] ウィンドウでテキストのサイズを変更する、**ビュー**メニューの [**小**または**大**、または F1 (小さい) または F2 (より大きい) キーを押します。 最小値または最大サイズに達するまで、テキストのサイズを増減する繰り返し押すことができます。

この機能では、欠陥ビューアーのみに影響します。 結果ウィンドウでテキストのサイズは変わりません。
