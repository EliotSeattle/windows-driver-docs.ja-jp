# <a name="c28268"></a>C28268


C28268 を警告します。関数の関数クラスが、ここで使用された typedef の関数クラスと一致しません

この警告は、関数のクラスに注釈が型の宣言で指定したとおり、関数の種類と一致しないことを示します。 この警告は、注釈で、分析するコードではなくエラーを示します。
