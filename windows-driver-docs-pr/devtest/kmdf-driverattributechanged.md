# <a name="driverattributechanged-rule-kmdf"></a>Driver属性 (kmdf)


**Driverの echanged**ルールでは、ドライバーが kmdf ドライバーの実行レベルまたは同期スコープを変更してはならないことを指定します。

|              |      |
|--------------|------|
| ドライバーモデル | KMDF |
