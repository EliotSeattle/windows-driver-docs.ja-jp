# <a name="driverattributechanged-rule-kmdf"></a>DriverAttributeChanged ルール (kmdf)


**DriverAttributeChanged**ルールでは、ドライバーは、KMDF ドライバーの実行レベルまたは同期スコープを変更できないことを指定します。

|--------------|------|
| ドライバー モデル | KMDF |
