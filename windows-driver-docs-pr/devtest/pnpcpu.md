# <a name="pnpcpu"></a>PNPCPU


PNPCPU (Pnpcpu.exe) は、Windows Server 2008 のインスタンスを実行するプロセッサのホット追加をシミュレートするコマンド ライン ツールです。

ドライバーとアプリケーション開発者は、このツールを使用して、ドライバーまたはアプリケーションが失敗しないようにプロセッサがシステムの実行時に追加されたときにテストすることができます。 機能の場合、ドライバーまたはアプリケーションが拡張されて、ホット追加プロセッサを使用するよう、PNPCPU がすべての関連するプラグ アンド プレイの通知が正しく処理されることを確認することできます。

PNPCPU では、Windows Vista WDK 以降に付属し、Windows Server 2008 で実行されます。 このツールを使用するには、Administrators グループのメンバーがあります。
