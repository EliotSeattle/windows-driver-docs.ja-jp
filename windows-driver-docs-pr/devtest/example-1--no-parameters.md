# <a name="example-1-no-parameters"></a>例 1:パラメーターなし

Tracefmt には、必要なパラメーターがありません。 そのため、最も簡単な Tracefmt コマンドのとおりです。

```
tracefmt
```

Tracefmt c: 既定のトレース メッセージの形式に応じて、\\% トレースで指定されたパスで TMF ファイルを使用してファイルを logfile.etl\_形式\_検索\_%path% 環境変数。 Fmtfile.txt という概要メッセージ ファイルがローカル ディレクトリに Fmtfile.txt.sum をという名前の出力ファイルを作成します。
