# <a name="how-to-set-basic-trace-session-options"></a>基本的なトレース セッションのオプションを設定する方法


この手順を使用すると、トレース セッションを作成するときに基本的なトレース セッションのオプションを設定できます。

1.  **ファイル** メニューのをクリックして**Create New Log Session**します。

2.  クリックして**プロバイダーの追加**プロバイダーを選択します。 必要に応じて、TMF ファイルまたは TMF ファイルのパスを指定します。

3.  追加のプロバイダーを追加するには、クリックして**プロバイダーの追加**します。 それ以外の場合、をクリックして**次**します。

4.  **ログ セッション オプション** ページで、基本的なトレース セッションのオプションを設定します。
