# <a name="how-to-set-advanced-trace-session-options"></a>高度なトレース セッションのオプションを設定する方法


トレース セッションを作成するときに、高度なトレース セッションのオプションを設定するのにには、次の手順を使用します。

1.  **ファイル** メニューのをクリックして**Create New Log Session**します。

2.  クリックして**プロバイダーの追加**プロバイダーを選択します。 必要に応じて、TMF ファイルまたは TMF ファイルのパスを指定します。

3.  複数のプロバイダーを追加するには、次のようにクリックします。**プロバイダーの追加**します。 この手順は省略可能です。

4.  **[次へ]** をクリックします。

5.  **ログ セッション オプション** ページで、必要な場合は、基本的なトレース セッションのオプションを設定します。

6.  をクリックして、**ログ セッション オプションの高度な**ボタンをクリックします。
