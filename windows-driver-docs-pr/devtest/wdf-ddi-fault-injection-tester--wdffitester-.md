# <a name="wdf-ddi-fault-injection-tester-wdffitester"></a>WDF DDI フォールト インジェクション テスト担当者 (WdfFiTester)


WdfFiTester では、WMI インターフェイスを使って、KMDF デバイス ドライバー インターフェイス (DDI) 関数の呼び出しにフォールト インジェクションを構成します。 WdfFiTester では、このような挿入されたエラーを正しく処理する、KMDF ドライバーの機能をテストします。
