# <a name="the-verifier-utility"></a>検証ツールのユーティリティ


## <span id="ddk_the_verifier_utility_tools"></span><span id="DDK_THE_VERIFIER_UTILITY_TOOLS"></span>


検証ユーティリティ、Verifier.exe は、アクティブ化して、Driver Verifier を監視します。 このユーティリティは、%windir% にある\\Windows の system32 ディレクトリ。

検証ツール ユーティリティを使用して 2 つの方法はあります。 Windows Vista と Windows の以降のバージョンでは、管理者特権が必要です。

-   **コマンド ライン。** コマンド プロンプト ウィンドウを開き、コマンド ラインで、Driver Verifier を使用する (**管理者として実行**)、型**verifier**コマンド ライン パラメーターを 1 つ以上続きます。

-   **ドライバー検証マネージャー。** ドライバー検証マネージャーのグラフィック ユーザー インターフェイスを開くには、コマンド プロンプト ウィンドウを開きます (**管理者として実行**)、型**verifier** (パラメーターなし) を使用します。

**注**  バージョンの Windows 2000 でのドライバー検証マネージャーから Windows XP 以降のドライバー検証マネージャーが若干異なります。
