# <a name="application-verifier"></a>アプリケーション検証ツール


Application Verifier (AppVerif.exe) は、*動的検証*ユーザー モード アプリケーション用のツール。 このツールは、アプリケーションの実行、アプリケーションにさまざまな負荷とテストをアプリケーションの実行または設計で潜在的なエラーに関するレポートを生成します。 アプリケーションのアクションを監視します。

Application Verifier は、ユーザー モード ドライバーなど、マネージ コードに基づいていない任意のユーザー モード アプリケーション内のエラーを検出できます。 標準的なアプリケーションのテストまたはドライバー テスト中に検出を困難に可能性のあるプログラミング エラーを検出します。

ユーザー モード デバッガーを使用して単独でまたは組み合わせて Application Verifier を使用できます。 このツールは、Microsoft Windows XP 以降のバージョンの Windows で動作します。 現在のユーザーは、コンピューターの Administrators グループのメンバーである必要があります。

Application Verifier に含まれている、 [Windows ソフトウェア開発キット]( https://go.microsoft.com/fwlink/p/?linkid=309790)(SDK) デスクトップ アプリから SDK をダウンロードする場合は個別のダウンロード可能なコンポーネントとして使用し[デスクトップ開発者向けダウンロードアプリ]( https://go.microsoft.com/fwlink/p/?linkid=309790)します。 ツール、独自のドキュメントが付属します。 Application Verifier を開始、マニュアルを読みとの間、**ヘルプ** メニューのをクリックして**ヘルプ**、または F1 キーを押します。
