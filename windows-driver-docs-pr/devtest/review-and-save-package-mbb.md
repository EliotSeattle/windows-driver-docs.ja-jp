# <a name="review-and-save-package-in-the-mobile-broadband-metadata-authoring-wizard"></a>確認して、モバイル ブロード バンド メタデータの作成ウィザードでパッケージを保存します。


すべてのタブで、ウィザードを完了後に確認し、パッケージを保存します。

## <a name="span-idtoreviewapackagespanspan-idtoreviewapackagespanspan-idtoreviewapackagespanto-review-a-package"></a><span id="To_review_a_package"></span><span id="to_review_a_package"></span><span id="TO_REVIEW_A_PACKAGE"></span>パッケージを確認するには


1.  をクリックして、**完了**タブ。
2.  すべての下に表示される情報の確認**概要情報**します。
3.  表示されるエクスペリエンスの ID を記録します。 StoreManifest.xml、Microsoft Store に送信された最終的な UWP アプリで含める必要があるファイルでは、エクスペリエンスの ID 情報を指定する必要があります。
4.  該当する場合は、赤のエラーを確認し、適切なタブでそれらを修正します。

## <a name="span-idtosaveapackagespanspan-idtosaveapackagespanspan-idtosaveapackagespanto-save-a-package"></a><span id="To_save_a_package"></span><span id="to_save_a_package"></span><span id="TO_SAVE_A_PACKAGE"></span>パッケージを保存するには


1.  **完了**] タブの [**名前を付けて保存**、パッケージ名の横に確認**Windows 8 パッケージ**します。
2.  横に**フォルダーの場所**、保存されたパッケージには、コンピューター上の場所を確認します。 または、 をクリック**変更**を変更します。
3.  お使いの PC にパッケージをテストする場合は、選択**システムのローカル メタデータ ストアにパッケージをコピー**します。
4.  **[Finish]**(完了) をクリックします。
