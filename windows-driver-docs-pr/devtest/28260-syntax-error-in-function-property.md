# <a name="c28260"></a>C28260


C28260 を警告します。関数内でのプロパティの解析中に、注釈で構文エラーが見つかりました

この警告は、注釈で、分析するコードではなくエラーを示します。 メッセージのテキストで説明されているテキストは、低レベルの注釈は、通常は、マクロとしてエンコードから抽出されます。 通常、ただし、構文エラーは、マクロの 1 つのパラメーターを式で。 構文エラーの通常の場所は、注釈付きの関数を含むヘッダー ファイルです。
