# <a name="c28736"></a>C28736

C28736 を警告します。禁止されている API 引数の使用方法

この警告は、従来のイベント ログまたは他のトレース API、または禁止されている引数を指定して、Perflib コンシューマー API が使用されていることを示します。 両方のパフォーマンスとセキュリティを向上させる、Windows のイベント トレース モデルで、これらの Api と引数が交換されています。
