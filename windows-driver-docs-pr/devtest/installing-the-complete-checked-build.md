# <a name="installing-the-complete-checked-build"></a>完全なチェック ビルドをインストールします。


## <span id="ddk_installing_the_complete_checked_build_tools"></span><span id="DDK_INSTALLING_THE_COMPLETE_CHECKED_BUILD_TOOLS"></span>


おそらくドライバーのデバッグのチェック ビルドを使用する最も簡単な方法は、完了チェック ビルドをコンピューターにインストールするは。 チェックのビルドをインストールするのにには、チェック ビルド配布メディアを使用します。 使用可能なすべてのチェックされているコンポーネントがコンピューターにインストールされます。

このインストールは、最大のデバッグ チェックで結果しますが、も、システムが、オペレーティング システムの無料のビルドがインストールされている場合よりもかなり低速に実行します。
