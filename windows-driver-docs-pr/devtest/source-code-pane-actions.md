# <a name="source-code-pane-actions"></a>[Source Code]\(ソース コード\) ウィンドウの操作


次の操作を行うことができます、**ソース コード**ウィンドウ。

-   **ソース コード ファイルを読み取り**

    ソース コード ファイルを読み取り、[ファイル] タブをクリックし、ファイルを移動する、マウスまたはキーボードを使用してください。 行を選択するか強調表示を移動すると、**ソース コード** ウィンドウで、対応するコード要素、[トレース ツリー ウィンドウ](trace-tree-pane.md)が強調表示されますおよび内の値、[状態ウィンドウ](state-pane.md)規則検証プロセスで、その時点での変数の値を反映するように変更します。

-   **ソース コード ファイルを切り替える**

    現在表示されていないソース コード ファイルを表示するには、そのファイルのタブをクリックします。 ファイルは、ファイル履歴の先頭に移動します。 タブをクリックした後、スタックの一番上にファイルを移動、右矢印と左矢印キーを使用できます。

-   **ソース コード ファイルをコピーします。**

    任意のタブからテキストをコピーすることができます、**ソース コード**ウィンドウと Microsoft Word やワードパッドなどのリッチ テキスト形 (式 RTF) をサポートするアプリケーションにファイルを貼り付けます。

    ソース コード ウィンドウで任意のタブに表示されているコードをコピーするには。

    1.  内をクリックし、**ソース コード**ウィンドウのタブ。
    2.  **編集**メニューの **すべて選択**(または ctrl キーを押しながら A キーを押します)。
    3.  **編集**メニューの **コピー** (または ctrl キーを押しながら C キーを押します)。
