# <a name="poolmon-commands"></a>PoolMon コマンド


## <span id="ddk_poolmon_commands_tools"></span><span id="DDK_POOLMON_COMMANDS_TOOLS"></span>


PoolMon には、2 つの独立したが、関連する一連のコマンドがあります。

-   [**PoolMon スタートアップ コマンド**](poolmon-startup-command.md):PoolMon を開始するには、スタートアップ コマンドの構文とパラメーターを使用してコマンドラインでコマンドを入力します。

-   [**PoolMon ランタイム コマンド**](poolmon-run-time-commands.md):実行中で PoolMon を構成するには、実行時コマンドを入力します。
