# <a name="c28735"></a>C28735


C28735 を警告します。禁止されている Crimson API の使用

この警告は、従来のイベント ログまたは他のトレース API を使用していることを示します。 これらの Api が両方のパフォーマンスとセキュリティを向上する Event Tracing for Windows (ETW) モデルに置き換えられました。 これらのレガシ API 呼び出しに使用を制限することによってイベントを使用して Windows のコンポーネントをデバッグするときに、特に管理者は、お客様が最適なエクスペリエンスをあること。
