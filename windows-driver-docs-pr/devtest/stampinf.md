# <a name="stampinf"></a>Stampinf


Stampinf (Stampinf.exe) は、ツールにドライバーをビルドするときに、共通の INF および INX ファイルのディレクティブを更新するためです。 プロジェクトの Stampinf オプションの設定方法の詳細については、[ドライバー プロジェクトのプロパティを Stampinf](https://docs.microsoft.com/windows-hardware/drivers/develop/stampinf-properties-for-driver-projects)を参照してください。 プロジェクト ファイルに Stampinf のサポートを追加する方法の詳細については、、 [WDK と Visual Studio ビルド環境](wdk-and-visual-studio-build-environment.md)と[Stampinf タスク](stampinf-task.md)を参照してください。

INX ファイルをは、INF ファイルに似ており、KMDF および UMDF ドライバーによって複数のプラットフォームおよび KMDF および UMDF バージョンをサポートするために使用します。 INX ファイルについては、[INX INF ファイルを作成するファイルを使用する](https://msdn.microsoft.com/library/windows/hardware/ff545473)を参照してください。
