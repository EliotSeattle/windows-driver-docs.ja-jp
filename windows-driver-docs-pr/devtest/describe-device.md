# <a name="describe-the-device-in-the-device-metadata-authoring-wizard"></a>デバイス メタデータ作成ウィザードでのデバイスの説明


Windows では、製造元とモデルの名前などのデバイスについて指定したわかりやすい情報を表示します。

### <a name="span-idtoprovidedescriptiveinformationaboutadevicespanspan-idtoprovidedescriptiveinformationaboutadevicespanspan-idtoprovidedescriptiveinformationaboutadevicespanto-provide-descriptive-information-about-a-device"></a><span id="To_provide_descriptive_information_about_a_device"></span><span id="to_provide_descriptive_information_about_a_device"></span><span id="TO_PROVIDE_DESCRIPTIVE_INFORMATION_ABOUT_A_DEVICE"></span>デバイスのわかりやすい情報を提供するには

1.  をクリックして、**説明**タブ。
2.  次のフィールドに入力します。
    -   **モデル名**します。 (必須) デバイスのモデル名
    -   **製造元**します。 (必須)、デバイスの製造元の名前
    -   **モデル番号**します。 (省略可能) デバイスのモデル番号
    -   **説明 1**します。 わかりやすいテキスト (省略可能)
    -   **説明 2**します。 (省略可能) 説明テキストを追加

**注**  を選択した場合**複数のロケール**で、**パッケージ定義**] タブの [複数表示**ロケール**タブ。 すべてのロケール情報を定義するには、各タブをクリックします。 クリックするとエラー メッセージが表示される各ロケール タブで、必須のフィールドを完了しない場合、**完了**のボタンでは、**完了**タブ。
