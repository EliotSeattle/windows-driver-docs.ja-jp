# <a name="pnpcpu-general-commands"></a>PNPCPU 一般的なコマンド


次の構文では、共通 PNPCPU のすべての操作です。

```
pnpcpu.exe -operation
```

### <a name="span-idparametersspanspan-idparametersspanparameters"></a><span id="parameters"></span><span id="PARAMETERS"></span>パラメーター

<span id="-install"></span><span id="-INSTALL"></span>**インストール**  
ローカル コンピューターのツールをインストールします。

<span id="-uninstall"></span><span id="-UNINSTALL"></span>**アンインストール**  
ツールをローカル コンピューターから削除し、コンピューターで実行する前にした状態に **-インストール**します。

これは、オフにすると後に発生する可能性がありますのあるすべてのプロセッサのエラー コードが含まれています。、 **-インストール**コマンドが実行され、システムが再起動されました。

<span id="-add"></span><span id="-ADD"></span>**-追加**  
実行、ホット アド操作ごとの論理プロセッサ上でのライセンスによってサポートされている最大までです。

<span id="-__or_-help"></span><span id="-__OR_-HELP"></span>**-?** または **-ヘルプ**  
このドキュメントのような使用状況情報を表示します。
