# <a name="opening-a-workspace"></a>ワークスペースを開く

トレース セッションのワークスペースを開くと、traceview では、ワークスペースに保存されているプロパティを持つ新しいトレース セッションを開始します。 トレース ログの表示用のワークスペースを開くと、構成したとおりに画面が表示されます。

### <a name="span-idtoopenaworkspacespanspan-idtoopenaworkspacespanto-open-a-workspace"></a><span id="to_open_a_workspace"></span><span id="TO_OPEN_A_WORKSPACE"></span>ワークスペースを開く

1.  **ファイル** メニューのをクリックして**ワークスペースを開く**します。

2.  **ワークスペースの保存**一覧で、ワークスペースの名前を選択します。

    または、**新しいワークスペース**ボックスに、ワークスペースの名前を入力します。

3.  **[OK]** をクリックします。

ワークスペースでは、パスとトレース セッションに関連付けられているファイル名が保存されているため、ファイルが再利用され、ワークスペースを開くたびに上書きされます。 ファイルを保持するためには、移動または各トレース セッションの後に名前を変更します。
