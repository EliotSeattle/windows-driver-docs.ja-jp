# <a name="using-traceview-workspaces"></a>TraceView ワークスペースの使用

Traceview でワークスペースとは一連のトレース セッションのプロパティとトレース ログの表示プロパティを保存して再利用できることができます。 ワークスペースでは、頻繁に使用されるログを表示したり、1 つの簡単な手順で慎重に構成済みのトレース セッションを開始できます。
