# <a name="adding-a-rule"></a>規則の追加


フィルターにルールを追加します。

1.  [トレース セッション リスト](trace-session-list.md)セッションで、任意のセルを右クリックし、クリックして**フィルターの管理**します。

2.  **フィルターの管理**ダイアログ ボックスで、ボックスの下部に、クリックして**新しいルール**します。

3.  **追加または変更するルール**領域で、、[要素ルールの](filter-rule-elements.md)します。

4.  クリックして**規則の追加**します。

使用する場合、[ワークスペース](using-traceview-workspaces.md)、する必要があります[ワークスペースを再度保存](saving-or-resaving-a-workspace.md)ルールを追加した後。 それ以外の場合、このルールは、ワークスペースには追加されません。
