# <a name="wdfrequestsendsyncatdispatch-rule-kmdf"></a>WdfRequestSendSyncAtDispatch ルール (kmdf)


**WdfRequestSendSyncAtDispatch**ルールを確認、 [ **WdfRequestSend** ](https://docs.microsoft.com/windows-hardware/drivers/ddi/content/wdfrequest/nf-wdfrequest-wdfrequestsend)関数は、適切な IRQL の優先度レベルで送信します。

場合は、ドライバーの設定、WDF\_要求\_送信\_オプション\_同期フラグは、呼び出す必要があります、 [ **WdfRequestSend** ](https://docs.microsoft.com/windows-hardware/drivers/ddi/content/wdfrequest/nf-wdfrequest-wdfrequestsend) IRQL で関数パッシブを =\_レベル。 このフラグが設定されていない場合、ドライバーを呼び出す必要があります、 **WdfRequestSend** IRQL で関数のディスパッチを =\_レベル。

> [!NOTE]
>
> このルールのテストと同じ条件、 [ **WdfRequestSendSyncAtDispatch2** ](wdfrequestsendsyncatdispatch2.md)規則が異なるコード パスをテストするさまざまなハーネスを使用します。
