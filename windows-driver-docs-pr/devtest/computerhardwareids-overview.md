# <a name="computerhardwareids-overview"></a>ComputerHardwareIds の概要


ComputerHardwareIds を実行するには、コマンド プロンプトでコマンドを入力します。 次の構文とパラメーターを使用します。

```
    ComputerHardwareIds
```
