# <a name="color-coding-in-the-trace-tree-pane"></a>Trace Tree (トレース ツリー) ウィンドウの色の設定


SDV は、すぐに表示される内容を解釈するための色を使用します。

-   **黄色の強調表示されたテキストを黒**ドライバーのコードを示します。

-   **黒のない強調表示されたテキスト**ドライバーには、最初の呼び出しと、ドライバーから返される呼び出しを示します。

-   **灰色のテキスト**SDV のコードを示します。

-   **青の強調表示**を示します、**トレース ツリー**で選択されているコード行に関連付けられているウィンドウ要素、**ソース コード**ウィンドウ。 青の強調表示でのコード行が選択した場合にのみが表示されます、**ソース コード**ウィンドウ。
