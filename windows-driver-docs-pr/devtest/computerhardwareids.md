# <a name="computerhardwareids"></a>ComputerHardwareIds


ComputerHardwareIds (ComputerHardwareIds.exe) は、仕入先のコンピューター システムの特定のモデルのハードウェア Id の生成に使用できるコマンド ライン ツールです。 仕入先は、これらのハードウェア Id を使用して、特定のコンピューター モデルへのデバイス メタデータ パッケージと一致します。

ComputerHardwareIds ツールは、Windows 7 または Windows オペレーティング システムの以降のバージョンを実行するシステムでサポートされます。 このツールはツールの下で Windows Driver Kit (WDK) に含まれる\\i ほかディレクトリ。 32 ビットの Windows プラットフォーム用の別のバージョンがあります (ツール\\他\\x86) および 64 ビット Windows プラットフォーム (ツール\\他\\ia64 とツール\\他\\amd64)。
