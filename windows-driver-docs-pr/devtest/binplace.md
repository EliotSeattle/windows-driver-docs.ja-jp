# <a name="binplace"></a>BinPlace


## <span id="ddk_binplace_tools"></span><span id="DDK_BINPLACE_TOOLS"></span>


BinPlace は、大きなコード プロジェクトを管理するためのツールです。 実行可能ファイル、シンボル ファイル、および他の種類のファイルに移動することができます。 実行可能ファイルからシンボルを抽出でき、シンボル ファイルからプライベート シンボルを削除することができます。

BinPlace は、多数のさまざまなパッケージで配布したりモジュールを構築している場合に便利です。 大規模なプロジェクトがない、まだ役立つ BinPlace シンボル ファイルを操作するためです。
