# <a name="c28162"></a>C28162


C28162 を警告します。浮動小数点演算ハードウェアを使用する権利を保持しているときに終了します。

<table>
<colgroup>
<col width="50%" />
<col width="50%" />
</colgroup>
<tbody>
<tr class="odd">
<td align="left"><p><strong>追加情報</strong></p></td>
<td align="left"><p>復元の浮動小数点で終了する、この関数の注釈が設定されました。</p></td>
</tr>
</tbody>
</table>

 

**\_カーネル\_float\_復元\_** 注釈が、浮動小数点数を使用する権利を解放するために使用に認識機能はありませんが、関数内のパスが検出されました操作が正常に呼び出されたことを実行します。 この警告になっているため、条件文 (**\_とき\_**) 注釈が必要なまたはコード エラーを示す可能性があります。
