# <a name="transport-minidrivers"></a>トランスポート ミニドライバー


このセクションには、独自の HID ミニドライバーを作成する必要のある仕入先の詳細が含まれています。 デバイスは、USB、Bluetooth、Bluetooth LE、I²C、トランスポートとしての GPIO を必要とする場合は、Microsoft から提供されたインボックス ドライバーを使用します。 インボックス トランスポート ミニドライバーの一覧を表示するには、[ **HID トランスポート**](hid-transports.md)を参照してください。

その他のトランスポートの場合は、トランスポート ミニドライバーを記述する必要があります。

HID ミニドライバーは、次のフレームワークの 1 つを使用して記述できます。

1.  UMDF – ユーザー モード ドライバー フレームワーク
2.  KMDF: カーネル モード ドライバー フレームワーク
3.  WDM – レガシ Windows Driver Model

**注**  ハードウェア ベンダーが、インボックス トランスポートのミニドライバー可能な限りの使用をお勧めします。 ただし、デバイスでは、サポートされていないトランスポートが必要とする場合は、ミニドライバーのドライバー モデルとしての Windows ドライバー フレームワーク (UMDF または KMDF) の使用を推奨します。 Windows ドライバー フレームワークでは、特定のトランスポートはサポートされていない場合のみ WDM ミニドライバーを作成する必要があります。

開発者が、開始点として、UMDF フレームワークを使用することをお勧めします。 機能が UMDF に利用できない場合にのみ、KMDF ドライバーの作成を検討してください。 2 つのドライバー フレームワークで機能の比較については、KMDF に UMDF 2 機能を比較するを参照してください。

次の表は、主な利点と課題に関連する HID トランスポート ミニドライバーは、2 つの WDF モデルに関連付けられているをキャプチャします。

|            |                                                                                                                  |                                                                                                                               |
|------------|------------------------------------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------|
|            | KMDF                                                                                                             | UMDF                                                                                                                          |
| 長所 | 必要なすべてのキーボードとマウスのフィルター ドライバーを WDF をサポートするすべての Windows プラットフォームで利用可能なサポートします。 | 簡単に開発および推奨のほとんどの垂直方向のデバイス クラスに対するこのドライバーでエラーのバグをオンにしないで、システム全体。 詳細については、[**書き込み UMDF ドライバーの利点**](../wdf/advantages-of-writing-umdf-drivers.md)を参照してください。   |
| チャレンジ | 適切に記述されて KDMF HID トランスポート ミニドライバーは、システムにクラッシュすることができます。                                              | ミニドライバーは UMDF HID トランスポートには、Windows 8 より前の Windows のバージョンがサポートされていません。 UMDF ドライバーでは、カーネル モード ドライバーからの I/O 要求を受信できます。 これらの遷移には、わずかなパフォーマンス影響を可能性があります。 |
