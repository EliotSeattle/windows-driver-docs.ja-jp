# <a name="transport-minidriver-overview"></a>Transport ミニドライバーの概要

ここでは、独自の HID ミニドライバーを作成する必要があるベンダーの詳細について説明します。 デバイスで USB、Bluetooth、Bluetooth LE、I ² C、GPIO がトランスポートとして必要な場合は、Microsoft が提供するインボックスドライバーを使用します。 インボックストランスポートミニドライバーの一覧については、「 [**HID**](hid-transports.md)transport」を参照してください。

その他のトランスポートの場合は、トランスポートミニドライバーを作成する必要があります。

HID ミニドライバーは、次のいずれかのフレームワークを使用して書き込むことができます。

1.  UMDF –ユーザーモードドライバーフレームワーク
2.  KMDF –カーネルモードドライバーフレームワーク
3.  WDM –レガシ Windows Driver Model

Microsoft では、可能な限り、ハードウェアベンダーがインボックスミニドライバーを使用することをお勧めします。   ただし、デバイスでサポートされていないトランスポートが必要な場合は、ミニドライバーのドライバーモデルとして Windows Driver Framework (UMDF または KMDF) を使用することをお勧めします。 特定のトランスポートが Windows ドライバーフレームワークでサポートされていない場合にのみ、WDM ミニドライバーを作成する必要があります。

Microsoft では、開発者が UMDF framework を出発点として使用することをお勧めします。 UMDF で機能を使用できない場合にのみ、KMDF ドライバーの記述を検討してください。 2つのドライバーフレームワークの機能の比較については、「UMDF 2 の機能と KMDF の比較」を参照してください。

次の表は、2つの WDF モデルが HID トランスポートミニドライバーに関連する主な利点と課題を示しています。

|            |                                                                                                                  |                                                                                                                               |
|------------|------------------------------------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------|
|            | KMDF                                                                                                             | UMDF                                                                                                                          |
| 長所 | すべてのキーボードおよびマウスフィルタードライバーに必要な WDF をサポートするすべての Windows プラットフォームでサポートされています。 | このドライバーでは、ほとんどのデバイスクラスのエラーを簡単に開発して推奨しています。システム全体を確認するバグはありません。 詳細については、「 [**UMDF ドライバーを作成する利点**](../wdf/advantages-of-writing-umdf-drivers.md)」を参照してください。   |
| チャレンジ | 適切に記述されていない KDMF HID トランスポートミニドライバーは、システムをクラッシュさせる可能性があります。                                              | UMDF HID transport ミニドライバーは、Windows 8 より前のバージョンの Windows ではサポートされていません。 UMDF ドライバーは、カーネルモードドライバーから i/o 要求を受け取ることができます。 これらの遷移は、わずかなパフォーマンスの影響を与える可能性があります。 |
