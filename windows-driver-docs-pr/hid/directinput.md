# <a name="directinput-overview"></a>DirectInput の概要

Microsoft DirectInput のドキュメントでは、Microsoft DirectX バージョン1.0、2.0、3.0、5.0、6.0、7.0、7.0 a、および8.0 の入力コンポーネントのドライバーを記述する方法について説明しています。 DirectX バージョン1.0 および2.0 には、元の Microsoft Windows 95 と同じジョイスティックサポートが含まれていますが、DirectX 3.0 ではマウスおよびキーボードアクセス用のコンポーネントオブジェクトモデル (COM) インターフェイスが追加され、ドライバーモデルを変更する。 DirectX 5.0 以降では、ジョイスティックアクセス、フォースフィードバック、およびジョイスティック構成情報へのアクセスを簡略化するための COM インターフェイスが追加されました。
