# <a name="directinput"></a>DirectInput





Microsoft DirectInput ドキュメントでは、Microsoft DirectX のバージョン 1.0、2.0、3.0、5.0、6.0、7.0、7.0a、および 8.0 の入力コンポーネント用のドライバーを記述する方法について説明します。 DirectX のバージョン 1.0 および 2.0 DirectX 3.0 がマウスとキーボード アクセス用のコンポーネント オブジェクト モデル (COM) インターフェイスを追加せずには、いくつかジョイスティック サポートの実装を改善中に元の Microsoft Windows 95 として同じジョイスティック サポートが含まれるドライバー モデルを変更します。 DirectX 5.0 以降では、ジョイスティック構成情報をフォース フィードバック ジョイスティック アクセスとアクセスの簡略化された COM インターフェイスを追加します。
