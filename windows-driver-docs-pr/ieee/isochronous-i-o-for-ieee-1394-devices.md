# <a name="isochronous-io-for-ieee-1394-devices"></a>IEEE 1394 デバイスの等時性 I/O





デジタル カメラなどのリアルタイムのマルチ メディア デバイスは、大量の帯域幅、安定でデータを送信する必要がありますが、配信の保証を必要としません。 (たとえば、デジタル カメラからドロップされたフレームには、信号の品質が低下するが、その意味を破棄が)。IEEE 1394 は、このようなデバイスでは、*アイソクロナス*転送では、保証された帯域幅を提供しますが、配送を保証しません。
