# <a name="minidriver-version-707-features"></a>ミニドライバー バージョン 7.07 の機能


このバージョンでは、次の機能が導入されました。

## <a name="span-idgenericinboxminidriverthatsupportsthegidscardedgespanspan-idgenericinboxminidriverthatsupportsthegidscardedgespanspan-idgenericinboxminidriverthatsupportsthegidscardedgespan-generic-inbox-minidriver-that-supports-the-gids-card-edge"></a><span id="_Generic_Inbox_minidriver_that_supports_the_GIDS__card_edge"></span><span id="_generic_inbox_minidriver_that_supports_the_gids__card_edge"></span><span id="_GENERIC_INBOX_MINIDRIVER_THAT_SUPPORTS_THE_GIDS__CARD_EDGE"></span> GID カードの端をサポートする汎用の受信トレイ ミニドライバー


-   ジェネリック ユーザー デバイスの仕様 (GID) カードの端をサポートする汎用の受信トレイ ミニドライバー

    Windows 8 では、GID カード エッジで Microsoft の電気のプロファイルをサポートする汎用ミニドライバーに含まれています。 参照してください[Windows 受信トレイのスマート カード ミニドライバー](windows-inbox-smart-card-minidriver.md)を特定し、受信トレイ ミニドライバーと GID のカードをペアリング プロセスの詳細については、Windows に依存します。 ジェネリックのミニドライバーも Windows 7 Service Pack 1 (SP1) 以降で使用できます。
