# <a name="smart-card-minidriver-versions"></a>スマート カード ミニドライバーのバージョン


スマート カード ミニドライバーは、特定のバージョンに準拠するように書き込まれます。 各バージョンでは、新機能と機能が導入されました。 特定のバージョンに準拠するには、ミニドライバーはすべての旧バージョンに含まれている、指定されたバージョンの機能をサポートします。
