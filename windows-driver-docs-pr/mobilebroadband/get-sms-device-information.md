# <a name="get-sms-device-information"></a>SMS デバイス情報の取得


モバイル ブロード バンド SMS プラットフォームは、アカウントの電話番号、状態、および携帯電話のクラスを含め、モバイル ブロード バンド デバイスに関する情報を提供します。

**SMS デバイスの携帯電話番号を取得するための JavaScript コードの例**

``` syntax
var mobileNumber = smsDevice.accountPhoneNumber;
```
