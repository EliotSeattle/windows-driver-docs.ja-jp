# <a name="hot-spot-authentication"></a>ホット スポット認証


可能であれば、Wi-fi ホット スポットへのデータ トラフィックの負荷を軽減する必要があります。 Windows 接続マネージャーを使用できる場合に、自動的に Wi-fi ネットワークに接続できます。 Wi-fi ネットワークに接続するには、Windows にホット スポットの資格情報を提供する必要があります。 アプリで独自の認証を実装することで、ホット スポットの資格情報を提供できます。
