# <a name="packageinfo-xml-schema-overview"></a>PackageInfo の XML スキーマの概要

[!include[MBAE deprecation warning](mbae-deprecation-warning.md)]

メタデータ パッケージには、オペレーティング システムを使用してメタデータ パッケージをインストールし、その内容を参照する情報を持つ 1 つの PackageInfo.xml ドキュメントが含まれています。

文書の書式設定 PackageInfo.xml 内のデータは、このセクションで説明されている PackageInfo XML スキーマに基づいています。

**注**   XML ドキュメントは UTF-8 エンコーディングで保存されている必要があります。
