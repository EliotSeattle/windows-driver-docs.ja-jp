# <a name="provisioning-api"></a>プロビジョニング API


プロビジョニング API を使用すると、モバイル ブロード バンド接続と Wi-fi では、必要な接続プロファイルを使用した Windows ベースのコンピューターをプロビジョニングでき、モバイル ブロード バンド プロファイルに関連付けられているコストを構成することができます。 プロビジョニング情報に含まれている XML ドキュメントで説明されている[メタデータを使用して、モバイル ブロード バンド エクスペリエンスを構成する](using-metadata-to-configure-mobile-broadband-experiences.md)します。 通常、またはコンピューターのプロビジョニング情報を更新するサブスクリプションの購入後に、この API を呼び出します。

Windows に提供される情報を更新するには、いつでもプロビジョニング API を呼び出すこともできます。 モバイル ブロード バンド アプリ、オペレーターのサーバーから更新されたコストとプランの状態を取得しますが、実行することも Wi-fi ホット スポット ネットワークまたはモバイル ブロード バンド ネットワーク構成を更新する必要があります、これは通常に行われます。

プロビジョニング API の詳細については、[ **ProvisioningAgent クラス**](https://msdn.microsoft.com/library/windows/apps/br207397)を参照してください。
