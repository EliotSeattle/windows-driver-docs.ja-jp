# <a name="communication-channels"></a>通信チャネル


次の図に示すように、モバイル ブロード バンド インターフェイスがコンピューターに直接接続されていないため、個人用ホット スポットのデバイスを使用して、演算子からの情報を交換する必要があります。

![通信チャネル](images/fig1-mb-commchannels.jpg)

デバイスによって提供されるデータに影響を与えるの 2 つの可能性があります: 時のセットアップ、および運用の有効期間中に、デバイスで使用される専用のバック チャネル (通常は web サービス) を使用して、ファームウェアでします。 これらのプロトコルは、Microsoft の設計の範囲外です。

必要なすべての機能は、ファームウェアまたは初期構成の一部として、通信事業者によって提供されるデータを使用して、デバイスの製造元によって実装できます。 省略可能な機能は、リアルタイムの更新プログラムの可用性を要求できます。
