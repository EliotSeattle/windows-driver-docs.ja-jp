# <a name="cosaapn-database"></a>COSA/APN データベース

このセクションには、COSA と apndatabase.xml (APN データベース) の両方に関する情報が含まれています。 COSA は、Windows 10、バージョン 1703 以降、APN データベースを使用して Windows 8 では、Windows 8.1、および Windows 10、Windows 10 バージョン 1703 より前に、のバージョンで使用される新しい形式です。

COSA と APN データベースは、エンドユーザーに提供しており、ユーザーのモバイル ブロード バンド デバイスに基づいて、APNs、使用可能な接続を試みると、シームレスな接続エクスペリエンスを提供する Windows 接続マネージャーなど、Windows ネットワーク コンポーネントによって使用されます。 COSA APN データベース情報が格納、モバイル ブロード バンド ネットワークに接続するために必要な Windows 最小限のユーザー入力を自動的に接続することができます。 アクセス文字列は、別のモバイル ネットワーク オペレーター、ユーザーの追加のソフトウェアやメタデータを取得する前に、オペレーターのネットワークへの接続を有効にするが維持します。 たとえば、モバイル ブロード バンド アプリをインストールしなくても接続できることをユーザー。

COSA と APN データベースには、情報をプロビジョニングするほか、アカウント エクスペリエンスの web サイトの URL も含まれます。 オペレーターのネットワークに接続すると自動的に、アカウントが発生する、ユーザーがサブスクリプションまたは 1 回限りのアクセスを購入できる既定のブラウザーで web サイトが開きます。
