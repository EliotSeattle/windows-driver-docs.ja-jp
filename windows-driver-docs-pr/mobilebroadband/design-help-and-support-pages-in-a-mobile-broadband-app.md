# <a name="design-help-and-support-pages-in-a-mobile-broadband-app"></a>モバイル ブロードバンド アプリでヘルプ ページとサポート ページを設計する


モバイル ブロード バンド サービスを使用して問題が発生したお客様は、ヘルプとサポート ページに移動します。

![ヘルプし、サポート ページ](images/mb-fig1-helpsupportpage.png)

ヘルプとサポート セクションでは、次の情報を表示することをお勧めします。

-   手順や、モバイル ブロード バンド サービスを使用する方法のチュートリアルのビデオ

-   モバイル ブロード バンド ハードウェアとサービスについてよく寄せられる質問

-   カスタマー サポートとオンライン チャット

-   カスタマー サポートの電話番号

-   ハードウェアとソフトウェアの診断情報

ヘルプのための設計を適切なし、ページのサポートします。

-   ヘルプとサポートのコンテンツを表示する必要があります、インターネットへのアクセスは必要ありません。 お客様は、サービスへの接続の問題が生じているために、インターネットへのアクセスがあります。

不適切なデザインのヘルプとサポートのページ。

-   外部の web ブラウザーで web ページを開くリンクが表示されません。

-   すべてのヘルプとサポートのコンテンツが、アプリ内で表示され、ユーザーがアプリを切り替える必要はありません。
