# <a name="subscriber-and-device-information-api"></a>サブスクライバーとデバイス情報 API


サブスクライバーとデバイス情報の API は、モバイル ブロード バンド アプリには、次のサブスクライバーとデバイスの次の情報を提供します。

-   **デバイス名**デバイスの名前。

-   **テクノロジ**GSM、CDMA と同様の情報。

-   **サブスクライバーとデバイスの ID**サブスクライバーとデバイスの識別情報 (IMSI、IMEI、電子的なシリアル番号\[ESN\]と同様の情報)。

-   **製造元**デバイスの製造元。

-   **携帯電話番号**デバイスに関連付けられている携帯電話の番号。

-   **SIM ID** SIM に関連付けられている ICCID します。

-   **デバイス機能**GSM バンドのクラスと CDMA 域外クラスなど、デバイス インターフェイスの機能です。

-   **ファームウェアとハードウェア バージョン**ファームウェアとハードウェアのバージョン。

サブスクライバーとデバイス情報 API の詳細については、次を参照してください。 [ **Windows.Networking.NetworkOperators 名前空間**](https://docs.microsoft.com/uwp/api/Windows.Networking.NetworkOperators)します。
