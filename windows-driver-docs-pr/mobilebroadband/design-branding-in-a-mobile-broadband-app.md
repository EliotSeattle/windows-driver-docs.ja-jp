# <a name="design-branding-in-a-mobile-broadband-app"></a>モバイル ブロード バンド アプリでのブランド化を設計します。


UWP アプリでは、次の方法でブランド化がお勧めします。

-   **色とフォントのカスタマイズ**を背景色とフォントの色をカスタマイズしている必要があります。 たとえば、アプリの外観やブランドの色を使用できます。

-   **ブランド画像を使用して、** Windows 接続マネージャーがメタデータと共にロゴを使用します。 保持すること、画像、アプリの招待の全体的な外観を維持する単純なことをお勧めします。 スプラッシュ画面とスタート画面のタイルには、ブランド画像を使用することができます。

モバイル ブロード バンド アプリでは、次の場所でブランド化を使用することができます。

-   スタート画面タイル (小)

    ![スタート画面の 1 x 1 タイル](images/mb-fig1-start-screen-small-tile.png)

-   スタート画面タイル (中)

    ![中規模のタイル](images/mb-design-start-screen-medium-tile.png)

-   スタート画面のタイル (幅)

    ![最初の画面 1 x 2 タイル](images/mb-fig2-start-screen-large-tile.png)

-   スタート画面のタイル (大)

    ![小さなタイル](images/mb-fig2-start-screen-2x1-tile.png)

-   [その他の Windows コンポーネントと統合して、モバイル ブロード バンド アプリ](integrate-a-mobile-broadband-app-with-other-windows-components.md#app-settings)ブランド名を示しています。

-   上のユーザー インターフェイス (UI) をアンインストール、**開始**メニューには、アイコンと名前が表示されます。

-   トースト通知

    ![トースト通知](images/mb-fig3-toast.png)

-   ブランド アイコンと名前を含むタスク マネージャー

-   コンピューターにモバイル ブロード バンド情報の設定 (オペレーター名を示しています)

-   スプラッシュ画面

    ![スプラッシュ スクリーン](images/mb-fig4-splash-screen.png)

-   Windows 接続マネージャー

    ![windows 接続マネージャー](images/mb-fig5-wcm.png)

Windows 10 のガイダンスをブランド化に関する詳細については、[UWP アプリのガイドラインをマーケティング](https://msdn.microsoft.com/library/windows/apps/hh694084)を参照してください。

## <a name="span-idquicksummaryspanspan-idquicksummaryspanspan-idquicksummaryspanquick-summary"></a><span id="Quick_summary"></span><span id="quick_summary"></span><span id="QUICK_SUMMARY"></span>簡単な概要


演算子のブランド化の適切なデザイン:

-   背景とフォントのアプリで使用してプライマリとセカンダリの企業ブランドの色します。

-   ブランド アイコンを使用して、適切にわかりやすくするための管理を UWP アプリのデザイン ガイドラインに従ってください。

演算子のブランド化の不適切なデザイン:

-   アイコンとイメージの空白文字を入力しません。
