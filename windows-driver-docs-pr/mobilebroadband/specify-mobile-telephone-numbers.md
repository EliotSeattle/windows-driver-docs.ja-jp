# <a name="specify-mobile-telephone-numbers"></a>携帯電話番号を指定する


電話番号は、ITU E.164 形式で指定する必要があります。 SMS クライアントは、ITU E.164 国際化の形式で電話番号を格納および ITU E.164 形式の電話番号を地域の一般的な記述形式に変換するライブラリを使用することが推奨されます。
