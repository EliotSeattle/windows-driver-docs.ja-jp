# <a name="sms-api"></a>SMS API


Windows では、演算子への超過料金とローミングの通知を受信、送信し、ユーザーにメッセージを受信およびメッセージングの構成を制御するには、SMS API を提供します。

SMS API は、次のユース ケースをサポートしています。

-   新しい SMS と新しい演算子に SMS 通知を送信します。

-   送信し、テキスト モードと (バイナリ) のプロトコル データ ユニット (PDU) モードでメッセージを読み取ります。

-   モバイル ブロード バンド デバイスのメッセージ ストアからメッセージを取得します。

-   モバイル ブロード バンド デバイスのメッセージ ストアからメッセージを削除します。

-   モバイル ブロード バンド デバイスのプロパティを取得 (アカウントの電話番号、モバイル通信用のグローバル システム\[GSM\]またはコード Division Multiple Access \[CDMA\]と同様の情報)。

SMS API についての詳細については、次を参照してください。 [ **Windows.Devices.Sms 名前空間**](https://msdn.microsoft.com/library/windows/apps/br206567)します。
