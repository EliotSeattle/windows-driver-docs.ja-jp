# <a name="open-the-networks-list"></a>ネットワークの一覧を開く


呼び出すことによって、ネットワークの一覧を開くことができます、 [ **ShowConnectionUI** ](https://docs.microsoft.com/uwp/api/Windows.Networking.NetworkOperators.MobileBroadbandNetwork#Windows_Networking_NetworkOperators_MobileBroadbandNetwork_ShowConnectionUI)アカウントの現在のネットワーク オブジェクトのメソッド。

次に、例を示します。

``` syntax
account.currentNetwork.showConnectionUI()
```
