# <a name="open-the-networks-list"></a>ネットワークの一覧を開く


呼び出すことによって、ネットワークの一覧を開くことができます、 [ **ShowConnectionUI** ](https://msdn.microsoft.com/library/windows/apps/hh921594)アカウントの現在のネットワーク オブジェクトのメソッド。

次に、例を示します。

``` syntax
account.currentNetwork.showConnectionUI()
```
