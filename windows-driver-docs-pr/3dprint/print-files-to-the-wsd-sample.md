# <a name="print-files-to-the-wsd-sample"></a>WSD サンプル ファイルを印刷します。


このセクションでは、WSD サンプルに印刷する方法について説明します。

## <a name="printing-the-files-to-the-wsd-sample"></a>WSD サンプル ファイルの印刷


1.  3D のビルダーのアプリケーションを起動します。

2.  3D モデルを選択し、[印刷] をクリックします。

    ![3d のビルダー アプリ](images/wsd-app-16.png)

3d のビルダー アプリから、プリンターに送信されたファイルが c: ドライブに移動\\Windows\\Temp フォルダーとは、zip ファイルです。

3D ビルダーから印刷 zip ファイルのフォルダー構造を次に示します。

![3d ビルダー zip ファイルのレイアウト](images/wsd-app-17.png)
