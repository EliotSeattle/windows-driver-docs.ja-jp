# <a name="serial-port-console-redirection-table--spcr"></a>シリアル ポート コンソール リダイレクト テーブル (SPCR)

このドキュメントでは、シリアル ポート コンソール リダイレクトのテーブルの内容を定義します。 このテーブルは、シリアル ポートまたは非レガシ UART インターフェイスがで Microsoft® Windows® 緊急管理サービス (EMS) を使用できるかどうかを示すために使用されます。

テーブルは、構成と、シリアル ポートまたは非レガシ UART インターフェイスの使用に関する情報を提供します。 場所、BIOS やシステム ファームウェアを使用して、シリアル ポート コンソール入力/出力システムでこのテーブルは、ファームウェアのコンソール出力と Windows の EMS の出力の間のシームレスな移行が、設定に関する情報を伝達するために使用する必要があります。

このテーブルは、他の ACPI テーブルのシステム メモリ内である必要があります、ACPI RSDT テーブルで参照する必要があります。 

> 特許の表示:Microsoft には特定の特許権が 2 つのオプション の下には、この仕様の実装を利用できるようにします。
> 1)  Microsoft の Community Promise で使用可能な<http://www.microsoft.com/openspecifications/en/us/programs/community-promise/default.aspx>; または
> 2)  Open Web Foundation 最終的な仕様許諾契約書バージョン 1.0 (「OWF 1.0」) 時点で、1、2012 年 10 月でご利用いただけます http://www.openwebfoundation.org/legal/the-owf-1-0-agreements/owfa-1-0します。
