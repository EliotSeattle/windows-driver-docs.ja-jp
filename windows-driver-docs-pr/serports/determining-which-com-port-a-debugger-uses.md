# <a name="determining-which-com-port-a-debugger-uses"></a>デバッガーを使用している COM ポートを決定します。


**KdComportInUse**グローバル変数には、カーネル デバッガーを使用している COM ポートに割り当てられているメモリ アドレスが含まれています。
