# <a name="determining-which-com-port-a-debugger-uses"></a>デバッガーで使用される COM ポートを特定する


**KdComportInUse**グローバル変数には、カーネル デバッガーを使用している COM ポートに割り当てられているメモリ アドレスが含まれています。
