# <a name="configuration-of-plug-and-play-serial-device-connected-to-an-rs-232-port"></a>Rs-232 ポートに接続されているプラグ アンド プレイ シリアル デバイスの構成





このセクションでは、ハードウェア、ドライバー、およびデバイスのプラグ アンド プレイ シリアル デバイスと rs-232 ポートに接続されているレガシ ポインター デバイス スタックの一般的な構成について説明します。 この構成は、マウス デバイス、ポインティング デバイス、タブレット、モデム、およびデジタル カメラなどのシリアル デバイスをサポートするために使用できます。

次の図は、トースターのプラグ アンド プレイ デバイスの一般的な構成を示します。

![プラグ アンド プレイ トースター デバイスのハードウェアとドライバー-と-デバイスのスタックの構成を示す図します。](images/ser2.png)

シリアル、Serenum が以前の構成で使用されます。 シリアルを作成し、接続関数デバイス オブジェクト (FDO)、rs-232 ポート スタック Serenum 作成し、rs-232 ポート スタックを上位レベルのフィルター デバイス オブジェクト (DO) をアタッチします。 Serenum、プラグ アンド プレイ マネージャは IRP を送信した後に、rs-232 ポートに接続されたデバイスを列挙する\_MN\_クエリ\_関係要求の種類の**BusRelations** rs-232 デバイス スタックにします。

Serenum がサポートされているデバイスを検出した後、(PDO) 物理デバイス オブジェクトを作成し、プラグ アンド プレイのマネージャーにデバイスを報告します。 構成マネージャーは、トースター デバイスのインストールを完了するのに INF ファイルとトースター デバイス用のインストーラーを使用します。 トースター ドライバーでは、FDO を作成し、トースター デバイス スタックに結び付けます。 DOs のフィルターは、トースター デバイス スタックにも追加できます。
