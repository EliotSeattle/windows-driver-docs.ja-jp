# <a name="installing-serenum-devices"></a>Serenum デバイスをインストールする





Serenum によって列挙されたデバイスをインストールするには、次を使用して*ハードウェア ID*デバイスの形式。

Serenum\\*XxxxYyyy*

各項目の意味は次のとおりです。*Xxxx* EISA 製造 ID; を指定する 4 つの ASCII 文字のフィールドであります。*Yyyy*プロダクト ID を指定する 4 つの ASCII 文字のフィールドであります。 Serenum Id が記載されて、[プラグ アンド プレイの外部 COM デバイスの仕様](https://msdn.microsoft.com/windows/hardware/gg463189.aspx)します。
