# <a name="powering-up-a-serial-device"></a>シリアル デバイスの電源を入れる





シリアル デバイスをオンする必要があります (デバイスの電源状態で**PowerDeviceD0**) のシリアル デバイスのハードウェアとの通信にします。 デバイスが有効でない場合シリアルは、デバイス ドライバーが要求を完了する前に有効にしようとします。
