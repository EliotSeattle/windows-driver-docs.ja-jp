# <a name="umdh-commands"></a>UMDH コマンド


## <span id="ddk_umdh_commands_dtools"></span><span id="DDK_UMDH_COMMANDS_DTOOLS"></span>


UMDH では、それぞれ独自のコマンド構文とパラメーターを持つ 2 つのモードがあります。

-   [**実行中のプロセスを分析**](analyze-a-running-process.md) -プロセスですべてのヒープ割り当てを表示します。

-   [**UMDH ログを分析する**](analyze-umdh-logs.md) -同じプロセスの 2 つの異なるタイミングで行われた割り当てのリストと比較します。 相違点を分析は、メモリ リークを特定するのに役立ちます。
