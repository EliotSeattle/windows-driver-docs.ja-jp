# <a name="writing-wdbgexts-extensions"></a>WdbgExts 拡張機能の作成


## <span id="ddk_writing_wdbgexts_extensions_dbwx"></span><span id="DDK_WRITING_WDBGEXTS_EXTENSIONS_DBWX"></span>


WdbgExts 拡張機能は、元の種類のデバッガー拡張機能です。 これらは DbgEng 拡張機能よりも強力ではありませんが、Microsoft Windows でユーザーモードまたはカーネルモードのデバッグを実行するときに、さまざまな機能を提供します。

Windows 用デバッグツールの完全インストールを実行した場合、「simplext」という名前のサンプル WdbgExts 拡張機能は、インストールディレクトリの\\samples\\simplext サブディレクトリにあります。
