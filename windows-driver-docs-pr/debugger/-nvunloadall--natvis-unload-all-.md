# <a name="nvunloadall-natvis-unload-all"></a>.nvunloadall (NatVis のすべてアンロード)

Nvunloadall コマンドは、デバッガー環境からすべての NatVis ファイルをアンロードします。

```dbgcmd
 .nvunloadall
```
