# <a name="logviewer"></a>LogViewer


## <span id="ddk_logviewer_dtoolq"></span><span id="DDK_LOGVIEWER_DTOOLQ"></span>


LogViewer ユーティリティでは、ロガーのツールで生成されたログの圧縮ファイル .lgv ファイルを操作できます。 ファイルを読み込むには、起動 Logviewer.exe またはコマンド プロンプト ウィンドウから Windows エクスプ ローラーを使用するだけです。 マニフェスト ファイルを解析するのにはしばらくをかかります。 これが完了すると呼び出す**ファイル |開いている**目的 .lgv ファイルを選択します。

LogViewer、.lgv ファイルが開くと、ログに記録された順序ですべての関数の一覧表示されます。 LogViewer に興味がない関数を除外するための強力なコマンドをサポートしています。 ログ ファイルのフィルター処理されたバージョンは、テキスト ファイルとして保存、こともできます。
