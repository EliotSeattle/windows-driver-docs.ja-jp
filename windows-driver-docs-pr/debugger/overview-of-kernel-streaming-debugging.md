# <a name="overview-of-kernel-streaming-debugging"></a>カーネル ストリーミングのデバッグの概要


カーネル ストリーミング デバッグ拡張機能は、拡張機能モジュール Ks.dll で確認できます。

Debug KS、AVStream、ポートのクラスやクラス ドライバーをストリームするのに Ks.dll を使用することができます。

Ks.dll で拡張機能コマンドの完全な一覧を参照してください。[カーネル ストリーミング拡張 (Ks.dll)](kernel-streaming-extensions--ks-dll-.md)します。

Windows 2000 でこれらの拡張機能を使用する場合は、w2kfre ディレクトリに kdexts ディレクトリから Ks.dll をコピーする必要があります。
