# <a name="edit--find-next"></a>編集 |次を検索します。


## <span id="ddk_edit_find_next_dbg"></span><span id="DDK_EDIT_FIND_NEXT_DBG"></span>


クリックして**次を検索**上、**編集**前回の検索を繰り返し、次の一致を検索するにはメニュー。

このコマンドは、f3 キーを押してと同じです。

逆方向に検索を繰り返すには、SHIFT + f3 キーを押します。

以前に検索を実行していない場合は、使用、**編集 |次を検索**コマンドや、f3 キーを押して開くには、SHIFT + F3 キーを押して、**検索** ダイアログ ボックス (のような[編集 |検索](edit---find.md)コマンド)。
