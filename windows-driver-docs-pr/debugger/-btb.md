# <a name="btb"></a>!btb


**! Btb**拡張機能には、現在のプロセッサに対する Itanium ベースのプロセッサ、ブランチトレースバッファー (btb) 構成、およびトレースレジスタが表示されます。

```dbgcmd
!btb
```

**重要**  このコマンドは、Windows デバッガーバージョン10.0.14257 以降では非推奨とされており、使用できなくなりました。

<p>Kdexts .dll</p></td>

この拡張機能コマンドは、Itanium ベースのターゲットコンピューターでのみ使用できます。
