# <a name="btb"></a>! btb


**! Btb**拡張機能には、Itanium ベースのプロセッサが表示されます、ブランチ トレース バッファー (BTB) 構成、およびトレースは、現在のプロセッサを登録します。

```dbgcmd
!btb
```

**重要な**  このコマンドが 10.0.14257 Windows デバッガーのバージョンでは非推奨以降されてし、は現在利用できません。
