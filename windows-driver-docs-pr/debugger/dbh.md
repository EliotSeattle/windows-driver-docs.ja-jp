# <a name="dbh"></a>DBH


DBH ツール (dbh.exe) は、シンボル ファイルの内容に関する情報を表示するコマンド ライン ツールです。

DBH は、当該のコマンド ライン インターフェイスを通じて DbgHelp API (dbghelp.dll) の機能を公開します。 そのため、DbgHelp が更新されると、その動作は変更できます。 DBH の 1 つのバージョンのソース コードでは、Windows 8 用 Windows ソフトウェア開発キット (SDK) で使用することができます。

このセクションの内容:

[DBH を使用します。](using-dbh.md)

[追加 DBH 例](additional-dbh-examples.md)

[**DBH コマンド ライン オプション**](dbh-command-line-options.md)

[DBH コマンド](dbh-commands.md)

DbgHelp API の詳細については、カスタムを実行する場合の Windows 内のデバッグ ツールの一部としてインストールされているドキュメントについては、インストールのデバッグ ヘルプ ライブラリを参照して選び、 **SDK**機能とそのサブ機能。
