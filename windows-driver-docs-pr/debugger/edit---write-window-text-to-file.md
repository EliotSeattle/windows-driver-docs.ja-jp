# <a name="edit--write-window-text-to-file"></a>編集 |ウィンドウのテキスト ファイルを書き込む


## <span id="ddk_edit_write_window_text_to_file_dbg"></span><span id="DDK_EDIT_WRITE_WINDOW_TEXT_TO_FILE_DBG"></span>


をクリックして**ウィンドウのテキストをファイルに書き込む**で、**編集**のすべてのテキストをファイルにアクティブなデバッグ情報ウィンドウに保存するメニュー。

このコマンドは、アクティブなウィンドウが場合にのみ使用可能な[デバッガー コマンド ウィンドウ](debugger-command-window.md)、[コール スタック ウィンドウ](calls-window.md)、またはスクラッチ パッド。

### <a name="span-iddialogboxspanspan-iddialogboxspandialog-box"></a><span id="dialog_box"></span><span id="DIALOG_BOX"></span>ダイアログ ボックス

クリックすると**Windows テキストをファイルに書き込む**、**ウィンドウのテキストをファイルに書き込む** ダイアログ ボックスが表示されます。 このダイアログ ボックスで、ウィンドウのテキストを保存するファイルの名前を入力します。 参照できます、**で保存**一覧ディレクトリにした場合または特定のファイルを上書きするを選択します。 既定のファイル名拡張子は .txt です。

をクリックして**保存**ファイルを保存する**キャンセル**を終了します。
