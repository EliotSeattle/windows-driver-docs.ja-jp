# <a name="entering-debugger-commands-in-visual-studio"></a>Visual Studio でのデバッガー コマンドの入力

> [!IMPORTANT]
> この機能は、Windows 10 バージョン1507以降のバージョンの WDK では使用できません。
> Visual Studio の代わりに、WinDbg Preview のコマンドウィンドウを使用します。 詳細については、「 [WinDbg プレビュー-ビューメニュー](windbg-view-preview.md) 」と「[デバッガーコマンド](debugger-commands.md)」を参照してください。
>

このトピックに記載されている手順では、Windows Driver Kit が Visual Studio に統合されている必要があります。 統合環境を利用するには、まず Microsoft Visual Studio をインストールしてから、Windows Driver Kit (WDK) をインストールします。 詳細については、「 [Windows Driver Development](https://docs.microsoft.com/windows-hardware/drivers/)」を参照してください。

Visual Studio では、デバッガーの [イミディエイト] ウィンドウにデバッガーコマンドを入力できます。 デバッガーの イミディエイト ウィンドウを開くには、**デバッグ** メニューの  **Windows&gt;イミディエイト** をクリックします。
