# <a name="bug-check-0x13c-invalidiobooststate"></a>バグ チェック 0x13C:無効な\_IO\_BOOST\_状態


無効な\_IO\_BOOST\_状態のバグ チェックが 0x0000013C の値を持ちます。 これは、無効な I/O boost 状態、スレッドが終了していることを示します。 スレッドの終了時にゼロがあります。
