# <a name="enabling-ndis-debug-tracing"></a>デバッグ トレースを有効にする NDIS


NDIS デバッグ トレースは、NDIS ドライバーをデバッグするため、主要な方法です。 NDIS のデバッグ トレースを設定するときは、実際に NDIS による DbgPrint ステートメントの 1 つまたは複数のレベルを有効にします。 結果として得られる情報は、ほとんどのネットワーク ドライバーの問題をデバッグするため十分です。

適切なレジストリ値を設定して、デバッグ トレースを有効にすることができます。 詳細については、[を有効にする NDIS デバッグ トレースによってレジストリ値の設定](enabling-ndis-debug-tracing-by-setting-registry-values.md)を参照してください。

ホスト コンピューターがインストールされている Ndis.sys 記号のチェック済みバージョンを持っていない場合でも、デバッグ トレース機能を有効にするレジストリ値を設定します。
