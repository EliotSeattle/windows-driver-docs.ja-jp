# <a name="pars"></a>! の部分から構成


**! の部分から構成**拡張機能は、指定されたプロセッサ アプリケーション レジスタ ファイルを表示します。

```dbgcmd
!pars Address
```

**重要な**  このコマンドが 10.0.14257 Windows デバッガーのバージョンでは非推奨以降されてし、は現在利用できません。
