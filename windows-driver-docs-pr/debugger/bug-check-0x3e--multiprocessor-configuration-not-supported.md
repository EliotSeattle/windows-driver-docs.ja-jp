# <a name="bug-check-0x3e-multiprocessorconfigurationnotsupported"></a>バグ チェック 0x3E の。マルチプロセッサ\_構成\_いない\_サポートされています。


マルチプロセッサ、\_構成\_いない\_サポートされているバグ チェックが 0x0000003E の値を持ちます。 これは、システムが複数のプロセッサが、相互の関連非対称、ことを示します。

**重要な**プログラマ向けのトピックです。 コンピューターを使用しているときに、エラー コードがブルー スクリーンが受信した顧客の場合を参照してください。[トラブルシューティング ブルー スクリーン エラー](https://windows.microsoft.com/windows-10/troubleshoot-blue-screen-errors)します。

## <a name="multiprocessorconfigurationnotsupported-parameters"></a>マルチプロセッサ\_構成\_いない\_サポートされているパラメーター


なし

<a name="cause"></a>原因
-----

対称であるするには、同じ種類とレベルのすべてのプロセッサがあります。 このシステムには、さまざまな種類 (たとえば、Pentium プロセッサおよび 80486 のプロセッサ) のプロセッサが含まれています。
