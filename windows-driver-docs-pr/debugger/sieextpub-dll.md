# <a name="sieextpubdll"></a>SieExtPub.dll


## <span id="ddk_oem_support_extensions_kdex2x86_dll__dbg"></span><span id="DDK_OEM_SUPPORT_EXTENSIONS_KDEX2X86_DLL__DBG"></span>


SieExtPub.dll 拡張モジュール Microsoft デバッグ ツールの Windows での一部でないし、SieExtPub.dll がデバッグ診断ツールに含まれていません。 SieExtPub.dll に含まれていた拡張機能コマンドの多くは、ext.dll になりました。 Ext.dll で拡張機能コマンドの一覧を表示するには、デバッガーで、次のコマンドを入力します。

**.extmatch/e ext \\***

**注**  、 **! critlist**拡張機能コマンドでは使用できなくします。
