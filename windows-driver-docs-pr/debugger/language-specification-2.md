# <a name="language-specification-2"></a>言語仕様 2


このパッケージは、.pdb ファイルの srcsrv データ ストリームで定義されている SRCSRVCMD せずに操作できる srcsrv.dll で出荷されます。 このような機能は、ソース管理システムから通常のファイルの抽出に使用できないのは、ファイル共有の UNC または HTTP サイトからの直接アクセスするため便利です。 参照してください[HTTP サイトと UNC 共有](http-sites-and-unc-shares.md)の詳細。
