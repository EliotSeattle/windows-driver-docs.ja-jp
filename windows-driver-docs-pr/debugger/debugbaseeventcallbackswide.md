# <a name="debugbaseeventcallbackswide-class"></a>Debugbaseeventの Swide クラス 

DebugbaseeventIDebugEventCallbacksWide Swide クラスは、 [](https://docs.microsoft.com/windows-hardware/drivers/ddi/dbgeng/nn-dbgeng-idebugeventcallbackswide)インターフェイスの基本実装を提供します。 

プログラムは、Debugbaseeventの Swide からイベントコールバッククラスを派生させ、必要なメソッドのみを実装できます。 

GetInterestMask を適切に実装するように注意してください。
