# <a name="logfile-display-log-file-status"></a>.logfile (ログ ファイル状態の表示)


**.Logfile**コマンドは、ログ ファイルが存在し、ファイルの状態を表示するかどうかを決定します。

```dbgcmd
    .logfile 
```
