# <a name="configuring-exceptions-and-events-in-cdb"></a>CDB で例外とイベントの構成


特定の方法で指定された例外とイベントに対応する CDB を構成できます。 例外ごとに、中断状態と処理の状態を設定できます。 各イベントに対して、中断状態を設定できます。

中断状態または処理の状態を構成するには、次のいずれかの方法します。

-   使用して、 [ **SXE**](sx--sxd--sxe--sxi--sxn--sxr--sx---set-exceptions-.md)、 **SXD**、 **SXN**、または**SXI**コマンド。

-   使用して、 **-x**、 **-xe**、 **-xd**、 **- xn**、または **-xi**オプション、 [ **CDB コマンドライン**](cdb-command-line-options.md)します。
-   使用して、 **sxe**または**sxd** Tools.ini ファイル内のキーワード。

例外とイベントの詳細については、次を参照してください。[を制御する例外とイベント](controlling-exceptions-and-events.md)します。
