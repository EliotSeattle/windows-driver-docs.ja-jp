# <a name="bug-check-0x163-workerthreadtestcondition"></a>バグ チェック 0x163:ワーカー\_スレッド\_テスト\_条件


ワーカー\_スレッド\_テスト\_条件のバグ チェックが 0x00000163 の値を持ちます。 これは、ワーカー スレッドがカーネルのテストにエラーが発生したことを示します。

> [!IMPORTANT]
> このトピックはプログラマーを対象としています。 コンピューターを使用しているときに、エラー コードがブルー スクリーンが受信した顧客の場合を参照してください。[トラブルシューティング ブルー スクリーン エラー](https://support.microsoft.com/help/14238/windows-10-troubleshoot-blue-screen-errors)します。

| パラメーター | 説明                                               |
|-----------|-----------------------------------------------------------|
| 1         | アクティブなテスト フラグ                                         |
| 2         | エラーを発生させたテストに対応するフラグ |
| 3         | 予約済み                                                  |
| 4         | 予約済み                                                  |
