# <a name="l"></a>L


<span id="live_kernel_mode_debugging"></span><span id="LIVE_KERNEL_MODE_DEBUGGING"></span>**ライブカーネルモードのデバッグ**  
ライブターゲットを使用したカーネルモードデバッグ。

「ライブユーザーモードのデバッグ」も参照してください。

<span id="live_target"></span><span id="LIVE_TARGET"></span>**ライブターゲット**  
(ダンプターゲットではなく) 現在操作可能なターゲットアプリケーションまたはターゲットコンピューター。

<span id="live_user_mode_debugging"></span><span id="LIVE_USER_MODE_DEBUGGING"></span>**ライブユーザーモードのデバッグ**  
ライブターゲットを使用したユーザーモードのデバッグ。

「ライブユーザーモードのデバッグ」も参照してください。

<span id="local_context"></span><span id="LOCAL_CONTEXT"></span>**ローカルコンテキスト**  
「スコープ」を参照してください。

<span id="local_debugging"></span><span id="LOCAL_DEBUGGING"></span>**ローカルデバッグ**  
これは、デバッガーとデバッグ対象のアプリケーションが同じコンピューター上に存在するデバッグセッションを指します。
