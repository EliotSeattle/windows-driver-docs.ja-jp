# <a name="bug-check-0x6f-session3initializationfailed"></a>バグ チェック 0x6F:SESSION3\_初期化\_失敗


SESSION3\_初期化\_失敗のバグ チェックが 0x0000006F の値を持ちます。 このバグ チェックでは、Microsoft Windows オペレーティング システムの初期化が失敗したことを示します。
