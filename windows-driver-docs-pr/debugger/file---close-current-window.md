## <span id="ddk_file_close_current_window_dbg"></span><span id="DDK_FILE_CLOSE_CURRENT_WINDOW_DBG"></span>


をクリックして**現在のウィンドウを閉じる**上、**ファイル** メニューのアクティブなデバッグ情報のウィンドウを閉じます。

このコマンドは、ctrl キーを押しながら F4 キーを押すと同じです。

クリックして、デバッグ情報 ウィンドウを閉じることも、**閉じます**WinDbg ウィンドウ内の情報ウィンドウの右上隅のボタンをクリックします。
