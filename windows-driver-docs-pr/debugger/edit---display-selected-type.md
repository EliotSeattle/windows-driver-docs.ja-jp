# <a name="edit--display-selected-type"></a>Edit | Display Selected Type (編集 | 選択された種類を表示)


## <span id="ddk_edit_display_selected_type_dbg"></span><span id="DDK_EDIT_DISPLAY_SELECTED_TYPE_DBG"></span>


をクリックして**表示タイプが選択されている**上、**編集**] メニューの [現在の選択範囲のデータ型を決定する、[ソース ウィンドウ](source-window.md)で型を表示して、 [デバッガー コマンド ウィンドウ](debugger-command-window.md)します。

このコマンドは、CTRL + SHIFT + Y キーを押すか、クリックしてに相当**選択した種類の表示**ソース ウィンドウのショートカット メニュー。

選択したテキストが 1 つのオブジェクトよりも多く含まれている、構文エラーやその他の不規則な結果が表示可能性があります。 ソース ウィンドウでテキストが選択されていない場合は、このコマンドを使用することはできません。
