# <a name="unloadall-unload-all-extension-dlls"></a>.unloadall (すべての拡張機能 DLL のアンロード)


**.Unloadall**コマンドは、ホスト システム上のデバッガーからのすべての拡張 Dll をアンロードします。

```dbgcmd
.unloadall
```
