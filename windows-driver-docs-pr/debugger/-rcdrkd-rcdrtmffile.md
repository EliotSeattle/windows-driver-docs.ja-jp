# <a name="rcdrkdrcdrtmffile"></a>!rcdrkd.rcdrtmffile


**! Rcdrkd.rcdrtmffile**拡張機能を設定またはトレース メッセージの形式 (TMF) ファイルの名前をクリアします。

```dbgcmd
!rcdrkd.rcdrtmffile [Filename] 
```

## <a name="span-idddkdevobjdbgspanspan-idddkdevobjdbgspanparameters"></a><span id="ddk__devobj_dbg"></span><span id="DDK__DEVOBJ_DBG"></span>パラメーター


<span id="_______Filename______"></span><span id="_______filename______"></span><span id="_______FILENAME______"></span> *ファイル名*   
TMF ファイルの名前。 このパラメーターが指定されていない場合、ファイル名はクリアされます。

## <a name="span-iddllspanspan-iddllspandll"></a><span id="DLL"></span><span id="dll"></span>DLL


Rcdrkd.dll
