# <a name="source-control-systems"></a>ソース管理システム


SrcSrv には、Visual SourceSafe (vss.pm)、Perforce (p4.pm)、Team Foundation Server (tfn.pm)、Subversion (svn.pm) プロバイダー モジュールが含まれています。 SrcSrv で使用するための同時実行バージョンのシステム (CVS) モジュールがありますが、広範囲にテストされていません。 管理システムの他のバージョンをサポートするために独自のモジュールを生成することもできます。
