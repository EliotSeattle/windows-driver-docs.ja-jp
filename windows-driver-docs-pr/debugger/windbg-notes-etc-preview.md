# <a name="windbg-preview---notes-command-memory-and-source-menus"></a>WinDbg Preview-メモ、コマンド、メモリ、およびソースメニュー

このセクションでは、WinDbg Preview で [メモ]、[コマンド]、[メモリ]、[ソース] の各メニューを操作する方法について説明します。

## <a name="notes"></a>注意

メニューを使用して、次のことを行います。

- メモファイルを開く
- メモファイルの保存

## <a name="command"></a>コマンド

[コマンド] メニューを使用して、次のことを行います。

- DML を優先する
- 現在のテキストの選択範囲の強調表示と強調解除 (CTRL + ALT + H)
- コマンドウィンドウのテキストをクリアする
- Dml ファイルにウィンドウテキストを保存する

## <a name="memory"></a>メモリ

[メモリ] メニューを使用して、次のことを行います。

- データモデルメモリクエリの設定
- メモリサイズをバイトまたは長などに設定します。
- 表示形式 (hex、signed など) を設定します。
- たとえば、テキストの表示形式を ASCII に設定します。

## <a name="source"></a>Source (ソース)

[ソース] メニューを使用して、次のことを行います。

- ソースファイルを開く
- 命令ポインターを設定する
- [カーソル行の前まで実行]
- すべてのソースウィンドウを閉じる
