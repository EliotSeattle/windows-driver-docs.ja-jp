# <a name="using-and-customizing-windbg-themes"></a>WinDbg テーマの使用とカスタマイズ


テーマは、デバッグ情報のウィンドウの便利な構成を含む事前に構成された WinDbg ワークスペースです。

すべてのテーマは、基本ワークスペースとして保存できます。 Windows のツールをデバッグ パッケージでのテーマは、(拡張子が .reg) のレジストリ ファイルのセットとして提供されます。 その他のデバッグ セッションが蓄積されると、さまざまな既定のワークスペースを自動的に設定します。 これらの既定のワークスペースは、開始点として、[基本] ワークスペースを使用します。 既定のワークスペースの詳細については、次を参照してください。[を作成して、ワークスペースを開く](creating-and-opening-a-workspace.md)します。
