# <a name="showsymfailures"></a>.show\_sym\_failures


**.Show\_sym\_エラー**コマンドを有効またはシンボル参照の失敗およびルックアップの失敗の種類の表示を無効にします。

```dbgcmd
.show_sym_failures /s 
.show_sym_failures /S
.show_sym_failures /t
.show_sym_failures /T
```

## <a name="span-idparametersspanspan-idparametersspanspan-idparametersspanparameters"></a><span id="Parameters"></span><span id="parameters"></span><span id="PARAMETERS"></span>パラメーター


<span id="________s______"></span><span id="________S______"></span> **/s**   
シンボル参照の失敗の表示を有効にします。

<span id="________S______"></span><span id="________s______"></span> **/S**   
シンボル参照の失敗の表示を無効にします。

<span id="________t______"></span><span id="________T______"></span> **/t**   
ルックアップの失敗の種類の表示を有効にします。

<span id="________T______"></span><span id="________t______"></span> **/T**   
ルックアップの失敗の種類の表示を無効にします。
