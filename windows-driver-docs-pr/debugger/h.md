# <a name="h"></a>H


<span id="host"></span><span id="HOST"></span>**ホスト**  
「ホストコンピューター」を参照してください。

<span id="handling-status"></span><span id="HANDLING_STATUS"></span>**処理状態**  
処理状態は、デバッガーエンジンが例外イベントに処理済みとしてフラグを付けるかどうかを指定します。 処理状態は例外フィルターの一部です。

「状態の中断」も参照してください。 詳細については、「[例外と](controlling-exceptions-and-events.md)イベントの制御」および「[イベントフィルター](event-filters.md) 」を参照してください。

<span id="host_computer"></span><span id="HOST_COMPUTER"></span>**ホストコンピューター**  
ホストコンピューターは、デバッグセッションを実行するコンピューターです。 コマンドと拡張機能の実行やシンボルの読み込みなど、すべてのデバッガー操作がホストコンピューター上で実行されます。

一般的な2システムカーネルデバッグセッションでは、デバッガーは、デバッグ対象のターゲットコンピューターに接続されているホストコンピューター上で実行されています。
