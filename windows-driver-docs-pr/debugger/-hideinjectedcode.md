# <a name="hideinjectedcode-hide-injected-code"></a>.hideinjectedcode (挿入されたコードを非表示にする)


**.Hideinjectedcode**ターンのコマンドまたはオフにすると、挿入されたコードの非表示します。

```dbgcmd
.hideinjectedcode { on | off } 
.hideinjectedcode help 
.hideinjectedcode
```

<span id="_______on_____________"></span><span id="_______ON_____________"></span> **on**   
挿入されたコードの非表示をオンにします。

<span id="_______off_"></span><span id="_______OFF_"></span> **オフ** ****   
挿入されたコードの非表示を解除します。

<span id="_______help______"></span><span id="_______HELP______"></span> **ヘルプ**   
このコマンドのヘルプを表示します。
