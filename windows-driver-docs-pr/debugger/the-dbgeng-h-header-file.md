# <a name="the-dbgengh-header-file"></a>dbgeng.h ヘッダー ファイル

## <span id="ddk_the_dbgeng_h_header_file_dbg"></span><span id="DDK_THE_DBGENG_H_HEADER_FILE_DBG"></span>

Dbgeng .h ヘッダーファイル内のルーチンは、DbgEng 拡張機能を記述するために使用されます。
