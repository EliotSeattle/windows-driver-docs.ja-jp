# <a name="oem-support-extensions-kdex2x86dll"></a>OEM のサポートの拡張機能 (kdex2x86.dll)


## <span id="ddk_oem_support_extensions_kdex2x86_dll__dbg"></span><span id="DDK_OEM_SUPPORT_EXTENSIONS_KDEX2X86_DLL__DBG"></span>


OEM のサポートの拡張機能は、w2kfre で見つかる\\kdex2x86.dll、Windows のツールをデバッグ インストール ディレクトリのサブディレクトリ。

これらの拡張機能は、Windows 2000 ターゲットでのみ使用できます。

これらの拡張機能は、このドキュメントでは説明しません。 これらの拡張機能の説明については、OEM サポート ツールのドキュメントを参照してください。 このドキュメントでは、OEM サポート ツール パッケージ全体と共に取得するには[Microsoft サポート記事 253066](https://go.microsoft.com/fwlink/p/?LinkId=241339)し、そのページの指示に従います。
