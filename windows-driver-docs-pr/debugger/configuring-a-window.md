# <a name="configuring-a-window"></a>ウィンドウの構成


## <span id="ddk_configuring_a_window_dbg"></span><span id="DDK_CONFIGURING_A_WINDOW_DBG"></span>


各デバッグ情報 ウィンドウには、ウィンドウのタイトル バーを右クリックして、またはタイトル バーの右上隅の近くのアイコンをクリックしてアクセスできるショートカット メニューがあります。 このショートカット メニューのコマンドを使用するには、ウィンドウを構成します。

多くのデバッグ情報ウィンドウには、ボタンが含まれているツールバーもされています。 これらのボタンのほとんどでは、ショートカット メニューのコマンドと同じである機能があります。
