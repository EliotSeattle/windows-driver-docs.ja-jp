# <a name="rrestart-register-for-restart"></a>.rrestart (登録の再起動)


**.Rrestart**コマンドは、コンピューターの再起動またはアプリケーション エラーが発生した場合の再起動のデバッグ セッションを登録します。

```dbgcmd
.rrestart
```

<a name="remarks"></a>注釈
-------

このコマンドは、管理者特権でのデバッガー セッションは機能しません。
