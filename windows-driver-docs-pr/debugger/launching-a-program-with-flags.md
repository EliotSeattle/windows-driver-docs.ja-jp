# <a name="launching-a-program-with-flags"></a>フラグを使ったプログラムの起動


## <span id="ddk_launching_a_program_with_flags_dtools"></span><span id="DDK_LAUNCHING_A_PROGRAM_WITH_FLAGS_DTOOLS"></span>


この機能は、指定したフラグを 1 回、プログラムを実行します。 これらの設定では、プログラムの開始のインスタンスのみに影響します。 レジストリでは保存されません。

**フラグを使用してプログラムを起動するには**

1.  をクリックして、**イメージ ファイル**タブ。

2.  **イメージ**ボックスで、実行可能ファイルまたはファイル名拡張子、およびプログラムの任意のコマンドを含む DLL の名前を入力し、TAB キーを押します。

    これをアクティブ化、**起動**ボタンとチェック ボックスを**イメージ ファイル**タブ。

3.  設定またはフラグをオンまたはオフ、フラグに関連付けられているチェック ボックスをオフにします。

4.  をクリックして、**起動**ボタンをクリックします。

    次のスクリーン ショットは、**起動**のボタンでは、**イメージ ファイル**Windows Vista タブ。

    ![windows vista のイメージ ファイル タブのスクリーン ショット ](images/gflags-launch.png)

**注**  レジストリに設定されているフラグが起動されるプログラムのインスタンスには影響しません。
イメージ ファイルのフラグが解除される場合でも、起動したインスタンスの ダイアログ ボックスで設定されているフラグが使用されます。
