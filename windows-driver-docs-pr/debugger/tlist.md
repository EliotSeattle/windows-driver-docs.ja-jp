# <a name="tlist"></a>TList


## <span id="ddk_tlist_dtools"></span><span id="DDK_TLIST_DTOOLS"></span>


TList (タスク リスト ビューアー)、Tlist.exe は、各プロセスに関する有用な情報と共に、ローカル コンピューター上で実行中のプロセスを表示するコマンド ライン ツールです。

TList が表示されます。

-   プロセス Id (Pid) と、コンピューターで実行されているすべてのプロセス。

-   各プロセスを作成するプロセスを示すツリー。

-   その仮想メモリの使用、プロセスを開始するコマンドなど、プロセスの詳細です。

-   スレッドはスレッド Id を含めた、各プロセスで実行されているエントリ ポイントでは、最後に報告されたエラー、およびスレッドの状態。

-   バージョン番号、属性、およびモジュールの仮想アドレスを含めた、各プロセスで実行されているモジュールです。

名または PID でプロセスを検索するか、指定されたモジュールが読み込まれているすべてのプロセスを検索するには、TList を使用することができます。

Windows XP および以降のバージョンの Windows では、TList がこれらのシステムのヘルプとサポートで説明されている TaskList (Tasklist.exe) によって置き換えられました。 デバッグ ツールの Windows タスク一覧へのアクセスを持たないユーザーをサポートする TList が含まれます。
