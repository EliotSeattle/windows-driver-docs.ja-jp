# <a name="debugging-tools-for-windows-new-for-windows-81"></a>Debugging Tools for Windows:Windows 8.1 の新機能


Windows 8.1、Windows ツールのデバッグにはには、これらの新機能が含まれます。

-   [GPIO 拡張機能](gpio-extensions.md)
-   [HID の拡張機能](hid-extensions.md)
-   ほとんどの[カーネル モード ドライバー フレームワークの拡張機能コマンド](kernel-mode-driver-framework-extensions--wdfkd-dll-.md)2 の UMDF および KMDF と作業のようになりました。 いくつかのコマンド (たとえば[ **! wdfkd.wdfumdevstacks**](-wdfkd-wdfumdevstacks.md)) サポート UMDF 2 具体的にはこのセットに追加されています。
-   ページング ファイルをメモリ ダンプ ファイルと共に CAB ファイルに含まれているようになりましたことができます。 参照してください[メモリ ダンプとページング ファイルを含む CAB ファイル](cab-files-that-contain-paging-files-along-with-a-memory-dump.md)
