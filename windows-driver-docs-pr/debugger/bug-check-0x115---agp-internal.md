# <a name="bug-check-0x115-agpinternal"></a>バグ チェック 0x115:AGP\_内部


AGP\_内部バグ チェックが 0x00000115 の値を持ちます。 これは、高速のグラフィックスのポート (AGP) ドライバーの違反が検出されたことを示します。

このバグ チェックが非常に少ない回数が表示されます。
