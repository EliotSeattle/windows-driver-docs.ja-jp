# <a name="v"></a>V


<span id="virtual_process"></span><span id="VIRTUAL_PROCESS"></span>**仮想プロセス**  
カーネルモードデバッグでは、デバッガーエンジンはターゲットのカーネルを表す単一の*仮想プロセス*を作成します。

<span id="virtual_thread"></span><span id="VIRTUAL_THREAD"></span>**仮想スレッド**  
カーネルモードのデバッグでは、デバッガーエンジンはターゲットコンピューターの各プロセッサに対して*仮想スレッド*を作成します。
