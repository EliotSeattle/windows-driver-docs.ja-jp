# <a name="ttd-range-objects"></a>TTD Range オブジェクト
## <a name="description"></a>説明
*TTD 範囲*オブジェクトを使用して、トレースで時間の範囲に関する情報を提供します。 これらはの有効期間を説明するものは一般に、 [TTD スレッド オブジェクト](time-travel-debugging-thread-objects.md)TTD セッション中にします。

## <a name="children"></a>Children

| オブジェクト | 説明 |
| --- | --- |
| MinPosition | A[位置オブジェクト](time-travel-debugging-position-objects.md)範囲に関連する最初の位置をについて説明します。 |
| MaxPosition | A[位置オブジェクト](time-travel-debugging-position-objects.md)範囲に関連する最新の位置をについて説明します。 |
