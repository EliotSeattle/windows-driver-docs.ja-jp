# <a name="view--windbg-command-line"></a>View | WinDbg Command Line (表示 | WinDbg コマンド ライン)


## <span id="ddk_view_windbg_command_line_dbg"></span><span id="DDK_VIEW_WINDBG_COMMAND_LINE_DBG"></span>


クリックして**WinDbg コマンド ライン**上、**ビュー** WinDbg の現在のセッションを開くために使用するコマンドを表示するメニュー。

コマンドは、サイズの小さなメッセージ ウィンドウに表示されます。 クリックして**OK**このウィンドウを閉じます。
