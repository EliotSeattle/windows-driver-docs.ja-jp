# <a name="bug-check-0x199-kernelstorageslotinuse"></a>バグ チェック 0x199:カーネル\_ストレージ\_スロット\_IN\_使用


カーネル\_ストレージ\_スロット\_IN\_使用のバグ チェックが 0x00000199 の値を持ちます。 これは、これを使用して、オブジェクトがあるため、ストレージ スロットを解放することはできませんを示します。
