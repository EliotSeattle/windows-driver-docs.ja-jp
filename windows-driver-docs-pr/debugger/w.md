# <a name="w"></a>W


<span id="wdbgexts_api"></span><span id="WDBGEXTS_API"></span>**WdbgExts API**  
拡張機能のデバッガーエンジンによって公開されるインターフェイス。 これには、デバッガーエンジン API の機能のサブセットが含まれており、拡張機能でのみ使用できます。

<span id="wdbgexts_extension"></span><span id="WDBGEXTS_EXTENSION"></span>**WdbgExts 拡張機能**  
Wdbgexts ヘッダーファイル内のプロトタイプに基づく拡張機能。 これらの拡張機能は、WdbgExts API を使用します。

<span id="wow64"></span><span id="WOW64"></span>**WOW64**  
32ビットの Windows アプリケーションを実行できる、64ビットの Windows のエミュレーションレイヤー。 64ビットの Windows で実行されている32ビットアプリケーションをデバッグする場合は、アプリケーション自体と WOW64 サブシステムを区別することが重要です。
