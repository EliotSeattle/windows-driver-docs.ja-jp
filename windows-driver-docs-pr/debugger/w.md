# <a name="w"></a>W


<span id="wdbgexts_api"></span><span id="WDBGEXTS_API"></span>**WdbgExts API**  
拡張機能のデバッガー エンジンによって公開されるインターフェイス。 デバッガー エンジン API の機能のサブセットが含まれていて、拡張機能でのみ使用できます。

<span id="wdbgexts_extension"></span><span id="WDBGEXTS_EXTENSION"></span>**WdbgExts 拡張機能**  
Wdbgexts.h ヘッダー ファイルでのプロトタイプに基づく拡張機能。 これらの拡張機能は、WdbgExts API を使用します。

<span id="wow64"></span><span id="WOW64"></span>**WOW64**  
32 ビット Windows アプリケーションを実行できる、64 ビット Windows でのエミュレーション層。 64 ビット Windows で実行されている 32 ビット アプリケーションをデバッグするときに、アプリケーション自体と、WOW64 サブシステム間で区別するために重要です。
