# <a name="logger-and-logviewer"></a>Logger と LogViewer


## <span id="ddk_logger_and_logviewer_dtoolq"></span><span id="DDK_LOGGER_AND_LOGVIEWER_DTOOLQ"></span>


*ロガー*と*LogViewer*ツールは、標準ユーザー モードのデバッグに代替手段を提供します。 ロガーは、ユーザー モードの対象アプリケーションのアクションを監視し、すべての API の呼び出しを記録します。 結果として得られる情報のデバッガーに表示、テキスト ファイルとして保存または LogViewer ツールによって、強力な対話形式で表示される.
