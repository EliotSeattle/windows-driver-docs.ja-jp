# <a name="bug-check-0x120-bitlockerfatalerror"></a>バグ チェック 0x120:BITLOCKER\_FATAL\_エラー


BITLOCKER\_FATAL\_エラーのバグ チェックが 0x00000120 の値を持ちます。 これは、BitLocker ドライブ暗号化に回復できない問題が発生したことを示します。
