# <a name="bug-check-0x100-loaderblockmismatch"></a>バグ チェック 0x100 の。ローダー\_ブロック\_が一致しません


ローダー\_ブロック\_の不一致のバグ チェックが 0x00000100 の値を持ちます。 これは、いずれか、ローダー ブロックが有効でないか、システムに読み込まれているのとは一致しませんを示します。
