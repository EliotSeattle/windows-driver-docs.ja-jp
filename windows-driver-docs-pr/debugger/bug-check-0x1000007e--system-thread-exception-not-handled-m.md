システム\_スレッド\_例外\_いない\_HANDLED\_M のバグ チェックが 0x1000007E の値を持ちます。 これは、システム スレッドがエラー ハンドラーをキャッチされなかった例外を生成することを示します。

バグ チェック 0x1000007E が同じ意味とパラメーターとして[**バグ チェック 0x7E** ](bug-check-0x7e--system-thread-exception-not-handled.md) (システム\_スレッド\_例外\_いない\_処理済み)。
