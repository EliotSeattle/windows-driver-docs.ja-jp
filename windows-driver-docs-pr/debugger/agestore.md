# <a name="agestore"></a>AgeStore


AgeStore ツール (agestore.exe) は、ディレクトリまたはその最後アクセス日に基づいて、ディレクトリ ツリー内のファイルを削除します。 このプログラムは、ディスク領域を節約するために、シンボル サーバーで使用されるダウン ストリームのストアから古いファイルを削除するために特に便利です。
