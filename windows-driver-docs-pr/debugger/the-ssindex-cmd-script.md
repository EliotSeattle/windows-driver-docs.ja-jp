# <a name="the-ssindexcmd-script"></a>Ssindex.cmd スクリプト


Ssindex.cmd スクリプトでは、各ファイルのバージョン情報と共にソース管理にチェックインするファイルの一覧を作成します。 この情報のサブセットをアプリケーションをビルドしたときに生成される .pdb ファイルに格納します。 SSIndex では、Perl モジュールのいずれかを使用して、ソース管理とのインターフェイスします。

-   p4.pm (Perforce)

-   vss.pm (Visual SourceSafe)

-   tfs.pm (Team Foundation Server)

-   svn.pm (Subversion)

詳細については、実行を: Ssindex.cmd、 **-でしょうか。** または **-[概要] タブ** (詳細なヘルプ) オプションか、スクリプトを調べます。
