# <a name="analyzing-a-kernel-mode-dump-file-with-kanalyze"></a>KAnalyze によるカーネルモード ダンプ ファイルの分析


## <span id="ddk_analyzing_a_kernel_mode_dump_file_with_kanalyze_dbg"></span><span id="DDK_ANALYZING_A_KERNEL_MODE_DUMP_FILE_WITH_KANALYZE_DBG"></span>


カーネル メモリ領域のアナライザー (KAnalyze、Kanalyze.exe) は、カーネル モードのダンプ ファイルを検証できるもう 1 つのツールです。

KAnalyze およびそのドキュメントは、OEM サポート ツール パッケージの一部です。

これらのツールをダウンロードするには[Microsoft サポート記事 253066](https://go.microsoft.com/fwlink/p/?linkid=241339)し、そのページの指示に従います。
