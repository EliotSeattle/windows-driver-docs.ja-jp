# <a name="logger"></a>Logger


## <span id="ddk_logger_dtoolq"></span><span id="DDK_LOGGER_DTOOLQ"></span>


ロガーのツールは、2 つの異なる媒体によってアクティブにできます。 1 つの方法では、スタンドアロンの Logger.exe プログラムを使用します。 CDB または WinDbg を開始して Logexts.dll デバッガー拡張機能を使用すると、もう一方が。 これらのメソッドの両方により、同じ種類のログ出力が生成されます。 ただし、NT ベースのオペレーティング システムで使用する最適な手段は CDB または WinDbg Logexts.dll 拡張機能のコマンドを使用します。

ロガーの車両の動作も、ロガーの電源とデバッガーの全機能をデバッガーを使用して用意されています。
