# <a name="application-verifier"></a>アプリケーション検証ツール


Application Verifier (AppVerif.exe) は、*動的検証*ユーザー モード アプリケーション用のツール。 このツールは、アプリケーションの実行、アプリケーションにさまざまな負荷とテストをアプリケーションの実行または設計で潜在的なエラーに関するレポートを生成します。 アプリケーションのアクションを監視します。

Application Verifier は、ユーザー モード ドライバーなど、マネージ コードに基づいていない任意のユーザー モード アプリケーション内のエラーを検出できます。 標準的なアプリケーションのテストまたはドライバー テスト中に検出を困難に可能性のあるプログラミング エラーを検出します。

ユーザー モード デバッガーを使用して単独でまたは組み合わせて Application Verifier を使用できます。 現在のユーザーは、コンピューターの Administrators グループのメンバーである必要があります。

Application Verifier は、Windows のツールのデバッグには含まれません。 Application Verifier は、Windows ソフトウェア開発キット (SDK) に含まれています。 ダウンロードと Windows SDK のインストールに関する情報を検索する[ここ](https://go.microsoft.com/fwlink/p?LinkID=271979)します。 Windows SDK をインストールした後 Appverif.exe Appverif.chm にが Windows\\System32 します。 Application Verifier を開始するには、コマンド プロンプト ウィンドウを開き、入力**appverif**します。

Application Verifier ツール、独自のドキュメントが付属します。 ドキュメントを読み取り、アプリケーション検証ツールの起動との間、**ヘルプ** メニューのをクリックして**ヘルプ**、またはキーを押します**F1**します。
