# <a name="view--source-language-file-extensions"></a>ビュー |ソース言語ファイルの拡張機能


をクリックして**ソース言語のファイル拡張子**上、**ビュー** WinDbg をソース ファイル名拡張子として認識するファイル名拡張子を制御するメニュー。 関連付けられているファイル名の拡張機能なプログラミング言語を指定することもできます。

### <a name="span-iddialogboxspanspan-iddialogboxspandialog-box"></a><span id="dialog_box"></span><span id="DIALOG_BOX"></span>ダイアログ ボックス

クリックすると**ソース言語のファイル拡張子**、**ソース言語のファイル拡張子** ダイアログ ボックスが表示されます。 このダイアログ ボックスで追加または内のカーソルを挿入することで、ファイル名拡張子を削除、**拡張機能と言語**ボックスと、適切な情報を入力します。 各ファイル名拡張子を適切なプログラミング言語を指定することを確認します。 たとえば、 **cxx である C + + =** .cxx 拡張子がソース ファイルであると、対応するその拡張子を持つすべてのファイルの言語のプログラミングが C++ であることを示します。 をクリックして **[ok]** 、変更を実装する**キャンセル**任意の変更を破棄します。
