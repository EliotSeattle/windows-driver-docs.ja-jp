# <a name="ending-a-debugging-session-in-visual-studio"></a>Visual Studio のデバッグ セッションの終了

> [!IMPORTANT]
> この機能は、Windows 10 バージョン 1507、以降のバージョンの WDK でご利用いただけません。
>

Microsoft Visual studio でデバッグ セッションを終了、**デバッグ** メニューの 選択**デバッグの停止**します。
