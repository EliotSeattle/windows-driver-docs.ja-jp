# <a name="amli-dh"></a>! amli dh


**! Amli dh**拡張機能は、AML インタープリターの内部ヒープのブロックを表示します。

構文

```dbgcmd
!amli dh [HeapAddress]
```

## <a name="span-idddkamlidhdbgspanspan-idddkamlidhdbgspanparameters"></a><span id="ddk__amli_dh_dbg"></span><span id="DDK__AMLI_DH_DBG"></span>パラメーター


<span id="_______HeapAddress______"></span><span id="_______heapaddress______"></span><span id="_______HEAPADDRESS______"></span> *HeapAddress*   
ヒープ ブロックのアドレスを指定します。 これを省略した場合は、グローバルのヒープが表示されます。
