# <a name="graphics-driver-extensions-gdikdxdll"></a>グラフィック ドライバー拡張機能 (Gdikdx.dll)


## <span id="ddk_graphics_driver_extensions_gdikdx_dll__dbg"></span><span id="DDK_GRAPHICS_DRIVER_EXTENSIONS_GDIKDX_DLL__DBG"></span>


グラフィックス ドライバー インターフェイス (GDI) のデバッグに役立つ拡張機能のコマンドは、Gdikdx.dll で確認できます。

この拡張機能の Windows 2000 バージョン w2kfre と w2kchk ディレクトリに DLL が表示されます。 この DLL XP 用の Windows のバージョンまたは以降のバージョンの Windows はありません。
