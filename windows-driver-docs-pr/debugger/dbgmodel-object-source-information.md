# <a name="source-information-objects"></a>オブジェクトの情報のソース 
## <a name="summary"></a>概要
マシン語命令に関連する情報のソース オブジェクトを使用してソース コード。
## <a name="object-properties"></a>オブジェクトのプロパティ
|名前|説明|
|--- |--- |
|FunctionName|記述されている関数の名前。|
|FunctionAddress|記述されている関数のベース アドレス。|
|FunctionOffset|関数の記述されている場所へのオフセット。|
