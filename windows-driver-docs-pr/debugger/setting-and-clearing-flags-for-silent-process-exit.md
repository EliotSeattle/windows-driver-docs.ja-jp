# <a name="configuring-silent-process-exit-monitoring"></a>プロセスの暗黙の終了の監視の構成


Windows 7 以降で、使用して、**サイレント プロセス終了** タブを有効にし、プロセスの終了をサイレントの監視を構成します。

## <span id="ddk_setting_and_clearing_image_file_flags_dtools"></span><span id="DDK_SETTING_AND_CLEARING_IMAGE_FILE_FLAGS_DTOOLS"></span>


指定する設定、**サイレント プロセスの終了** タブがレジストリに保存されに変更するまでの有効なままです。

**有効にしてサイレント プロセス終了の監視を構成するには**

1.  をクリックして、**サイレント プロセスの終了**タブ。

    次のスクリーン ショットは、**サイレント プロセスの終了** タブでは、Windows 8。

    ![windows vista のイメージ ファイル タブのスクリーン ショット ](images/gflagssilentprocessexit01.png)

2.  **イメージ**ボックス、ファイル名拡張子を含めて、実行可能ファイルの名前を入力して TAB キーを押します。

    上のチェック ボックスがアクティブになります、**サイレント プロセスの終了**タブ。

3.  選択するか、チェック ボックスをオフにすると、テキスト ボックスに値を入力して、設定を指定します。

4.  すべての設定を指定すると、クリックして**適用**します。
