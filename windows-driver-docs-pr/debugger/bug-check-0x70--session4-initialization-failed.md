# <a name="bug-check-0x70-session4initializationfailed"></a>バグのチェックは 0x70 です。SESSION4\_初期化\_失敗


SESSION4\_初期化\_失敗のバグ チェックが 0x00000070 の値を持ちます。 このバグ チェックでは、Microsoft Windows オペレーティング システムの初期化が失敗したことを示します。
