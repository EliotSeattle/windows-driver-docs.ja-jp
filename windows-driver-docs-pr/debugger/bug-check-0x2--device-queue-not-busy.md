# <a name="bug-check-0x2-devicequeuenotbusy"></a>バグ チェック 0x2:デバイス\_キュー\_いない\_ビジー


デバイス\_キュー\_いない\_ビジー状態のバグ チェックが 0x00000002 の値を持ちます。

このバグ チェックが非常に少ない回数が表示されます。
