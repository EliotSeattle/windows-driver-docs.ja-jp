# <a name="usbkdusbpdoxls"></a>!usbkd.usbpdoxls


**! Usbkd.usbpdoxls**コマンドは、USB ハブのドライバーによって作成されたすべての物理デバイス オブジェクト (Pdo) に関する情報を表示します。 各 pdo は、このコマンドは、コンマ区切り値の行を表示します。

```dbgcmd
!usbkd.usbpdoxls
```
