# <a name="dbgerr003-mismatched-symbols"></a>dbgerr003:Mismatched Symbols


## <span id="ddk_dbgerr003_dbg"></span><span id="DDK_DBGERR003_DBG"></span>


デバッガーのエラー **dbgerr003** 、メッセージが表示されます"*ファイル*記号が一致していません"。 このエラーは、DbgHelp にあるファイルで表されるシンボル ファイルが、シンボル ファイルは、正しいバージョンではありません。 またはその DbgHelp がシンボル ファイルが適切なバージョンを確認することはできませんのことを示します。

デバッガーでは、パス内の他の要件に応じて、このエラーに関係なく、指定されたシンボル ファイルを読み込む可能性があります。
