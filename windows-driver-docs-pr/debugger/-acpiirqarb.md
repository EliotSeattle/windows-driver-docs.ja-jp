# <a name="acpiirqarb"></a>!acpiirqarb


**! Acpiirqarb**拡張機能は、Advanced Configuration and Power Interface (ACPI) IRQ アービター構造体のシステムの割り込みコント ローラーの入力とプロセッサへの I/O デバイスの構成を含む内容を表示します。ディスパッチ テーブル (IDT) エントリを中断します。

```dbgcmd
!acpiirqarb
```
