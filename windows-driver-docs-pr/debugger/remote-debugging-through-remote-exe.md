# <a name="remote-debugging-through-remoteexe"></a>Remote.exe によるリモート デバッグ


## <span id="ddk_remote_debugging_through_remote_exe_dbg"></span><span id="DDK_REMOTE_DEBUGGING_THROUGH_REMOTE_EXE_DBG"></span>


Remote.exe を通じてリモートにデバッグするには、デバッガーをリモート コンピューターで実行されていると、ローカル コンピューターで remote.exe ツールを実行する必要があります。

リモート コンピューターとローカル コンピューターに任意の Windows オペレーティング システムを実行できます。

**注**   remote.exe はコンソール アプリケーションにのみ動作するため、WinDbg をリモートで制御するのには使用できません。
