異常な\_リセット\_検出されたバグ チェックが 0x00000147 の値を持ちます。 これは、Windows に異常のリセットが生じたことを示します。 コンテキストまたは例外のレコードが保存されないと、バグチェック コールバックは呼び出されませんでした。
