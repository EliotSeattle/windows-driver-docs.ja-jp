# <a name="preferdml-prefer-debugger-markup-language"></a>.prefer\_dml (デバッガーのマークアップ言語の優先)


**.Prefer\_dml**コマンドは、デバッガーのマークアップ言語 (DML) の形式で出力を提供することができるコマンドの既定の動作を設定します。

```dbgcmd
.prefer_dml 0
.prefer_dml 1
```

## <a name="span-idparametersspanspan-idparametersspanspan-idparametersspanparameters"></a><span id="Parameters"></span><span id="parameters"></span><span id="PARAMETERS"></span>パラメーター


<span id="_______0"></span> 0  
既定では、すべてのコマンドはプレーン テキストの出力を提供します。

<span id="_______1"></span> 1  
既定では、DML の出力を提供することができるコマンドは DML の出力を提供します。
