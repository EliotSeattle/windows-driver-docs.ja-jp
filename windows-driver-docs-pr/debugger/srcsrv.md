# <a name="srcsrv"></a>SrcSrv


SrcSrv ツール (Srcsrv.dll) は、アプリケーションの構築に使用されたソース ファイルの正確なバージョンを取得するクライアントを使用できます。 モジュールのソース コードは、バージョン間と年にわたるに変更するためには、対象のモジュールのバージョンのビルド時に存在していたソース コードを見る必要があります。

SrcSrv は、ソース管理から、適切なファイルを取得します。 SrcSrv を使用するには、ソースにインデックス付けされたアプリケーションをされている必要があります。
