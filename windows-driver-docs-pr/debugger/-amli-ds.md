# <a name="amli-ds"></a>!amli ds


**! Amli ds**拡張機能により、AML スタックが表示されます。

構文

```dbgcmd
    !amli ds [/v] [Address] 
```

## <a name="span-idddk__amli_ds_dbgspanspan-idddk__amli_ds_dbgspanparameters"></a><span id="ddk__amli_ds_dbg"></span><span id="DDK__AMLI_DS_DBG"></span>パラメータ


<span id="________v______"></span><span id="________V______"></span> **/v**   
表示を詳細にします。 

<span id="_______Address______"></span><span id="_______address______"></span><span id="_______ADDRESS______"></span>*アドレス*   
スタックが必要なコンテキストブロックのアドレスを指定します。 *Address*を省略した場合は、現在のコンテキストが使用されます。
