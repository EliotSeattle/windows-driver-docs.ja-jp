# <a name="symbols-for-windows-debugging-windbg-kd-cdb-ntsd"></a>Windows デバッグのシンボル (WinDbg、KD、CDB、NTSD)


Windows デバッガー (WinDbg、KD、CDB、NTSD) のシンボルは、パブリック シンボル サーバーで取得できます。 

## <span id="symbols"></span><span id="SYMBOLS"></span>


このセクションの内容:

[シンボルの概要](introduction-to-symbols.md)

[デバッグ用シンボルへのアクセス](accessing-symbols-for-debugging.md)

[デバッガーによるシンボルの認識方法](how-the-debugger-recognizes-symbols.md)

[デバッグ中のシンボルに関する問題](symbol-problems-while-debugging.md)

これらのトピックでは、シンボルの概要、デバッグ セッション中にシンボルにアクセスする方法、デバッガーのシンボル オプションとシンボルの一致を制御する方法、およびデバッグ中に遭遇するシンボル関連の問題に対応する方法について説明します。

独自のプログラムと Windows のシンボルにアクセスするようにデバッガーを構成するだけの場合は、情報がより簡潔な、概要に関するトピックである「[シンボル パス](symbol-path.md)」および「[シンボル サーバーの使用](using-a-symbol-server.md)」をお読みいただく方が早いかもしれません。
