# <a name="symbols-for-windows-debugging-windbg-kd-cdb-ntsd"></a>Windows デバッグ用シンボル (WinDbg、KD、CDB、NTSD)


Windows デバッガー (WinDbg、KD、CDB、NTSD) のシンボルは、パブリックシンボルサーバーから使用できます。

独自のプログラムと Windows 用のシンボルにアクセスするようにデバッガーを構成するだけの場合は、詳細な入門トピックの[シンボルパス](symbol-path.md)と[シンボルサーバーの使用](using-a-symbol-server.md)をお勧めします。
