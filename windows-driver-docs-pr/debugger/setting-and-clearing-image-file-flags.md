# <a name="setting-and-clearing-image-file-flags"></a>イメージ ファイル フラグの設定とクリア


## <span id="ddk_setting_and_clearing_image_file_flags_dtools"></span><span id="DDK_SETTING_AND_CLEARING_IMAGE_FILE_FLAGS_DTOOLS"></span>


イメージ ファイルの設定では、コマンドが完了した後に開始する、指定したプログラムのインスタンスに影響します。 レジストリに保存され、それらを変更するまでの有効なまま。

**設定し、イメージ ファイルのレジストリ フラグをクリア**

1.  をクリックして、**イメージ ファイル**タブ。

    次のスクリーン ショットは、**イメージ ファイル**Windows Vista タブ。

    ![windows vista のイメージ ファイル タブのスクリーン ショット ](images/gflags-image.png)

2.  **イメージ**ボックスで、実行可能ファイルまたはファイル名拡張子を含む DLL の名前を入力し、TAB キーを押します。

    上のチェック ボックスがアクティブになります、**イメージ ファイル**タブ。

3.  設定またはフラグをオンまたはオフ、フラグに関連付けられているチェック ボックスをオフにします。

4.  選択またはクリアするフラグはすべて、クリックして**適用**します。
