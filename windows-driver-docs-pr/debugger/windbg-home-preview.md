# <a name="windbg-preview---home-menu"></a>WinDbg プレビュー-ホームメニュー

![ビットパターンを使用した windbg プレビューの小さいロゴ](images/windbgx-preview-logo.png)

このトピックでは、ホームメニューを使用する方法について説明します。

![デバッガーの [表示] メニュー](images/windbgx-home-menu.png)

## <a name="flow-control"></a>フロー制御

[*フロー制御*] ボタンを使用して、接続されたデバッグターゲットを中断し、ターゲットでコードの実行を再開して、コードにステップインまたはステップアウトします。

## <a name="reverse-flow-control"></a>フロー制御の反転

時間をさかのぼって移動するには、[*反転フロー制御*] ボタンを使用します。 詳細については、「[Time Travel Debugging - 概要](time-travel-debugging-overview.md)」を参照してください。

## <a name="end"></a>[終了]

[*終了*] ボタンを使用して、再起動、デタッチ、およびデバッグを停止します。

## <a name="preferences"></a>[設定]

*基本設定*ボタンを使用して、ソースコードとアセンブリビューを切り替えたり、 **[設定]** メニューにアクセスしたりします。

## <a name="help-support"></a>ヘルプ (サポート)

*ヘルプ*ボタンを使用して、次の操作を行います。
- ローカルヘルプを確認する-オフライン
- オンラインヘルプ-最新

### <a name="feedback-hub"></a>フィードバック Hub

- フィードバックの送信 (WinDbg を改善するためのフィードバックの送信の詳細については、「[フィードバックの提供](debugging-using-windbg-preview.md#providing-feedback)」を参照してください)。
