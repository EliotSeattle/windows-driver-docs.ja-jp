# <a name="windbg-preview---home-menu"></a>WinDbg Preview - ホーム メニュー 

このトピックでは、[ホーム] メニューを操作する方法について説明します。

![デバッガーでの表示 メニュー](images/windbgx-home-menu.png)


## <a name="flow-control"></a>フロー制御

使用して、*フロー制御*に接続されているデバッグ ターゲットでは、分割ボタンを選択し、ターゲット上でコードが実行を再開して、コードからステップします。

## <a name="end"></a>[終了]

使用して、*エンド*を再起動して、ボタンのデタッチし、デバッグを停止します。

## <a name="source-mode"></a>ソース モード

使用*ソース モード*を切り替えるボタンのソース コードとアセンブリのビュー。

## <a name="help-support"></a>ヘルプ (サポート)

使用*ヘルプ*以下を実行するボタン。
- ヘルプを確認してください。
- フィードバックの送信 (WinDbg を向上させるフィードバックの送信の詳細については、次を参照してください[フィードバックを提供する](debugging-using-windbg-preview.md#providing-feedback)。)。
