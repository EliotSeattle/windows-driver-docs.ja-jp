# <a name="logger-extensions-logextsdll"></a>Logger 拡張機能 (Logexts.dll)


## <span id="ddk_logger_extensions_logexts_dll__dbg"></span><span id="DDK_LOGGER_EXTENSIONS_LOGEXTS_DLL__DBG"></span>


Logger と LogViewer ツールに関連する拡張機能のコマンドは、Logexts.dll で確認できます。

この DLL は、winext ディレクトリに表示されます。 すべての Windows オペレーティング システムで使用できます。
