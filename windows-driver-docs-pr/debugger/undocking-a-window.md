# <a name="undocking-a-window"></a>ウィンドウのドッキング解除


## <span id="ddk_undocking_a_window_dbg"></span><span id="DDK_UNDOCKING_A_WINDOW_DBG"></span>


ウィンドウのドッキングを解除し、そのフローティング ウィンドウは、次のいずれかの操作を行います。

-   ウィンドウのタイトル バーをダブルクリックします。

-   ウィンドウのタイトル バーを右クリックし、タブ付きのコレクションの一部である場合は、ウィンドウのタブを右クリックして、またはウィンドウの右上隅にあるアイコンをクリックしてショートカット メニューを開き、クリックして**ドッキング解除**します。

-   WinDbg ウィンドウで、上、**ウィンドウ** メニューのをクリックして**すべてドッキング解除**します。 このコマンドは、すべてのドッキング ウィンドウをフローティング ウィンドウに変更します。

上記の方法のいずれかによって、ウィンドウをドッキング解除するときに、ウィンドウを前のドッキング位置を返します。

ドッキング ウィンドウのタイトル バーをクリックしてドラッグすることもできます。 このアクションを使用すると、別のドッキング位置にウィンドウを移動またはドッキング解除できます。 (フローティング ウィンドウの新しい位置にドラッグすることとまったく同じ機能の新しい位置にドッキング ウィンドウをドラッグします。 フローティング ウィンドウを新しい位置にドラッグします詳細については、次を参照してください。[ウィンドウをドッキング](docking-a-window.md)。)。

ドッキングを解除またはこれらのメソッドのいずれかで、タブ付きウィンドウをドラッグするときは、タブ付きのコレクション内のアクティブなウィンドウだけが移動します。
