# <a name="windbg-preview---installation"></a>WinDbg プレビュー - インストール


## <a name="installation"></a>インストール

このセクションでは、プレビューの WinDbg デバッガーをインストールする方法について説明します。

プレビューの WinDbg デバッガーは、Microsoft Store で使用できます。 インストールする Windows 10 Anniversary Update が必要です。 これをインストールする "WinDbg Preview"を検索して Microsoft Store を開くかクリックして[ここ](
https://www.microsoft.com/store/apps/9pgjgd53tn86)します。

ダウンロードしてインストールすることで、アプリが、配置されているをクリックします。


## <a name="checking-for-updates"></a>更新プログラムの確認

1. ストア アプリを開き、アカウントの画像検索ボックスの横にあるをクリックします。 

2. をクリックして**がダウンロードされ更新**更新プログラムを確認します。

3. ダウンロードおよび更新プログラム ページで、次のように選択します。**更新プログラムを入手**します。


## <a name="debugger-coexistence"></a>デバッガーの共存  

WinDbg のプレビューは、同時に両方のバージョンで動作するため、同じマシン上のクラシック WinDbg デバッガーと共存します。
