# <a name="bug-check-0x152-invalidpushlockflags"></a>バグ チェック 0x152:無効な\_プッシュ\_ロック\_フラグ


無効な\_プッシュ\_ロック\_フラグのバグ チェックが 0x00000152 の値を持ちます。 これは、プッシュ ロック Api のいずれか 1 つのフラグが有効でないことを示します。

| パラメーター | 説明                              |
|-----------|------------------------------------------|
| 1         | 呼び出し元によって指定された無効なフラグ |
| 2         | プッシュのロックのアドレス             |
| 3         | 予約済み                                 |
