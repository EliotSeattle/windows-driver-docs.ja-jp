# <a name="the-srctool-utility"></a>SrcTool ユーティリティ


(Srctool.exe) を付けて SrcTool ユーティリティは、すべてのファイルが .pdb ファイル内のインデックスを一覧表示します。 各ファイルの完全なパス、ソース管理サーバー、およびファイルのバージョン番号が一覧表示します。 この情報は、参照は使用できます。

.Pdb ファイルから生のソース ファイル情報を一覧表示を付けて SrcTool を使用することもできます。 これを行うには、使用、 **-s**コマンド ライン スイッチします。

他のオプションを付けて SrcTool が。 使用して、**でしょうか。** 表示を切り替えます。 最も関心のある、バージョン コントロールからすべてのソース ファイルを抽出するこのユーティリティを使用できることができます。 これは、 **-x**スイッチします。

**注**  このプログラムの以前のバージョンは、ファイルを抽出するときに、現在のディレクトリの下の src をという名前のディレクトリを作成します。 これは、ケースではなくなりました。 Src ディレクトリを使用する場合は、自分で作成し、そのディレクトリからコマンドを実行し、する必要があります。
