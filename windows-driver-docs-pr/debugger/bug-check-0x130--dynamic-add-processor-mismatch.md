# <a name="bug-check-0x130-dynamicaddprocessormismatch"></a>バグ チェック 0x130:動的\_追加\_プロセッサ\_が一致しません


動的\_追加\_プロセッサ\_の不一致のバグ チェックが 0x00000130 の値を持ちます。 このバグチェックでは、システムに追加された新しいプロセッサの現在の構成と互換性がないことを示します。
