# <a name="bthkdbthdevinfo"></a>!bthkd.bthdevinfo


**! Bthkd.bthdevinfo**コマンドは、指定された作成 BTHENUM デバイス PDO に関する情報を表示します。

```dbgsyntax
!bthkd.bthdevinfo addr
```

<span id="_______addr______"></span><span id="_______ADDR______"></span> *addr*   
BTHENUM のアドレスは、デバイスの PDO 拡張機能を作成します。
