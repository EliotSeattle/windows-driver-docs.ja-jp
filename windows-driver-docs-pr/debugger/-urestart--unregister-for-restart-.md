# <a name="urestart-unregister-for-restart"></a>.urestart (再起動の登録解除)


**.Urestart**コマンドは、コンピューターの再起動またはアプリケーション エラーが発生した場合の再起動のデバッグ セッションを登録解除します。

```dbgcmd
.urestart
```

<a name="remarks"></a>注釈
-------

このコマンドは、管理者特権でのデバッガー セッションは機能しません。
