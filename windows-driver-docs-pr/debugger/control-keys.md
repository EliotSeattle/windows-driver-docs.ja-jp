# <a name="control-keys"></a>Ctrl キー


## <span id="ddk_control_keys_dbg"></span><span id="DDK_CONTROL_KEYS_DBG"></span>


参照のこのセクションでは、デバッガーで使用できるさまざまなコントロール キーについて説明します。

これらは、キーの作業で KD と場合によっては、CDB を制御します。 これらの多くは、WinDbg (ctrl キーだけではなく、CTRL + ALT キーを使用) でも動作します。

WinDbg も、CTRL キー、ALT キー、および F キーを押して、キーとして使用ショートカット メニュー オプションを切り替えます。 参照してください[ショートカット キー](keyboard-shortcuts.md)その意味の一覧についてはします。
