# <a name="ndiskdnwadapter"></a>!ndiskd.nwadapter


**! Ndiskd. nwadapter**拡張機能には、1つまたは複数の nwifi に関する情報が表示されます。構造を調整します。 パラメーターを使用せずにこの拡張機能を実行すると、! ndiskd はすべての nwifi!構造を調整します。

```console
!ndiskd.nwadapter [-handle <x>] 
```
