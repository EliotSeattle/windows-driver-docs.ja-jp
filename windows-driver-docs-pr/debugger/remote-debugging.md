# <a name="remote-debugging"></a>リモート デバッグ


## <span id="ddk_remote_debugging_dbg"></span><span id="DDK_REMOTE_DEBUGGING_DBG"></span>


2 台のコンピューターは、リモート ユーザー モードのデバッグ:*クライアント*と*server*します。 サーバーは、デバッグするアプリケーションを実行するコンピューターです。 また、サーバーは、ユーザー モード デバッガーまたはプロセス サーバーを実行します。 クライアントは、リモートでデバッグ セッションを制御するコンピューターです。

3 台のコンピューターは、リモートのカーネル モードのデバッグ:*クライアント*、 *server*、および*ターゲット コンピューター*します。 ターゲット コンピューターでは、デバッグするコンピューターです。 サーバーは、カーネル デバッガーまたは KD 接続のサーバーを実行するコンピューターです。 クライアントは、リモートでデバッグ セッションを制御するコンピューターです。
