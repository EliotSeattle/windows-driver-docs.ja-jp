# <a name="bug-check-0x1b-pfnsharecount"></a>バグ チェック 0x1B:PFN\_共有\_数


PFN\_共有\_数のバグ チェックが 0x0000001B の値を持ちます。

このバグ チェック コードは、Windows オペレーティング システムでは使用されません。
