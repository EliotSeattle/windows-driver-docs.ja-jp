# <a name="f"></a>F


<span id="free_build"></span><span id="FREE_BUILD"></span>**無料ビルド**  
NT ベースのオペレーティングシステムごとに、次の2つの異なるビルドが存在します。

-   Windows の (または) は、オペレーティングシステムのエンドユーザーバージョンです。 システムとドライバーが完全に最適化された状態でビルドされ、デバッグアサートが無効になり、デバッグ情報がバイナリから削除されます。 システムとドライバーの空き容量は小さく、高速で、使用するメモリも少なくなります。
-   Windows の (または) は、テストおよびデバッグの支援として機能します。 詳細については、「チェック済みビルド」を参照してください。

オペレーティングシステムの無償ビルドを含む配布メディアには特別なラベルがありません。つまり、無料のビルドを含む CD には、Windows のバージョン名と共にラベルが付けられ、ビルドの種類への参照は含まれません。

<span id="first_chance_exception"></span><span id="FIRST_CHANCE_EXCEPTION"></span>**初回例外**  
例外を処理する最初の機会。 例外が最初の営業案件のハンドラーによって処理されない場合、ハンドラーには2つ目の確率が与えられます。
