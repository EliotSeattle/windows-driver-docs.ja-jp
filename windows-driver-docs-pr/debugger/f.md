# <a name="f"></a>F


<span id="free_build"></span><span id="FREE_BUILD"></span>**無料のビルド**  
各 NT ベースのオペレーティング システムの 2 つの異なるビルドが存在します。

-   (または) の Windows オペレーティング システムのエンドユーザーのバージョン。 システムとドライバーは、最大限の最適化で構築された、アサートのデバッグは無効にし、デバッグ情報がバイナリから削除されます。 無料のシステムとドライバーが小さく、高速より少ないメモリを使用しています。
-   Windows の役割をテストおよびデバッグに役立つとしての (または)。 詳細については、チェック ビルドを参照してください。

配布メディアを格納しているオペレーティング システムの無料のビルドには、任意の特殊なラベルはありません。--つまり、無料のビルドを含む CD だけという名前になります、Windows のバージョンの名前とへの参照がビルドの種類。

<span id="first_chance_exception"></span><span id="FIRST_CHANCE_EXCEPTION"></span>**初回例外**  
例外を処理する最初のチャンスです。 最初の機会に、ハンドラーによって例外が処理されない場合、ハンドラーには、2 番目のチャンスが与えられます。
