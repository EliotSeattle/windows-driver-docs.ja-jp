# <a name="finding-a-user-mode-memory-leak"></a>ユーザー モード メモリ リークを検出

ユーザー モードのメモリ リークの原因を特定するのにには、次の手法を使用します。

- [パフォーマンス モニターを使用して、ユーザー モード メモリ リークの検出](using-performance-monitor-to-find-a-user-mode-memory-leak.md)
- [UMDH を使用してユーザー モードのメモリ リークを検出するには](using-umdh-to-find-a-user-mode-memory-leak.md)

最初の手法は、メモリをリークしているプロセスを決定します。 のどのプロセスの関係を確認した後、2 番目の手法は問題がある特定のルーチンを判断できます。
