# <a name="setting-and-clearing-system-wide-flags"></a>システム全体のフラグの設定とクリア


## <span id="ddk_setting_and_clearing_system_wide_flags_dtools"></span><span id="DDK_SETTING_AND_CLEARING_SYSTEM_WIDE_FLAGS_DTOOLS"></span>


システム全体のレジストリ設定では、Windows で実行されているすべてのプロセスに影響します。 レジストリに保存され、それらを変更するまでの有効なまま。

**設定し、システム全体のレジストリ フラグをクリア**

1.  をクリックして、**システム レジストリ**タブ。

    次のスクリーン ショットは、Windows Vista では、システム レジストリ タブを示します。![windows vista では、[レジストリ] タブのスクリーン ショット](images/gflags-registry.png)

2.  設定またはフラグをオンまたはオフ、フラグに関連付けられているチェック ボックスをオフにします。

3.  選択またはすべてのフラグをオフになっている、クリックして**適用**します。

4.  変更を有効にする Windows を再起動します。
