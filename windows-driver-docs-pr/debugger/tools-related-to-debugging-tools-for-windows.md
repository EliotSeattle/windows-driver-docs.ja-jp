# <a name="tools-related-to-debugging-tools-for-windows"></a>Windows 用デバッグ ツールに関連するツール


このセクションでは、デバッグに関連するが、Windows のツールをデバッグ パッケージに含まれないツールについて説明します。

<span id="Application_Verifier"></span><span id="application_verifier"></span><span id="APPLICATION_VERIFIER"></span>[アプリケーション検証ツール](application-verifier.md)  
アプリケーション アクションの監視、アプリケーションの実行、サブジェクトの負荷とテストをさまざまなアプリケーションとアプリケーションの実行または設計で潜在的なエラーに関するレポートを生成します。 Application Verifier は、Windows 用 Windows ソフトウェア開発キット (SDK) を含まれています。 ダウンロードと Windows の Windows SDK のインストールに関する情報を検索する [ここ](https://go.microsoft.com/fwlink/p?LinkID=271979)します。

<span id="Windows_Error_Reporting"></span><span id="windows_error_reporting"></span><span id="WINDOWS_ERROR_REPORTING"></span>[Windows エラー報告](windows-error-reporting.md)  
Windows エラー報告 (WER) ユーザー モードのコードで例外やその他のエラーが発生した場合、ユーザー モード ダンプ ファイルを書き込むを構成することができます。 WER は、Windows Vista および Windows の以降のバージョンに含まれます。
