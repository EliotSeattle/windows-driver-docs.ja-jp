# <a name="usbkdusbdevobj"></a>!usbkd.usbdevobj


**! Usbkd.usbdevobj**コマンドは、USB デバイス オブジェクトからの情報を表示します。

```dbgcmd
!usbkd.usbdevobj DeviceObject
```
