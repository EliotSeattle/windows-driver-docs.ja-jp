# <a name="debug--modules"></a>デバッグ |モジュール


## <span id="ddk_debug_modules_dbg"></span><span id="DDK_DEBUG_MODULES_DBG"></span>


クリックして**モジュール**上、**デバッグ**読み込まれたモジュールの現在の一覧を表示するメニュー。

### <a name="span-iddialogboxspanspan-iddialogboxspandialog-box"></a><span id="dialog_box"></span><span id="DIALOG_BOX"></span>ダイアログ ボックス

クリックすると**モジュール**、**モジュール一覧** ダイアログ ボックスが表示されます。 このダイアログ ボックスでは、メモリに現在読み込まれているすべてのモジュールが一覧表示します。

**モジュール一覧**は、次の列に分かれています。

-   **名前**列が、モジュール名を指定します。

-   **開始**と**エンド**列は、モジュールのメモリのイメージの最初と最後のアドレスを指定します。

-   **タイムスタンプ**列は、モジュールのビルド日時を指定します。

-   **チェックサム**列は、チェックサム値を指定します。

-   **シンボル**列には、このモジュールを使用するシンボルに関する情報が表示されます。 この列に表示される値の詳細については、次を参照してください。[シンボルの状態の省略形](symbol-status-abbreviations.md)します。

-   **シンボル ファイル**列が関連付けられているシンボル ファイルのパスとファイル名を指定します。 デバッガーがシンボル ファイルの対応でない場合は、実行可能ファイルの名前が指定されました。

列のタイトル バーをクリックした場合は、その列のデータで、表示が並べ替えられます。 再びのタイトル バーをクリックした場合、並べ替え順序を反転させます。

行を選択し、クリックすると**再読み込み**モジュールのシンボル情報が再度読み込まれます。

行を選択して ctrl キーを押しながら C キーを押して、行全体がクリップボードにコピーします。

クリックして**閉じます**このダイアログ ボックスを閉じます。
