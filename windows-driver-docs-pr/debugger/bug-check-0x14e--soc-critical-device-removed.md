# <a name="bug-check-0x14e-soccriticaldeviceremoved"></a>バグ チェック 0x14E:SOC\_重大\_デバイス\_削除しました


SOC\_重大\_デバイス\_削除済みのバグ チェックが 0x0000014E の値を持ちます。 これは、重要な SOC デバイスが予期せず削除されたが失敗したことを示します。
