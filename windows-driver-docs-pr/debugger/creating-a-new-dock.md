# <a name="creating-a-new-dock"></a>新しいドッキング ステーションを作成します。


## <span id="ddk_creating_new_docks_dbg"></span><span id="DDK_CREATING_NEW_DOCKS_DBG"></span>


WinDbg が開始されると、WinDbg ウィンドウは、可能な唯一のドッキング場所です。

新しいドッキング ステーションに上に作成する、**ウィンドウ** メニューのをクリックして**開いてドッキング**します。 このドッキング ステーションとは、最大化、最小化、またはデスクトップ上の他のウィンドウのようにドラッグできる独立したウィンドウです。

使用して新しいドッキング ステーションを作成することも、**新しいドッキングするのには移動**既に開いているウィンドウのショートカット メニューのオプション。 この選択は、ウィンドウを閉じるされ、新しいドッキング ステーションで開きます。

必要に応じて多くのドッキングを作成できます。

ドッキング ステーションとがドッキングされているすべてのデバッグ情報ウィンドウを閉じる をクリックして、**閉じます**ドッキングの右上隅のボタンをクリックします。
