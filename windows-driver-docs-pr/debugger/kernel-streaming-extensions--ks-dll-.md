# <a name="kernel-streaming-extensions-ksdll"></a>カーネルの拡張機能 (Ks.dll) のストリーミング


ストリーミングのカーネル ドライバーと AVStream ドライバーのデバッグに役立つ拡張機能のコマンドは、Ks.dll で確認できます。

これらのリファレンス ページのサンプル出力のほとんどは、フィルターを中心としたサンプル Avssamp.sys をデバッグして生成されました。 このサンプルは、Windows Driver Kit に含まれます。

Windows 2000 で Ks.dll 拡張機能を使用する場合は、w2kfre ディレクトリに kdexts ディレクトリから Ks.dll をコピーする必要があります。

この拡張機能を使用する特別なシンボルが必要です。 詳細については、次を参照してください。、[ツールを Windows のデバッグ](https://go.microsoft.com/fwlink/p/?linkid=8708)Web サイト。

引数なしでデバッガーにコマンドを入力するだけで、このセクションでは多くの拡張機能コマンドの追加情報を取得できます。

詳細については、[ストリーミングのカーネル デバッグ](kernel-streaming-debugging.md)を参照してください。
