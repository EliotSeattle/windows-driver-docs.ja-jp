# <a name="secure-mode"></a>保護モード


## <span id="ddk_secure_mode_dbg"></span><span id="DDK_SECURE_MODE_DBG"></span>


カーネル モードのデバッグを実行している場合は、デバッガーで実行できる*保護モード*します。 これにより、デバッガーが、ホスト コンピューターに影響を与えることを防ぎますまだ対象のコンピュータをデバッグするには、その自由が大幅に低下しません。

リモート クライアントのデバッグ セッションに参加できるようにする場合は、セキュリティで保護されたモードを使うことをお勧めします。
