# <a name="edit--select-all"></a>編集 |[すべて選択] します。


## <span id="ddk_edit_select_all_dbg"></span><span id="DDK_EDIT_SELECT_ALL_DBG"></span>


をクリックして**すべて選択**で、**編集**、アクティブなすべてのテキストを選択するメニュー[デバッガー コマンド ウィンドウ](debugger-command-window.md)、[逆アセンブル ウィンドウ](disassembly-window.md)、 [ソース ウィンドウ](source-window.md)、またはダイアログ ボックス。

このコマンドは、ctrl キーを押しながら A キーを押すと同じです。

### <a name="span-idadditionalinformationspanspan-idadditionalinformationspanadditional-information"></a><span id="additional_information"></span><span id="ADDITIONAL_INFORMATION"></span>追加情報

選択する方法の詳細については、コピー、切り取り、およびテキストを貼り付けるし、についてどのようにこれらの操作は異なるウィンドウを[切り取りと貼り付けのテキスト](cutting-and-pasting-text.md)を参照してください。
