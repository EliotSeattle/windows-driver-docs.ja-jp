# <a name="wdfkdwdfcollection"></a>!wdfkd.wdfcollection


**! Wdfkd.wdfcollection**拡張機能では、すべての WDFCOLLECTION 構造体に格納されているオブジェクトが表示されます。

```dbgcmd
!wdfkd.wdfcollection Handle
```
