# <a name="plotter-driver-user-interface"></a>プロッター ドライバーのユーザー インターフェイス





プロッターのユーザー インターフェイスを採用しています[CPSUI](common-property-sheet-user-interface.md)次の 2 つのプロパティ シートのページを作成します。

-   **デバイス設定**、ユーザーが選択したときに表示されるプリンターのプロパティ シートのページ、**プロパティ**プリンター フォルダーまたはプリンターのウィンドウからメニュー項目。 このページには、プリンターに固有の構成設定が一覧表示されます。

-   **レイアウト**、**用紙/品質**、および**詳細**ユーザーを選択したときに表示される、ドキュメントのプロパティ シートのページ、**ドキュメントの既定値**プリンター フォルダーまたはプリンターのウィンドウから、またはアプリケーションを呼び出すときにメニュー項目、 **PrinterProperties**または**DocumentProperties**関数 (で説明されている、Microsoft Windows SDK のドキュメント)。 このページには、ドキュメントに固有の構成設定が一覧表示されます。

これらのプロパティ シートには、プロッター機能とプロッターのミニドライバーで指定されたオプションが含まれます。 ユーザー オプション値を変更することもできます。

プロッターのユーザー インターフェイスはユーザー モードとして[プリンター インターフェイス DLL](printer-interface-dll.md)します。 CPSUI、と共に、この DLL 内のコードでは、プロパティ シートのページの内容を指定します。 DLL は、どのプロッターのオプションを組み合わせて、ミニドライバーの情報に基づいて制約を適用します。 また、ユーザーがプロッターにインストールされていないオプションを選択しないでください。
