# <a name="configuration"></a>構成


スキーマのパス:\\Printer.Configuration

ノード型: プロパティ

構成プロパティには、デバイスの構成方法に関する情報が含まれています。 メモリ、ハード ディスクがインストールされているかどうか、および双方向の単位がインストールされているかどうかの量。 このデータは、のみ、デバイスの物理的な構成が変更されたとき、通常、実際のハードウェアでの変更によって変更します。

構成プロパティは、すべてのプロパティおよび構成セクション内の値の見出しです。

構成プロパティは次のプロパティの親であります。

[メモリ](memory.md)

[ハード ディスク](harddisk.md)

[DuplexUnit](duplexunit.md)
