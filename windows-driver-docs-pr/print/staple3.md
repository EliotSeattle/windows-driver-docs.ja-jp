# <a name="staple"></a>Staple


スキーマのパス:\\Printer.Finishing.Staple

ノード型: プロパティ

説明: このプロパティには、ホチキス止めのデバイスを記述する情報がすべて含まれます。

ホチキス止めプロパティには、子の値が含まれています。**インストール済み**、の親であると、[場所](location3.md)と[角度](angle2.md)プロパティ。

### <a name="span-idinstalledspanspan-idinstalledspaninstalled"></a><span id="installed"></span><span id="INSTALLED"></span>インストールされています。

スキーマのパス:\\Printer.Finishing.Staple:Installed

ノード型: 値

データ型: BIDI\_BOOL

説明: 印刷デバイスがホチキス止め機能がインストールされているかどうかを決定します。 場合**TRUE**、ホチキス止めの機能がインストールされている; 場合**FALSE**、ホチキス止めの機能がインストールされていません。
