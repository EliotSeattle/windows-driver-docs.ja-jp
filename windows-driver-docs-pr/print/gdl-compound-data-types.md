# <a name="gdl-compound-data-types"></a>GDL 複合データ型


GDL 複合データ型が定義されている[GDL プリミティブ データ型](gdl-primitive-data-types.md)または以前の複合データ型を定義します。 複合型は、配列、合成、および複数のパーソナリティの種類に分類できます。 複合型は、作成時の高い複雑なデータ型のある他の複合型を組み込むことができます。

次の複合データ型は、GDL テンプレートで使用されます。

[配列](array-template-data-types.md)

[複合](composite-template-data-types.md)

[複数のパーソナリティ](multiple-personality-template-data-types.md)
