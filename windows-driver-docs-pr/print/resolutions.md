# <a name="resolutions"></a>解決策


スキーマのパス:*\\Printer.Layout.Resolutions*

ノード型: プロパティ

説明: このプロパティには、デバイスでサポートされている印刷の解像度に関連するすべての値のエントリが含まれています。 すべての解像度はドット/インチ (dpi です)。

解像度のプロパティには、2 つの子の値が含まれています。*CurrentValue*と*サポート*します。

### <a name="span-idcurrentvaluespanspan-idcurrentvaluespancurrentvalue"></a><span id="currentvalue"></span><span id="CURRENTVALUE"></span>*CurrentValue*

スキーマのパス:*\\Printer.Layout.Resolutions:CurrentValue*

ノード型: 値

データ型: BIDI\_INT

説明: デバイス印刷の解像度、ドット/インチ (dpi) での現在の (既定値) の値。

### <a name="span-idsupportedspanspan-idsupportedspansupported"></a><span id="supported"></span><span id="SUPPORTED"></span>*サポートされています*

スキーマのパス:*\\Printer.Layout.Resolutions:Supported*

ノード型: 値

データ型: BIDI\_文字列

説明: A 解像度、ドット/インチ (dpi) でサポートされているすべての値のコンマ区切り一覧。
