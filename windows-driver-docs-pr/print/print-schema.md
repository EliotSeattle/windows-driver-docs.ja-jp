# <a name="print-schema"></a>印刷スキーマ


印刷チケットと印刷機能のテクノロジは基本的に、印刷スキーマに基づきます。 印刷スキーマには、組織と、PrintTicket と PrintCapabilities ドキュメントを使用して、これらのテクノロジの内容を定義します。

印刷スキーマの設計では、特定のキーワードをパブリックの機能をマップしもお客様はイノベーションにベンダーを有効にするプライベート拡張機能をサポートします。 印刷チケット フレームワークと、印刷機能のフレームワークは、印刷スキーマのフレームワーク全体のサブセットです。

印刷機能およびプロパティを記述する共通のスキーマを使用して、特定の印刷ドライバーまたはアプリケーションに接続しなくても印刷機能がアプリケーションにアクセスできます。 共通のスキーマでは、プリンターの機能と属性名と定義の一貫性のあるセットを提供することで、あいまいさの名前と機能も減少します。

印刷スキーマは、Microsoft Windows SDK ドキュメントに記載されています。
