# <a name="deviceinfo-section"></a>DeviceInfo セクション


DeviceInfo セクションでは、 [DeviceInfo](deviceinfo.md)プロパティと、全体的なデバイスに関する情報を提供するいくつかの子の値。

```cpp
Printer
 DeviceInfo
 FriendlyName
 Manufacturer
 ModelName
 Location
    FirmwareVersion
    IEEE1284DeviceID
    NetworkingInfo
 Comment
```
