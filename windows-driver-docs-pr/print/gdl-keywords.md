# <a name="gdl-keywords"></a>GDL キーワード


A *GDL キーワード*トークンのセットの文字から成る\[A ~ Z、a ~ z、0-9、 \*、 \_、ですか?\]します。 慣例により、ほとんどのキーワードはアスタリスクで始まる (\*)。 キーワードの長さに制限はありません。

GDL キーワードは、任意の GDL エントリに必要です。 省略可能なキーワードに続けて[値](gdl-values.md)します。 値は、指定した場合、キーワードと値は、コロン (:) で区切る必要があります。 省略可能な[空白文字](gdl-whitespace-characters.md)キーワードと値の間も表示できます。
