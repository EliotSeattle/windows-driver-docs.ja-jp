# <a name="pnpxid"></a>PnPXID


`PnPXID`コマンド、プラグ アンド プレイ (PnP) 識別子 (ID) 形式で返します関数検出キー、鍵の\_PNPX\_id。 このキーは、サービスまたはデバイスの一意の識別子を返します。 出力バッファーには、識別子を表す NULL で終わる文字列が含まれています。

探索のファンクション キーの詳細については、Microsoft Windows SDK のドキュメントを参照してください。
