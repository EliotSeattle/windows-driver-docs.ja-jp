# <a name="else-conditional-preprocessor-directive"></a>\#それ以外の場合の条件付きプリプロセッサ ディレクティブ


```GDL
#Else:
```

\#Else ディレクティブは、前のセクションの終わりと新しい条件付きセクションの先頭を定義します。 セクションには、コンストラクトの前のセクションが保持されていない場合は保持されます。 それ以外の場合、削除されます。 シンボルは使用されません。
