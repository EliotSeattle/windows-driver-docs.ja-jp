# <a name="miscellaneous-gdl-directives"></a>その他の GDL ディレクティブ


GDL には、その他の 2 つのディレクティブが用意されています。

-   \*IgnoreBlock 値と、コンストラクトの内容を無視する GDL パーサーをによりします。

-   \*% (アスタリスクとパーセント記号) は、行の残りの部分を無視するためにパーサーをによりします。 コメントを追加するため、このディレクティブを使用することができます。
