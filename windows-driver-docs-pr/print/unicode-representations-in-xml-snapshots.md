# <a name="unicode-representations-in-xml-snapshots"></a>XML のスナップショットでの Unicode 表現


GDL パーサーでは、生成されるスナップショットやスキーマで utf-8 または utf-16 を使用して、Unicode を表すことができます。 ユーザーは、対応するインターフェイスの関数で、オプション フラグを使ってこの Unicode を選択できます。 UTF エンコード スナップショットでは、表示しやすくするための任意のテキスト エディターに読み込まれることができ、通常は utf-16 のバージョンよりも小さきます。
