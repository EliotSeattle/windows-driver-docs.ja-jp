# <a name="symbolic-names"></a>シンボル名





シンボリック名とは、識別子として使用される文字列です。 シンボル名には、次の特性があります。

-   最初の文字は、大文字または小文字の文字である必要があります。

-   最初の文字の後に名前はの大文字と小文字、数字、およびアンダー スコア文字で構成できます ( \_ )。

-   名前は大文字小文字を区別します。

-   名前の文字数が制限されています。

シンボル名の数が多いを GPD 言語事前に定義します。 定義済みの名前が存在[プリンター属性](printer-attributes.md)、[コマンド名](command-names.md)、[標準機能](standard-features.md)、[標準オプション](standard-options.md)、および[標準変数](standard-variables.md)します。

シンボル名を定義する[機能をカスタマイズ](customized-features.md)、[オプションをカスタマイズ](customized-options.md)、および[マクロ](macros.md)します。
