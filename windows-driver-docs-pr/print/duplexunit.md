# <a name="duplexunit"></a>Duplex ユニット


スキーマのパス:\\Printer.Configuration.DuplexUnit

ノード型: プロパティ

説明: デバイスで両面印刷ユニットの値のエントリ。

DuplexUnit プロパティには、1 つの子の値が含まれています。インストールされています。

### <a name="span-idinstalledspanspan-idinstalledspan-installed"></a><span id="installed"></span><span id="INSTALLED"></span> インストールされています。

スキーマのパス:\\Printer.Configuration.DuplexUnit:Installed

ノード型: 値

データ型: BIDI\_BOOL

説明: 両面印刷ユニットが、デバイスにインストールされているかどうかを示します。
