# <a name="device-stage-document-device-development-kit"></a>デバイス ステージのドキュメント デバイス開発キット

デバイス ステージ ドキュメント デバイス開発キットには、次の情報が含まれています。

- **Windows 7 デバイス ステージ ドキュメント デバイス開発ガイド**-互換性のあるタスクの追加、状態とその他の作成のトピックの詳細を使用してデバイスを識別する方法のドキュメントのデバイス固有の開発のトピックに関するドキュメントを詳しく説明します。

- **Windows 7 デバイス ステージの計画ワークシート**- ドキュメントのデバイスの計画ワークシートの一覧が必要なし、省略可能な値のために必要な計画および Device Stage を構築する際に発生します。

- **デバイスのチュートリアルを文書化**-デバイス テンプレートで新しいデバイスのカスタム Device Stage エクスペリエンスを作成するドキュメントを編集する方法のステップ バイ ステップ ガイドです。

- **ドキュメント デバイス テンプレート**-ドキュメントのデバイスを Device Stage テンプレート。
