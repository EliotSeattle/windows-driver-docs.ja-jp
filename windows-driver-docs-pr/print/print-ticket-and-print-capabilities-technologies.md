# <a name="print-ticket-and-print-capabilities-technologies"></a>印刷チケットと印刷機能のテクノロジ


Microsoft Windows Vista には、印刷機能と印刷チケットテクノロジが導入されています。 これらの新しいテクノロジでは、より柔軟で拡張可能な印刷構成が提供され、以前の方法よりもドキュメントの外観が一貫して印刷されます。 これらのテクノロジは、Microsoft Windows Presentation Foundation (WPF) と Microsoft Win32 アプリケーションの両方で使用できます。また、グラフィックスデバイスインターフェイス (GDI) に基づく XPSDrv 印刷ドライバーと印刷ドライバーで使用することもできます。
