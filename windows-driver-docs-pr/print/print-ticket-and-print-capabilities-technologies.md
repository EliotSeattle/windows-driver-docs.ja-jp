# <a name="print-ticket-and-print-capabilities-technologies"></a>印刷チケットと印刷機能のテクノロジ


Microsoft Windows Vista には、印刷機能および印刷チケット テクノロジが導入されています。 これらの新しいテクノロジより柔軟で拡張可能なプリンターの構成と前のメソッドよりもより一貫性のある印刷ドキュメントの外観を提供します。 これらのテクノロジは、Microsoft Windows Presentation Foundation (WPF) と Microsoft Win32 アプリケーションの両方で使用できると XPSDrv ドライバーとグラフィックス デバイス インターフェイス (GDI) に基づくドライバーを印刷します。
