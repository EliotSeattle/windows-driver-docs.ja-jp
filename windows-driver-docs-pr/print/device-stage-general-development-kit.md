# <a name="device-stage-general-development-kit"></a>デバイス ステージの一般的な開発キット


デバイス ステージの一般的な開発キットには、次の情報が含まれています。

-   **Windows 7 デバイス ステージのデザイン ガイド**-お客様の会社の企業を反映する Device Stage エクスペリエンスを作成する方法と製品ブランド。

-   **Windows 7 デバイス ステージ開発ガイド**- Device Stage エクスペリエンスを作成するために必要な一般的な手順を使用して、Device Stage 開発プロセスの概要について説明します。

-   **Windows 7 デバイス ステージ リファレンス ガイド**- を開発するためのガイドを参照および Device Stage をアセンブルなど、エクスペリエンスは開発トピック高度な。

-   **Windows 7 デバイス ステージの XML スキーマ**-スキーマ ファイル Device Stage エクスペリエンスの XML ファイルを編集するために便利です。

-   **Windows 7 デバイス ステージ開発のサンプル**-サンプル グラフィックおよび Device Stage の開発を支援するためのツールのコレクションが発生します。
