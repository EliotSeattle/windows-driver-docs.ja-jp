# <a name="rectangles"></a>矩形





GPD ファイル内の四角形を記述するには、RECT キーワードを使用します。 形式です。

**RECT** (*左*、*上部*、*右*、*下部*)

場所*左*、*上部*、*右*、および*下部*を表す[数値](numeric-values.md)で[ユニットをマスター](master-units.md)します。
