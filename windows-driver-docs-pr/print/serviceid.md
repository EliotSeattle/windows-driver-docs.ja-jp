# <a name="serviceid"></a>ServiceID


`ServiceID`コマンドでは、サービス識別子 (ID) を返します関数検出キー、PNP の形式で\_PNPX\_ServiceID します。 出力バッファーには、URI を表す NULL で終わる文字列が含まれています。

探索のファンクション キーの詳細については、Microsoft Windows SDK のドキュメントを参照してください。
