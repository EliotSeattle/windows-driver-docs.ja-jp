# <a name="printer-bidi-interfaces"></a>プリンター双方向インターフェイス

これらのインターフェイスではこと、V4 印刷ドライバー モデルを確立し、USB プリンターとの双方向通信を維持できます。 これらのインターフェイスでは、印刷スプーラで実行される JavaScript コードを使用してアクセスできるのみです。 

詳細については、スプーラーは、次を参照してください。[印刷スプーラー アーキテクチャ](https://docs.microsoft.com/windows-hardware/drivers/print/print-spooler-architecture)と[印刷スプーラー コンポーネント](https://docs.microsoft.com/windows-hardware/drivers/print/print-spooler-components)。
