# <a name="constants"></a>定数





定数は名前付きの GPD 言語で定義済みの値。 通常、定数名は大文字で構成されます。 GPD 定数の例は、ページ、シリアル ポート、および TTY、値を割り当てることができるは、 \* **PrinterType**属性。
