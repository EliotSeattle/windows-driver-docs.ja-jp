# <a name="device-stage-for-document-devices"></a>ドキュメント デバイスのデバイス ステージ

Windows デバイス エクスペリエンスは、ユーザーを検出して各自の PC に接続されているデバイスを使用する方法を向上させる機能のコレクションです。 これらのユーザー機能は、デバイス メーカーが、デバイスとの関連アプリケーションと Windows でのサービスを表示する方法を定義するシステムとプラットフォームの機能でサポートされます。

Device Stage は、開発およびデバイス固有のエクスペリエンスを配布するための豊富なプラットフォームを提供します。 Device Stage、XML ファイルとグラフィックスを定義することでブランドに合わせてエクスペリエンス、機能、およびデバイスのサービスを作成できます。 Device Stage は限られた数のデバイス クラスのサポートされています。
