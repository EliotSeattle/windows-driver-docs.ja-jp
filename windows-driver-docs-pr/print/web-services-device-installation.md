# <a name="installing-printers-that-support-web-services-for-devices"></a>デバイスの Web サービスをサポートするプリンターをインストールします。


Web Services for Devices (WSD) を検出しをペアになっているに有効になっているプリンターをネットワークに接続されて、 [Windows.Devices.Enumeration](https://docs.microsoft.com/uwp/api/Windows.Devices.Enumeration) API 名前空間。

WSD プリンターはペアリングされていると、 [WSDMON ポート モニター](wsdmon-port-monitor.md)プリンターを自動的にインストールされます。

使用するデバイスの列挙とペアリングの例については**Windows.Devices.Enumeration**を参照してください、[デバイスの列挙とサンプルをペアリング](https://github.com/Microsoft/Windows-universal-samples/tree/master/Samples/DeviceEnumerationAndPairing)GitHub でします。

詳細については、次を参照してください。、[をネットワーク経由でデバイスを列挙](https://docs.microsoft.com/windows/uwp/devices-sensors/enumerate-devices-over-a-network)Windows SDK のドキュメント記事。
