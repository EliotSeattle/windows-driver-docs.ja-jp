# <a name="enumerations-and-xsd-defined-data-types"></a>列挙体と XSD 定義のデータ型


GDL テンプレートを使用して定義されている列挙型では、GDL パーサーによって生成される XSD スキーマに表示される列挙データ型の定義が発生します。 GDL テンプレートを使用する XSD 定義データ型の場合、GDL パーサーによって生成される XSD スキーマに表示される XSD データ型の定義。
