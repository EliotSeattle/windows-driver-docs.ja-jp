# <a name="print-driver-manifests"></a>印刷ドライバー マニフェスト





印刷ドライバー パッケージには、マニフェストの 2 つの種類を含めることができます。

-   **アプリケーション マニフェスト**セキュリティ機能として、すべての Windows Vista のドライバーに必要な。 アプリケーション マニフェストの詳細については、デバイスのインストール アプリケーションの作成のガイドラインを参照してください。

-   **テーマのマニフェスト**は印刷ドライバーをプリント アプリケーション UI の外観を制御するためのオプション。

テーマのマニフェストの詳細については、[テーマのマニフェストを使用して](using-theme-manifests.md)を参照してください。
