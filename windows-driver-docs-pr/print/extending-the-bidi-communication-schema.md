# <a name="extending-the-bidi-communication-schema"></a>双方向通信のスキーマを拡張します。


Bidi 拡張機能は、新しいスキーマを定義する XML ファイルを使用して、双方向通信のスキーマを拡張できます。 これには、双方向の拡張機能フレームワーク、Windows Vista および Windows の以降のバージョンで使用できる XSD 標準で定義されている構成要素が適用されます。

双方向の拡張機能フレームワークの正確な定義は、Windows Driver Kit (WDK) で提供される tcpbidi.xsd ファイルで確認できます。

スキーマの拡張の詳細については、次を参照してください。[プリンター ポート モニターをカスタマイズする](customizing-the-printer-port-monitors.md)します。
