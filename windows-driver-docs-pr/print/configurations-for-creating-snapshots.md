# <a name="configurations-for-creating-snapshots"></a>スナップショットを作成するための構成


構成は、パラメーターと各パラメーターに割り当てられている現在の値の一覧にすぎません。 GDL パーサーのインターフェイスには、XML では、あるが、簡略化の構成が必要ですが、次の擬似コード例は、パラメーター値のペアのリストを使用して、構成の一般的な表現を示します。

```cpp
Today: Sunday
Weather: Cloudy
```
