GPD 言語を記述するためのエントリを提供する[ハードウェア常駐フォント](hardware-resident-fonts.md)、[フォント カートリッジ](font-cartridges.md)と[フォントの置き換え](font-substitution.md)の TrueType フォントのようなハードウェアに常駐またはカートリッジのフォント。

ソフト フォントを処理するため、カスタマイズ コードを必要に応じて指定できます。 このカスタマイズされたコードについては、次を参照してください。[フォントのカスタマイズした管理](customized-font-management.md)します。
