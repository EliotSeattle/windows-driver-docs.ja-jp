# <a name="gdl-directives-for-macros"></a>マクロの GDL ディレクティブ


GDL には、マクロを定義するディレクティブがあります。 マクロへの参照は、展開されたインライン パーサーによって解釈されるエントリを取得します。

GDL には、次のマクロ ディレクティブが含まれています。

-   **\*マクロ**マクロの値のグループを定義します。

-   **\*BlockMacro**ブロック マクロを定義します。

-   **\*InsertBlock**ブロック マクロを参照

-   **=** (等号) は、値マクロの名前が続く場合、値の符号を参照します。

マクロ ディレクティブの詳細については、次を参照してください。 [GDL マクロ](gdl-macros.md)します。
