プリント サーバーに接続されているプリント クライアントは、ポイントを使用し、インストール用の全体のドライバー パッケージのコピーを印刷できます。 使用する利点は、ポイントし、そのパッケージを使用した印刷します。

-   ドライバーの実行可能なすべてのコンポーネントは、印刷のクライアントにインストールされます。

-   印刷のクライアントでは、ドライバーの署名とドライバーの整合性がチェックされます。

-   ポイント アンド プリントがより信頼できると、管理された環境での管理者がより適切に制御できます。

ポイントと印刷プロセスは、ドライバー パッケージの導入により以前のバージョンの Windows と Windows Vista では異なります。 Windows Vista は、ドライバーのインストールの推奨される方法として、パッケージのインストールを使用します。 Windows Vista より前のバージョンの Windows でポイント アンド プリントは、クライアントでのみ基本的な印刷機能をインストールでした。
