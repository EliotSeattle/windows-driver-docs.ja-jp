プリント サーバーはプリント サーバーの Web ページに関する情報を更新するためのサーバー プッシュをサポートしていません。 クライアント プル操作では、(ユーザーのブラウザーに HTML 指定「更新」パラメーターが認識されると仮定)、サポートされ、カスタマイズされたページに含めることができます。 (クライアント プルの詳細についてを参照してください、適切な HTML ドキュメントです。)
