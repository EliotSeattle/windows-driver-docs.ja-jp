# <a name="gdl-attributes"></a>GDL 属性


GDL 属性は、キーワードと値のペアです。 *値*データの一部は、および*キーワード*のデータを識別するために使用されるタグだけです。 GDL 構文規則に従っている限り、任意のキーワードとデータのペアが許可されます。

次のコード例では、GDL 属性を示します。

```cpp
*Weight: 330 pounds
```

前の例では、"\*重み"、キーワードは、「330 ポンド」文字列値。
