# <a name="attribute-types"></a>属性の型





構成されるいくつかの種類の属性が定義されます。

-   [一般的な属性](general-attributes.md)

    一般的な属性は、プリンター デバイスに関連付けられます。

-   [コマンドの属性](command-attributes.md)

    コマンドの属性は、プリンターのコマンドに関連付けられます。

-   [機能の属性](feature-attributes.md)

    機能の属性は、特定のプリンターの機能に関連付けられます。

-   [オプションの属性](option-attributes.md)

    オプションの属性は、特定の機能に属しているオプションに関連付けられます。
