# <a name="gdl-source-file-structure"></a>GDL ソース ファイル構造


GDL エントリは、[属性](gdl-attributes.md)または[構築](gdl-constructs.md)します。 *属性*は、[キーワード](gdl-keywords.md)-[値](gdl-values.md)ペア。 A*構築*(が属性または構成要素またはその両方の組み合わせがあります)、0 個以上の GDL エントリを保持できるオブジェクトは、それ自体に関連付けられているキーワードと値のペア。
