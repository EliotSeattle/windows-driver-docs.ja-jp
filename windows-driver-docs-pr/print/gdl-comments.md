# <a name="gdl-comments"></a>GDL コメント


A*コメント*の任意の位置を[リテラルでない空白](gdl-whitespace-characters.md)表示されることができます。 コメントは、アスタリスク、パーセント記号で始まる (\*%)linebreak シーケンスまたは継続 linebreak が発生したときに終了します。

すべてのコメントは、リテラルでない空白文字として扱われます。 コメントの一部でない linebreak の終端文字のシーケンス。 コメントは、入れ子になったコンテキスト内で表示できますが、コメント内のコンテキストは認識されません。
