Microsoft プロッター Driver (MSPlot) は、次の機能を提供します。

-   モデルに固有のプロッターを使用して、Hewlett-Packard グラフィックス言語の HPGL/2 のバージョンを使用しているすべてのプロッター サポート[プロッター ドライバー ミニドライバー](plotter-driver-minidrivers.md)します。

-   A[プロッター ドライバーのユーザー インターフェイス](plotter-driver-user-interface.md)を基に、TreeView コントロールとプロパティ シート、すべてのプロッターの整合性があります。

-   A[プロッター ドライバー レンダラー](plotter-driver-renderer.md)の GDI グラフィックス エンジンと共に変換 Win32 GDI 呼び出しアプリケーションから印刷スプーラに送信できるプリンター コマンドにします。

新しい HPGL/2 準拠デバイスの種類のサポートを提供するには、デバイスを記述するミニドライバーに提供を行う必要があるすべてが。
