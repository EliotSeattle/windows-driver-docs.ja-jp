# <a name="hp-gl2-supported-fonts"></a>サポートされるフォントの HP-GL/2





HP-GL/2 では、ラスター モード用に定義されているものと同じフォント リソースをサポートします。 ベクターとラスターの両方のモードで使用できるデバイス フォントは、GPD とリソース ファイルに追加されます。
