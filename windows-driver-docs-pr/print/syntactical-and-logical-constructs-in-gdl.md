# <a name="syntactical-and-logical-constructs-in-gdl"></a>GDL の構文から見た論理的コンストラクト


GDL 文字どおり GDL ソース ファイル内のエントリによって定義されるコンストラクトと GDL の内部データでそれらのコンストラクトの形式を区別します。 前者は、構文の表現と、後者は、論理的に表したもの。 形式は、ソース ファイルの構造が定義されている場合は異なります。 GDL パーサーは、特定の構成要素の型と構造タグ、そのようなコンストラクトが GDL ソース ファイルで定義されている回数に関係なく構造の 1 つだけの論理表現を作成します。
