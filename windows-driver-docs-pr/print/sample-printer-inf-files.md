# <a name="sample-printer-inf-files"></a>サンプル プリンター INF ファイル





Windows Driver Kit (WDK) にあるサンプル プリンター INF ファイルを指定、 \\src\\印刷\\inf\\driver5 サブディレクトリ。 このファイルは、Unidrv と Pscript ミニドライバーをインストールするために使用される INF ファイルのレイアウトを説明するテンプレートです。 プレース ホルダー ファイル名をドライバー ファイルの名前に置き換えます。
