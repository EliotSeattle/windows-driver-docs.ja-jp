# <a name="macros"></a>マクロ





GPD 言語には、2 種類のマクロがサポートされています。

-   [マクロ ブロック](block-macros.md)、GPD ファイル内の任意の場所に挿入できる GPD ファイルのエントリのセットを指定できます。

-   [マクロを値](value-macros.md)、GPD ファイルにどこにでも個別に挿入できる値のセットを指定できます。
