# <a name="stress-testing"></a>ストレス テスト


ストレス テストを大規模に送信することによって (100 MB 以上)、long を使用してプリンター (150 以上のページ)、および視覚的に処理を要する印刷ジョブを実行する必要があります。 画像を多用するジョブの場合は、さまざまなサイズの組み合わせおよび可能な限りグラフィック イメージの種類を変更します。

### <a name="specialized-bus-stress"></a>特殊な Bus ストレス

ストレス ケースを作成するときに、特殊なバスで発生する可能性を考慮対象としてストレス要因を考慮します。 多くのデバイスでは、同じバスを共有、これを必ずテストの同じ型またはモデルでは、デバイスまたは共通のデバイスをユーザーがシステムに接続が通常でした--同じバス上の他のデバイスと、デバイスが正常に共存できます。 たとえば、プリンターが接続されている同じ USB ハブと同時に多数のデバイスを接続する可能性があります。

次のテストがお勧めします。

1.  ハブに他のいくつかのデバイスを接続することを確認するか、システムは、デバイスに影響しません。

2.  複数のハブとデバイスをチェーン化し、デバイスは正常に機能することを確認します。

3.  同じバス上のすべてのデバイスを同時に操作できることを確認します。

4.  いることを確認、接続されているハブがシステムから外れている場合、デバイスが正常にアンインストールします。

Bluetooth 対応のプリンターのように印刷ジョブ ストレス テストを実行します。

1.  無線に応じて 30 ~ 90 フィート (10 ~ 30 m)、通常、プリンター、Bluetooth 無線の最大範囲との間を移動します。

2.  Bluetooth 無線の範囲内のプリンターにジョブを送信します。

3.  Bluetooth 無線の最大の範囲外のプリンターにジョブを送信します。 エラー状態が発生するまで待機し、プリンターが正常に回復することを確認する範囲内に戻します。
