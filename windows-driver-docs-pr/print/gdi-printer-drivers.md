# <a name="gdi-printer-drivers"></a>GDI プリンター ドライバー





すべての Windows 2000 およびそれ以降のプリンター ドライバーは、次のコンポーネントで構成されます。

-   A[プリンター グラフィックス DLL](printer-graphics-dll.md)を表示、印刷ジョブで GDI を支援し、印刷スプーラーに表示されたデータ ストリームを送信します。

-   A[プリンター インターフェイス DLL](printer-interface-dll.md)ドライバーの構成パラメーターを両方のユーザー インターフェイスを提供して、システムの印刷関連イベントのドライバーに通知する、スプーラーのインターフェイスを呼び出すことができます。

また、ことを Microsoft から提供されたプリンター ドライバーの使用[プリンターのデータ ファイル](printer-data-files.md)します。
