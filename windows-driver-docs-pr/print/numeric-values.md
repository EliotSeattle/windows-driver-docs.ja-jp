# <a name="numeric-values"></a>数値





エントリの値または GPD ファイル内のパラメーター値として指定されているすべての数値は整数である必要があります。 小数点が許可されていないテキスト文字列の中を除きます。

数値の値は、頭にマイナス記号しない限り、正の値と見なされます。

数値は符号なし 16 進数の値は後者 0x、しない限り、10 進数であると見なされます。

アスタリスク文字 (\*) GPD ファイルのエントリを特定のコンテキスト内で該当する場合、無限の値または「不要」の値のいずれかを示すために使用できます。
