# <a name="callback-objects"></a>コールバック オブジェクト





カーネルのコールバック機構では、要求し、特定の条件が満たされたときに通知を提供するドライバーの一般的な方法を提供します。

ドライバーが、コールバック オブジェクトを作成し、他のドライバーは、このドライバーの定義済みのコールバックに関連付けられた条件の通知を要求できます。 さらに、システムは、ドライバーを使用するための 2 つのコールバック オブジェクトを定義します。

すべてのコールバック オブジェクトには、名前とオブジェクトの作成時に定義されている属性のセットがあります。 システム定義のコールバック オブジェクトの名前は**\\コールバック\\SetSystemTime**、 **\\コールバック\\PowerState**、および**\\コールバック\\ProcessorAdd**はドライバー定義のコールバックにこれらの名前が重複していない必要があります。

システムまたはドライバーの定義済みのコールバックからの通知を要求するには、ドライバーは、コールバック オブジェクトを開きし、コールバック ルーチンを登録します。 コールバックを定義された条件が true になると、作成者は、通知をトリガーします。 さらに、システムは、コールバックの登録されたすべてのコールバック ルーチンを呼び出します。
