# <a name="wdm-versions"></a>WDM のバージョン


以降にリリースされたオペレーティング システムが追加 WDM をサポートしている WDM 増分の変更保存されましたが、Microsoft Windows 2000 および以降のバージョンの NT ベース オペレーティング システム、Windows 98 および Windows Me と共にすべて WDM のさまざまなバージョンのサポート、以前のリリースをサポートしない機能です。 WDM の以降のバージョンは一般に WDM; の以前のバージョンで使用可能なすべての機能をサポートします。これは WDM の新しい各バージョンでは、WDM の以前のバージョンのスーパー セットがあります。 可能であれば、システム間 driver が任意のオペレーティング システムで最も低い WDM バージョンに従う必要があります。
