# <a name="starting-a-device-in-a-filter-driver"></a>フィルター ドライバーでのデバイスの起動





上位レベルのフィルター ドライバーは、いずれかの関数のドライバーの開始アクティビティを補強可能性があります。

通常、下位レベルのフィルターは、デバイスの機能を強化し、デバイスの開始に参加可能性があります。
