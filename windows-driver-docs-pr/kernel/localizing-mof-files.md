# <a name="localizing-mof-files"></a>MOF ファイルのローカライズ





WMI プロパティ修飾子は、表示可能な文字列では多くの場合であるために、Windows XP およびそれ以降のバージョンのオペレーティング システムがこれらの修飾子をローカライズするメカニズムを提供します。
