[デバイスへのアクセスを制御します。](controlling-device-access.md)

[デバイス Namespace へのアクセスを制御します。](controlling-device-namespace-access.md)

[デバイス オブジェクトの SDDL](sddl-for-device-objects.md)

-   デバイスの名前空間へのアクセス。 詳細については[デバイス Namespace のアクセスを制御する](controlling-device-namespace-access.md)します。
