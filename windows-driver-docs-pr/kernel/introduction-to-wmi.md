# <a name="introduction-to-wmi"></a>WMI の概要





WMI プロバイダーをドライバーを行うことができます。

-   カスタム データを WMI コンシューマーが使用できるようにします。

-   カスタム コントロール パネル アプリケーションではなく、標準的なインターフェイスを使用してデバイスを構成する WMI コンシューマーを許可します。

-   ドライバー定義イベントの WMI コンシューマーをポーリングまたは Irp を送信するコンシューマーを必要とせずに通知します。

-   収集して 1 つの宛先に要求されたデータのみを送信するには、ドライバーのオーバーヘッドを削減します。

-   わかりやすいドライバー定義のクラス名と、WMI クライアントが列挙し、およびユーザーに表示できる説明は省略可能なデータとイベントのブロックの注釈を設定します。
