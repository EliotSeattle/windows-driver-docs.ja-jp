# <a name="acpi-bios"></a>ACPI BIOS





Microsoft Windows オペレーティング システムでサポートされている、統合された電源管理機能は、Advanced Configuration and Power Interface (ACPI) BIOS がコンピューターでのみ使用できます。

Windows Server 2003、Windows XP、および Windows 2000、ACPI BIOS が 1999 年 1 月 1 日の日付以降にすることが必要です。 ただし、これらの Windows バージョンの 1 つは、ACPI 問題が表面化する BIOS が既知である判断した場合、ローダーは ACPI を無効にし、アドバンスト パワー マネージメント (APM) の代わりに使用します。 Windows Vista 以降、オペレーティング システムには、ACPI 準拠の BIOS であり、1999 年 1 月 1 日の日付またはそれ以降を使用しているコンピューターのみがサポートしています。

デバイス マネージャーは、個々 のコンピューターが ACPI をサポートしているかどうかを示します。 ドライバー情報の確認、**コンピューター**デバイス カテゴリ。

ACPI の詳細については、、 [ACPI 5.0 仕様](https://www.uefi.org/specifications)を参照してください。
