# <a name="system-working-state-s0"></a>システム稼働状態 S0





システムの電源状態 S0、システムの稼働状態には、次の特徴があります。

<a href="" id="power-consumption"></a>**電力消費量**  
最大値。 ただし、個々 のデバイスの電源の状態はできます電源の節約は、デバイスごとに、動的に変更します。 未使用のデバイスの電源を切断しに応じて投入できます。

<a href="" id="software-resumption"></a>**ソフトウェアの再開**  
適用できません。

<a href="" id="hardware-latency"></a>**ハードウェアの遅延**  
なし。

<a href="" id="system-hardware-context"></a>**システム ハードウェアのコンテキスト**  
すべてのコンテキストが保持されます。
