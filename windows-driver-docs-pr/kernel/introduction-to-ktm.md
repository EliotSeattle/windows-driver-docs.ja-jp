# <a name="introduction-to-ktm"></a>KTM の概要


カーネル トランザクション マネージャー (KTM) とは、ユーザー モードまたはカーネル モード (または両方) でトランザクション処理システム (TP) を作成することができます、トランザクション管理サービスです。

KTM は、Microsoft Windows オペレーティング システムのカーネル モードのサービスです。 KTM は、Windows Vista と Windows の以降のバージョンで使用できます。
