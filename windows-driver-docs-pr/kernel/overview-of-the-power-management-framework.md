# <a name="overview-of-the-power-management-framework"></a>電源管理フレームワークの概要


Windows 8 以降では、実行時の電源管理フレームワーク (PoFx) は、コンポーネント (またはサブデバイス) レベルでの電源とクロックの管理をサポートします。 デバイス ドライバーは、デバイスの個々 のコンポーネントでの電力使用状況を個別に管理する PoFx を登録します。 PoFx では、Windows のポータブル コンピューター、タブレット、電話、またはその他のモバイル デバイスがバッテリの充電で実行できる時間を拡張するために必要な細かい制御を提供します。 PoFx では常に、常に接続されているモバイル デバイスの外観を維持する方法で電力使用量が減少します。

コンポーネント、またはサブデバイスは、オンまたは同じデバイスで他のコンポーネントとは無関係に低電力状態に切り替えることができるデバイスの機能のハードウェア ユニットです。 たとえば、オーディオ デバイスには、再生して状態を個別に管理できる電力を記録するための個別のコンポーネントがあります。 再生コンポーネントを使用している場合、記録のコンポーネントがアイドル状態、再生コンポーネントの動作を妨げることがなく、記録コンポーネントを低電力状態に切り替えることができます。
