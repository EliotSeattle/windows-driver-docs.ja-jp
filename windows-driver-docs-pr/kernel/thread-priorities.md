一部のドライバーでは、独自のドライバーまたはデバイス専用システム スレッドを作成し、リアルタイムの優先度の最小値に、スレッドの基本優先順位を設定します。 その他の最上位レベルのドライバー、特にファイル システム ドライバーは、通常は最も高い変数の優先度の値に設定する基本優先順位でシステム ワーカー スレッドを使用します。 カーネルは、システムのほぼすべてのユーザー モード スレッドを含む変数の優先順位を持つすべてのスレッドの前に実行するリアルタイム最低の優先度のスレッドをスケジュールします。
