# <a name="wi-fi-support-in-firmware"></a>ファームウェアでの Wi-fi のサポート 


Wi-fi サポートは、UEFI 仕様 2.6 で明確 UEFI 仕様の 2.5 で最初に定義されています。

Windows オペレーティング システムと Windows プレインストール環境 (WinPE) は、UEFI のこれらの機能をフル活用するためのサポートを現在がありません。 

メールを送信して<SAUEFI@Microsoft.com>質問または懸念事項がある場合、または会社でこの機能に興味がある場合。
