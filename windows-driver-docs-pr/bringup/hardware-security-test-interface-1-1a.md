# <a name="hardware-security-test-interface-hsti-11a"></a>ハードウェア セキュリティ テスト インターフェイス (HSTI) 1.1a


HSTI では、セキュア ブートの約束 (たとえば、SPI flash または eMMC とパーティション分割のロック、適切な SMM 構成、Intel ブート Guard が正しく構成されているなど) を適用するセキュリティ テクノロジを独自のプラットフォーム用の標準的なテスト インターフェイスを指定します。 シリコン チップと BIOS のベンダーは、指定し、ファームウェアのリリースで、組み込みの自己テストとしては提供されているために必要なテスト_ケースを実装します。

テスト インターフェイスのファームウェアに含めることには、知識の豊富なコンシューマーのファームウェアのセキュリティ機能の有無を確認することができます。

HSTI のサポートは、次のバージョンの Windows に含まれています。

-   Windows Server Technical Preview 2016

-   Windows 10、バージョン 1607 およびそれ以降のバージョン
