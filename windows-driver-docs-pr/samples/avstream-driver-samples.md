# <a name="avstream-driver-samples"></a>AVStream ドライバーのサンプル

このディレクトリの AVStream ドライバーサンプルは、デバイス用のカスタムストリーミングメディアドライバーを作成するための開始点となります。

| サンプル | 説明 |
| --- | --- |
| [AvsCamera-AVStream カメラサンプルドライバー](https://docs.microsoft.com/samples/microsoft/windows-driver-samples/avscamera) | さまざまなフレームレートで、RGB24、RGB32、YUY2、NV12 の各形式で、シミュレートされたキャプチャを240または 640 x 480 で実行する、シミュレートされた前面および背面カメラ用の、ピン中心の AVStream キャプチャドライバーを提供します。 |
| [AVStream のシミュレートされたハードウェアサンプルドライバー (Avshws)](https://docs.microsoft.com/samples/microsoft/windows-driver-samples/avstream-simulated-hardware-sample-driver-avshws) | Pin を中心としたキャプチャドライバーを提供し、AV キャプチャハードウェアをシミュレートする、シミュレートされたハードウェアサンプルドライバー。 |
| [AVStream フィルター中心のシミュレートされたキャプチャサンプルドライバー (Avssamp)](https://docs.microsoft.com/samples/microsoft/windows-driver-samples/avstream-filter-centric-simulated-capture-sample-driver-avssamp/) | 機能オーディオを持つ AVStream フィルター中心のシミュレートされたキャプチャサンプルドライバー。 |
| [ドライバーデバイス変換のサンプル](https://docs.microsoft.com/samples/microsoft/windows-driver-samples/driver-device-transform-sample) | メディアファンデーションを使用して Avstream ベースのカメラデバイスをストリーミングするプロセスで読み込まれるドライバーデバイス変換の例。 |
| [ドライバー MFT のサンプル](https://docs.microsoft.com/samples/microsoft/windows-driver-samples/driver-mft-sample) | カメラの UWP デバイスアプリで使用するドライバー MFT。 ドライバー MFT は、ビデオをキャプチャするときに特定のカメラで使用されるメディアファンデーション変換です。 |
