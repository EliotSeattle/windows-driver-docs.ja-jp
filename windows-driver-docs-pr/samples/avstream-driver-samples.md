# <a name="avstream-driver-samples"></a>AVStream ドライバーのサンプル


AVStream ドライバー サンプルでは、このディレクトリは、デバイス用のカスタム ストリーミング メディア ドライバーを記述するための出発点を提供します。

## <a name="avstream"></a>AVStream


| サンプル名                    | ソリューション                                                           | 説明                                                                                                                                                           |
|--------------------------------|--------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| AVStream                       | [avshws](https://go.microsoft.com/fwlink/p/?LinkId=620185)          | AV キャプチャ ハードウェアをシミュレートするために pin を中心としたキャプチャ ドライバーを提供するシミュレートされたハードウェア サンプル ドライバー。                                                            |
| AVSSamp                        | [avssamp](https://go.microsoft.com/fwlink/p/?LinkId=620186)         | フィルターを中心とした AVStream キャプチャ機能のオーディオ ドライバーのサンプルをシミュレートします。                                                                                    |
| ドライバー デバイス変換のサンプル | [sampledevicemft](https://go.microsoft.com/fwlink/p/?LinkId=620187) | Streaming Media Foundation を使用して、カメラの Avstream ベース デバイスのプロセスで読み込むドライバー デバイス変換の例を説明します。                         |
| Driver MFT                     | [SampleMft0](https://go.microsoft.com/fwlink/p/?LinkId=617126)      | カメラの UWP デバイス アプリで使用するための driver MFT。 Driver MFT は、ビデオをキャプチャするときに、特定のカメラで使用 Media Foundation 変換です。 |
