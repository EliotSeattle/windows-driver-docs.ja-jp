# <a name="windows-management-instrumentation-wmi-driver-samples"></a>Windows Management Instrumentation (WMI) ドライバーのサンプル

このディレクトリのドライバーサンプルは、デバイスのカスタム WMI ドライバーを作成するための開始点となります。

| サンプル | 説明 |
| --- | --- |
| [WMI ACPI](https://docs.microsoft.com/samples/microsoft/windows-driver-samples/wmi-acpi-sample) | Acpi bios および Microsoft Windows Management Instrumentation (WMI) のサンプルコードが含まれており、acpi BIOS のインストルメンテーションを ACPI ソース言語 (ASL) コード内から実行できます。 |
| [WMI Data Provider](https://docs.microsoft.com/samples/microsoft/windows-driver-samples/sample-kmdf-driver-implementing-a-wmi-data-provider) | KMDF で WMI プロバイダーを登録し、プロバイダーインスタンスを作成する方法と、デバイスに送信される WMI クエリを処理する方法を示します。 |
