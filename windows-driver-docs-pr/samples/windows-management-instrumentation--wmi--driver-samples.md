# <a name="windows-management-instrumentation-wmi-driver-samples"></a>Windows Management Instrumentation (WMI) ドライバーのサンプル


このディレクトリにドライバーのサンプルでは、デバイスのカスタムの WMI ドライバーを記述するための開始点を提供します。

## <a name="windows-management-instrumentation-wmi"></a>Windows Management Instrumentation (WMI)


| サンプル名       | ソリューション                                                   | 説明                                                                                                                                                                      |
|-------------------|------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| WMIACPI           | [wmiacpi](https://go.microsoft.com/fwlink/p/?LinkId=618006) | ACPI ソース言語 (ASL) コード内から、ACPI BIOS のインストルメンテーションをできるようにする ACPI BIOS および Microsoft Windows Management Instrumentation (WMI) のサンプル コードが含まれています。 |
| WMI データ プロバイダ | [wmisamp](https://go.microsoft.com/fwlink/p/?LinkId=618007) | KMDF に WMI プロバイダーを登録し、プロバイダーのインスタンスを作成する方法と、デバイスに送信される WMI クエリを処理する方法について説明します。                                             |
