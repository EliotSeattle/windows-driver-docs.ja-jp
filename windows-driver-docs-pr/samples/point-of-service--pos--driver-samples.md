# <a name="point-of-service-pos-driver-samples"></a>Point of Service (POS) ドライバーのサンプル

このディレクトリのドライバーサンプルは、デバイスのカスタム POS ドライバーを作成するための開始点となります。

| サンプル | 説明 |
| --- | --- |
| [バーコードスキャナードライバーのサンプル](https://docs.microsoft.com/samples/microsoft/windows-driver-samples/barcode-scanner-driver-sample) | この UMDF 2 サンプルは、バーコードスキャナドライバーを作成するためのテンプレートとして機能し、デバイスに排他アクセスを要求したり有効にしたりするなどの基本的な機能を示しています。 |
| [磁気ストライプリーダードライバーのサンプル](https://docs.microsoft.com/samples/microsoft/windows-driver-samples/magnetic-stripe-reader-driver-sample) | この UMDF 2 サンプルは、磁気ストライプリーダードライバーを作成するためのテンプレートとして機能し、デバイスに対して排他アクセスを要求したり有効にしたりするための基本的な機能を示しています。 |
