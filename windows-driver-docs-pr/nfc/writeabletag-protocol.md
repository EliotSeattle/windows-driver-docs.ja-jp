# <a name="writeabletag-protocol"></a>WriteableTag プロトコル


"WriteableTag"サブスクリプションを返します、 **DWORD**タグ全体を (現在使用できるこれらのバイト列を含む) の書き込み可能な最大バイト数を格納している書き込み可能なタグが近接を入力するたびにします。
