# <a name="disabling-nfp"></a>NFP を無効にします。


クライアントは、サブスクリプション、パブリケーション、およびプレゼンスに一時的に無効にできます。

サブスクリプション、パブリケーション、およびプレゼンスを一時的に無効には送信することによって行われます[ **IOCTL\_NFP\_を無効にする**](https://msdn.microsoft.com/library/windows/hardware/jj853315)ハンドルにします。 これは、機能は、クライアントが近接通信機能を無効にするには迅速に再度有効にするために必要なときに割り当てられたリソースを保持する場合に便利です。
