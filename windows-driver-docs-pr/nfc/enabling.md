# <a name="enabling-nfp"></a>NFP の有効化


クライアントは、以前に無効にしたサブスクリプション、パブリケーション、および[**IOCTL\_NFP\_有効**](https://docs.microsoft.com/windows-hardware/drivers/ddi/nfpdev/ni-nfpdev-ioctl_nfp_enable)にすることができます。
