# <a name="nfc-glossary"></a>NFC 用語集


このトピックでは、NFC class extension (CX) の用語集について説明します。

<a href="" id="nfpdrivers-glossary"></a>**CD**  
カードエミュレーション

<a href="" id="nfpdrivers-cx"></a>**シリーズ**  
クラスの拡張

<a href="" id="nfpdrivers-ddi"></a>**DDI**  
ドライバー開発インターフェイス

<a href="" id="nfpdrivers-fw"></a>**ファームウェア**  
NFCC ファームウェア

<a href="" id="nfpdrivers-hce"></a>**HCE**  
ホストカードエミュレーション

<a href="" id="nfpdrivers-ihv"></a>**IHV**  
独立系ハードウェアベンダー

<a href="" id="nfpdrivers-isv"></a>**ISV**  
独立系ソフトウェアベンダー

<a href="" id="nfpdrivers-kmdf"></a>**KMDF**  
カーネルモードドライバーのフレームワーク

<a href="" id="nfpdrivers-nci"></a>**NCI**  
NFC コントローラーインターフェイス

<a href="" id="nfpdrivers-nfc"></a>**NFC**  
近距離無線通信

<a href="" id="nfpdrivers-nfcc"></a>**NFCC**  
NFC コントローラー

<a href="" id="nfpdrivers-nfcee"></a>**NFCEE**  
NFC 実行環境

<a href="" id="nfpdrivers-p2p"></a>**ツー**  
ピアツーピア

<a href="" id="nfpdrivers-rf"></a>**RF**  
無線周波数

<a href="" id="nfpdrivers-se"></a>**専門**  
NFC secure 要素

<a href="" id="nfpdrivers-t1t"></a>**T1T**  
NFC フォーラムタイプ1タグ

<a href="" id="nfpdrivers-t2t"></a>**T2T**  
NFC フォーラムタイプ2タグ

<a href="" id="nfpdrivers-t3t"></a>**T3T**  
NFC フォーラムタイプ3タグ

<a href="" id="nfpdrivers-umdf"></a>**UMDF**  
ユーザーモードドライバーフレームワーク2.0

<a href="" id="nfpdrivers-wdf"></a>**WDF**  
Windows ドライバーフレームワーク
