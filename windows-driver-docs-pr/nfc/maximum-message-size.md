# <a name="nfp-maximum-message-size"></a>NFP 最大メッセージ サイズ


各プロバイダーの実装では、パブリケーションおよびサブスクリプションのメッセージの最大サイズを指定できます。 Windows では、このプロバイダーでサポートされているメッセージの最大サイズが 10 KB 以上である必要があります。
