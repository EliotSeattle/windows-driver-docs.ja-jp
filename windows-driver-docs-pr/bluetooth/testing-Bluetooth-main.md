# <a name="bluetooth-driver-testing-resources"></a>Bluetooth ドライバーのテストリソース

これらのリソースは、Bluetooth ドライバーのテストをサポートしています。

- [Windows Hardware Lab Kit](https://docs.microsoft.com/windows-hardware/test/hlk/)は、テスト環境を構築し、ドライバーとシステムのテストを自動化し、 [Windows ハードウェア互換性プログラム](https://docs.microsoft.com/windows-hardware/design/compatibility/)に参加するために必要な送信パッケージを作成するためのテストフレームワークを提供します。

- [Windows パフォーマンスレコーダー](https://docs.microsoft.com/previous-versions/windows/it-pro/windows-8.1-and-8/hh448205(v=win.10))は、WINDOWS イベントトレーシング (ETW) に基づくパフォーマンス記録ツールです。 Bluetooth に固有のトレーススクリプトとログスクリプトの作成については[、GitHub の Bus tools For Windows リポジトリ](https://github.com/microsoft/busiotools/blob/master/bluetooth/tracing/readme.md)で詳しく説明されています。

- [Bluetooth テストプラットフォーム (BTP)](testing-BTP-Overview.md)は、bluetooth ハードウェア、ドライバー、およびソフトウェアのテストを自動化するように設計されています。 BTP を使用すると、ホスト (PC) と周辺機器の両方で Bluetooth ラジオを試すことができます。
