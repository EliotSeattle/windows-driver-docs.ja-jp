# <a name="device-metadata"></a>デバイス メタデータ


認定を受けるためにデバイスを提出するときに、デバイス メタデータ パッケージも提出できます。 このパッケージには、Windows でユーザーがデバイスを識別して操作できるようにするためにデバイス エクスペリエンス機能で利用される、.xml ファイルとグラフィック ファイルが格納されます。
