# <a name="download-failure-cabs"></a>エラー CAB のダウンロード

[Get Report Data](get-report-data.md) API の一部として受け取ったレポートから取得した applicationId と cabIdHash を使用してドライバーのエラー CAB をダウンロードするには、この方法を使います

- [OEM ハードウェア エラーの CAB ファイルをダウンロードする](https://docs.microsoft.com/windows/uwp/monetize/download-the-cab-file-for-an-oem-hardware-error)

- [Windows 10 のドライバー エラーに関する CAB ファイルをダウンロードする](https://docs.microsoft.com/windows/uwp/monetize/download-the-cab-file-for-a-windows-10-driver-error)

さらに、[ドライバーの CAB ファイルの一括ダウンロード](bulk-download-failure-cabs.md)を使用して、エラー ハッシュで使用可能なすべての CAB ファイルをダウンロードすることもできます
