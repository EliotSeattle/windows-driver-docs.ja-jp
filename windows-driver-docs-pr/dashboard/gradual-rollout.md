# <a name="gradual-rollout"></a>段階的なロールアウト

段階的なロールアウトの目的は、Windows エコシステムは、お客様に最良のエクスペリエンスがあることを確認するテレメトリを監視することによってシステムの使用に最適なドライバーを持つようにするためです。 ドライバーが表示されます異常な段階的なロールアウト フェーズ中にする Microsoft こともできます**を一時停止**調査のためのドライバーの配布、ドライバーの Microsoft によるキャンセル (を含め、適切な修復をシークまたは_有効期限_)。

>[!NOTE]

> - 最終的には、Windows Update に送信されたすべてのドライバーは段階的なロールアウトを移動します。

## <a name="main-concepts-of-gradual-rollout"></a>段階的なロールアウトの主な概念

- 30 日間**監視**期間

- **ドライバー調整**期間
  - 1% から製品版 Windows の母集団の 100% の範囲を調整します。
  - Automation のルールは、割合の進行状況を管理します。
  - スロットルされた割合が 100% にある場合は、WU に完全にライブ実行されているすべてのシステム**Windows 10 バージョン 1709 以降**します。  Microsoft は引き続き**モニター** 30 日の期間の残りの部分をリリースします。

このプロセス全体の中には、出荷ラベルの状態が、としてまま**段階的なロールアウト**します。 これは通常とには、ドライバーが Windows Update で利用します。
