# <a name="trusted-certificates"></a>信頼できる証明書


信頼できる証明書は、初期セットアップ時にモバイル ネットワーク事業者の Web ページまたはキャプティブ ポータルから配信されたプロビジョニング ファイルのデジタル署名を Windows で検証するために使われます。 これにより、ユーザーはモバイル ブロードバンド アプリをインストールする前にサービスを購入できます。
