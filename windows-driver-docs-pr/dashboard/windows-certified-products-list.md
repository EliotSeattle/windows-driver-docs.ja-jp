# <a name="windows-certified-products-list"></a>Windows 認定製品リスト

Windows 認定製品リストは、認定されたデバイスおよびシステムの包括的なリストです。

一覧をフィルター処理するか、特定の製品名の検索を実行して、デバイスおよびシステムを見つけることができます。

> [!IMPORTANT]
> Sysdev が廃止されたため、Windows 認定製品リストはオフラインになっています。  2018 年秋に提供が再開されます。

## <a name="to-find-a-product-by-searching"></a>製品を検索して見つけるには

製品 (ここでは、Arc Mouse) を検索して見つけるには、次の手順を実行します。

1. **[Select a group]** (グループの選択) で **[デバイス]** を選択します。 これにより、**[Select a product type]** (製品タイプの選択) と **[Select a feature or AQ]** (機能または AQ の選択) のリストの表示内容が決定されます。 一覧で **[システム]** を選択して、Arc Mouse の検索を実行した場合は、結果が返されません。

2. **[Enter a search value]** (検索値の入力) ボックスに「**Arc Mouse**」と入力し、**[検索]** をクリックします。 このテキスト文字列を含むすべてのハードウェア デバイスの結果セットが返されます。 この一覧は、該当する列見出しをクリックして、**[製品名]**、**[会社]**、または **[証明書]** の順に並べ替えることができます。

3. 検索範囲を限定し、返されるレコード数を減らすには、**[Select an OS]** (OS の選択) リストで **[Windows 8.1 Client]** (Windows 8.1 クライアント) を選び、**[検索]** をクリックします。

4. 製品名をクリックして、製品に関する詳しい情報を表示します。 詳細のリストには、製品名、会社、互換性の状態、製品タイプ、および認定の状態が含まれています。

> [!Note]
> 検索の対象は、**"製品名"** と **"会社"** に制限されます。
> 
> 検索は、**[検索]** ボックスに入力した各語句に対して AND 処理で実行されます。
> 
> 検索では、引用符で囲まれた複数の単語が、1 つの検索文字列として処理されます。

## <a name="to-find-products-by-filtering"></a>製品をフィルター処理で見つけるには

Windows 8.1 x64 のすべてのキーボード製品を見つけるには、次の手順に従います。

1. **[Select a group]** (グループの選択) で **[デバイス]** を選択します。 これにより、**[Select a product type]** (製品タイプの選択) と **[Select a feature or AQ]** (機能または AQ の選択) のリストの表示内容が決定されます。

2. **[Select an OS]** (OS の選択) で **[Windows 8.1 Client x64]** (Windows 8.1 クライアント x64) を選択します。

3. **[Select a product type]** (製品タイプの選択) で **[キーボード]** を選択し、**[検索]** をクリックします。

Windows 8.1 x 64 用キーボード製品の一覧が返されます。 この一覧は、該当する列見出しをクリックして、**[製品名]**、**[会社]**、または **[証明書]** の順に並べ替えることができます。

**[Select a feature or AQ]** (機能または AQ の選択) リストを使って、機能でフィルター処理することもできます。 この一覧内の値は、[Windows ハードウェア互換性プログラムの要件に関するページ](https://go.microsoft.com/fwlink/p/?LinkID=219232)に記載された機能に対応しています。

## <a name="to-sort-your-results"></a>検索結果を並べ替えるには

すべての検索結果は、製品名、会社、認定に基づいて並べ替えることができます。

1. いずれかの見出しをクリックすると、対応する属性の昇順で結果が並べ替えられます。

2. その見出しを再度クリックすると、対応する属性の降順で結果が並べ替えられます。

## <a name="to-view-a-windows-certification-verification-report"></a>Windows 認定検証レポートを表示するには

製品の詳細ページには、製品のすべての認定に関する Windows 認定検証レポートへのリンクが含まれています。 このリンクは、提出 ID と認定完了日で構成されています。

Windows 認定検証レポートを表示するには、次の手順に従います。

- 製品の [詳細] ページで、**[Verification report]** (検証レポート) セクションにあるリンクのいずれかをクリックします。 認定検証レポートは PDF 形式で表示されます。
