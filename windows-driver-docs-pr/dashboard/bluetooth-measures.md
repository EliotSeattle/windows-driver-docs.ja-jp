# <a name="bluetooth-measures"></a>Bluetooth の測定値

## <a name="description"></a>説明

Bluetooth は、コンピューター、携帯電話、携帯デバイス、マウス デバイス、キーボード、プリンターなど、さまざまなデバイスの間でワイヤレス接続を実現する業界標準のプロトコルです。 Microsoft の Windows マシンはこのプロトコルに対応しています。 [Bluetooth プロファイル ドライバーの概要](https://docs.microsoft.com/windows-hardware/drivers/bluetooth/bluetooth-profile-drivers-overview)に関するページでは、マシンでの Bluetooth 操作を有効にするために必要な各種の要件、ドライバー、サポート ファイルが示されています。
