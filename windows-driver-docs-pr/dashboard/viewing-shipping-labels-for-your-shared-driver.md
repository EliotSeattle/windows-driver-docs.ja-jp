# <a name="view-partner-shipping-labels-for-a-shared-driver"></a>パートナーによる共有ドライバーの出荷ラベルを表示する


ドライバーをパートナーと共有すると、パートナーはドライバーの出荷ラベルを作成できます。 パートナーによるドライバーの出荷ラベルの一覧を表示するには、以下の手順を実行します。

1.  共有ドライバーが含まれている[ハードウェア提出を検索します](manage-your-hardware-submissions.md)。

2.  ハードウェア提出の **[配布]** セクションに移動します。 この共有ドライバーの出荷ラベルの一覧を表示することができます。

    ![共有された提出の出荷ラベルを示すスクリーンショット](images/publish-view-label.png)
