# <a name="ecosystem-netflix-hwdrm-quality-comparison"></a>エコシステム Netflix HWDRM 品質比較

## <a name="description"></a>説明

ハードウェア デジタル著作権管理 (HWDRM) は、複数のデバイス プラットフォームで高解像度および超高解像度のコンテンツの安全な再生を可能にする機能です。 この機能は、秘密キーやコンテンツ キーなどのデジタル キーのほか、圧縮および非圧縮のビデオとオーディオを保護します。 HWDRM エラーが発生すると、ユーザーは Netflix などの特定のサービスでビデオを再生できなくなります。 ドライバーのセッション失敗率が、小売市場で同じ IHV から入手した全ドライバーの平均セッション失敗率の 2 標準偏差以内である場合に合格です。  ドライバーのセッション失敗率がこれより高い場合は不合格です。  

## <a name="measure-attributes"></a>測定値の属性

|属性|Value|
|----|----|
|**オーディエンス**|エコシステム|
|**期間**|28 日間のスライディング ウィンドウ|
|**測定基準**|インスタンスの集計|
|**最小母集団**|1 つのインスタンス|
|**合格基準**|ドライバーのセッション失敗率が、全ドライバーの平均セッション失敗率の 2 標準偏差以内にある|
|**測定 ID**|19170127|

## <a name="calculation"></a>計算

1.  この測定値は、28 日間のスライディング ウィンドウからのテレメトリを**バイナリ値 (0 または 1)** として集計したもので、ドライバーのパフォーマンスが、小売市場で同じ IHV から入手した全ドライバーの集計パフォーマンスの 2 標準偏差以内であるかどうかを示します

    i.  測定結果が 0 の場合は成功です。この結果は、ドライバーのパフォーマンスが、すべての小売ドライバーのパフォーマンスの 95% 以内にあることを意味します 

2.  "*評価されるドライバーのセッション失敗率 = 失敗したセッションの数 / 総セッション数*"
3.  "*小売ドライバーのセッション失敗率 = 失敗したセッションの数 / 総セッション数*"

    i.  小売市場の同じ IHV から入手したすべての他のグラフィックス ドライバーを対象に 1 回計算

4.  "*小売ドライバーの失敗率の平均 = 小売ドライバーのセッション失敗率の平均*"
5.  "*小売ドライバーの失敗率の標準偏差 = 小売ドライバーのセッション失敗率の標準偏差*"
6.  "*標準偏差 2 の上限 = 平均 + (小売ドライバーの失敗率の標準偏差 * 2)* "

### <a name="final-calculation"></a>最終的な計算
8.  "*Netflix HWDRM 小売比較 =* " 
    
    i.  "*評価されるドライバーの標準失敗率 =< 標準偏差 2 の上限の場合、0*"

    ii. "*それ以外の場合、Netflix HWDRM 小売比較 = 1*"
