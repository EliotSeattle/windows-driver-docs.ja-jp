# <a name="service-number"></a>サービス番号


サービス番号は、特定の移動体通信事業者を一意に識別するもので、サービス メタデータ パッケージを作成するときの必須フィールドです。 このフィールドは GUID によって表され、xxxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxx という形式 (中かっこは不要) にする必要があります。

[ServiceNumber](https://docs.microsoft.com/windows-hardware/drivers/mobilebroadband/servicenumber) は、サービス メタデータ パッケージの ServiceInfo.xsd ファイルの必須要素です。
