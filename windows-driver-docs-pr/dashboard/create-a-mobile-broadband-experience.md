# <a name="create-a-mobile-broadband-experience"></a>モバイル ブロードバンド エクスペリエンスの作成

モバイル ブロードバンド エクスペリエンスでは、サービス メタデータ パッケージを作成して、UWP モバイル ブロードバンド アプリをハードウェア ドングルに関連付けることができます。

ここでは、次のトピックについて説明します。

- [開発者モードとプレビュー モード](https://docs.microsoft.com/windows-hardware/drivers/dashboard/)

- [サービス アイコンの要件](https://docs.microsoft.com/windows-hardware/drivers/dashboard/)

- [サービス番号](https://docs.microsoft.com/windows-hardware/drivers/dashboard/)

- [信頼できる証明書](https://docs.microsoft.com/windows-hardware/drivers/dashboard/)

サービス メタデータ パッケージに関する詳しい情報や、サービス メタデータ パッケージをパートナー センターに申請する方法については、次のリンク先をご覧ください。

- [サービス メタデータの作成に関する開発者向けガイド](https://docs.microsoft.com/windows-hardware/drivers/mobilebroadband/developer-guide-for-creating-service-metadata) – パートナー センターにサービス メタデータ パッケージを申請するプロセスについて説明した手順ガイド。

- [サービス メタデータ パッケージのスキーマ リファレンス](https://docs.microsoft.com/windows-hardware/drivers/mobilebroadband/service-metadata-package-schema-reference) – サービス メタデータ パッケージに含まれている XML スキーマのリファレンスです。
