# <a name="create-a-mobile-broadband-experience"></a>モバイル ブロードバンド エクスペリエンスの作成

モバイル ブロードバンド エクスペリエンスでは、サービス メタデータ パッケージを作成して、UWP モバイル ブロードバンド アプリをハードウェア ドングルに関連付けることができます。

ここでは、次のトピックについて説明します。

- [開発者モードとプレビュー モード](https://msdn.microsoft.com/library/windows/hardware/dn282082.aspx)

- [サービス アイコンの要件](https://msdn.microsoft.com/library/windows/hardware/dn236416.aspx)

- [サービス番号](https://msdn.microsoft.com/library/windows/hardware/dn236413.aspx)

- [信頼できる証明書](https://msdn.microsoft.com/library/windows/hardware/dn486876.aspx)

サービス メタデータ パッケージに関する詳しい情報や、サービス メタデータ パッケージをパートナー センターに申請する方法については、次のリンク先をご覧ください。

- [サービス メタデータの作成に関する開発者向けガイド](https://msdn.microsoft.com/library/windows/hardware/dn973080) – パートナー センターにサービス メタデータ パッケージを申請するプロセスについて説明した手順ガイド。

- [サービス メタデータ パッケージのスキーマ リファレンス](https://msdn.microsoft.com/library/windows/hardware/dn973175) – サービス メタデータ パッケージに含まれている XML スキーマのリファレンスです。
