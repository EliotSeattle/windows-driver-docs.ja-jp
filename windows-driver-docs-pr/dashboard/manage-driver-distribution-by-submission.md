# <a name="manage-driver-distribution-with-shipping-labels"></a>出荷ラベルによるドライバーの配布の管理


ハードウェアの提出を作成し、ドライバー パッケージをアップロードした後、出荷ラベルを作成してドライバーを配布できます。 出荷ラベルで次の操作を行うことができます。

-   ドライバーを Windows Update に公開する

-   ドライバーをパートナーと共有する

-   共有しているドライバーを公開する

-   パートナーの代わりにドライバーを公開する
