# <a name="how-to-read-the-driver-test-results-log"></a>ドライバーのテスト結果ログを読み取る方法

**[Driver Test Group Explorer (ドライバー テスト グループ エクスプローラー)]** で、ドライバー テストの結果の表示、テスト結果のフォルダーへの保存、フォルダーからの結果の読み込みを実行できます。

### <a name="span-idopen_the_test_resultsspanspan-idopen_the_test_resultsspanspan-idopen_the_test_resultsspanopen-the-test-results"></a><span id="Open_the_test_results"></span><span id="open_the_test_results"></span><span id="OPEN_THE_TEST_RESULTS"></span>テスト結果を開く

1. **[Driver Test Group Explorer] (ドライバー テスト グループ エクスプローラー)** ウィンドウで、対象のドライバー テスト グループをクリックします。
2. 特定のテスト実行の結果をクリックします。

   テスト グループの結果は、**Results (** <em>computer</em> **-** <em>date</em> **:** <em>time</em> **)** という名前のフォルダーとして示されます

3. そのテスト実行内のテストの結果が一覧表示されます。 結果ログを表示するテストの名前をクリックします。

結果ログには、実行されたテストの要約と、テスト ケース別に生成された詳細が含まれます。 記録される情報には、各テストの実行日時、テストを実行したリモート テスト コンピューターの名前、テスト結果 (実行したテストの数、成功、失敗) が含まれます。

![[テスト結果] ウィンドウには、実行されたテストの要約と、各テスト ケースの詳細が表示される。](images/ert-test-results.png)

### <a name="span-idsave_the_test_results_to_a_folderspanspan-idsave_the_test_results_to_a_folderspanspan-idsave_the_test_results_to_a_folderspansave-the-test-results-to-a-folder"></a><span id="Save_the_test_results_to_a_folder"></span><span id="save_the_test_results_to_a_folder"></span><span id="SAVE_THE_TEST_RESULTS_TO_A_FOLDER"></span>テスト結果をフォルダーに保存する

1.  **[Driver Test Group Explorer (ドライバー テスト グループ エクスプローラー)]** ウィンドウで、対象のテスト実行結果を右クリックします。
2.  **[保存]** をクリックすると、結果をフォルダーに保存するためのダイアログ ボックスが開きます。
