# <a name="ksnodetypeadc"></a>KSNODETYPE\_ADC


## <span id="ddk_ksnodetype_adc_ks"></span><span id="DDK_KSNODETYPE_ADC_KS"></span>


KSNODETYPE\_ADC ノードが、アナログ/デジタル変換 (ADC) を表します。 ADC は、1 つの入力ストリームと 1 つの出力ストリームがあります。

KSNODETYPE\_ADC ノードは、次の省略可能なプロパティをサポートできます。

[**KSPROPERTY\_オーディオ\_サンプリング\_率**](ksproperty-audio-sampling-rate.md)

[**KSPROPERTY\_オーディオ\_動的\_サンプリング\_率**](ksproperty-audio-dynamic-sampling-rate.md)
