# <a name="ksnodetypesrc"></a>KSNODETYPE\_SRC


## <span id="ddk_ksnodetype_src_ks"></span><span id="DDK_KSNODETYPE_SRC_KS"></span>


KSNODETYPE\_SRC ノードは、デジタル署名の再入力ストリームのバージョンである出力ストリームを生成する、サンプル速度コンバーターを表します。 SRC ノードは 1 つの入力ストリームを備え、1 つの出力ストリームと同じ数のチャネル。

KSNODETYPE\_SRC ノードは、次の省略可能なプロパティをサポートできます。

[**KSPROPERTY\_オーディオ\_サンプリング\_率**](ksproperty-audio-sampling-rate.md)

[**KSPROPERTY\_オーディオ\_動的\_サンプリング\_率**](ksproperty-audio-dynamic-sampling-rate.md)

[**KSPROPERTY\_オーディオ\_品質**](ksproperty-audio-quality.md)
