# <a name="ksnodetypedemux"></a>KSNODETYPE\_DEMUX


## <span id="ddk_ksnodetype_demux_ks"></span><span id="DDK_KSNODETYPE_DEMUX_KS"></span>


KSNODETYPE\_DEMUX ノードが、デマルチプレクサーを表します。 このコントロールには、1 つの入力ストリームと複数の出力ストリームがあります。

KSNODETYPE\_DEMUX ノードは、次のプロパティをサポートする必要があります。

[**KSPROPERTY\_AUDIO\_DEMUX\_DEST**](ksproperty-audio-demux-dest.md)
