# <a name="bass-boost"></a>低音ブースト


低音機能が制限付きのスピーカーがあるラップトップなどのシステムで低音スピーカーでサポートされる周波数の範囲で応答を混ぜてオーディオの知覚品質を向上させることがあります。 音は、中間低音の範囲のゲインを増やすことで非常に小さなスピーカーのモバイル デバイスで音を向上します。
