# <a name="speaker-phantoming"></a>スピーカーのファントム


通常、center およびサテライト loudspeakers を含め、マルチ チャネル システム内のすべての loudspeakers では、常に存在します。 ただし、ユーザーに必要なすべての loudspeakers いない可能性があります。 またはそれらの 1 つ以上を有効にすることを選択可能性があります。 一般的な例は、center スピーカーが不足しているマルチ チャネル システムです。 スピーカーのファントムは隣接する loudspeakers 間で分割することにより、通常不足しているスピーカーからサウンドを再現します。 ただし、ファントムは背面左と背面右または左にある側右スピーカーなどの他の組み合わせにも使用できます。
