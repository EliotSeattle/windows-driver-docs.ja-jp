# <a name="kspropsetidrtaudio"></a>KSPROPSETID\_RTAudio


`KSPROPSETID_RTAudio`プロパティ セットが WaveRT オーディオ デバイスのプロパティを指定します。 Windows Vista 以降の Windows オペレーティング システムでは、これらのプロパティがサポートされています。

次のプロパティの定義にプロパティが取得専用と、ターゲットは、暗証番号 (pin)。

-   このプロパティのすべてのプロパティ セットのサポート**取得**プロパティがなく、クライアントから要求**設定**プロパティ要求。

-   このセットのすべてのプロパティ、クライアントがプロパティの要求を送信する先のターゲットは、暗証番号 (pin) インスタンスです。 (その他の KS プロパティ設定、その一方で、フィルター インスタンスでプロパティをサポートします。)

`KSPROPSETID_RTAudio`プロパティ セットには、次のプロパティが含まれています。

[**KSPROPERTY\_RTAUDIO\_バッファー**](ksproperty-rtaudio-buffer.md)

[**KSPROPERTY\_RTAUDIO\_BUFFER\_WITH\_NOTIFICATION**](ksproperty-rtaudio-buffer-with-notification.md)

[**KSPROPERTY\_RTAUDIO\_CLOCKREGISTER**](ksproperty-rtaudio-clockregister.md)

[**KSPROPERTY\_RTAUDIO\_GETREADPACKET**](ksproperty-rtaudio-getreadpacket.md)

[**KSPROPERTY\_RTAUDIO\_HWLATENCY**](ksproperty-rtaudio-hwlatency.md)

[**KSPROPERTY\_RTAUDIO\_PACKETCOUNT**](ksproperty-rtaudio-packetcount.md)

[**KSPROPERTY\_RTAUDIO\_POSITIONREGISTER**](ksproperty-rtaudio-positionregister.md)

[**KSPROPERTY\_RTAUDIO\_プレゼンテーション\_位置**](ksproperty-rtaudio-presentation-position.md)

[**KSPROPERTY\_RTAUDIO\_クエリ\_通知\_サポート**](ksproperty-rtaudio-query-notification-support.md)

[**KSPROPERTY\_RTAUDIO\_登録\_通知\_イベント**](ksproperty-rtaudio-register-notification-event.md)

[**KSPROPERTY\_RTAUDIO\_SETWRITEPACKET**](ksproperty-rtaudio-setwritepacket.md)

[**KSPROPERTY\_RTAUDIO\_登録解除\_通知\_イベント**](ksproperty-rtaudio-unregister-notification-event.md)
