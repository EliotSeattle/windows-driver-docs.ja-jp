# <a name="ksnodetypedevspecific"></a>KSNODETYPE\_DEV\_特定


## <span id="ddk_ksnodetype_dev_specific_ks"></span><span id="DDK_KSNODETYPE_DEV_SPECIFIC_KS"></span>


KSNODETYPE\_DEV\_特定のノードは、デバイス固有のコントロールを表します。 デバイス固有のコントロールには、複数の入力ストリームと複数の出力ストリームを持つことができます。 特定のオーディオ デバイスに固有の関数のセットを実行します。

KSNODETYPE\_DEV\_特定のノードは、次の必要なプロパティをサポートする必要があります。

[**KSPROPERTY\_オーディオ\_DEV\_特定**](ksproperty-audio-dev-specific.md)
