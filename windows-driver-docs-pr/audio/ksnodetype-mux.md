# <a name="ksnodetypemux"></a>KSNODETYPE\_MUX


## <span id="ddk_ksnodetype_mux_ks"></span><span id="DDK_KSNODETYPE_MUX_KS"></span>


KSNODETYPE\_MUX ノードは、マルチプレクサー (マルチプレクサー) を表します。 MUX は、複数の入力ストリームと、同じデータ形式をすべて 1 つの出力ストリームがあります。 一度に 1 つだけの入力ストリームは、出力ストリームにルーティングされます。

KSNODETYPE\_MUX ノードは、次の必要なプロパティをサポートする必要があります。

[**KSPROPERTY\_AUDIO\_MUX\_SOURCE**](ksproperty-audio-mux-source.md)
