# <a name="kseventsetidsounddetector"></a>KSEVENTSETID\_SoundDetector


KSEVENTSETID\_SoundDetector イベントのセットは、サウンドの検出機能の一致が検出されたことをオペレーティング システムに通知するオーディオ ドライバーによって使用されます。

このセット内のイベントの項目が次のように指定された**KSEVENT\_SOUNDDETECTOR**列挙値。

-   [**KSEVENT\_SOUNDDETECTOR\_MATCHDETECTED**](ksevent-sounddetector-matchdetected.md)
