# <a name="low-frequency-protection"></a>低周波の保護


頻度の低い保護は、一種の[ベースの管理を転送](bass-management.md)します。 サブウーファーがないと、すべてのスピーカーが小さいし、低音機能がないときに使用されます。 頻度の低い保護 APO を適用するときに、APO のアルゴリズムで指定したカットオフ周波数の下にあるオーディオ信号の一部の削除になります。
