# <a name="ksnodetypetone"></a>KSNODETYPE\_トーン


## <span id="ddk_ksnodetype_tone_ks"></span><span id="DDK_KSNODETYPE_TONE_KS"></span>


KSNODETYPE\_トーン ノードがトーン コントロールを表します。 トーン コントロールが 1 つの入力ストリームと 1 つの出力ストリーム。2 つのストリームでは、同じデータ形式があります。 低音の量を減衰できます高音、または出力ストリームの中頻度。 さらに、音または向上を必要に応じてサポートできます。

KSNODETYPE\_トーン ノードは、次のプロパティの少なくとも 1 つをサポートする必要があります。

[**KSPROPERTY\_オーディオ\_低音**](ksproperty-audio-bass.md)

[**KSPROPERTY\_オーディオ\_MID**](ksproperty-audio-mid.md)

[**KSPROPERTY\_AUDIO\_TREBLE**](ksproperty-audio-treble.md)

[**KSPROPERTY\_オーディオ\_低音\_BOOST**](ksproperty-audio-bass-boost.md)
