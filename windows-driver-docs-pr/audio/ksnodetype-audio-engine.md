# <a name="ksnodetypeaudioengine"></a>KSNODETYPE\_オーディオ\_エンジン


KSNODETYPE\_オーディオ\_エンジンのオーディオのエンドポイントが Windows 8 および Windows の以降のバージョンで提供される新しいエンドポイント。 KSNODETYPE\_オーディオ\_エンジンのオーディオのエンドポイントがストリーミング (KS) オーディオ フィルター、Wave カーネル内に含まれ、オーディオ ドライバー、ハードウェアのオーディオ エンジンの機能を公開することができます。

KSNODETYPE\_オーディオ\_エンジンのオーディオのエンドポイントが必要なエンドポイントとその概要行われた、新しいエンドポイントの直接の接続を新しいデータ シンク暗証番号 (pin) ファクトリを開発するために必要です。

KSNODETYPE\_オーディオ\_エンジンのオーディオのエンドポイントをサポートする必要があります、 [ **KSPROPERTY\_AUDIOENGINE** ](ksproperty-audioengine.md)次 KS だけでなく値の列挙プロパティ:

[**KSPROPERTY\_オーディオ\_ミュート**](ksproperty-audio-mute.md)

[**KSPROPERTY\_AUDIO\_PEAKMETER**](ksproperty-audio-peakmeter.md)

[**KSPROPERTY\_オーディオ\_VOLUMELEVEL**](ksproperty-audio-volumelevel.md)

[KSPROPSETID\_AudioEngine](kspropsetid-audioengine.md)
