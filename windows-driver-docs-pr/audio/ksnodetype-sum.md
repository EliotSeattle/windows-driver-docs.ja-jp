# <a name="ksnodetypesum"></a>KSNODETYPE\_合計


## <span id="ddk_ksnodetype_sum_ks"></span><span id="DDK_KSNODETYPE_SUM_KS"></span>


KSNODETYPE\_合計ノードは、単純な集計 (SUM) 関数を表します。 合計ノードには、複数の入力ストリームと同じデータ形式をすべて 1 つの出力ストリームがあります。 入力ストリームが出力ストリームを生成するために均等に混在しています。

KSNODETYPE\_合計ノードは、プロパティをサポートする必要はありません。
