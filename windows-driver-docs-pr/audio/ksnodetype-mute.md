# <a name="ksnodetypemute"></a>KSNODETYPE\_ミュート


## <span id="ddk_ksnodetype_mute_ks"></span><span id="DDK_KSNODETYPE_MUTE_KS"></span>


KSNODETYPE\_ミュート ノードがミュート用のコントロールを表します。 このノードが 1 つの入力ストリームと 1 つの出力ストリームの 2 つのストリームは、同じデータ形式。 入力は、オンまたはオフに切り替えます。

KSNODETYPE\_ミュート ノードは、次の必要なプロパティをサポートする必要があります。

[**KSPROPERTY\_オーディオ\_ミュート**](ksproperty-audio-mute.md)
