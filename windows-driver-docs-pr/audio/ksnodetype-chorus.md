# <a name="ksnodetypechorus"></a>KSNODETYPE\_コーラス


## <span id="ddk_ksnodetype_chorus_ks"></span><span id="DDK_KSNODETYPE_CHORUS_KS"></span>


KSNODETYPE\_コーラス ノードが、出力ストリームにコーラス効果を追加、コーラス効果プロセッサを表します。 コーラス ノードが 1 つの入力ストリームと 1 つの出力ストリーム、およびこれら 2 つのストリームが、同じデータ形式を共有します。

KSNODETYPE\_コーラス ノードは、次の必須プロパティをサポートする必要があります。

[**KSPROPERTY\_オーディオ\_コーラス\_レベル**](ksproperty-audio-chorus-level.md)

[**KSPROPERTY\_オーディオ\_コーラス\_変調\_深さ**](ksproperty-audio-chorus-modulation-depth.md)

[**KSPROPERTY\_オーディオ\_コーラス\_変調\_率**](ksproperty-audio-chorus-modulation-rate.md)
