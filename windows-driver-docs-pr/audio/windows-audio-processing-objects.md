# <a name="windows-audio-processing-objects"></a>Windows オーディオ処理オブジェクト

オーディオ処理オブジェクト (APOs) は、Windows オーディオストリーム用のカスタマイズ可能なソフトウェアベースのデジタル信号処理を提供します。

ここでは、次のトピックについて説明します。

-   [オーディオ処理オブジェクトのアーキテクチャ](audio-processing-object-architecture.md)
-   [オーディオ処理オブジェクトの実装](implementing-audio-processing-objects.md)
-   [ハードウェアオフロードの APO 効果の実装](implementing-hardware-offloaded-apo-effects.md)

 
APOs は Windows Vista で初めて導入されましたが、以前のシステムに関する参照が表示される場合があります。 詳細については、「 [Windows Vista でのカスタムオーディオ効果](https://go.microsoft.com/fwlink/p/?linkid=106024)」を参照してください。 このホワイトペーパーでは、以前の[COM](https://go.microsoft.com/fwlink/p/?linkid=106293)および[UI の開発](https://go.microsoft.com/fwlink/p/?linkid=106294)に関するトピックを参照できます。
