# <a name="ksnodetypedelay"></a>KSNODETYPE\_遅延


## <span id="ddk_ksnodetype_delay_ks"></span><span id="DDK_KSNODETYPE_DELAY_KS"></span>


KSNODETYPE\_遅延のノードが遅延するコントロールを表します。 遅延コントロールが 1 つの入力ストリームと 1 つの出力ストリーム、およびこれら 2 つのストリームが、同じデータ形式を共有します。 遅延コントロールは、一定の時間によって、入力ストリームの遅れを出力ストリームを発生させます。

KSNODETYPE\_遅延ノードは、次の必要なプロパティをサポートする必要があります。

[**KSPROPERTY\_オーディオ\_遅延**](ksproperty-audio-delay.md)
