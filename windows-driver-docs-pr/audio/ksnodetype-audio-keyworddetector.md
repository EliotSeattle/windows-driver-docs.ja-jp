# <a name="ksnodetypeaudiokeyworddetector"></a>KSNODETYPE\_オーディオ\_KEYWORDDETECTOR


KSNODETYPE\_オーディオ\_KEYWORDDETECTOR オーディオ エンドポイントは、Windows 10 と以降のバージョンの Windows で使用可能である新しいエンドポイント。 KSNODETYPE\_オーディオ\_KEYWORDDETECTOR により、検出機能をサポートするために、オーディオ キャプチャ ドライバー。

このフィルターは、キャプチャ デバイスと共有することはできません。 フィルターがピン留めするカテゴリ KSNODETYPE をある KS pin ファクトリ\_オーディオ\_KEYWORDDETECTOR します。 KS フィルター インスタンスでこの KS 暗証番号 (pin) カテゴリを持つ 1 つ以上のピン留めするファクトリを設定することはできません。

KSNODETYPE\_オーディオ\_KEYWORDDETECTOR オーディオ エンドポイントは、次の KS プロパティをサポートする必要があります。

[**KSPROPERTY\_SOUNDDETECTOR\_SUPPORTEDPATTERNS**](ksproperty-sounddetector-supportedpatterns.md)

[**KSPROPERTY\_SOUNDDETECTOR\_パターン**](ksproperty-sounddetector-patterns.md)

[**KSPROPERTY\_SOUNDDETECTOR\_故障**](ksproperty-sounddetector-armed.md)

[**KSPROPERTY\_SOUNDDETECTOR\_MATCHRESULT**](ksproperty-sounddetector-matchresult.md)
