# <a name="wdm-audio-platform-differences"></a>WDM 音声プラットフォームの違い


## <span id="wdm_audio_platform_differences"></span><span id="WDM_AUDIO_PLATFORM_DIFFERENCES"></span>


このセクションでは、Windows のさまざまなバージョンで提供されている Microsoft Windows Driver Model (WDM) オーディオ ドライバーのサポートのレベルについて説明します。

次のトピックでは、利用できる各種の Windows リリースでのオーディオの WDM のサポートについて説明します。

[WDM オーディオの Windows Vista のサポート](windows-vista-support-for-wdm-audio.md)

WDM プラットフォームの違いの詳細については、Windows Driver Kit (WDK) ドキュメントの他の場所で使用できます。

-   Windows のバージョン間で相違点 PortCls 関数とインターフェイスのサポートについては、[オペレーティング システムによって PortCls サポート](portcls-support-by-operating-system.md)を参照してください。

-   Windows のバージョン間で PortCls アダプター ドライバーのバージョン番号の違いについては、[オーディオ ドライバーのバージョン番号](version-numbers-for-audio-drivers.md)を参照してください。

-   どの WDM ドライバーのサポートは、一般に Windows のバージョンでは、ドライバーは、浮動小数点の問題の説明などによって異なります。 については、[WDM バージョン間の違い](https://msdn.microsoft.com/library/windows/hardware/ff543216)を参照してください。
