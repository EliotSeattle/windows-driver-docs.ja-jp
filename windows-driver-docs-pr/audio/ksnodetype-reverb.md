# <a name="ksnodetypereverb"></a>KSNODETYPE\_リバーブ


## <span id="ddk_ksnodetype_reverb_ks"></span><span id="DDK_KSNODETYPE_REVERB_KS"></span>


KSNODETYPE\_リバーブ ノードがリバーブ (または「リバーブ」) のコントロールを表します。 リバーブ ノードが 1 つの入力ストリームと 1 つの出力ストリームの入力と出力ストリームが、同じデータ形式。 ノードは、出力ストリームにリバーブを追加します。

KSNODETYPE\_リバーブ ノードは、次の必須プロパティをサポートする必要があります。

[**KSPROPERTY\_オーディオ\_リバーブ\_レベル**](ksproperty-audio-reverb-level.md)

[**KSPROPERTY\_オーディオ\_リバーブ\_時間**](ksproperty-audio-reverb-time.md)
