# <a name="ksnodetypefmrx"></a>KSNODETYPE\_FM\_RX


**KSNODETYPE\_FM\_RX**ノードがシステムで FM キャプチャ オーディオ エンドポイントを表すし、オーディオのエンドポイントのすべてのプロパティがサポートされます (を参照してください[KSPROPSETID\_オーディオ](kspropsetid-audio.md)). KSNODETYPE にキャプチャ暗証番号 (pin) を作成する\_FM\_RX は、FM 受信者からオーディオ キャプチャを許可します。

関連付けられているフィルター、 **KSNODETYPE\_FM\_RX**ノードは、次の追加プロパティをサポートする必要があります。

[**KSPROPERTY\_FMRX\_ANTENNAENDPOINTID**](ksproperty-fmrx-antennaendpointid.md)

[**KSPROPERTY\_FMRX\_ENDPOINTID**](ksproperty-fmrx-endpointid.md)

[**KSPROPERTY\_FMRX\_状態**](ksproperty-fmrx-state.md)

[**KSPROPERTY\_FMRX\_ボリューム**](ksproperty-fmrx-volume.md)
