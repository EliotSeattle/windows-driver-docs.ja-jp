# <a name="kseventsetidpincapschange"></a>KSEVENTSETID\_PinCapsChange


Windows 7 および Windows オペレーティング システムでは、以降のバージョンで、`KSEVENTSETID_PinCapsChange`イベントのセットは、オーディオ形式、ジャック情報またはオーディオ デバイスのさまざまなプロパティが変更されたオペレーティング システムに通知するオーディオ ドライバーによって使用されます。

このセット内のイベント項目は、次の KSEVENT として指定\_PINCAPS\_通知列挙値。

[**KSEVENT\_PINCAPS\_段落**](ksevent-pincaps-formatchange.md)

[**KSEVENT\_PINCAPS\_JACKINFOCHANGE**](ksevent-pincaps-jackinfochange.md)
