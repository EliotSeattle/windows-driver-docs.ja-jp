# <a name="kspropsetidtelephonytopology"></a>KSPROPSETID\_TelephonyTopology


`KSPROPSETID_TelephonyTopology`プロパティ セットが使用されているアクセスのエンドポイントに関連付けられた携帯電話の呼び出しのボリュームと携帯電話のオーディオ ルーティングします。

`KSPROPSETID_TelephonyTopology`プロパティ セットには、次のプロパティが含まれています。

-   [**KSPROPERTY\_テレフォニー\_ENDPOINTIDPAIR**](ksproperty-telephony-endpointidpair.md)

-   [**KSPROPERTY\_テレフォニー\_ボリューム**](ksproperty-telephony-volume.md)
