# <a name="ksnodetypestereowide"></a>KSNODETYPE\_ステレオ\_ワイド


## <span id="ddk_ksnodetype_stereo_wide_ks"></span><span id="DDK_KSNODETYPE_STEREO_WIDE_KS"></span>


KSNODETYPE\_ステレオ\_ワイド ノードは、コントロールをステレオ ワイドかどうかを表します。 この種類のコントロールは、ステレオの画像の明らかな幅を増やすことで、既存のステレオ (2 つのチャネル) ストリームに広がりを追加できます。 結果のステレオ イメージでは、いくつかのサウンドは、左右のスピーカーで囲まれている領域の外側の位置から表示されます。

ワイドかどうかのノードには、次の必要なプロパティをサポートする必要があります。

[**KSPROPERTY\_オーディオ\_ワイドかどうか**](ksproperty-audio-wideness.md)
