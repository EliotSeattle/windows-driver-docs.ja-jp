# <a name="ksnodetypeloudness"></a>KSNODETYPE\_ラウドネス


## <span id="ddk_ksnodetype_loudness_ks"></span><span id="DDK_KSNODETYPE_LOUDNESS_KS"></span>


KSNODETYPE\_ラウドネスのノードが出力ストリームの動的範囲を調整するラウドネス コントロールを表します。 ラウドネス コントロールが 1 つの入力ストリームと 1 つの出力ストリームの 2 つのストリームは、同じデータ形式。

KSNODETYPE\_ラウドネス ノードは、次のプロパティの少なくとも 1 つをサポートする必要があります。

[**KSPROPERTY\_オーディオ\_ラウドネス**](ksproperty-audio-loudness.md)

[**KSPROPERTY\_オーディオ\_動的\_範囲**](ksproperty-audio-dynamic-range.md)
