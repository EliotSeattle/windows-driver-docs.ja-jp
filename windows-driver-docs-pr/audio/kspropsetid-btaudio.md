# <a name="kspropsetidbtaudio"></a>KSPROPSETID\_BtAudio


`KSPROPSETID_BtAudio`プロパティ セットが (KS) フィルターまたは pin をストリーミングする Bluetooth のオーディオ カーネルに Bluetooth オーディオ固有のプロパティを送信するために使用します。

`KSPROPSETID_BtAudio`プロパティ セットには、次のプロパティが含まれています。

[**KSPROPERTY\_ONESHOT\_切断**](ksproperty-oneshot-disconnect.md)

[**KSPROPERTY\_ONESHOT\_再接続**](ksproperty-oneshot-reconnect.md)
