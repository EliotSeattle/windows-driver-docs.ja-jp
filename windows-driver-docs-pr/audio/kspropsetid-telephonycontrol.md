# <a name="kspropsetidtelephonycontrol"></a>KSPROPSETID\_TelephonyControl


`KSPROPSETID_TelephonyControl`プロパティ セットがテレフォニーのコントロールのプロパティを操作するために使用します。 これらのプロパティは、呼び出しの状態を確認して、保留中の呼び出しを配置するか、ローカルのマイクのミュートなどの特定の操作を実行に使用できます。

`KSPROPSETID_TelephonyControl`プロパティ セットには、次のプロパティが含まれています。

-   [**KSPROPERTY\_テレフォニー\_CALLCONTROL**](ksproperty-telephony-callcontrol.md)

-   [**KSPROPERTY\_テレフォニー\_CALLHOLD**](ksproperty-telephony-callhold.md)

-   [**KSPROPERTY\_テレフォニー\_CALLINFO**](ksproperty-telephony-callinfo.md)

-   [**KSPROPERTY\_テレフォニー\_ミュート\_TX**](ksproperty-telephony-mute-tx.md)

-   [**KSPROPERTY\_テレフォニー\_PROVIDERCHANGE**](ksproperty-telephony-providerchange.md)

-   [**KSPROPERTY\_テレフォニー\_PROVIDERID**](ksproperty-telephony-providerid.md)
