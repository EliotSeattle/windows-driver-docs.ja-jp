# <a name="overview-of-using-xperf-with-usb-etw"></a>Xperf を USB ETW による使用の概要


このトピックでは、Netmon を Xperf を使用して、USB トレース データを分析する方法について説明します。

Xperf は、トレース データを分析する、または USB トレースとして同一のタイムライン上のカーネル イベントを分析する Netmon を使用できます。 Xperf は、Windows 8 用 Windows ソフトウェア開発キット (SDK) の一部では、Windows パフォーマンス ツールキットでです。 SDK をダウンロードする[この Web サイト](https://msdn.microsoft.com/windows/hardware/hh852363)します。
