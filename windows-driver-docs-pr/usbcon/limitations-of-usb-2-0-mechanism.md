USB 2.0 仕様では、使用しない場合、中断状態の入力デバイス (またはハブ) は、電源を節約するためのメカニズムを定義します。 このメカニズムと呼ばれる[セレクティブ サスペンド](https://go.microsoft.com/fwlink/p/?linkid=230962)します。 セレクティブ サスペンドは、終了の待機時間を数十ミリ秒単位では電力を節約する強力なメカニズムです。 セレクティブ サスペンドには、ソフトウェアをデバイスにすべての転送がキャンセルし、中断状態にデバイスを明示的に送信する必要があります。 そのため、このメカニズムは実用的では、デバイスが通常は数秒で長時間アイドル状態だった場合にのみです。

セレクティブ サスペンドも中断状態のデバイスで厳密な電力消費量に制限を課します。 これらの制限は稼働状態にあるときに、デバイスに課せられる制限よりも大幅に減少します。 デバイスは、制限が課せられるに自体を制限するときに、目的のウェイク アップ機能を維持できない、セレクティブ サスペンドを送信できません。
