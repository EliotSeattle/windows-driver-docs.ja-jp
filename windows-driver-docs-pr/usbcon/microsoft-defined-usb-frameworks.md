Microsoft では、一部の種類が自分の USB デバイス クラスの指定がない USB デバイスのドライバー フレームワークを提供します。 ベンダーがこれらの種類のデバイスを開発するには、デバイスの種類の指定のフレームワークを使用するデバイス ドライバーを開発する必要があります。
