# <a name="microsoft-defined-usb-driver-frameworks"></a>Microsoft による USB ドライバー フレームワーク


このトピックでは、独自の USB デバイス クラスの指定がない USB デバイスの Microsoft 提供のドライバー フレームワークについて説明します。

Microsoft では、一部の種類が自分の USB デバイス クラスの指定がない USB デバイスのドライバー フレームワークを提供します。 ベンダーがこれらの種類のデバイスを開発するには、デバイスの種類の指定のフレームワークを使用するデバイス ドライバーを開発する必要があります。

現在マイクロソフトは、次の USB デバイスの次のドライバー フレームワークを提供します。

-   USB 生体認証デバイス

    マイクロソフトは、Windows 生体認証フレームワークを提供することで、USB 生体認証デバイス (指紋リーダー) をサポートします。 詳細については、[生体認証フレームワークの概要](https://msdn.microsoft.com/library/windows/desktop/dd560897.aspx)を参照してください。
