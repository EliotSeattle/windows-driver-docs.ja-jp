# <a name="tearing-down-a-call"></a>呼び出しの破棄





接続指向のクライアントは、発信通話を加えられたまたは承諾が着信呼び出しの終了処理を開始できます。 同様に、リモート パーティは、発信通話を加えられたまたは承諾が着信呼び出しの終了処理を開始できます。
