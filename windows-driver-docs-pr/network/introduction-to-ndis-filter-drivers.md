# <a name="getting-started-with-ndis-filter-drivers"></a>NDIS フィルタードライバーの概要

フィルタードライバーは、ミニポートドライバー用のフィルター処理サービスを提供します。 NDIS ドライバースタックにはミニポートドライバーとプロトコルドライバーが含まれている必要があり、必要に応じてフィルタードライバーを含めることもできます。 NDIS ドライバーとドライバースタックの詳細については、「[ドライバースタック管理](driver-stack-management.md)」を参照してください。

次のアプリケーションでは、フィルタードライバーが必要になる場合があります。

-   セキュリティまたはその他の目的のためのデータフィルター処理アプリケーション。

-   ネットワークデータの統計情報を監視および収集するアプリケーション。
