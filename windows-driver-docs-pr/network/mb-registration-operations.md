# <a name="mb-registration-operations"></a>MB 登録操作


ミニポート ドライバーを使用して、 [OID\_WWAN\_登録\_状態](https://msdn.microsoft.com/library/windows/hardware/ff569834)の両方を処理する*クエリ*と*設定*登録操作。 たとえば、これらの操作は、ネットワーク プロバイダーに登録する、プロバイダーのネットワークとデバイスの登録状態を照会および登録の状態が変更されたときに通知を送信する可能性があります。
