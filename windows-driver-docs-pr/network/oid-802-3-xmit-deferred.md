# <a name="oid8023xmitdeferred"></a>OID_802_3_XMIT_DEFERRED

NIC の後に正常に送信されたフレームの数は、少なくとも 1 回の伝送を延期します。

| | |
| --- | --- |
| バージョン | Windows Vista 以降 |
| Header | Ntddndis.h (include Ndis.h) |
