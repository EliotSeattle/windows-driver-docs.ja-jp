# <a name="changing-the-qos-of-an-active-vc"></a>アクティブ VC の QoS の変更





接続指向のクライアントは、送信または受信呼び出しをアクティブな接続 (VC) 上のサービス (QoS) の品質の変更を要求できます。 リモート クライアントは、このような要求をこともできます。 この場合、コール マネージャーまたは MCM ドライバーには、QoS を変更するリモート クライアントの受信要求を示します。
