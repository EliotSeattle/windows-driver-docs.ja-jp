# <a name="faster-filter-drivers"></a>高速のフィルター ドライバー





NDIS 6.x フィルター ドライバー モデルには、NDIS 5.x フィルター中間ドライバー モデルよりも優先されます。 新しいフィルター ドライバー モデルには、いくつかの方法でシステムのパフォーマンスが拡張されています。

-   NDIS とは異なり中間ドライバーのフィルター、NDIS 6.0 または以降のバージョンのフィルター ドライバーとして実装されていない組み合わせミニポート ドライバーとプロトコル ドライバー。 ミニポートとプロトコル ドライバーに似ていますが、ドライバー スタックで渡される情報をフィルター処理するために最適化された一意のインターフェイスがあります。

-   NDIS 6.0 とそれ以降のフィルター ドライバーは、バイパス機能をサポートできるように、このような処理が必要な場合に、ドライバーがデータを処理しません。

-   NDIS 6.0 とそれ以降のフィルター ドライバーが動的に挿入したりバインド解除を行うことがなく実行時にドライバー スタックから削除します。 このような動的操作が発生する前に、NDIS 6.0 には、スタック内のすべての NDIS ドライバーが一時停止します。 NDIS は、再構成が完了すると、スタックを再起動します。

NDIS 6.0 フィルター ドライバーの詳細については、次を参照してください。 [NDIS 6.0 のフィルター ドライバー](ndis-filter-drivers.md)します。
