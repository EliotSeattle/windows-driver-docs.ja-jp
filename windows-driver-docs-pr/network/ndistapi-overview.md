# <a name="ndistapi-overview"></a>NDISTAPI の概要





NDISTAPI は、システム指定のドライバーで、TAPI サービスへの NDISWAN、NDIS WAN ミニポート ドライバーのインターフェイスです。 NDIS 5.0 およびそれ以降のドライバーは、NDIS WAN ミニポート ドライバーはサポートされていません。 新しいドライバーは、いる CoNDIS WAN のドライバーのアーキテクチャに基づいている必要があります。
