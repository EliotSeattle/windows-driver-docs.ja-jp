# <a name="offloading-ipsec-tasks"></a>IPsec タスクのオフロード

\[IPsec タスクオフロード機能は非推奨とされているため、使用しないでください。\]




ここでは、インターネットプロトコルセキュリティ (IPsec) タスクのオフロードについて説明します。

**注**  IPsec オフロード (OOB) データは、 [**NET\_バッファー\_リスト**](https://docs.microsoft.com/windows-hardware/drivers/ddi/ndis/ns-ndis-_net_buffer_list)情報配列に格納されます。 OOB データの詳細については、「 [Tcp/ip オフロード NET\_BUFFER\_LIST 情報へのアクセス](accessing-tcp-ip-offload-net-buffer-list-information.md)」を参照してください。

 

[IPsec オフロードバージョン 2](ipsec-offload-version-2.md)は、NDIS 6.1 以降でサポートされています。

[IPsec オフロードバージョン 1](ipsec-offload-version-1.md)は、NDIS 6.0 以降でサポートされています。

ここでは、次のトピックについて説明します。

-   [IPsec オフロードバージョン1](ipsec-offload-version-1.md)
-   [IPsec オフロードバージョン2](ipsec-offload-version-2.md)
