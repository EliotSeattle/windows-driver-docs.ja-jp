# <a name="mstcpiph"></a>Mstcpip.h

このセクションには、カーネル モードのネットワーク ドライバーに関する Mstcpip.h ヘッダーが含まれています。 このヘッダーは、ユーザー モードのアプリケーション ネットワークと共有されても、Windows SDK に含まれます。

Mstcpip.h ヘッダーには、コア Winsock の定義を Microsoft に固有の拡張機能の定義が含まれています。
