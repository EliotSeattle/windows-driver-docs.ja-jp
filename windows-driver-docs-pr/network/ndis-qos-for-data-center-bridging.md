# <a name="introduction-to-ndis-qos-for-data-center-bridging"></a>データセンターブリッジングの NDIS QoS の概要


このセクションでは、IEEE 802.1 データセンターブリッジング (DCB) インターフェイスの NDIS Quality of Service (QoS) について説明します。 ミニポートドライバーは、DCB に準拠しているネットワークアダプターを経由した送信パケットまたは送信パケットのトラフィックの優先順位付けに NDIS QoS を使用します。

DCB の NDIS QoS は、Windows Server 2012 の NDIS 6.30 で導入されました。
