# <a name="wdi_tlv_indication_can_sustain_ap"></a>WDI\_TLV\_\_\_AP を維持できることを示し\_ます。


WDI\_TLV\_\_\_\_は、ap が AP を表すことができる理由を含む TLV です。

## <a name="tlv-type"></a>TLV 型


0xE7

## <a name="length"></a>長さ


UINT32 のサイズ (バイト単位)。

## <a name="values"></a>値


| タスクバーの検索ボックスに   | 説明                                                                                                                        |
|--------|------------------------------------------------------------------------------------------------------------------------------------|
| UINT32 | は AP の理由を維持できます。 考えられる理由値については、「 [**WDI\_が\_AP\_\_を維持する**](https://docs.microsoft.com/windows-hardware/drivers/ddi/wditypes/ne-wditypes-_wdi_can_sustain_ap_reason)」を参照してください。 |
