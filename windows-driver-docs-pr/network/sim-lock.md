# <a name="sim-locking"></a>SIM ロック


## <a name="supported-sim-lock-features"></a>サポートされている SIM ロック機能


Windows 10 Mobile には、PIN1、暗証番号 2、およびこれらの Perso ロック機能がサポートされています。

-   3 gpp ネットワーク ロック

-   3 gpp のネットワーク サブセットのロック

-   3 gpp サービス プロバイダーのロック

-   3 gpp 企業ロック

-   3 gpp SIM/USIM ロック

-   3 gpp 2 ネットワークの種類が 1 のロック (MCC、mnc も)

-   3 gpp 2 ネットワークの種類が 2 つのロック (IRM ベースの最小値)

-   3 gpp 2 HRPD ロック

-   3 gpp 2 サービス プロバイダーのロック

-   3 gpp 2 企業ロック

-   3 gpp 2 R UIM ロック

Microsoft では、SIM ロック UI を提供します。 OEM カスタマイズは、この ui に表示される必要はありません。

**注**   SIM ロック UI は Oem によって構成可能ではありません。
