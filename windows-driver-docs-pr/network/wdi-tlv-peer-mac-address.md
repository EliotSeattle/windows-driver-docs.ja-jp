# <a name="wdi_tlv_peer_mac_address"></a>WDI\_TLV\_ピア\_MAC\_アドレス


WDI\_TLV\_ピア\_MAC\_アドレスは、ピアの MAC アドレスを含む TLV です。

## <a name="tlv-type"></a>TLV 型


0x4C

## <a name="length"></a>長さ


[**WDI\_MAC\_アドレス**](https://docs.microsoft.com/windows-hardware/drivers/ddi/dot11wdi/ns-dot11wdi-_wdi_mac_address)構造のサイズ (バイト単位)。

## <a name="values"></a>値


| タスクバーの検索ボックスに                                              | 説明                                  |
|---------------------------------------------------|----------------------------------------------|
| [**WDI\_MAC\_アドレス**](https://docs.microsoft.com/windows-hardware/drivers/ddi/dot11wdi/ns-dot11wdi-_wdi_mac_address) | ピアの Wi-fi MAC アドレスを指定します。 |
