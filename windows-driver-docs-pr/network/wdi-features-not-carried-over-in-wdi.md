# <a name="features-not-carried-over-in-wdi"></a>WDI に引き継がれません機能


次の機能は、前のネイティブ WLAN ドライバー モデルで使用可能なが WDI では使用できません。

-   Soft AP
-   IBSS
-   ネットワーク モニター

次の論理的な AP 機能は、Wi-Fi Direct に移植する必要があります。

-   追加した 802.11b です。
-   Wi-Fi Direct には、ソフト アジア太平洋のすべてのケーブルの要件を移動します。
