# <a name="wdi_tlv_p2p_listen_state"></a>WDI\_TLV\_P2P\_リッスン\_状態


WDI\_TLV\_P2P\_リッスン\_状態は、Wi-fi ダイレクトリッスン状態を含む TLV です。

## <a name="tlv-type"></a>TLV 型


0x81

## <a name="length"></a>長さ


含まれているすべての要素のサイズの合計 (バイト単位)。
