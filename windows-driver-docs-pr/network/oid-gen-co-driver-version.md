# <a name="oidgencodriverversion"></a>OID_GEN_CO_DRIVER_VERSION

NIC ドライバーによって使用中で NDIS バージョンです。 この OID は 2 つのバイトの長さです。上位バイトは、メジャー バージョン番号と下位バイトはマイナー バージョン番号。

## <a name="requirements"></a>要件

| | |
| --- | --- |
| バージョン | Windows Vista 以降 |
| Header | Ntddndis.h (include Ndis.h) |
