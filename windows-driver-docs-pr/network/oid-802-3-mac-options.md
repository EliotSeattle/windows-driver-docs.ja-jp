# <a name="oid8023macoptions"></a>OID\_802\_3\_MAC\_オプション





プロトコルは、この OID を使用して、イーサネットをエミュレートすることが、基になるドライバーでサポートされる機能を決定できます。

基になるドライバーのオプション サポートなしを示す、0 を返します。

**注**  この OID は 6 の NDIS ドライバーの廃止されています。
