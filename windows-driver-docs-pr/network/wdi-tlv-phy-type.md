# <a name="wdi_tlv_phy_type"></a>WDI\_TLV\_PHY\_型


WDI\_TLV\_PHY\_型は、PHY 型を含む TLV です。

## <a name="tlv-type"></a>TLV 型


0x122

## <a name="length"></a>長さ
