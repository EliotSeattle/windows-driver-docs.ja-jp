# <a name="wditlvphytype"></a>WDI\_TLV\_PHY\_型


WDI\_TLV\_PHY\_型は PHY 型を含む TLV します。

## <a name="tlv-type"></a>TLV 型


0x122

## <a name="length"></a>長さ


Uint32 型のサイズをバイト単位で。

|------------------------------------------------------|---------------|
| [**WDI\_PHY\_型**](https://msdn.microsoft.com/library/windows/hardware/dn926105) (UINT32) | PHY 型。 |
