# <a name="wdichannelnumber"></a>WDI_CHANNEL_NUMBER

WDI_CHANNEL_NUMBER のデータ型は、チャネルの数を定義する UINT32 値です。

```c++
typedef UINT32 WDI_CHANNEL_NUMBER;
```
