# <a name="oidgencorcvpduserror"></a>OID_GEN_CO_RCV_PDUS_ERROR

OID_GEN_CO_RCV_PDUS_ERROR OID には、NIC が受信したが、バインド エラーのためのプロトコルを指定していなかった Pdu の数を指定します。

## <a name="requirements"></a>要件

| | |
| --- | --- |
| バージョン | Windows Vista 以降 |
| Header | Ntddndis.h (include Ndis.h) |
