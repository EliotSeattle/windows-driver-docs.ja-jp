# 

> [!IMPORTANT]
> Windows 10 バージョン1709以降では、Wi-fi ホットスポットオフロード機能は非推奨とされており、使用しないでください。 代わりに、UWP アプリを作成し、Wi-fi ホットスポット認証 API (Windows の NetworkOperators) を使用することをお勧めします。
>
> コードサンプルと詳細については、 [wi-fi ホットスポット認証のサンプル](https://go.microsoft.com/fwlink/p/?linkid=854004)を参照してください。
