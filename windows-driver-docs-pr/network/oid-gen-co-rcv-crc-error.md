# <a name="oidgencorcvcrcerror"></a>OID_GEN_CO_RCV_CRC_ERROR

OID_GEN_CO_RCV_CRC_ERROR OID は、Pdu の巡回冗長と受信の数 (CRC) エラーを確認するを指定します。

## <a name="requirements"></a>必要条件

| | |
| --- | --- |
| バージョン | Windows Vista 以降 |
| Header | Ntddndis.h (include Ndis.h) |
