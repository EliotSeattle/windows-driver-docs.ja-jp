# <a name="provider-module"></a>プロバイダー モジュール


A*プロバイダー モジュール*は、[ネットワーク モジュール](network-module.md)のプロバイダー側の実装をサポートする、[ネットワーク プログラミング インターフェイス (NPI)](network-programming-interface.md)します。 プロバイダー モジュールとしてネットワーク モジュールのレジストラーで登録を*プロバイダー* NPI でサポートされるのです。 プロバイダー モジュールは、1 つ以上の NPI のプロバイダーとして自体を登録できます。 ネットワーク モジュールは、両方のプロバイダー モジュールには、[クライアント モジュール](client-module.md)します。
