# <a name="wdi_tlv_ft_pmkr0name"></a>WDI\_TLV\_FT\_PMKR0NAME


WDI\_TLV\_FT\_PMKR0NAME は、PMKR0Name または PMKR1Name (802.11 r) を含む TLV です。

## <a name="tlv-type"></a>TLV 型


0x107

## <a name="length"></a>長さ


[**PMK\_名前構造\_WDI\_型**](https://docs.microsoft.com/windows-hardware/drivers/ddi/wditypes/ns-wditypes-_wdi_type_pmk_name)のサイズ (バイト単位)。
