# <a name="managing-installed-hyper-v-extensible-switch-extensions"></a>インストールされている Hyper-V 拡張可能スイッチ拡張機能の管理


Windows Server 2012 以降では、各拡張可能スイッチのインスタンス内の HYPER-V 拡張可能スイッチの拡張機能は、PowerShell コマンドレットを使用管理できます。 これらのコマンドレットを使用、拡張機能を有効または無効にできるし、拡張可能スイッチ ドライバー スタックでは、同じクラス内での拡張機能の順序を変更できます。 これにより、各拡張可能スイッチのインスタンスを異なる有効な拡張機能のセットと同じクラスの拡張機能の一意の順序になります。

このセクションには、HYPER-V 拡張可能スイッチの拡張機能を管理する PowerShell コマンドレットを使用する方法を説明する次のトピックが含まれています。

-   [HYPER-V 拡張可能スイッチのインスタンスを列挙します。](enumerating-hyper-v-extensibility-switch-instances.md)
-   [HYPER-V 拡張可能スイッチの拡張機能を列挙します。](enumerating-installed-hyper-v-extensibility-switch-extensions.md)
-   [HYPER-V 拡張可能スイッチの拡張機能を有効にします。](enabling-hyper-v-extensibility-switch-extensions.md)
-   [HYPER-V 拡張可能スイッチの拡張機能を無効にします。](disabling-hyper-v-extensibility-switch-extensions.md)
-   [HYPER-V 拡張可能スイッチの拡張機能の並べ替え](reordering-hyper-v-extensibility-switch-extensions.md)
