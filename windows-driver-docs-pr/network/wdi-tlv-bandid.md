# <a name="wditlvbandid"></a>WDI\_TLV\_BANDID


WDI\_TLV\_BANDID がバンドの ID を含む TLV

## <a name="tlv-type"></a>TLV 型


0x39

## <a name="length"></a>長さ


Uint32 型のサイズをバイト単位で。

## <a name="values"></a>値


| 種類   | 説明                  |
|--------|------------------------------|
| UINT32 | バンドの識別子。 |
