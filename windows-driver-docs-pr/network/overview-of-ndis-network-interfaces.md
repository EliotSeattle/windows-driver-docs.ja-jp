# <a name="overview-of-ndis-network-interfaces"></a>NDIS ネットワーク インターフェイスの概要





NDIS ネットワーク インターフェイスでは、Microsoft Windows をサポートするさまざまなネットワーク インターフェイスのすべての一貫性のある表現を提供します。 なし、NDIS ネットワーク インターフェイスのサービスでは、すべてのネットワーク インターフェイスは、コンピューターの管理者に表示されないし、表示されているインターフェイスは必ずしも管理情報ベース (MIB) サポートしません。 また、NDIS ネットワーク インターフェイスのサービスでは、インターフェイス、管理者に表示されるリレーションシップの階層化が有効にします。
