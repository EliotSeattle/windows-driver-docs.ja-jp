# <a name="oidgencobytesxmit"></a>OID_GEN_CO_BYTES_XMIT

OID_GEN_CO_BYTES_XMIT OID にはエラーなしで送信された Pdu のバイト数を指定します。

## <a name="requirements"></a>要件

| | |
| --- | --- |
| バージョン | Windows Vista 以降 |
| Header | Ntddndis.h (include Ndis.h) |
