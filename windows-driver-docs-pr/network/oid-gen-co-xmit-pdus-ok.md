# <a name="oidgencoxmitpdusok"></a>OID_GEN_CO_XMIT_PDUS_OK

OID_GEN_CO_XMIT_PDUS_OK OID には、エラーなしで送信された Pdu の数を指定します。

| バージョン | Windows Vista 以降 |
