# <a name="filter"></a>フィルター


A*フィルター* TCP/IP ネットワーク データとすべてのフィルター条件を満たす場合に、データに対して実行されるアクションをフィルター処理するためのいくつかのフィルター条件を定義します。 指定できるかどうか、フィルターには、ネットワーク データの追加の処理が必要とする、[コールアウト](callout.md)フィルターのアクション。 このようなフィルターのフィルター条件がすべて true の場合、[フィルター エンジン](filter-engine.md)追加の処理のコールアウトの指定されたネットワーク データを渡します。
