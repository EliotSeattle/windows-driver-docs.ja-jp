# <a name="introduction-to-ndis-packet-coalescing"></a>NDIS パケット合体の概要


NDIS 6.30 以降では、ネットワークアダプターは NDIS パケット合体をサポートできます。 この機能により、ランダムなブロードキャストまたはマルチキャストパケットの受信によって、ホストシステムでの処理のオーバーヘッドと電力消費が軽減されます。
