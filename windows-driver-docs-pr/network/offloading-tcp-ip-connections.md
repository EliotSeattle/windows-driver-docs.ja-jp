# <a name="offloading-tcpip-connections"></a>TCP/IP 接続のオフロード





TCP/IP 接続のオフロードには、NDIS 6.0 以降はサポートされています。

NDIS TCP/IP 接続のオフロード インターフェイスは、TCP chimney オフロードなどのサービスを使用します。

TCP/IP 接続をオフロードの詳細については、概要の TCP Chimney オフロードで参照してください、[完全な TCP オフロード](full-tcp-offload.md)します。
