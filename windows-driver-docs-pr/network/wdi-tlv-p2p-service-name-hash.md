# <a name="wditlvp2pservicenamehash"></a>WDI\_TLV\_P2P\_サービス\_名前\_ハッシュ


WDI\_TLV\_P2P\_サービス\_名前\_ハッシュは、サービス名のハッシュを含む TLV します。

## <a name="tlv-type"></a>TLV 型


0xED

## <a name="length"></a>長さ


サイズ (バイト単位) で、 [ **WDI\_P2P\_サービス\_名前\_ハッシュ**](https://msdn.microsoft.com/library/windows/hardware/dn926103)構造体。

## <a name="values"></a>値


| 種類                                                                    | 説明                      |
|-------------------------------------------------------------------------|----------------------------------|
| [**WDI\_P2P\_サービス\_名前\_ハッシュ**](https://msdn.microsoft.com/library/windows/hardware/dn926103) | WFDS サービス名のハッシュです。 |
