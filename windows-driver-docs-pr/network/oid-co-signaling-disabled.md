# <a name="oidcosignalingdisabled"></a>OID_CO_SIGNALING_DISABLED

OID_CO_SIGNALING_DISABLED OID は、通話または着信呼び出しをディスパッチできないことを示すためにコール マネージャーによって送信されます。 コール マネージャーは、この OID をコール マネージャーで開き、アドレス ファミリを持つ各クライアントに送信します。

## <a name="requirements"></a>必要条件

| | |
| --- | --- |
| バージョン | Windows Vista 以降 |
| Header | Ntddndis.h (include Ndis.h) |
