# <a name="wdi-usb-resume-sequence"></a>WDI USB 再開シーケンス


オペレーティング システムでは、NIC を使用する必要があるときに、アイドル状態をキャンセルし、D0 の稼働状態に NIC を再開します。 NDIS を開始、再開します。
