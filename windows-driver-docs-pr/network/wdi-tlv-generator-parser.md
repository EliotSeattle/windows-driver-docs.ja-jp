# <a name="wdi-tlv-generatorparser-topics"></a>WDI TLV generator/パーサーに関するトピック


TLV generator およびパーサーの共有ライブラリを使用すると、IHV ドライバーは、TLVs を厳密C++に型指定された C/構造体に正しく解析したり、逆に構造体から TLV バイト blob を生成したりできます。 また、バージョン管理のセマンティクスを処理するので、IHV はこれを必要としません。
