# <a name="wditlvp2pservicestatus"></a>WDI\_TLV\_P2P\_サービス\_状態


WDI\_TLV\_P2P\_サービス\_状態は、サービスのサービスの状態を含む TLV します。

## <a name="tlv-type"></a>TLV 型


0xFB

## <a name="length"></a>長さ


UINT8 のサイズをバイト単位で。

## <a name="values"></a>値


| 種類  | 説明                      |
|-------|----------------------------------|
| UINT8 | サービスのサービスの状態。 |
