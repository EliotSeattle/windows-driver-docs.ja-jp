# <a name="mb-multimode-multicarrier"></a>MB マルチモード マルチキャリア


このトピックでは、Windows 8 モバイル ブロード バンドに統合されている MB マルチモード マルチのサポートについて説明します。 Windows 8 には、通信事業者にロックが解除のモバイル ブロード バンド デバイスのユーザーは、そのを選択してから、Windows UI 内ではサポートされている運送すべてへの接続 (米国外のほとんどのモバイル ブロード バンドのユーザー) の切り替えのサポートの運送業者のネイティブ サポートが導入されています。

これらの機能は有効にすると、このシナリオのサポートを実装する Ihv を対象とします。 これらは、マルチモードおよび multicarrier のシナリオを MB をサポートするために必要なデバイスの動作と Cid を使用して、USB NCM (MBIM) 仕様 (現在のバージョン 1.0) を補足します。
