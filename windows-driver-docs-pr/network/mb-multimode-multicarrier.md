# <a name="mb-multimode-multicarrier"></a>MB マルチモード マルチキャリア


このトピックでは、Windows 8 用モバイルブロードバンドに統合されている MB マルチモードマルチキャリアのサポートについて説明します。 Windows 8 では、通信事業者によるロックが解除されたモバイルブロードバンドデバイスのユーザーが、通信事業者の交代 (米国外のほとんどのモバイルブロードバンドユーザー) を使用して、Windows UI 内からサポートされる任意の通信会社を選択して接続できるようにするネイティブサポートが導入されています。

これらの機能は、このシナリオのサポートを実装できるように、Ihv を対象としています。 これらは、MB のマルチモードマルチキャリアシナリオをサポートするために必要な Cid とデバイスの動作を使用する USB NCM (MBIM) 仕様 (現在のバージョン 1.0) を補完します。

詳細については、以下のトピックを参照してください。

-   [MB プロバイダーの操作](mb-provider-operations.md)

-   [OID\_WWAN\_優先\_マルチキャリア\_プロバイダー](https://docs.microsoft.com/windows-hardware/drivers/network/oid-wwan-preferred-multicarrier-providers)

-   [**NDIS\_ステータス\_WWAN\_優先\_マルチキャリア\_プロバイダー**](https://docs.microsoft.com/windows-hardware/drivers/network/ndis-status-wwan-preferred-multicarrier-providers)

-   [**NDIS\_WWAN\_優先\_マルチキャリア\_プロバイダー**](https://docs.microsoft.com/windows-hardware/drivers/ddi/ndiswwan/ns-ndiswwan-_ndis_wwan_preferred_multicarrier_providers)
