# <a name="client-module"></a>クライアント モジュール


A*クライアント モジュール*は、[ネットワーク モジュール](network-module.md)のクライアント側の実装をサポートする、[ネットワーク プログラミング インターフェイス (NPI)](network-programming-interface.md)します。 クライアント モジュールとしてネットワーク モジュールのレジストラーで登録します、*クライアント*NPI でサポートされるのです。 クライアント モジュールは、NPI の 1 つ以上のクライアントとして自身を登録できます。 ネットワーク モジュールは、両方のクライアント モジュールは、[プロバイダー モジュール](provider-module.md)します。
