# <a name="halting-a-vf-miniport-driver"></a>VF ミニポート ドライバーの停止


VF ミニポートドライバーは、[*ミニ Porthaltex*](https://docs.microsoft.com/windows-hardware/drivers/ddi/ndis/nc-ndis-miniport_halt)関数が呼び出されたときに、NDIS ミニポートドライバーと同じ手順に従います。 これらの手順の詳細については、「[ミニポートアダプターの停止](halting-a-miniport-adapter.md)」を参照してください。
