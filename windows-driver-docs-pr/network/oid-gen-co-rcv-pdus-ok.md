# <a name="oidgencorcvpdusok"></a>OID_GEN_CO_RCV_PDUS_OK

OID_GEN_CO_RCV_PDUS_OK OID には、NIC を選択し、エラーなしで受信したバインド プロトコルに示される Pdu の数を指定します。

## <a name="requirements"></a>必要条件

| | |
| --- | --- |
| バージョン | Windows Vista 以降 |
| Header | Ntddndis.h (include Ndis.h) |
