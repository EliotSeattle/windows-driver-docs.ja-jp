# <a name="oid_gen_co_rcv_pdus_ok"></a>OID_GEN_CO_RCV_PDUS_OK

OID_GEN_CO_RCV_PDUS_OK OID は、NIC がエラーなしで受信し、バインドされたプロトコルを示す Pdu の数を指定します。

## <a name="requirements"></a>要件

| | |
| --- | --- |
| バージョン | Windows Vista 以降 |
| Header | Ntddndis (Ndis .h を含む) |
