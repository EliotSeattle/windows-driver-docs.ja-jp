# <a name="ndisstatuswdiindicationopencomplete"></a>NDIS\_状態\_WDI\_INDICATION\_オープン\_完了


ミニポート ドライバーを使用して、NDIS\_状態\_WDI\_INDICATION\_オープン\_の完了を示すために完了[OID\_WDI\_タスク\_オープン](oid-wdi-task-open.md)します。

| オブジェクト  |
|---------|
| [アダプター] |

 

## <a name="payload-data"></a>ペイロード データ


この表示には、追加のデータが含まれていません。 ヘッダー内のデータで十分です。
