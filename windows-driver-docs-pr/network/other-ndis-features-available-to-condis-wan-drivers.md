# <a name="other-ndis-features-available-to-condis-wan-drivers"></a>いる CoNDIS WAN ドライバーが利用できるその他の NDIS 機能





いる CoNDIS WAN ミニポート ドライバーまたは MCMs を次の機能の利用にできます。

-   ミニポート ドライバーのプラグ アンド プレイ (PnP) イベント通知

-   統計カウンターの 64 ビット

-   送信パケットのキャンセル

-   登録、 [ *MiniportShutdownEx* ](https://msdn.microsoft.com/library/windows/hardware/ff559449)関数

-   新しいミニポート ドライバーの属性

-   安全な関数
