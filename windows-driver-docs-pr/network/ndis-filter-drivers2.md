# <a name="ndis-filter-drivers"></a>NDIS フィルター ドライバー





NDIS フィルター ドライバーは、NDIS 6.0 以降でサポートされています。 次のセクションでは、フィルター ドライバーを紹介し、記述して、NDIS フィルター ドライバーをインストールする方法について説明します。

-   [NDIS フィルター ドライバーを開発するためのロードマップ](roadmap-for-developing-ndis-filter-drivers.md)
-   [NDIS フィルター ドライバーの概要](introduction-to-ndis-filter-drivers.md)
-   [NDIS フィルター ドライバーの記述](writing-ndis-filter-drivers.md)
-   [NDIS フィルター ドライバーのインストール](ndis-filter-driver-installation.md)

**注**  中間の NDIS フィルター ドライバーではなく、NDIS フィルター ドライバーを使用する必要があります。 中間の NDIS フィルター ドライバーは廃止されました。
