# <a name="taltxrxhandle"></a>TAL_TXRX_HANDLE

TAL_TXRX_HANDLE のデータ型では、NDIS_HANDLE 値です。

```c++
typedef NDIS_HANDLE TAL_TXRX_HANDLE, *PTAL_TXRX_HANDLE;
```
