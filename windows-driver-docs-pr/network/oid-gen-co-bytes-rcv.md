# <a name="oidgencobytesrcv"></a>OID_GEN_CO_BYTES_RCV

OID_GEN_CO_BYTES_RCV OID にはエラーなしで受信した Pdu のバイト数を指定します。

| | |
| --- | --- |
| バージョン | Windows Vista 以降 |
