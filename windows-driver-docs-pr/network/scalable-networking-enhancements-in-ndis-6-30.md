# <a name="scalable-networking-enhancements-in-ndis-630"></a>NDIS 6.30 のスケーラブル ネットワークの機能強化


スケーラブルなネットワークには、NDIS 6.30 以降のドライバーを次の機能強化が含まれています。

[セグメント Coalescing (RSC) が表示される](receive-segment-coalescing--rsc-.md)
[ネットワーク ダイレクト カーネル プロバイダー インターフェイス (NDKPI)](network-direct-kernel-programming-interface--ndkpi-.md)
[Generic Routing Encapsulation (NVGRE) タスクを使用してネットワーク仮想化オフロード](network-virtualization-using-generic-routing-encapsulation--nvgre--task-offload.md)
[IPsec オフロード バージョン 2](ipsec-offload-version-2.md) (IPsecOV2) 更新[Receive Side Scaling](ndis-receive-side-scaling2.md) (RSS) 更新プログラム

**注**  NetDMA は Windows 8 および Windows オペレーティング システムの以降のバージョンでサポートされていません。
