# <a name="hyper-v-extensible-switch"></a>Hyper-V 拡張可能スイッチ


HYPER-V 拡張可能スイッチを NDIS フィルター ドライバーのインスタンスのためのインターフェイスをサポートしています (と呼ばれる*拡張可能スイッチの拡張機能*)、拡張可能スイッチ ドライバー スタック内でバインドします。 バインドされ、有効にした後の拡張機能できます監視、変更、および拡張可能スイッチ ポートにパケットを転送します。 これにより、拡張機能を元に戻す、リダイレクトすると、またはパケットを HYPER-V パーティションによって使用されるポートの発信がもできます。

HYPER-V 拡張可能スイッチは、Windows Server 2012 で NDIS 6.30 以降はサポートされています。
