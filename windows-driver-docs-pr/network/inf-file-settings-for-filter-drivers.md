# <a name="inf-file-settings-for-filter-drivers"></a>フィルター ドライバーの INF ファイル設定





フィルター ドライバーの INF ファイルの設定は、フィルター ドライバーの特性を指定します。 フィルター ドライバーを変更することや監視などの必須またはオプションを指定できます。 INF ファイルには、一般的な構成パラメーターとフィルター ドライバーを特定のミニポート アダプターに関連付けるには、その他の情報も指定します。 フィルター ドライバーは、フィルターの一意の id を GUID (FID GUID) によって識別されます。
