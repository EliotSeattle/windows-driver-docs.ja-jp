# <a name="pldr"></a>PLDR


## <a name="recovery-of-pldr"></a>PLDR の回復


突然削除後にデバイス オブジェクトを削除して再 (バス) によって列挙できるように、ドライバー (UE と LE) がすべてのリソースを解放する必要があります。 そうでない場合は、デバイスが再列挙されない、し、したがって回復されません。
