# <a name="wditlvchannelnumber"></a>WDI\_TLV\_チャネル\_数


WDI\_TLV\_チャネル\_数がチャンネル番号を含む TLV します。

## <a name="tlv-type"></a>TLV 型


0x121

## <a name="length"></a>長さ


UINT8 のサイズをバイト単位で。

## <a name="values"></a>値


| 型  | 説明         |
|-------|---------------------|
| UINT8 | チャネルの数。 |
