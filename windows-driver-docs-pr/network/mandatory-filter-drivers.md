# <a name="mandatory-filter-drivers"></a>必須フィルター ドライバー





必須のフィルター ドライバーは、ドライバー スタックが正常に機能するのに必要なフィルター ドライバーです。 必須のフィルター モジュールが関連付けられていない場合、ドライバー スタックの残りの部分は破棄されます。 [変更またはフィルター ドライバーを監視](types-of-filter-drivers.md)必須にすることができます。 すべての中間フィルター ドライバーは省略可能です。

ドライバー スタックに必須のフィルター モジュールを添付するには、NDIS はすべてのプロトコル バインディングをバインド解除、フィルター モジュールをアタッチし、すべてのプロトコル バインドを再確立しました。 ドライバーが関連付けられていない場合は、NDIS、基になるドライバー スタックを廃棄します。

NDIS ドライバー スタックからの必須のフィルター モジュールをデタッチするには、すべてのプロトコル バインドをバインド解除、フィルター モジュールをデタッチし、プロトコル バインドを再確立しました。 オプションのフィルター モジュールをデタッチするには、NDIS は、スタックを一時停止し、プロトコル ドライバーのバインドを解除せずに再起動します。

コンピューターを再起動すると、アダプターに関連付けられているすべての必須のフィルター モジュールがミニポート アダプターにアタッチできません NDIS はプロトコル ドライバー ミニポート アダプターにバインドしません。

必須のフィルター ドライバーをインストールするには、0x00000001 の値を指定する必要があります**FilterRunType** INF ファイルにします。 オプションのフィルター ドライバーをインストールするには、0x00000002 の値を指定する必要があります**FilterRunType** INF ファイルにします。
