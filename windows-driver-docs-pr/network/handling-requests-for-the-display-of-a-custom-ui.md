# <a name="handling-requests-for-the-display-of-a-custom-ui"></a>カスタム UI の表示の要求を処理




 

このセクションでは、802.11 IHV UI 拡張機能のネイティブ DLL で、次のいずれかでカスタム ユーザー インターフェイス (UI) を表示する方法について説明します。

-   ネイティブの 802.11 IHV 拡張 DLL からの要求。 たとえば、ネイティブの 802.11 IHV 拡張 DLL では、ワイヤレス LAN (WLAN) イベントのユーザー通知のカスタム UI を要求します。

-   802.11 IHV UI 拡張機能のネイティブ DLL に表示できるカスタム UI があるかどうかを決定する、オペレーティング システムによって行われたクエリです。

このセクションでは、次のトピックがあります。

[カスタム UI の表示を要求します。](requesting-the-display-of-a-custom-ui.md)

[カスタム UI を表示するためのクエリ](querying-for-the-display-of-a-custom-ui.md)

[バルーン通知内のカスタム UI ページを表示します。](displaying-custom-ui-pages-within-a-balloon-notification.md)

[ネットワークの接続ウィザード内のカスタム UI ページを表示します。](displaying-custom-ui-pages-within-the-network-connection-wizard.md)

[プロファイルとコンテキスト データにアクセスします。](accessing-profile-and-context-data.md)

ネイティブの 802.11 IHV 拡張 DLL の詳細については、[802.11 IHV 拡張機能のネイティブの DLL](native-802-11-ihv-extensions-dll4.md)を参照してください。
