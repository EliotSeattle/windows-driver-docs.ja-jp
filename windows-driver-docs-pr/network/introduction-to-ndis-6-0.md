# <a name="introduction-to-ndis-60"></a>NDIS 6.0 の概要





NDIS 6.0 は、Network Driver Interface Specification (NDIS) ライブラリの次のメジャー バージョンです。 NDIS は、ネットワークのカーネル モード ドライバーとオペレーティング システムの間の標準のインターフェイスを指定します。 NDIS には、複数層のネットワーク ドライバー、それによってネットワーク トランスポートなどの上位レベルのドライバーからハードウェアを管理する下位レベルのドライバーを抽象化の間の標準のインターフェイスも指定します。 NDIS 6.0 には、Windows Vista オペレーティング システムが含まれます。

このセクションでは、NDIS 6.0 の概要について説明し、主要な設計の目標、これらの目標を満たしている方法、および NDIS 6.0 の利点について説明します。
