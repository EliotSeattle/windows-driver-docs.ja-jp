# <a name="text-mode-phase-of-the-network-upgrade-process"></a>ネットワークのアップグレード プロセスのテキスト モードのフェーズ





**注**  Microsoft Windows XP では、ベンダーから提供されたネットワークのアップグレードはサポートされていない (SP1 以降)、Microsoft Windows Server 2003、および以降のオペレーティング システム。

 

セットアップでは、応答ファイルからすべてのコメントを削除し、System32 ディレクトリの名前の $winnt$.sif$ .inf 下に、応答ファイルを書き込みます。 GUI モードのセットアップに、システムでブートされます。 テキスト モードのフェーズでは、特定のネットワークの処理は行われません。
