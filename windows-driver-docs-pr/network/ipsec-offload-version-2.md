# <a name="ipsec-offload-version-2"></a>IPsec Offload Version 2

\[IPsec タスク オフロード機能は非推奨し、は使用できません。\]




インターネット プロトコル セキュリティ (IPsec) タスク オフロードには、コンピューターが IPsec オフロードに対応したネットワーク インターフェイス カード (Nic) に IPsec の計算処理を要する暗号化操作ができるようにします。 Windows Server 2008 および NDIS 6.1 拡張 IPsec で提供されるサービス 2 (IPsecOV2) オフロードのバージョン 1 (IPsecOV1) ネットワーク パフォーマンスの向上を提供する IPsec オフロード バージョンをサポートします。 IPsecOV2 は、追加の暗号化アルゴリズム、IPv6、および大量送信オフロード バージョン 2 (LSOv2) との共存をサポートします。
