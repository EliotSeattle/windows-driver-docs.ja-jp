# <a name="remote-ndis-data-message"></a>リモートの NDIS データ メッセージ





リモートの NDIS デバイスには、NDIS パケット データ チャネル経由で転送がカプセル化します。 データのメッセージは、帯域外の (OOB) のデータやパケット単位での情報があるため、この操作に使用されます。

データ チャネル経由で転送するためのデータをカプセル化するために使用されるデータのメッセージは、次のトピックについて説明します。

[**リモート\_NDIS\_パケット\_メッセージ**](remote-ndis-packet-msg.md)
