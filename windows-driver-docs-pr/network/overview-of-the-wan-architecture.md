# <a name="overview-of-the-wan-architecture"></a>WAN アーキテクチャの概要





WAN のアーキテクチャ、コンポーネントは、主に、WAN ミニポート ドライバーに直接そのインターフェイスが構成されています。 ただし、WAN のアーキテクチャは最適なより広範な RAS アーキテクチャのコンテキスト内で導入されました。 RAS アーキテクチャには、WAN のアーキテクチャの高レベルの理解に不可欠ですが、それ以外の場合、Microsoft Windows Driver Kit (WDK) と Windows ドライバー開発キット (DDK) のスコープ外では一部のコンポーネントが含まれています。
