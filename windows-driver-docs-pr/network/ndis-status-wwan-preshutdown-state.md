# <a name="ndisstatuswwanpreshutdownstate"></a>NDIS\_状態\_WWAN\_PRESHUTDOWN\_状態


NDIS\_状態\_WWAN\_PRESHUTDOWN\_状態通知がホストに、MBB ドライバーからの一方向の通知。 MBB ドライバーは、モデムがシャット ダウンする前に必要なすべての操作を完了すると、この通知を送信します。

この通知を使用して、 [ **NDIS\_WWAN\_PRESHUTDOWN\_状態**](https://msdn.microsoft.com/library/windows/hardware/mt593234)構造体。
