# <a name="oidgencoxmitpduserror"></a>OID_GEN_CO_XMIT_PDUS_ERROR

OID_GEN_CO_XMIT_PDUS_ERROR OID には、NIC が送信できませんでした Pdu の数を指定します。

| | |
| --- | --- |
| バージョン | Windows Vista 以降 |
| Header | Ntddndis.h (include Ndis.h) |
