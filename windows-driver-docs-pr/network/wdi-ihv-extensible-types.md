# <a name="wdi-ihv-extensible-types"></a>WDI IHV 拡張型


WDI モデルには、カスタム PHY 型のサポート、認証用のアルゴリズムと暗号アルゴリズムを IHV が使用できます。 PHY 型 IHV の拡張機能では、レポート目的で主に使用され、オペレーティング システムの動作は変わりません。 IHV 拡張機能の認証と暗号アルゴリズムは、IHV 機能拡張のモジュールおよび IHV プロファイルで使用されます。 これらの接続を使用している場合、ホストには任意に一致するセキュリティ設定の候補をアダプターに BSS リストに転送する前に実行されません。
