# <a name="wditlvp2padvertisementid"></a>WDI\_TLV\_P2P\_広告\_ID


WDI\_TLV\_P2P\_広告\_ID は、サービス インスタンスを一意に識別する ID を含む TLV します。

## <a name="tlv-type"></a>TLV 型


0 xea

## <a name="length"></a>長さ


Uint32 型のサイズをバイト単位で。

## <a name="values"></a>値


| 型   | 説明                                        |
|--------|----------------------------------------------------|
| UINT32 | サービス インスタンスを一意に識別する ID です。 |
