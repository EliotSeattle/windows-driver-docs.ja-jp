# <a name="learning-about-intermediate-drivers"></a>中間ドライバーについて





コネクションレス型、または接続指向の下端のいずれかを含む中間のドライバーを記述できます。 次の一覧では、WDK ドキュメントのセクションをお読みください、記述する中間のドライバーの種類に応じてについて説明します。

<a href="" id="intermediate-drivers-that-have-a-connectionless-lower-edge"></a>**コネクションレスの下端の中間のドライバー**  
下端コネクションレスのミニポート ドライバー インターフェイスを提供する中間のドライバーを作成する場合は、次の読み取り。

-   [NDIS 中間ドライバ](ndis-intermediate-drivers.md)

<a href="" id="intermediate-drivers-that-have-a-connection-oriented-lower-edge"></a>**接続指向の下端の中間のドライバー**  
下端は、接続指向のミニポート ドライバー インターフェイスを提供します。 中間のドライバーを作成する場合は、次の読み取り。

-   [NDIS 中間ドライバ](ndis-intermediate-drivers.md)

-   [接続指向の NDIS](connection-oriented-ndis.md)
