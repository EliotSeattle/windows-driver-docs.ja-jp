# <a name="windows-driver-documentation"></a>Windows ドライバーのドキュメント

Windows ドライバー ドキュメント リポジトリへようこそ。ここには、公式の Windows Driver Kit ドキュメント ([Docs.microsoft.com](https://docs.microsoft.com/windows-hardware/drivers)) のソースが格納されています。 

ドライバーのリファレンス (API/DDI) ドキュメントは、別のリポジトリにあります。
「 https://github.com/MicrosoftDocs/windows-driver-docs-ddi 」を参照してください。
